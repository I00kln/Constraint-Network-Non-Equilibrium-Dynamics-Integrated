# Causal Information Dynamics Simulator V2.0

严格按照《数值仿真计划v5.md》v2.0版本设计实现。

## 设计原则

1. **允许理论失败**：明确区分"结构涌现"与"复杂系统通用伪影"
2. **强制零模型**：任何观测信号必须与三种对照组比较
3. **避免激活函数伪影**：用图拉普拉斯谱截断替代sigmoid
4. **谱图理论优先**：路径模稳定性通过拉普拉斯本征模分析
5. **有限尺度标度**：所有结论需外推到N→∞
6. **规范结构严格化**：边权升级为U(1)或SU(2)相位
7. **几何涌现可操作定义**：光锥用响应函数，引力用离散Ricci曲率
8. **相图与普适性**：扫描参数空间，寻找不依赖于参数细节的普适区

## 核心修订

### 边权升级
- 每条边携带U(1)相位 e^{iθ} 或SU(2)矩阵
- 初始相位均匀随机 [0,2π)

### 谱粗化（替代sigmoid）
- 计算图拉普拉斯本征值和本征向量
- 宏观场：投影到低本征模子空间
- 微观场：ρ - Φ
- 无sigmoid，分解是线性的、正交的

### 三种零模型
| 零模型 | 修改 | 目的 |
|--------|------|------|
| NM-A | 去掉谱分解 | 检验宏微观分解必要性 |
| NM-B | 随机重连 | 检验规范信号是否伪影 |
| NM-C | 静态拓扑 | 检验谱结构来源 |

### 观测量
- **拉普拉斯本征谱**：识别稳定模和三代候选
- **Wilson loop**：计算holonomy分布
- **响应函数**：检测光锥涌现
- **Ollivier-Ricci曲率**：检验爱因斯坦方程

## 编译与运行

```bash
# 编译
make

# 运行测试
make test

# 单次模拟
./causal_sim_v2 single -N 500 -steps 1000 -eta 0.1 -lambda 1.0 -kc 0.1

# 零模型对比
./causal_sim_v2 null

# 帮助
./causal_sim_v2 --help
```

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| -N | 500 | 节点数 |
| -steps | 1000 | 最大步数 |
| -eta | 0.1 | 传播速率 |
| -lambda | 1.0 | 微观压缩因子 |
| -kc | 0.1 | 截止比例（低本征模占比） |
| -alpha | 0.5 | 重连密度依赖强度 |
| -sigma | 0.1 | 相位噪声 |
| -model | 0 | 模型类型（0=Full, 1=NM-A, 2=NM-B, 3=NM-C） |

## 验证流程

每个观测信号必须通过以下检验：

1. **信号存在性**：在目标模型中观测到
2. **零模型消融**：在NM-A/B/C中显著减弱或消失
3. **参数稳定性**：在普适区内改变参数，信号定性不变
4. **有限尺度外推**：N→∞时信号保持
5. **非平凡性**：信号无法简化为已知网络通用性质

只有通过全部5项，才能声称"因果自指机制产生了物理结构"。

## 理论命运判定

| 模拟结果 | 对理论的意义 |
|----------|----------------|
| 三代谱在零模型中不出现，且N→∞稳定 | **强支持**：三代机制合理 |
| Wilson loop分布出现离散峰（非均匀） | **支持**：规范群涌现候选 |
| 响应函数显示光锥，爱因斯坦关系成立 | **支持**：几何涌现机制成立 |
| 谱簇数量随N增长 | **证伪**：三代猜想不成立 |
| 信号在零模型中也出现 | **无效**：需重新设计机制 |
