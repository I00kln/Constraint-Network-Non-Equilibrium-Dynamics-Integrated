# 谱维度计算问题分析与修正

## 问题描述

**观察到的现象**：
- 3D系统谱维度 d_s = 3.322（t=0.5）
- 预期值 d_s = 3.0
- 偏差约 10%

## 问题根源

**关键发现**：t=0.5 不在3D系统的平台区域！

### 谱维度随t的变化规律

```
3D系统 (L=16) 完整曲线:

  t       d_s      dd_s/dt   平台判断
  -----   ------   ------   -----------
  0.50    3.322    -0.418
  1.00    3.627    -0.359
  1.50    3.420    -0.286
  2.00    3.275    -0.206
  2.50    3.199    -0.111
  3.00    3.155    -0.067
  3.50    3.128    -0.045   *** 接近平台
  4.00    3.109    -0.032   ***** 平台
  4.50    3.094    -0.025   ***** 平台
  5.00    3.083    -0.021   ***** 平台
```

**最佳平台位置**：t = 5.0, d_s = 3.083

## 理论解释

### 1. 小t极限 (t → 0)

$$P(t) \sim (4\pi t)^{-d/2}$$

$$d_s \to d \text{ (拓扑维度)}$$

但有限尺寸下，小t区域被截断。

### 2. 中间区域

- 存在平台，但位置取决于系统尺寸
- 平台值接近拓扑维度
- **不同维度需要不同的t值**

### 3. 大t极限 (t → ∞)

- 只有零模贡献
- d_s → 0

### 关键洞察

对于有限系统，谱维度曲线有峰值：
- 峰值位置随系统尺寸移动
- 平台区域应在峰值附近
- **t=0.5 对于3D系统太小**

## 修正方案

### 方案1：寻找平台区域

```
3D系统: t ~ 5.0, d_s = 3.083 (偏差 2.8%)
2D系统: t ~ 4.9, d_s = 2.057 (偏差 2.9%)
```

### 方案2：热力学极限外推

```
使用 t = 5.0 读取:

  L      d_s(平台)   偏差(%)
  ----   --------   --------
  12     3.005      0.2
  14     3.073      2.4
  16     3.083      2.8
  20     3.084      2.8
  24     3.084      2.8

热力学极限: d_s = 3.084
```

### 方案3：自适应t值选择

根据系统尺寸L和维度d，选择合适的t值：
- t ~ L/d 可以获得较好的平台
