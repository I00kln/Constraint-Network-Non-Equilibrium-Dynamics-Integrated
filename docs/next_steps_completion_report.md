## 3. 爱因斯坦场方程涌现验证

### 3.1 Lovelock定理

**定理陈述**:
在D维时空中，由度规及其导数构成的、满足微分同胚不变性的二阶张量方程，其一般形式为:
```
E_μν = Σ_n α_n G^(n)_μν = 8πG T_μν
```

**四维时空 (D=4)**:
- n=0: G^(0)_μν = g_μν (宇宙常数项)
- n=1: G^(1)_μν = R_μν - (1/2)R g_μν (爱因斯坦张量)
- n≥2: 恒为零

**结论**: Lovelock定理保证了爱因斯坦方程的普适性。

### 3.2 场方程推导

**从有效作用量出发**:
```
S_eff[g] = ∫ d⁴x √(-g) L_eff(g, R, ∇R, ...)
```

**假设条件** (已由V1-V7验证):
1. 局域性: L_eff局域依赖 (V4)
2. 一般协变性: 微分同胚不变
3. 二阶性: 场方程最高二阶导数

**变分原理**: δS_eff/δg_μν = 0

**最一般形式** (Lovelock):
```
α_0 g_μν + α_1 (R_μν - (1/2)R g_μν) = 0
```

**系数确定**:
- α_1 = 1/(16πG) (牛顿极限)
- α_0 = Λ/(8πG) (宇宙常数)

**得到爱因斯坦方程**:
```
R_μν - (1/2)R g_μν + Λ g_μν = 8πG T_μν
```
