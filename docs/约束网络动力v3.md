# 约束关系网络动力学：从单一生成原理到时空涌现

## 零. 总纲：理论已从“构造”转向“推导”

**修正前**，理论由多个独立模块拼接而成：辛结构 + 手调闭合项 + 人工耗散 + 现象学维度流。  
**修正后**，所有结构必须来自一个**统一生成原理**，否则视为失败。

### 0.1 生成原理（取代公设0–1）

**唯一动力学生成泛函**，定义在群流形上：

\[
S = \sum_{\triangle} \mathrm{Tr}(1 - U_\triangle), \qquad U_\triangle = U_{ij}U_{jk}U_{ki}
\]

其中 \(U_e \in G\) 是边上的群变量，底空间为有向图 \(G=(V,E)\)。相空间天然是**离散余切丛** \(T^*G\)，正则共轭对为 \((A_e, E_e)\)，满足：

\[
\{A_{ij}, E^{kl}\} = \delta_i^k\delta_j^l - \delta_j^k\delta_i^l
\]

这一选择**不是出于类比**，而是因为：\(S\) 是离散 BF 作用量在 \(G=U(1)\) 时的自然约化，且天然携带规范对称与拓扑扇区。  
所有后续结构——约束、闭合、耗散、维度——必须从此唯一生成量中**涌现或消失**。

## 一. 基础层：严格约束辛系统（取缔原公设2）

### 1.1 哈密顿量（测地距离定义，取缔 cos 展开）

从作用量 \(S\) 过渡到哈密顿形式时，不使用截断 Fourier 展开，而用群流形上的测地距离平方：

\[
H = \sum_e \frac{1}{2} E_e^2 + \beta \sum_{\triangle} d_G(U_\triangle, \mathbf{1})^2
\]

对 \(G=U(1)\)，\(d_G(e^{iF},1) = |F|_{2\pi}\)（最小弧长）。此定义的后果：
- **无需 Fourier counterterms**：闭合是代数的，非拟合的
- **UV 更干净**：不引入格点赝象
- **RG 更稳定**：不存在为压低 anomaly 而必须精细调节的高阶项

### 1.2 约束系统（仅保留生成原理强制要求的）

**高斯约束**（规范对称，来自 \(S\) 的规范不变性）：
\[
\mathcal{G}_i = \sum_j E^{ij} \approx 0
\]

**哈密顿约束**（时间重参数化，来自测地距离定义，非人为加项）：
\[
\mathcal{H}_{\text{tot}} = \sum_e \frac{1}{2}E_e^2 + \beta \sum_{\triangle} d_G(U_\triangle,\mathbf{1})^2 \approx 0
\]

局域密度定义为：
\[
H_i = \frac{1}{\deg(i)}\sum_{e\ni i} \frac{1}{2}E_e^2 + \beta \sum_{\triangle\ni i} d_G(U_\triangle,\mathbf{1})^2
\]

**扩散约束已删除**。冻结 \(q_{ij}=1\)，尺度涌现完全交付 RG 流。

**候选空间约束已降级为研究注释**：目前无证据表明需要离散微分同胚生成元。待闭合性计算判决是否需要，如需，则必须构造真正的图变群胚生成元，而非中心泛函。

### 1.3 生死线：约束代数闭合性（取代原“异常耗散的Lyapunov稳定性”为首位）

计算 \(\{H_i, H_j\}\)。  
在唯一生成原理下，预期结果为：

\[
\{H_i, H_j\} = \sum_k f_{ij}^k \mathcal{G}_k \quad \text{（在约束面上）}
\]

即闭合为高斯约束的线性组合。此非数值吻合，而必须是**代数恒等式**——structure function \(f_{ij}^k\) 需从泊松几何自动出现。

**失败条件：出现无法写为 \(\sum (\text{约束}) \times (\text{结构函数})\) 的异常项。**  
此门不通，即宣告此实现失败。不允许以耗散项吸收异常。

**若闭合仅当补充额外空间生成元时才成立**，则必须构造真正的离散 cotangent bundle 上的空间微分同胚生成元。该生成元必须满足超曲面变形代数 \(\{H[\xi], H[\eta]\} = D[\xi\partial\eta - \eta\partial\xi]\) 的离散形式。

## 二. 涌现层：粗粒化自动产生耗散、维度流与时间箭头（取缔原公设3的人工耗散）

### 2.1 耗散来自积分掉微观自由度

不手工加 \(-\gamma \partial C\)。取而代之地，对一部分网络自由度取迹：

\[
\rho_{\text{eff}} = \mathrm{Tr}_{\text{env}} \rho
\]

得到有效演化为 **Lindblad 型**：

\[
\dot{\rho}_{\text{eff}} = -i[H_{\text{eff}}, \rho_{\text{eff}}] + \sum_a \left(L_a \rho_{\text{eff}} L_a^\dagger - \frac{1}{2}\{L_a^\dagger L_a, \rho_{\text{eff}}\}\right)
\]

跳变算符 \(L_a\) 的结构完全由粗粒化方案决定，不是自由参数。  
约束 violation 的单调衰减——若出现——是此方程的自然后果，非人工设计的梯度下降。

### 2.2 时间箭头：RG 流的不可逆性

不再声称“时间箭头来自熵增”。改为：  
**不可逆性涌现自 RG 变换的非幺正性**。系统流向约束吸引子是粗粒化必然产物，方向等同于因果深度 \(\tau\) 的增长方向。

原“因果深度”定义保留，但降级为规范不变的序参数，不再赋予“时间本身”的本体论地位。

### 2.3 谱维度流：从算子出发，非现象拟合

不再直接测热核斜率后反推维度。改为：

**先定义有效传播子**，从基本变量导出：

\[
G(k) \sim \frac{1}{k^{2-\eta(k)}}
\]

其中 \(\eta(k)\) 是反常维度，直接由粗粒化的 RG 方程解出。  
**然后**，谱维度由反常维度给出：

\[
d_s(k) = \frac{2d}{2-\eta(k)}
\]

热核法降级为交叉检验。  
维度流中的“噪声”因此被吸收进 \(\eta(k)\) 的系统标度行为，而非来自图各向异性的赝象。

## 三. 约束面几何：规范不变度量（保留原公设4核心，删除“空间决定怎么走”的过度表述）

空间几何在 \(\mathcal{G}_i \approx 0\) 的约束面上读取。所有几何量必须规范不变：

- **曲率**：\(\mathrm{Ric}_\triangle = d_G(U_\triangle,\mathbf{1})^2\)（来自生成原理，非人为定义）
- **距离**：Wilson 型退相干成本或 Connes 谱距离
- **谱维度**：如上，从反常维度导出，热核法验证

“空间决定怎么走”弱化为：约束面上的切演化由哈密顿约束生成，可逆；与时间方向的不可逆性构成对比。

## 七. 科学状态声明（取代原“核心结论”）

本框架现为**单一生成原理**：群流形上的离散 BF 型作用量 \(S = \sum \mathrm{Tr}(1-U_\triangle)\)。所有结构——约束代数、耗散、维度流、Lorentz 恢复——必须是此原理在粗粒化下的必然产物，而非独立添加的模块。

若闭合成立，则进入开放系统 RG 检验；若失败，**框架本身需被放弃或根本性重构**。

这就是成熟的标志：理论现在定义了自己确切的死亡条件。
