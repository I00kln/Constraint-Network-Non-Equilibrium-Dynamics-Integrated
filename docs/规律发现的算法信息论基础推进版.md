# 规律发现的算法信息论基础：从最小描述长度到闭包结构与尺度相变  
**版本 14.0（严格变分流、unifilar 因果态与资源约束闭包几何）**  

---

## 摘要

在算法信息论与计算力学的基座上，建立观察者资源约束下规律发现的严格变分压缩理论。核心对象为由观测数据、受限于最大状态数 \(R\) 与历史深度 \(L\) 的粗粒化映射及宏观预测模型构成的多层三元组。定义统一变分泛函  

\[
\mathcal{F}(C,P)=I(Z_{t+1};\overleftarrow{Z}_t\mid Y_t)+\mathbb{E}_\mu\!\bigl[D_{\mathrm{KL}}(\mu(\cdot\mid Y_t)\,\|\,P(\cdot\mid Y_t))\bigr],
\]

在平稳遍历且满足 **有限 unifilar 因果态、有限记忆与一步可辨识性** 的过程上，将其与期望编码超额量精确恒等。规律选择提炼为最小化 \(\mathcal{F}\) 与描述长度加权的理论压缩代价。所有主要定理——超额编码分解、MDL一致收敛、因果态极小充分性——均作为该泛函的变分性质自然导出。进而，在有限历史窗口划分格上，严格构造受限于资源 \(R\) 与深度 \(L\) 的 **变分闭包算子**，揭示其不动点结构；证明因果态划分是唯一的零缺陷全局极小不动点，建立 \(\mathcal{F}\) 作为 Lyapunov 函数的严格下降性质，并分析资源约束下必然出现的资源壁垒不动点（亚稳闭包结构）。严格区分路径存在性与策略依赖性：证明存在一条从任意初态到达因果态的有限分裂路径，但局部变分算子可能陷入亚稳态。定义尺度依赖的压缩算子族，证明有限因果态过程的尺度饱和定理。利用统一泛函刻画序参量并描述资源约束诱导的 **闭包阈值转变**：在有限划分模型类中表现为容量阈值的刚性阶梯转变，给出亚稳态结构的出现条件。提炼闭包结构作为普适对象，提出嵌套 MDL 粗粒化流中的闭包半群与重整化群嵌入猜想。明确区分理想数学动力学与可计算算法近似，并给出基于信息几何 Bregman 投影及 KL 梯度流的严格化推进路线。全文明确区分已证明结果与开放问题。

## 第一部分：严格基础

### 1. 基本定义与符号

#### 1.1 概率空间与过程  
设 \((\Omega,\mathcal{F},\mu)\) 为标准 Borel 概率空间，\(\mu\) 为平稳遍历概率测度。考虑双向无限序列空间 \(\Sigma^{\mathbb{Z}}\)，字母表 \(\Sigma\) 有限。坐标过程 \(Z_t(\omega)=\omega_t\)。记过去半无限序列 \(\overleftarrow{Z}_t = (\dots, Z_{t-1}, Z_t)\)。平稳性保证左移算子保持测度不变。由标准 Borel 空间的性质，存在关于 \(\overleftarrow{Z}_t\) 的正则条件概率 \(\mu(\cdot\mid\overleftarrow{Z}_t)\)，且对几乎所有轨道唯一确定有限维边缘。  
过程的 **熵率** 定义为  

\[
h_\mu = \lim_{n\to\infty} \frac{1}{n} H(Z_{1:n}) = H(Z_{t+1}\mid\overleftarrow{Z}_t),
\]

其中条件熵理解为 \(\mathbb{E}[H(\mu(\cdot\mid\overleftarrow{Z}_t))]\)。在平稳遍历且字母表有限下，极限存在且等式成立。

**假设 1（有限 unifilar 因果态、有限记忆与一步可辨识性）**  
过程 \(\mu\) 具有 **有限因果态**，个数 \(N = |\mathcal{S}| < \infty\)，且因果态转移具有 **unifilar** 性质：存在可测函数 \(\lambda:\mathcal{S}\times\Sigma\to\mathcal{S}\)，使得若当前因果态为 \(\sigma\)，观测到符号 \(a\)，则下一时刻的因果态唯一确定为 \(\lambda(\sigma,a)\)。进而，存在有限整数 \(L_0 < \infty\)，使得对几乎所有过去 \(\overleftarrow{z}\)，其因果态仅由最近 \(L_0\) 个符号 \(z_{-L_0+1:0}\) 完全决定（有限记忆）。最后，过程满足 **一步可辨识性**：对任意两个不同的因果态 \(\sigma \neq \sigma'\)，它们的一步预测分布不同，即 \(\mu(\cdot\mid\sigma) \neq \mu(\cdot\mid\sigma')\)。由 unifilarity 与有限记忆，一步预测分布不同等价于所有有限步预测分布不同。全文在此假设下进行。

此假设是计算力学中 ε-machine 的标准设定：因果态构成一个 unifilar 的隐马尔可夫模型，且状态数有限、记忆有限、一步可区分。它为将一步闭包缺陷为零提升为完美闭包提供了必要且充分的基础。

**注 1.1（一步可辨识性的地位）**  
本文结果严格依赖于一步可辨识性假设。对于一般因果态仅通过整个未来分布区分的过程，需要将一步闭包缺陷 \(\iota_1\) 替换为完整未来闭包缺陷 \(\iota_\infty(C)=I(\overrightarrow{Z}_t;\overleftarrow{Z}_t\mid Y_t)\)，相关变分结构可类似建立，但技术细节更复杂，且当前格上的有限间隙引理等需重新论证。这留作第六部分所列的开放问题。本文在此强假设下展开，以获得最清晰的数学图像，并不失一般性，因为所建立的闭包结构范式可迁移至更广情形。

#### 1.2 观测与描述长度  
观察者获得有限观测序列 \(D = z_{1:T}\)，由 \(\mu\) 生成。固定通用前缀图灵机 \(U\)。有限串 \(x\) 的前缀柯尔莫哥洛夫复杂度 \(K(x)\)，条件复杂度同理。概率分布 \(P\) 称为可计算，若存在图灵机对任意有限历史及有理误差可在有限步输出条件概率的近似。模型的描述长度记为其程序前缀码长 \(K(H)\)，或使用显式编码方案的代理复杂度 \(\mathcal{K}(\Pi)\)。

#### 1.3 观察者约束与合法理论类  
**定义 1.1（观察者与资源约束）**  
观察者 \(\mathcal{O}\) 由最大允许宏观状态数 \(R\in\mathbb{N}\) 和最大历史深度 \(L\in\mathbb{N}\) 共同刻画。\(R\) 为可稳定操作的最大粗粒化状态数目，\(L\) 为可编码的最大回溯长度。

**定义 1.2（受约束粗粒化映射与理论对）**  
给定 \(L,R\)，粗粒化映射 \(C:\Sigma^L \to \mathcal{Y}\) 将长度为 \(L\) 的过去窗口映射至宏观状态，\(\mathcal{Y}\) 有限且 \(|\mathcal{Y}| \le R\)。\(C\) 可由有限描述定义，因而具有有限柯氏复杂度。所有此类映射的集合记为 \(\mathscr{C}_{R,L}\)。一个理论对 \(\Pi = (C, P)\) 由 \(C\in\mathscr{C}_{R,L}\) 和宏观预测模型 \(P\) 构成，\(P\) 为给定宏观状态输出下一个符号的可计算条件分布。合法理论对全体记为 \(\mathcal{M}_{R,L}\)。由于 \(L\) 固定且字母表有限，\(\Sigma^L\) 有限，从而 \(\mathscr{C}_{R,L}\) 及 \(\mathcal{M}_{R,L}\) 均为有限集合。

#### 1.4 信息投影结构与统一泛函  
对任意 \(C\in\mathscr{C}_{R,L}\)，令 \(Y_t = C(Z_{t-L+1:t})\)。定义 **闭包缺陷** 为条件互信息  

\[
\iota_1(C) = I(Z_{t+1} ; \overleftarrow{Z}_t \mid Y_t).
\]

由平稳性，此值不依赖于 \(t\)。利用条件互信息与 KL 散度的关系，

\[
\iota_1(C) = \mathbb{E}_{\overleftarrow{Z}_t}\!\left[ D_{\mathrm{KL}}\big( \mu(\cdot\mid\overleftarrow{Z}_t) \,\big\|\, \mu(\cdot\mid Y_t) \big) \right],
\]

它度量了用宏观状态 \(Y_t\) 替代完整过去所带来的信息几何投影损失。

**定义 1.3（统一变分泛函）**  
对理论对 \(\Pi=(C,P)\)，定义  

\[
\mathcal{F}(C, P) = \iota_1(C) + \mathbb{E}_{Y_t}\!\left[ D_{\mathrm{KL}}\big(\mu(\cdot\mid Y_t)\,\big\|\, P(\cdot\mid Y_t) \big) \right].
\tag{1.1}
\]

当预测模型取为真实条件分布 \(P(\cdot\mid Y_t)=\mu(\cdot\mid Y_t)\) 时，模型误设项为零，此时  

\[
\mathcal{F}(C, \mu_C) = \iota_1(C),
\]

其中 \(\mu_C\) 表示由 \(C\) 导出的真实宏观条件分布。因此 \(\iota_1(C)\) 即是在理想预测下的统一泛函，成为粗粒化质量的核心度量。

**定义 1.4（完美闭包与零缺陷）**  
若 \(\iota_1(C)=0\)，称 \(C\) 具有完美一步闭包。由假设1中的 unifilarity 与一步可辨识性（见引理3.0），此时亦满足完整未来独立于过去给定 \(Y_t\)（即 \(I(\overrightarrow{Z}_t; \overleftarrow{Z}_t \mid Y_t)=0\)），简称完美闭包。此时称粗粒化映射 \(C\) 为零缺陷的。

#### 1.5 粗粒化格与精炼单调性  
**定义 1.5（精炼偏序与格）**  
固定 \(L\)，在 \(\mathscr{C}_{R,L}\) 上定义精炼关系：\(C' \preceq C\)（称 \(C'\) 是 \(C\) 的精炼）当且仅当存在满射 \(g:\mathcal{Y}'\to\mathcal{Y}\) 使得对任意 \(w\in\Sigma^L\)，有 \(C(w) = g(C'(w))\)。即 \(C'\) 的划分是 \(C\) 的划分的细分。\(\mathscr{C}_{R,L}\) 在此偏序下构成有限格，最大元为平凡映射（单一状态），最小元为恒等映射（每个历史为一状态，但受限于 \(R\) 可能被截断）。格的高度和宽度均有限。

**命题 1.1（精炼单调性）**  
若 \(C',C\in\mathscr{C}_{R,L}\) 满足 \(C'\preceq C\)，则  

\[
\iota_1(C') \le \iota_1(C).
\]

*证明*：因为 \(C'\) 是 \(C\) 的精炼，有 \(\sigma(Y_t) \subseteq \sigma(Y'_t)\)（均包含于 \(\sigma(\overleftarrow{Z}_t)\)）。计算条件互信息：  

\[
\iota_1(C) = H(Z_{t+1}\mid Y_t) - H(Z_{t+1}\mid \overleftarrow{Z}_t).
\]  

由于 \(Y'_t\) 是比 \(Y_t\) 更细的函数，\(H(Z_{t+1}\mid Y'_t) \le H(Z_{t+1}\mid Y_t)\)。而第二项不依赖于 \(C\)。故 \(\iota_1(C') = H(Z_{t+1}\mid Y'_t) - H(Z_{t+1}\mid \overleftarrow{Z}_t) \le \iota_1(C)\)。\(\square\)

#### 1.6 可辨识间隙与强可辨识性  
**引理 1.2（有限格间隙）**  
设 \(\mathcal{C}\) 为一个有限集合，\(\iota_1:\mathcal{C}\to[0,\infty)\) 为其上的函数。则集合 \(\{\iota_1(C) > 0 : C\in\mathcal{C}\}\) 若不为空，必有正的最小值。记  

\[
\Delta_0(R,L) = \min\{ \iota_1(C) : C\in\mathscr{C}_{R,L}, \iota_1(C) > 0 \}.
\]

对于固定的有限 \(R,L\)，该最小值存在且为正。本文主要工作在固定资源下展开，故常简写为 \(\Delta_0\)。需注意此间隙依赖于有限模型类；当 \(L\to\infty\) 或 \(R\to\infty\) 时，\(\Delta_0\) 的行为是开放问题。

**定义 1.6（强可辨识性）**  
真实过程 \(\mu\) 在模型类 \(\mathcal{M}_{R,L}\) 上称为 **强可辨识**，若：  
- (I1) 因果态模型所在的等价类 \([\Pi_0]\) 中所有理论对均满足 \(\mathcal{F}=0\)；且存在 \(\Delta>0\)，使得任意 \(\Pi\notin[\Pi_0]\) 有 \(\mathcal{F}(\Pi) \ge \Delta\)；  
- (I2) \([\Pi_0]\) 内所有粗粒化划分的 \(\sigma\)-代数与因果态划分的 \(\sigma\)-代数同构（在重标记意义下）。

在假设1下，若取 \(L\ge L_0\) 且 \(R\ge N\)，因果态模型属于 \(\mathcal{M}_{R,L}\)。由于模型类有限，由引理1.2，可取 \(\Delta = \min\{\mathcal{F}(\Pi)>0\}\)，强可辨识性自动满足。为行文清晰，我们将其列为设定成立的性质。

#### 1.7 显式模型编码与理论压缩代价  
对状态数 \(m\le R\)、概率精度 \(\delta_0\) 的有限状态模型，存在自定界编码方案使  

\[
\mathcal{K}(\Pi) \le \log m + m|\Sigma|(\log(1/\delta_0) + c_1) + c_2 .
\]

定义经验压缩代价  

\[
\mathcal{L}_{\text{emp}}(\Pi) = \mathcal{K}(\Pi) - \log P_{\Pi}(D),
\]

渐近期望形式为 \(\overline{\mathcal{L}}(\Pi) = T(h_\mu + \mathcal{F}(C, P_\Pi)) + \mathcal{K}(\Pi)\)。

### 2. 超额编码分解与闭包缺陷的恒等式

**命题 2.1（每步期望编码代价的统一分解）**  
设 \(\mu\) 平稳遍历，具有正则条件概率。对 \(\Pi = (C,P)\)，有  

\[
\mathbb{E}_\mu[-\log P(Z_{t+1}\mid Y_t)] = h_\mu + \mathcal{F}(C, P). \tag{2.1}
\]

*证明*：由交叉熵分解，  

\[
\mathbb{E}_\mu[-\log P(Z_{t+1}\mid Y_t)] = H(Z_{t+1}\mid Y_t) + \mathbb{E}_{Y_t}\!\left[ D_{\mathrm{KL}}(\mu(\cdot\mid Y_t) \,\|\, P(\cdot\mid Y_t)) \right].
\]  

由条件熵的链式法则，  

\[
H(Z_{t+1}\mid Y_t) = H(Z_{t+1}\mid Y_t, \overleftarrow{Z}_t) + I(Z_{t+1};\overleftarrow{Z}_t\mid Y_t).
\]  

因 \(Y_t\) 是 \(\overleftarrow{Z}_t\) 的函数，\(H(Z_{t+1}\mid Y_t,\overleftarrow{Z}_t) = H(Z_{t+1}\mid\overleftarrow{Z}_t)\)。在平稳遍历下，\(H(Z_{t+1}\mid\overleftarrow{Z}_t) = h_\mu\)。代入即得。\(\square\)

**推论 2.1**  期望编码超额量恰好分解为结构缺陷 \(\iota_1\) 与模型误设 KL 期望之和。最小化压缩代价等价于折衷模型复杂度与联合泛函 \(\mathcal{F}\)。

**附加结构（双投影几何解释）**  
上述分解可被视为条件分布流形上的两阶段信息投影结构：  

\[
\mu(\cdot\mid\overleftarrow{Z}_t) \xrightarrow{\text{Blackwell 投影}} \mu(\cdot\mid Y_t) \xrightarrow{\text{KL 投影}} P(\cdot\mid Y_t),
\]

其中第一段投影误差即 \(\iota_1(C)\)，第二段误差为模型误设 KL。\(\mathcal{F}\) 因此是这两个投影误差之和，揭示了信息几何与变分压缩的内在联系。将第一步严格化为 Bregman 投影是第六部分列出的开放问题之一。

### 3. 因果态、统计复杂度与统一泛函下的最优性

#### 3.1 因果态与极小充分统计量  
**定义 3.1（因果态）**  
对过去序列 \(\overleftarrow{z},\overleftarrow{z}'\in\Sigma^{\mathbb{Z}_{<0}}\)，定义等价关系 \(\sim_\epsilon\)：\(\overleftarrow{z}\sim_\epsilon \overleftarrow{z}'\) 当且仅当对任意未来有限序列 \(a\)，\(\mu(Z_{1:|a|}=a\mid\overleftarrow{z}) = \mu(Z_{1:|a|}=a\mid\overleftarrow{z}')\)（几乎处处）。等价类称为因果态，映射记为 \(\epsilon:\overleftarrow{z}\mapsto\sigma\)。因果态集合 \(\mathcal{S}\) 有限，\(N=|\mathcal{S}|\)。统计复杂度 \(C_\mu = H(\epsilon)\)。由假设1，存在 \(L_0\) 使 \(\epsilon\) 可由长度 \(L_0\) 的历史窗口确定，且状态转移 unifilar。

**引理 3.0（一步闭包升华为完美闭包）**  
在假设1（有限记忆、一步可辨识性、unifilar 因果态）下，若粗粒化 \(C\in\mathscr{C}_{R,L}\) （其中 \(L\ge L_0\)）满足 \(\iota_1(C)=0\)，则对任意 \(k\ge 1\) 有 \(I(Z_{t+1:t+k}; \overleftarrow{Z}_t \mid Y_t) = 0\)，即实现完美闭包。

*证明*：设 \(\iota_1(C)=0\)，即 \(I(Z_{t+1};\overleftarrow{Z}_t\mid Y_t)=0\)。这意味着给定 \(Y_t\) 后，\(Z_{t+1}\) 与完整过去 \(\overleftarrow{Z}_t\) 条件独立。对任意两个过去 \(\overleftarrow{z}, \overleftarrow{z}'\) 映射到同一 \(Y_t\) 值，必有 \(\mu(\cdot\mid\overleftarrow{z}) = \mu(\cdot\mid\overleftarrow{z}')\)（否则会有信息通过一步预测泄露）。设它们对应的因果态分别为 \(\sigma, \sigma'\)。由于一步可辨识性，若 \(\sigma \neq \sigma'\) 则 \(\mu(\cdot\mid\sigma)\neq\mu(\cdot\mid\sigma')\)，与前述相等矛盾。故 \(\sigma=\sigma'\)。因此，每个 \(Y_t\) 值对应的过去全部属于同一因果态，即存在函数 \(f\) 使得因果态 \(\sigma = f(Y_t)\) 几乎处处成立。  
现在考虑未来。由 unifilarity，给定当前因果态 \(\sigma_t\) 和未来符号序列 \(Z_{t+1}, Z_{t+2}, \dots\)，整个未来分布仅依赖于 \(\sigma_t\)。因为 \(\sigma_t = f(Y_t)\)，故未来分布仅依赖于 \(Y_t\)，即 \(I(Z_{t+1:t+k}; \overleftarrow{Z}_t \mid Y_t)=0\)。\(\square\)

#### 3.2 因果态的最优性  
**定理 3.1（极小充分性）**  
设 \(L\ge L_0\)，\(R\ge N\)。在所有 \(\mathscr{C}_{R,L}\) 中满足完美闭包（即 \(\iota_1(C)=0\)）的粗粒化映射中，因果态划分 \(\epsilon\)（视为 \(L\) 窗口上的映射）具有最小的状态数及最小的状态熵 \(H(\epsilon)\)。若 \(C\in\mathscr{C}_{R,L}\) 满足 \(\iota_1(C)=0\)，则 \(\epsilon\) 是 \(C\) 的粗化（即存在 \(f\) 使 \(\epsilon = f\circ C\)），从而 \(H(\epsilon)\le H(C)\)。

*证明*：由引理3.0，\(\iota_1(C)=0\) 推出 \(Y_t\) 是因果态的函数，故每个 \(Y\) 值对应的窗口完全属于同一因果态。因此因果态划分的等价类是 \(C\)-划分块的并集，即 \(\epsilon = f\circ C\)。因此状态数 \(|Y_C|\ge N\)，且若 \(|Y_C|=N\) 则 \(f\) 为双射。熵的单调性推出 \(H(\epsilon)\le H(C)\)。因果态划分在所有零缺陷划分中状态数最小。\(\square\)

**定理 3.2（统一泛函全局最优性）**  
在假设1下，取 \(L\ge L_0\)，\(R\ge N\)，令 \(\Pi_\epsilon = (\epsilon, \mu(\cdot\mid\epsilon))\)。则 \(\mathcal{F}(\Pi_\epsilon)=0\)，且任何 \(\Pi\in\mathcal{M}_{R,L}\) 若满足 \(\mathcal{F}(\Pi)=0\)，则其粗粒化划分的 \(\sigma\)-代数与因果态 \(\sigma\)-代数同构（即存在双射重标记），且 \(P\) 几乎处处等于 \(\mu(\cdot\mid Y_t)\)（属于等价类 \([\Pi_\epsilon]\)）。在等价类内，因果态划分具有最小状态数。

*证明*：\(\mathcal{F}(\epsilon,\mu)=0\) 显然。若 \(\Pi\) 也达到零，则 \(\iota_1(C)=0\) 且 KL 期望为零，故几乎处处 \(P(\cdot\mid Y_t)=\mu(\cdot\mid Y_t)\)。由定理3.1，\(\epsilon = f\circ C\)。由于 \(C\) 的状态数不超过 \(R\) 且因果态数为 \(N\)，必有 \(|Y_C|\ge N\)。若 \(|Y_C| > N\)，则 \(C\) 是因果态的严格精炼，但通过合并相同因果态的状态可得到状态数更少的等价模型，其 \(\mathcal{F}\) 仍为零，且状态数为 \(N\)。因果态划分就是这样的最小代表。由强可辨识性(I2)，等价类内所有划分的 \(\sigma\)-代数均同构于因果态划分。\(\square\)

## 第二部分：闭包变分算子与吸引子结构

### 4. 资源约束下的变分闭包算子、Lyapunov 函数与不动点分析

在粗粒化映射格上构造一个基于统一泛函最小化的确定性变分算子，建立 \(\iota_1\) 作为 Lyapunov 函数的严格框架，完整刻画不动点集，并严格区分路径存在性与策略依赖性。

**定义 4.1（预测等价与合并算子 \(M\)）**  
对 \(C\in\mathscr{C}_{R,L}\)，两个状态 \(y_1,y_2\in\mathcal{Y}\) 称为 **预测等价**，若  
\[
\mu(\cdot\mid Y_t = y_1) = \mu(\cdot\mid Y_t = y_2) \quad \text{几乎处处。}
\]  
定义合并算子 \(M: \mathscr{C}_{R,L} \to \mathscr{C}_{R,L}\)：将 \(C\) 的状态按预测等价类合并，得到商映射 \(M(C)\)，其状态数等于预测等价类的数目。显然 \(M\) 是幂等的：\(M(M(C))=M(C)\)，且状态数不增加：\(|Y_{M(C)}| \le |Y_C|\)。合并仅归并条件分布相同的状态，条件互信息不变：

\[
\iota_1(M(C)) = \iota_1(C).
\]

称 \(C\) 为 **约化** 的，若 \(M(C)=C\)，即内部无不同状态具有相同预测分布。

**定义 4.2（有效分裂的存在性与严格凹性证明）**  
设 \(C\) 是约化的，且 \(\iota_1(C)>0\)。由于 \(\iota_1(C) = H(Z_{t+1}\mid Y_t) - H(Z_{t+1}\mid\overleftarrow{Z}_t) > 0\)，宏观状态无法完全捕获过去的全部预测信息，故必存在某个状态 \(y_0\in\mathcal{Y}\) 和两个过去窗口子集 \(A,B\subset C^{-1}(y_0)\)，使得 \(\mu(\cdot\mid A) \neq \mu(\cdot\mid B)\)。在假设1下，这等价于这些窗口来自至少两个不同的因果态。因 \(C\) 是约化的，状态 \(y_0\) 中无两个子集具有完全相同的预测分布，故我们可取 \(A\) 为某一因果态在 \(y_0\) 内的全部窗口，\(B = C^{-1}(y_0)\setminus A\)。两者均非空。定义新映射 \(C'\)：将状态 \(y_0\) 分裂为 \(A\) 与 \(B\)，保持其他状态不变。显然 \(|Y_{C'}| = |Y_C| + 1\)。现在比较条件熵。令 \(w_A = \mu(C^{-1}(y_0)\cap A)/\mu(C^{-1}(y_0))\)，\(w_B = 1-w_A\)，则原状态 \(y_0\) 上的局部条件分布为混合分布 \(p = w_A p_A + w_B p_B\)。该状态对总条件熵的贡献为 \(\pi(y_0) H(p)\)（其中 \(\pi(y_0)=\mu(Y_t=y_0)\)）。分裂后，两个新状态分别贡献 \(\pi(A) H(p_A)\) 和 \(\pi(B) H(p_B)\)。由于熵的严格凹性，当 \(p_A \neq p_B\) 时，

\[
H(p) > w_A H(p_A) + w_B H(p_B).
\]

两边乘以 \(\pi(y_0)\)，并注意到 \(\pi(A) = w_A \pi(y_0)\)，\(\pi(B) = w_B \pi(y_0)\)，可得

\[
\pi(y_0) H(p) > \pi(A) H(p_A) + \pi(B) H(p_B).
\]

因此整个条件熵 \(H(Z_{t+1}\mid Y_t)\) 严格下降，从而 \(\iota_1(C') = H(Z_{t+1}\mid Y'_t) - H(Z_{t+1}\mid\overleftarrow{Z}_t) < \iota_1(C)\)。由引理1.2，下降量至少为 \(\Delta_0\)。称此操作为 **有效分裂**。  

**定义 4.3（分裂算子 \(S\) 与变分闭包算子 \(\mathcal{V}_{R,L}\)）**  
在约化映射子集 \(\mathscr{C}^{\text{red}}_{R,L} = \{C\in\mathscr{C}_{R,L} : M(C)=C\}\) 上定义分裂算子 \(S\)：对 \(C\in\mathscr{C}^{\text{red}}_{R,L}\)，
- 若 \(\iota_1(C)=0\) 或 \(|Y_C| = R\)，则 \(S(C) = C\)；
- 否则（即 \(\iota_1(C)>0\) 且状态数小于 \(R\)），选取一次有效分裂所得之映射作为 \(S(C)\)（若存在多个候选分裂，按固定确定性规则选取其一，例如分割最大化条件熵下降者）。

定义 **变分闭包算子**  

\[
\mathcal{V}_{R,L} = S \circ M : \mathscr{C}_{R,L} \to \mathscr{C}_{R,L}.
\]

该算子由两步构成：先合并预测等价状态至约化形式（不改变 \(\iota_1\)），再在约化表示上依据资源约束和缺陷执行分裂操作（严格下降 \(\iota_1\) 或停止）。

**命题 4.1（算子的基本性质与 Lyapunov 函数）**  
对任意 \(C\in\mathscr{C}_{R,L}\)，记 \(C' = \mathcal{V}_{R,L}(C)\)。则：
- (i) \(\iota_1(C') \le \iota_1(C)\)，且等号成立当且仅当 \(S(M(C))=M(C)\)；
- (ii) 若 \(C\) 不是不动点（即 \(C' \neq C\)），则 \(\iota_1(C') \le \iota_1(C) - \Delta_0\)；
- (iii) 算子输出的状态数始终 \(\le R\)；
- (iv) \(\iota_1\) 是此变分动力学的严格 Lyapunov 函数：迭代序列 \(C_{k+1} = \mathcal{V}_{R,L}(C_k)\) 使 \(\iota_1\) 单调不增，且严格下降直到抵达不动点。

*证明*：(i) \(M\) 不改变 \(\iota_1\)，故 \(\iota_1(M(C)) = \iota_1(C)\)。若 \(S\) 执行了分裂，则由有效分裂性质，\(\iota_1\) 严格下降；若未分裂，则不变。(ii) 若非不动点，则必有 \(M(C)\neq C\) 或 \(S(M(C))\neq M(C)\)。若 \(M(C)\neq C\)，则状态数减少而 \(\iota_1\) 不变，下一步算子会作用于约化状态，若此时仍非不动点，则必然在 \(S\) 步发生分裂，使 \(\iota_1\) 严格下降至少 \(\Delta_0\)。因此综合两步后，\(\iota_1\) 至少下降 \(\Delta_0\)。(iii) 合并步不增状态数，分裂步仅在状态数 \(<R\) 时执行，故总状态数 \(\le R\)。(iv) 由 (i)(ii) 直接可得。\(\square\)

**定理 4.2（有限步收敛与不动点分类）**  
设过程满足假设1，且 \(L\ge L_0\)。对任意初始 \(C_0\in\mathscr{C}_{R,L}\)，迭代序列 \(C_{k+1} = \mathcal{V}_{R,L}(C_k)\) 在有限步内终止于某个不动点 \(C_\infty = \mathcal{V}_{R,L}(C_\infty)\)。所有不动点分为且仅分为两类：

1. **零缺陷不动点**：\(\iota_1(C_\infty)=0\)。此时 \(C_\infty\) 满足完美闭包，其状态数与 \(\sigma\)-代数同构于因果态划分（在重标记意义下），状态数恰为 \(N\)。因果态划分（当 \(R\ge N\) 时属于 \(\mathscr{C}_{R,L}\)）是此类中唯一的极小代表。

2. **资源壁垒不动点（亚稳闭包结构）**：\(\iota_1(C_\infty)>0\)。此时必满足 \(M(C_\infty)=C_\infty\)（状态分布两两互异）且状态数 \(|Y_{C_\infty}| = R\)。因资源耗尽而无法通过分裂进一步降低缺陷，但缺陷严格大于零。

当 \(R < N\) 时，零缺陷不动点不可达（状态数不足），所有收敛子必为资源壁垒不动点。

*证明*：因 \(\iota_1\) 取值于有限集，每次严格下降至少 \(\Delta_0\)，故迭代只能经历有限次严格下降，从而在有限步内终止，即达到不动点。设不动点为 \(C_\infty\)。

若 \(\iota_1(C_\infty)=0\)：由引理3.0，\(C_\infty\) 实现完美闭包。由定理3.1，因果态划分 \(\epsilon\) 是 \(C_\infty\) 的粗化（\(\epsilon = f\circ C_\infty\)），且 \(C_\infty\) 满足 \(M(C_\infty)=C_\infty\)（否则合并会改变它，与不动点矛盾）。故其状态均预测互异。由因果态极小性，\(|Y_{C_\infty}| \ge N\)。若 \(|Y_{C_\infty}| > N\)，则由鸽巢原理必有两个状态对应同一因果态，从而预测分布相同，与 \(M(C_\infty)=C_\infty\) 矛盾。故 \(|Y_{C_\infty}| = N\) 且 \(f\) 为双射，即 \(\sigma\)-代数同构。

若 \(\iota_1(C_\infty)>0\)：因是不动点，必满足 \(M(C_\infty)=C_\infty\) 且 \(S(C_\infty)=C_\infty\)。由 \(S\) 的定义，这意味着要么 \(\iota_1=0\)（矛盾），要么状态数已达 \(R\)。因此为资源壁垒不动点。\(\square\)

**定理 4.3（路径存在性与局部变分流陷阱）**  
设 \(R \ge N\)，\(L \ge L_0\)。

- **(存在性)** 对任意初态 \(C_0 \in \mathscr{C}_{R,L}\)，存在一条由合并与有效分裂构成的有限操作序列，使得从 \(C_0\) 可到达一个零缺陷不动点（因果态划分等价类）。具体构造：首先将 \(C_0\) 强制合并为平凡映射（单状态，状态数 1 ≤ R），然后逐次分离不同的因果态窗口，每次分裂增加一个纯因果态，最终在状态数达到 \(N\) 时得到零缺陷不动点。
- **(局部变分流限制)** 变分算子 \(\mathcal{V}_{R,L}\) 使用局部邻域搜索，仅采取不增加 \(\iota_1\) 的移动（合并）或严格降低 \(\iota_1\) 的移动（分裂），不会主动执行增加缺陷的强制合并。因而可能在某些初态下收敛到资源壁垒不动点（亚稳态），未能到达全局零缺陷不动点。此现象表明，从给定初态出发的局部变分流不一定收敛到全局最优，取决于初始状态和确定性选择规则。亚稳区域的大小与结构由资源约束和过程细节决定，其严格刻画为开放问题。

*证明*：存在性由显式构造保证：强制合并虽可能暂时增大 \(\iota_1\)，但作为外部的全局操作重置了状态数为1，然后通过定向分裂（每次分离出一个因果态的所有历史）逐步纯化，状态数单调增至 \(N\)，过程中始终满足 ≤ R，且最后达到零缺陷。此路径不局限于算子 \(\mathcal{V}_{R,L}\) 的单步局部下降，而是在更大的变换空间中存在。局部变分流只允许 \(\iota_1\) 不变或下降，因此不会执行增大 \(\iota_1\) 的强制合并，从而可能陷入局部极小。\(\square\)

### 5. 局部变分压缩与 MDL 能量景观

将模型类 \(\mathcal{M}_{R,L}\) 视为离散构型空间，能量函数 \(\mathcal{E}(\Pi) = \mathcal{L}_{\text{emp}}(\Pi)\)。定义邻域 \(\mathcal{N}(\Pi)\) 包含通过合并或分裂一个状态（或调整概率参数）而得到的模型，并保持状态数 \(\le R\)。定义局部变分压缩算子 \(\widetilde{\mathcal{V}}\) 为在邻域内选取能量最小者（若存在多个极小按固定规则选一）。

**定理 5.1（严格单调下降与有限步终止）**  
\(\mathcal{M}_{R,L}\) 有限，能量取值集有限，存在最小分辨率 \(\delta>0\)。若非不动点，则 \(\mathcal{E}(\widetilde{\mathcal{V}}(\Pi)) \le \mathcal{E}(\Pi) - \delta\)。迭代必在有限步内终止于压缩不动点。

*证明*：因模型类有限，能量只能取有限个值，取 \(\delta\) 为最小非零能量差。每次严格下降至少减少 \(\delta\)，有限步后终止。\(\square\)

**定理 5.2（因果态模型的局部极小性与大样本性质）**  
设过程满足假设1，且 \(L\ge L_0\)，\(R\ge N\)。则存在 \(T_0\)，当样本量 \(T>T_0\) 时，因果态模型 \(\Pi_0\) 以大概率成为其邻域内严格能量极小点。同时，任何资源壁垒不动点对应的模型，若其 \(\mathcal{F}>0\)，其能量在 \(T\to\infty\) 时以概率1严格高于 \(\Pi_0\) 的能量，差距至少为 \(T\Delta/2\)（\(\Delta\) 为强可辨识性下界）。

*证明*：与之前版本类似，利用一致收敛和强可辨识性。\(\square\)

## 第三部分：一致收敛与有限样本保证

### 6. 有限样本 MDL 误差上界

**设定**：真实过程 \(\mu\) 平稳且满足指数 \(\alpha\)-混合，对数损失一致有界（字母表有限）。模型类 \(\mathcal{M}=\mathcal{M}_{R,L}\) 有限。

**引理 6.1（指数混合过程的均匀浓度不等式）**  
存在常数 \(c_1,C_1>0\)，使对任意 \(\varepsilon>0\)，

\[
\mathbb{P}\!\left( \sup_{\Pi\in\mathcal{M}} \left| \frac{1}{T}\sum_{t=1}^T f_t(\Pi) - \ell(\Pi) \right| > \varepsilon \right)
\le 2|\mathcal{M}| \exp( -c_1 T \varepsilon^2 ).
\]

*证明概要*：利用 Berbee 耦合将序列分解为独立块，对每个块应用 Hoeffding 不等式，再以联合界覆盖有限模型类。具体构造参见 Rio (1999) 及 Grünwald (2007) 中 MDL 一致收敛的标准论证。\(\square\)

**定理 6.1（经验压缩代价一致收敛）**  
\[
\sup_{\Pi\in\mathcal{M}} \left| \frac{1}{T}\mathcal{L}_{\text{emp}}(\Pi) - \left(\ell(\Pi) + \frac{\mathcal{K}(\Pi)}{T}\right) \right| \xrightarrow{\text{a.s.}} 0,
\]
速率为 \(O(\sqrt{\log |\mathcal{M}| / T})\)。由 Borel–Cantelli 直接可得。

**定理 6.2（全局经验最小化器的一致性）**  
在相同条件下，令 \(\Pi_T^* = \arg\min_{\Pi\in\mathcal{M}} \mathcal{E}(\Pi)\)。则几乎必然有 \(\lim_{T\to\infty} \mathcal{F}(\Pi_T^*) = 0\)，且其粗粒化划分的 \(\sigma\)-代数收敛到因果态划分的 \(\sigma\)-代数（重标记意义）。

*证明*：由一致收敛，\(\frac{1}{T}\mathcal{E}(\Pi)\to \ell(\Pi)\) 一致。对足够大 \(T\)，全局最小化器必然落入真实等价类 \([\Pi_0]\)，因为其他模型的经验代价高出至少 \(T\Delta - O(1)\)，其中 \(\Delta\) 为强可辨识性下界。等价类内 \(\mathcal{F}=0\)，由强可辨识性，划分的 \(\sigma\)-代数同构于因果态。\(\square\)

## 第四部分：闭包阈值转变与嵌套 MDL 粗粒化流

### 7. 资源诱导的闭包阈值转变

**定义 7.1（最优联合泛函与序参量）**  
对固定的 \(L\)，定义  

\[
\mathcal{F}_L^*(R) = \inf_{\Pi \in \mathcal{M}_{R,L}} \mathcal{F}(C_\Pi, P_\Pi).
\]

序参量向量 \(\mathbf{\Omega}(\Pi)=(\iota_1(C), H(C), \mathcal{K}(\Pi))\)。

**定理 7.1（刚性闭包阈值转变）**  
设过程满足假设1，且 \(L\ge L_0\)。则 \(\mathcal{F}_L^*(R)\) 是关于 \(R\) 的阶梯函数：

\[
\mathcal{F}_L^*(R) = \begin{cases}
\ge \Delta_0 > 0, & R < N,\\
= 0, & R \ge N,
\end{cases}
\]

其中 \(\Delta_0\) 为模型类中非零最小缺陷（依赖于 \(R,L\)）。当 \(R\) 跨过临界值 \(N\) 时，零缺陷闭包解突然涌现，此现象称为 **资源诱导的闭包阈值转变**。在有限模型类中，它体现为严格的刚性阶梯，而非热力学极限下的连续相变。当 \(R > N\) 时，虽然全局下确界为零，但局部变分压缩动力学可能存在资源壁垒亚稳不动点，其缺陷严格大于零，构成亚稳闭包相。这些亚稳态是资源过剩情形下局部变分流可能陷入的陷阱。

*证明*：当 \(R < N\)，任何模型状态数不足，必然合并某些不同因果态。由一步可辨识性，这些被合并的因果态具有不同的一步预测分布，故给定粗粒化状态后过去仍包含区分信息，从而 \(\iota_1>0\)。由引理1.2，正缺陷有一致下界 \(\Delta_0\)，故 \(\mathcal{F}_L^*(R) \ge \Delta_0\)。当 \(R\ge N\)，因果态模型可实现，达到零。当 \(R > N\)，资源壁垒不动点可能存在，此时 \(\iota_1>0\) 且为局部极小（在算子 \(\mathcal{V}_{R,L}\) 的意义下不可继续下降），但全局下确界仍为零。\(\square\)

**注记 7.1（与统计物理相变的类比与界限）**  
本文的“阈值转变”在有限系统内是组合突变，严格依赖于有限划分格的间隙 \(\Delta_0\)。真正的统计物理相变通常要求热力学极限下的奇异性。但在资源受限的观察者理论中，这种有限容量阈值结构已经是捕获规律突变的有效原型。将 \(L\to\infty\) 或允许状态数连续化后是否存在真正的奇异性，是开放问题。

### 8. 尺度依赖压缩算子与饱和

固定 \(R\) 充分大 (\(R\ge N\))，现在变化历史窗口长度 \(L\)。定义模型类 \(\mathcal{M}_{R,L}\) 及相应最优期望压缩代价  

\[
\overline{\mathcal{L}}_R^*(L) = \inf_{\Pi\in\mathcal{M}_{R,L}} \overline{\mathcal{L}}(\Pi).
\]

**定理 8.1（有限因果态过程的尺度饱和）**  
若真实过程满足假设1，记忆长度 \(L_0\) 足够确定因果态，则对所有 \(L \ge L_0\)，

\[
\overline{\mathcal{L}}_R^*(L) = \overline{\mathcal{L}}_R^*(L_0),
\]

且最优模型等价于因果态模型。尺度饱和点 \(L_s = L_0\)。

*证明*：当 \(L\ge L_0\)，因果态划分可由窗口 \(L\) 表达，其代价已达全局下界 \(T h_\mu + \mathcal{K}(\epsilon)\)。增大 \(L\) 不增加新的区分能力，下确界不再降低。\(\square\)

**定义 8.1（尺度依赖压缩算子族）**  
对 \(L_1 < L_2\)，定义嵌入映射 \(\mathfrak{E}_{L_2,L_1} : \mathcal{M}_{R,L_1} \to \mathcal{M}_{R,L_2}\)，将基于短历史的模型扩展为长历史模型（忽略多余符号）。定义压缩算子 \(\mathcal{R}_{L_2,L_1} = \mathcal{V}_{R,L_2} \circ \mathfrak{E}_{L_2,L_1}\)，在扩展后的模型类上迭代变分闭包算子至不动点。

**猜想 8.2（嵌套 MDL 粗粒化流的半群性与重整化群）**  
在 \(T\to\infty\) 极限下，算子族近似满足半群性质  

\[
\mathcal{R}_{L_3,L_2} \circ \mathcal{R}_{L_2,L_1} \simeq \mathcal{R}_{L_3,L_1}.
\]

在因果态不动点处线性化，可定义相关/无关方向，与统计场论中重整化群变换形成对偶。该半群的严格构造及其谱分解是核心开放问题（见第六部分）。

## 第五部分：闭包结构公理与双流架构

### 9. 闭包结构作为普适阈值转变公理

上述定理提炼出 **闭包结构**：在给定资源 \((R,L)\) 约束下，变分压缩动力学的稳定宏观态划分（不动点）。所有局部闭包结构构成空间 \(\mathbb{S}(R,L)\)。闭包阈值转变即 \(\mathbb{S}\) 在参数空间中的分岔。我们将其提炼为以下公理：

> **公理（闭包结构）**：对于平稳遍历且具有有限 unifilar 因果态的随机过程，在观察者受记忆深度 \(L\) 与状态容量 \(R\) 约束的条件下，可行的稳定粗粒化划分恰为变分压缩算子 \(\mathcal{V}_{R,L}\) 的不动点。当资源跨越过程的固有复杂度与特征尺度时，不动点集合发生阈值式跃变：零缺陷完美闭包结构在 \(R\ge N, L\ge L_0\) 时涌现，并可能伴随亚稳闭包相（资源壁垒不动点）。该公理独立于具体过程细节，仅依赖于因果态数目 \(N\) 和特征记忆尺度 \(L_0\)，属于规律发现的普适结构。

该公理统合计算力学、MDL 与阈值转变理论，为提取普适的规律发现相变类提供了严格数学语言。

### 10. 理想动力学与可计算算法：双流架构

清晰区分两个层面：

- **理想数学流**：基于真实测度和精确互信息的变分闭包算子与局部压缩算子，定义在有限模型格上的确定性动力系统。其不动点的数学性质（分类、Lyapunov 稳定性、路径存在性）已在上文得到严格证明。理想流为算法提供了真值基准。
- **可计算算法流**：实际算法需使用有限样本、统计检验与近似推断。闭包缺陷分解为统计误差、模型类受限误差和计算截断误差。理想吸引子在噪声下可能退化为亚稳态。两者在有限状态遍历过程的大样本极限下重合（由定理6.2保证）。

### 7.1 Golden Mean Process（黄金均值过程）

#### 7.1.1 过程定义与因果态
字母表 \(\Sigma = \{0,1\}\)。过程禁止连续两个 \(1\)，并在所有合法序列上赋予均匀测度（即拓扑熵最大测度）。其最小 unifilar ε‑machine 具有两个因果态  
\[
\mathcal{S} = \{A, B\},
\]
转移概率与符号输出为  
\[
\begin{aligned}
A &\xrightarrow{0:\; \frac{1}{2}} A, \quad A \xrightarrow{1:\; \frac{1}{2}} B,\\
B &\xrightarrow{0:\; 1} A, \quad B \xrightarrow{1:\; 0} \text{（禁止）}.
\end{aligned}
\]
状态 \(A\) 表示上一符号为 \(0\)，状态 \(B\) 表示上一符号为 \(1\)。一步预测分布  
\[
\mu(\cdot|A) = \bigl(\tfrac{1}{2},\,\tfrac{1}{2}\bigr),\qquad 
\mu(\cdot|B) = (1,\,0),
\]
满足一步可辨识性。因果态数 \(N=2\)，且有限记忆 \(L_0 = 1\)：因果态由最近一个符号完全确定。平稳分布  
\[
\pi_A = \frac{2}{3},\quad \pi_B = \frac{1}{3},
\]
熵率 \(h_\mu = \frac{2}{3}\) bit。该过程严格满足 **假设1** 的全部条件。

#### 7.1.2 观察者设置与窗口划分格
取历史深度 \(L=2 \ge L_0\)。长度为2的合法窗口集合  
\[
\mathcal{W} = \{00,\,01,\,10\},
\]
各窗口的平稳概率均为 \(\frac{1}{3}\)。粗粒化映射集合 \(\mathscr{C}_{R,2}\) 为所有将 \(\mathcal{W}\) 映射到至多 \(R\) 个宏观状态的函数。  
以下按资源约束 \(R\) 分别计算闭包缺陷的下确界与不动点结构，完整检验定理7.1与定理4.2。

##### 情形1：\(R = 1\)（容量不足）
唯一粗粒化映射为平凡映射 \(C_{\text{triv}}\)：所有窗口映射到单一状态 \(y_0\)。宏观状态不提供任何信息，给定 \(Y_t=y_0\) 的一步预测分布为窗口预测分布的混合：
\[
\mu(\cdot|y_0) = \frac{1}{3}\mu(\cdot|00) + \frac{1}{3}\mu(\cdot|01) + \frac{1}{3}\mu(\cdot|10).
\]
因条件预测仅由窗口末位符号决定，且
\[
\mu(\cdot|00)=\bigl(\tfrac12,\tfrac12\bigr),\quad
\mu(\cdot|01)=(1,0),\quad
\mu(\cdot|10)=\bigl(\tfrac12,\tfrac12\bigr),
\]
故
\[
\mu(\cdot|y_0) = \frac{2}{3}\bigl(\tfrac12,\tfrac12\bigr) + \frac{1}{3}(1,0) = \Bigl(\frac{2}{3},\,\frac{1}{3}\Bigr).
\]
条件熵  
\[
H(Z_{t+1}\mid Y_t) = H\!\Bigl(\frac{2}{3},\frac{1}{3}\Bigr) \approx 0.918 \text{ bit},
\]
而 \(H(Z_{t+1}\mid\overleftarrow{Z}_t)=h_\mu=\frac{2}{3}\approx0.667\) bit。因此闭包缺陷  
\[
\iota_1(C_{\text{triv}}) = 0.918 - 0.667 = 0.251 > 0.
\]
由于 \(R=1<N=2\)，零缺陷解不可达，最优联合泛函下确界 \(\mathcal{F}_2^*(1)=0.251>0\)，确认定理7.1的刚性阈值左侧。

##### 情形2：\(R = 2\)（容量充足）
此时最大状态数等于因果态数。我们枚举所有约化的（即无预测等价状态）两状态粗粒化划分。将 \(\mathcal{W}\) 划分为两个非空块的方法共三种（不计标号对称），分别对应以下窗口划分：
1. **因果态划分 \(C_\epsilon\)**：\(\{00,10\}\) 与 \(\{01\}\)。该划分恰好按最后一个符号（从而按因果态）区分窗口。两个状态的预测分布  
   \[
   \mu(\cdot|y_0)=\bigl(\tfrac12,\tfrac12\bigr),\quad \mu(\cdot|y_1)=(1,0),
   \]
   故给定宏观状态后不再残留过去的信息：\(\iota_1(C_\epsilon)=0\)。这是唯一的零缺陷不动点（完美闭包结构）。

2. **非因果态划分 \(C^{(1)}\)**：\(\{00\}\) 与 \(\{01,10\}\)。块预测分布  
   \[
   \mu(\cdot|y_0)=\bigl(\tfrac12,\tfrac12\bigr),\qquad
   \mu(\cdot|y_1)=\frac{1}{2}(1,0)+\frac{1}{2}\bigl(\tfrac12,\tfrac12\bigr) = (0.75,\,0.25).
   \]
   条件熵  
   \[
   H(Z_{t+1}\mid Y_t) = \frac{1}{3}H\!\bigl(\tfrac12,\tfrac12\bigr) + \frac{2}{3}H(0.75,0.25) \approx 0.874\text{ bit},
   \]
   故 \(\iota_1(C^{(1)})=0.874-0.667=0.207>0\)。

3. **非因果态划分 \(C^{(2)}\)**：\(\{10\}\) 与 \(\{00,01\}\)。对称计算得 \(\iota_1(C^{(2)})=0.207\)。

所有约化的两状态划分中，除 \(C_\epsilon\) 外缺陷均为正（约0.207）。由于 \(R=2\)，这些正缺陷划分的状态数已达上限，无法执行有效分裂；且内部无预测等价状态，合并算子 \(M\) 也不改变它们。因此它们是变分闭包算子 \(\mathcal{V}_{2,2} = S\circ M\) 的 **资源壁垒不动点**（亚稳闭包结构）。全局下确界为零，局部极小与全局极小共存，完全验证定理4.2的不动点分类。

#### 7.1.3 静态与动力学谱间隙
在 \(\mathscr{C}_{2,2}\) 中，所有可能出现的 \(\iota_1\) 值构成有限集合
\[
\{0,\;0.207,\;0.251\}.
\]
定义该集合上的 **静态谱间隙** 为所有相异值间的最小正差：
\[
\Delta_{\mathrm{spec}} = \min\{|a-b| : a,b\in\{\iota_1(C)\},\, a\neq b\} = 0.044.
\]
然而，变分流的有限步收敛并不直接依赖静态谱间隙，而需依赖 **动力学谱间隙**——在允许局部移动下实际可实现的能量下降量的最小值。形式地，对每个 \(C\in\mathscr{C}_{2,2}\)，考虑其邻域 \(\mathcal{N}(C)\)（依定义4.3，包含合并与有效分裂），定义
\[
\Delta_{\mathrm{dyn}} = \min_{\substack{C'\in\mathcal{N}(C),\; \iota_1(C')<\iota_1(C)}} \bigl(\iota_1(C)-\iota_1(C')\bigr),
\]
其中的最小值遍历所有存在严格下降的邻接边。在本例中：
- \(C_{\text{triv}}\)（缺陷0.251）可通过一次有效分裂变为 \(C^{(1)}\) 或 \(C^{(2)}\)（缺陷0.207），下降量 \(0.251-0.207=0.044\)；也可直接变为 \(C_\epsilon\)，下降0.251。故在 \(C_{\text{triv}}\) 处的最小下降量为 \(0.044\)。
- \(C^{(1)}\) 与 \(C^{(2)}\) 均无任何严格下降的邻域移动（合并返回平凡映射将增大能量，不在考虑之列）。
因而整个景观上的动力学谱间隙 \(\Delta_{\mathrm{dyn}} = 0.044 = \Delta_{\mathrm{spec}}\)。一般而言，\(\Delta_{\mathrm{dyn}} \ge \Delta_{\mathrm{spec}}\)，且对于有限模型类两者均为正。正是 \(\Delta_{\mathrm{dyn}}>0\) 保证了从任何初态出发，若存在严格下降序列，则必在有限步内终止（定理4.2）。

#### 7.1.4 离散Morse指数与亚稳态吸引域
在闭包能量景观中，可仿照Morse理论为每个粗粒化映射赋予一个 **离散Morse指数**：
\[
\operatorname{ind}(C) = \#\{\,C'\in\mathcal{N}(C) : \iota_1(C') < \iota_1(C)\,\}.
\]
指数直观上表示从该状态出发的下降方向数量。
- \(C_\epsilon\)：\(\iota_1=0\)，无任何邻域可再降低能量，\(\operatorname{ind}(C_\epsilon)=0\)（全局极小）。
- \(C^{(1)}\) 与 \(C^{(2)}\)：无严格下降的邻接移动，\(\operatorname{ind}=0\)（局部极小，资源壁垒亚稳态）。
- \(C_{\text{triv}}\)：具有多个下降方向（可分裂为三种两状态划分），\(\operatorname{ind}(C_{\text{triv}})\ge 3\)，属不稳定点或鞍点。

指数为零的状态构成变分流的不动点集合。局部极小（亚稳态）具有吸引域：凡在局部下降流中落入其邻域且无更低能量方向可达的初态，均被捕获。亚稳态的存在表明，资源约束可在景观中制造出无法被贪婪下降克服的障碍。

#### 7.1.5 越障路径与闭包自由能视角
定理4.3保证存在一条从 \(C^{(1)}\) 到 \(C_\epsilon\) 的全局路径，但它不是局部下降流——它需要先“越障”将 \(C^{(1)}\) 强制合并为平凡映射 \(C_{\text{triv}}\)（能量从0.207升至0.251），然后再分裂至 \(C_\epsilon\)。该路径的 **越障高度**（barrier height）为 \(0.251-0.207=0.044\)。这正是动力学谱间隙的值。

从资源代价的角度，可引入一个 **闭包自由能** 泛函，将预测缺陷与表示复杂度置于统一的竞争框架中：
\[
\Phi_{\beta,\alpha}(C) = \iota_1(C) + \beta\,\Psi(C),
\]
其中 \(\Psi(C)\) 度量粗粒化的资源代价，例如状态数 \(|Y_C|\) 或状态熵 \(H(Y_C)\)，\(\beta\ge0\) 为资源惩罚强度。当 \(\beta=0\) 时，全局极小即因果态划分（零缺陷）。当 \(\beta>0\) 时，平凡映射或亚稳态可能成为有限温度下的全局极小，从而产生真正的“相变”：随着资源约束的逐渐收紧（\(\beta\) 增大），最优闭包结构会发生非连续跃迁。这一图景与统计力学中自由能竞争驱动的相变完全对应，为规律发现的资源依赖性提供了深刻的物理类比。

### 7.2 Even Process（偶数过程）—— 有限因果态、无限记忆的边界情形

#### 7.2.1 过程定义与因果态结构
Even Process 的字母表为 \(\{0,1\}\)，禁止长度为奇数的 \(1\)-游程后接 \(0\)。其最小 unifilar ε‑machine 具有两个因果态  
\[
S_0,\; S_1,
\]
转移规则：从 \(S_0\) 输出 \(0\) 并停留于 \(S_0\)（概率 \(p\)），或输出 \(1\) 并转移到 \(S_1\)（概率 \(1-p\)）；从 \(S_1\) 输出 \(1\) 并回到 \(S_0\)（概率1）。两个因果态的一步预测分布分别为  
\[
\mu(\cdot|S_0) = (p,\,1-p),\qquad \mu(\cdot|S_1) = (0,\,1),
\]
当 \(p\in(0,1)\) 时满足一步可辨识性。因果态数 \(N=2\)，但该过程 **不具有有限记忆**：要确定当前状态是 \(S_0\) 还是 \(S_1\)，需要知道自上一个 \(0\) 以来出现了奇数还是偶数个 \(1\)，这可能需要回溯无限远的过去。因此 \(L_0 = \infty\)，假设1中的有限记忆条件被违背。

#### 7.2.2 有限历史深度下的最优闭包缺陷
取 \(p=\frac12\)，固定资源上限 \(R=2\)（等于因果态数）。考察模型类 \(\mathcal{M}_{2,L}\) 中可达到的最优闭包缺陷 \(\mathcal{F}_L^*(2) = \inf_{\Pi} \mathcal{F}(\Pi)\)。关键观察：
> 对任意有限 \(L\)，存在两个过去序列 \(\overleftarrow{z}, \overleftarrow{z}'\)，它们在最近 \(L\) 步完全相同，但一个对应于 \(S_0\)，另一个对应于 \(S_1\)。这是因为状态的区分依赖于奇偶性计数，而任何有限窗口只能看到有限个 \(1\)，无法消除奇偶性歧义。

因此，任何基于长度为 \(L\) 的窗口的粗粒化映射 \(C\in\mathscr{C}_{2,L}\) 必须将这样两个过去映射到同一宏观状态，导致该宏观状态下的一步预测分布为两个不同因果态预测分布的混合。由一步可辨识性，混合分布不同于任一纯粹因果态的分布，故给定宏观状态后过去仍包含对未来的预测信息，从而 \(\iota_1(C) > 0\)。因此，对于所有有限 \(L\)，必有
\[
\mathcal{F}_L^*(2) > 0.
\]
但另一方面，随着 \(L\) 增大，歧义窗口的比例逐渐减小，最优粗粒化可以越来越精确地逼近因果态。可以证明（详细推导略，此处给出数值结果）：
- \(L=1\)：窗口为 \(\{0,1\}\)。最优两状态划分为将 \(0\) 和 \(1\) 分开，但 \(1\) 对应的宏观状态内同时包含来自 \(S_0\) 和 \(S_1\) 的过去，缺陷约为 \(0.17\)。
- \(L=2\)：窗口为 \(\{00,01,10,11\}\)。最优划分按“是否为01”等规则，缺陷约 \(0.08\)。
- \(L=3\)：缺陷约 \(0.04\)。
- 当 \(L\to\infty\) 时，\(\mathcal{F}_L^*(2) \to 0\) 但永不为零。

这精确展示了 **尺度非饱和** 现象：尽管因果态数目有限，由于缺乏有限记忆，零缺陷闭包无法在任何有限窗口深度下实现。Golden Mean Process 在 \(L=1\) 时即饱和为零，而 Even Process 则展现出随尺度缓慢衰减的残余缺陷，形成了鲜明的对比。

#### 7.2.3 双复杂度的分离与二维资源相图
Even Process 揭示了两种内在复杂度的根本区别：
- **状态复杂度（横向）**：统计复杂度 \(C_\mu = H(\mathcal{S})\) 有限（此处很小）。
- **深度复杂度（纵向）**：实现完美预测所需的最小历史深度 \(L_0\) 为无限。

这一区分表明，**“统计复杂度有限”并不蕴含“规律可在有限记忆资源下完美闭包”**。由此，资源 \((R,L)\) 自然地构成一个二维相图，决定了闭包行为的全域相结构：
- 当 \(L \ge L_0\) 且 \(R \ge N\) 时，存在零缺陷闭包（**完美闭包相**）。
- 当 \(L \ge L_0\) 但 \(R < N\) 时，零缺陷不可达，只存在资源壁垒亚稳态（**容量受限相**）。
- 当 \(L < L_0\)（包括 \(L_0=\infty\)）时，即便 \(R\) 充分大，零缺陷闭包亦不可达，缺陷随 \(L\) 衰减，形成 **深度受限相**。
- 当 \(R\) 过小且 \(L\) 过浅，缺陷最大，属 **严重欠资源相**。

Golden Mean 与 Even Process 分别位于相图中的不同区域：前者在 \(L=1,R=2\) 已进入完美闭包相；后者由于 \(L_0=\infty\)，在任何有限 \(L\) 和 \(R=2\) 下均处于深度受限相，仅在 \(L\to\infty\) 的极限下逼近完美闭包边界。这个二维资源相图为未来研究有限样本与近似算法提供了系统的分类框架。

### 7.3 闭包能量景观的形式化提炼与扩展

上述实例自然引出对闭包理论的普适几何结构的形式化提炼。

**定义 7.2（闭包能量景观）**  
固定资源 \((R,L)\)，在有限粗粒化映射集合 \(\mathscr{C}_{R,L}\) 上定义三元组
\[
\mathfrak{L}_{R,L} = \bigl(\mathscr{C}_{R,L},\; \iota_1,\; \mathcal{N}\bigr),
\]
称为 **闭包能量景观**。其中：
- **节点** 为所有容许的粗粒化映射 \(C\)；
- **能量** 为闭包缺陷 \(\iota_1(C)\)；
- **允许局部操作** 构成邻域结构 \(\mathcal{N}\)：包括预测等价合并（不增能量）以及约化后的有效分裂（严格降能量且状态数 \(\le R\)）。变分闭包算子 \(\mathcal{V}_{R,L}\) 即在该景观中执行局部能量下降移动。

在此基础上，可定义：
- **全局极小**：对应完美闭包结构（零缺陷）。
- **局部极小**（除全局极小外）：资源壁垒亚稳态，\(\iota_1>0\) 且无邻接严格下降。
- **静态谱间隙**：\(\Delta_{\mathrm{spec}} = \min\{|a-b| > 0 : a,b\in\{\iota_1(C)\}\}\)。
- **动力学谱间隙**：\(\Delta_{\mathrm{dyn}} = \min_{C,C':\,\iota_1(C')<\iota_1(C),\,C'\in\mathcal{N}(C)} (\iota_1(C)-\iota_1(C'))\)，保证每次局部下降至少降低 \(\Delta_{\mathrm{dyn}}\)。
- **离散Morse指数**：\(\operatorname{ind}(C) = \#\{C'\in\mathcal{N}(C): \iota_1(C')<\iota_1(C)\}\)。指数为零即局部极小。
- **越障高度**：从一个局部极小到更低能量极小所需经过路径的最大能量增量，量度亚稳态的动力学稳定性。

**定义 7.3（闭包自由能）**  
引入资源惩罚函数 \(\Psi:\mathscr{C}_{R,L}\to\mathbb{R}_{\ge0}\)（例如 \(\Psi(C)=|Y_C|\) 或 \(H(Y_C)\)）与约束参数 \(\beta\ge0\)，定义
\[
\Phi_{\beta}(C) = \iota_1(C) + \beta\,\Psi(C).
\]
\(\Phi_\beta\) 称为 **闭包自由能**。当 \(\beta=0\)，其全局极小为因果态划分；当 \(\beta>0\)，表示成本与预测精度的折衷，极小值可能从完美闭包转移至资源消耗更低的亚稳态或粗粒化，从而引发资源诱导的相变。这一构造为规律发现中的“奥卡姆剃刀”原则提供了严格的变分基础。

Golden Mean Process 即为 \(\beta=0\)、\(R\ge N\) 下的景观：全局极小零缺陷，同时存在由资源壁垒造成的亚稳态局部极小。Even Process 则展示了当 \(L\) 有限时，景观中不存在零缺陷状态，全局极小为正缺陷且随尺度降低，为连续极限下的重整化群流提供了明确原型。

### 7.4 展望：连续极限与热力学相变

当前理论建立在有限模型类 \(\mathcal{M}_{R,L}\) 上，一切收敛性、谱间隙和不动点分类均严格且构造性。然而，两个实例均指向一个核心问题：当资源约束趋于无穷（\(L\to\infty\) 或 \(R\to\infty\)）时，离散景观将过渡为连续或无穷维景观，可能出现：
- 静态与动力学谱间隙趋于零，收敛速度降低，亚稳态寿命发散；

- 闭包自由能 \(\Phi_\beta\) 的极小值在热力学极限下发生真正的非解析相变，属于某种普适类。

Even Process 已为此提供了一个清晰的出发点：在有限 \(L\) 下，缺陷严格正且随 \(L\) 衰减，恰好对应“有限截断近似”中的 running coupling；若将其视作重整化群流，则 \(L\) 扮演了倒格距的角色，因果态为不动点。未来工作可沿以下路线推进：

3. 研究 \(\beta>0\) 时闭包自由能的相图，寻找资源参数驱动的相变边界，并与统计力学中的一阶或连续相变建立严格对偶。

### 补充 7.A  动力学谱间隙与有限步收敛

在第七部分中，我们区分了静态谱间隙与动力学谱间隙。以下给出形式定义，并证明动力学谱间隙严格正，从而保证变分流有限步终止。

**定义 7.A.1（局部严格下降量与动力学谱间隙）**  
设 \(\mathcal{N}\) 为定义 4.3 中变分闭包算子所依赖的邻域结构（仅包含单步合并与单步有效分裂，不含复合操作）。对任意 \(C\in\mathscr{C}_{R,L}\)，定义其 **局部严格下降量** 为  
\[
\delta(C) = \min\{\iota_1(C)-\iota_1(C') : C'\in\mathcal{N}(C),\; \iota_1(C')<\iota_1(C)\},
\]  
若不存在严格下降邻域，则规定 \(\delta(C)=\infty\)。  
若 \(\mathscr{C}_{R,L}\) 中至少存在一个非不动点（即 \(\mathcal{V}_{R,L}(C)\neq C\)），则定义整个景观上的 **动力学谱间隙** 为  
\[
\Delta_{\mathrm{dyn}} = \min_{\substack{C\in\mathscr{C}_{R,L}\\ \delta(C)<\infty}} \delta(C).
\]  
若所有状态均为不动点，则 \(\Delta_{\mathrm{dyn}}\) 未定义（此时收敛显然成立）。

**命题 7.A.2**  对任意满足假设1的过程及有限资源 \((R,L)\)，若存在非不动点，则 \(\Delta_{\mathrm{dyn}} > 0\)。

*证明*：\(\mathscr{C}_{R,L}\) 为有限集，\(\iota_1\) 仅取有限个实数值。对任意 \(C\)，集合 \(\{\iota_1(C)-\iota_1(C')\mid C'\in\mathcal{N}(C),\ \iota_1(C')<\iota_1(C)\}\) 若非空，则为有限个正实数，其最小值存在且为正，此即 \(\delta(C)\)。当存在非不动点时，至少有一个 \(C\) 满足 \(\delta(C)<\infty\)。\(\Delta_{\mathrm{dyn}}\) 为有限个正数的下确界（实为最小值），故严格正。\(\square\)

**推论 7.A.3（严格下降保证有限步收敛）**  
若从某个初始 \(C_0\) 出发，变分算子 \(\mathcal{V}_{R,L}\) 生成的序列严格下降（即每一步均有 \(\iota_1(C_{k+1})<\iota_1(C_k)\)），则该序列必在不超过  
\[
K \le \frac{\iota_1(C_0)}{\Delta_{\mathrm{dyn}}} + 1
\]  
步内终止于不动点。

*证明*：每次严格下降至少减少 \(\Delta_{\mathrm{dyn}}\)，且 \(\iota_1\) 非负，故只能进行有限次下降。\(\square\)

**注 7.A.4**  
静态谱间隙 \(\Delta_{\mathrm{spec}}\) 只保证能量值的离散性，不能排除两个不相邻的状态之间能量差极小却无法通过局部移动实现。动力学谱间隙才是保证迭代收敛的精确量。在 Golden Mean 实例中，经计算 \(\Delta_{\mathrm{dyn}}=\Delta_{\mathrm{spec}}=0.044\)，但一般情形下可能有 \(\Delta_{\mathrm{dyn}} > \Delta_{\mathrm{spec}}\)。

### 补充 7.B  下降指数与局部极小性质

**定义 7.B.1（下降指数）**  
对 \(C\in\mathscr{C}_{R,L}\)，定义其 **下降指数** 为  
\[
\operatorname{desc}(C) = \left|\{C'\in\mathcal{N}(C) : \iota_1(C') < \iota_1(C)\}\right|,
\]  
其中 \(\mathcal{N}(C)\) 仅包含单步合并与单步有效分裂。若 \(\operatorname{desc}(C)=0\)，则称 \(C\) 为 **局部极小**（含全局极小）。

**命题 7.B.2（不动点的等价刻画）**  
在变分闭包算子 \(\mathcal{V}_{R,L} = S\circ M\) 的作用下，\(C\) 为不动点的充分必要条件是 \(M(C)=C\) 且 \(\operatorname{desc}(C)=0\)。

*证明*：  
\((\Rightarrow)\) 若 \(\mathcal{V}_{R,L}(C)=C\)，则必有 \(M(C)=C\)。进而 \(S(C)=C\)，这意味着不存在任何有效分裂可使 \(\iota_1\) 严格下降。若存在单步合并使能量严格下降，则该合并必然归并了预测等价状态，这与 \(M(C)=C\) 矛盾。故所有单步邻域移动均不能降低 \(\iota_1\)，即 \(\operatorname{desc}(C)=0\)。  
\((\Leftarrow)\) 若 \(M(C)=C\) 且 \(\operatorname{desc}(C)=0\)，则因 \(C\) 已约化，任何分裂均不降能量，且合并亦不降能量。故 \(S(C)=C\)，从而 \(\mathcal{V}_{R,L}(C)=C\)。\(\square\)

**实例计算**（Golden Mean，\(R=2\)，\(L=2\)）  
- \(C_\epsilon\)：\(\operatorname{desc}=0\)（全局极小）。  
- \(C^{(1)}\)：邻域包含与 \(C_\epsilon\) 的合并（能量上升）、分裂到三状态（因 \(R=2\) 不允许），以及合并成平凡映射（能量上升）。故无下降邻域，\(\operatorname{desc}=0\)，为局部极小（资源壁垒亚稳态）。  
- \(C_{\text{triv}}\)：可分裂为 \(C^{(1)}, C^{(2)}, C_\epsilon\)，三者均严格下降，故 \(\operatorname{desc}\ge 3\)，为非稳定点。

### 补充 7.C  闭包自由能的形式构造与变分性质

**定义 7.C.1（闭包自由能）**  
设 \(\Psi:\mathscr{C}_{R,L}\to\mathbb{R}_{\ge0}\) 为资源代价泛函，例如 \(\Psi(C)=|Y_C|\) 或 \(\Psi(C)=H(Y_C)\)。对参数 \(\beta\ge0\)，定义 **闭包自由能**  
\[
\Phi_\beta(C) = \iota_1(C) + \beta \Psi(C).
\]  
此处 \(\beta\) 为表示成本的拉格朗日乘子，可解释为“逆温度”或“资源惩罚强度”。

**注 7.C.2（约束形式的对偶）**  
当 \(\beta=0\) 时，极小化 \(\Phi_0=\iota_1\) 即寻求完美闭包结构（若资源允许）。当 \(\beta>0\) 时，极小化 \(\Phi_\beta\) 等价于在约束 \(\Psi(C)\le \rho\) 下极小化 \(\iota_1\) 的拉格朗日松弛，其中 \(\rho\) 由 \(\beta\) 隐式决定。此构造与信息瓶颈及率失真理论形式对偶，但本质区别在于：此处的“失真”为闭包缺陷 \(\iota_1\)，度量的是宏观状态丢失的预测信息，而非符号重建误差，因而具有天然的因果解释。

**命题 7.C.3（硬约束与软约束的分离）**  
主体文档中的理论首先采用硬约束 \(\mathscr{C}_{R,L}\)（状态数 \(\le R\)，窗口长度 \(=L\)），在此离散集合上直接优化 \(\iota_1\)。该硬约束理论可视为 \(\beta\to\infty\) 时软约束理论的极限情形（在 \(\Psi(C)=|Y_C|\) 且不允许超出 \(R\) 时）。两种形式等价于下述事实：对任意固定的有限 \(R\)，存在充分大的 \(\beta\) 使得 \(\Phi_\beta\) 的全局极小始终位于状态数不超过 \(R\) 的区域内。

*证明概要*：因 \(\iota_1\) 有界，取 \(\beta > \max \iota_1 / \min_{|Y|>R} (\Psi(C)-R)\) 即可强制状态数不超过 \(R\)。详细构造略。

### 补充 7.D  Even Process 中严格正缺陷的严格证明

**命题 7.D.1**  
设 Even Process 的参数 \(p\in(0,1)\)，固定任意有限窗口深度 \(L\)。对任意粗粒化映射 \(C:\Sigma^L\to\mathcal{Y}\)，必有 \(\iota_1(C) > 0\)。

*证明*：Even Process 的两个因果态 \(S_0\) 与 \(S_1\) 具有不同的一步预测分布 \(\mu_0=(p,1-p)\) 与 \(\mu_1=(0,1)\)。取自然数 \(K > L\)，构造两个双向无限序列的过去段如下：

- \(\overleftarrow{z}^{(0)}\)：以 \(0\) 后接 \(2K\) 个 \(1\) 结尾。即最近 \(2K\) 个符号均为 \(1\)，再往前一位是 \(0\)。
- \(\overleftarrow{z}^{(1)}\)：以 \(0\) 后接 \(2K+1\) 个 \(1\) 结尾。即最近 \(2K+1\) 个符号均为 \(1\)，再往前一位是 \(0\)。

由于 \(K > L\)，这两个序列的最近 \(L\) 个符号完全相同（全部为 \(1\)）。因此任何基于长度为 \(L\) 的窗口的粗粒化映射 \(C\) 将它们映射到同一宏观状态 \(y^*\)。然而，\(\overleftarrow{z}^{(0)}\) 自上一个 \(0\) 以来经历了偶数个 \(1\)（\(2K\) 个），故处于因果态 \(S_0\)；\(\overleftarrow{z}^{(1)}\) 自上一个 \(0\) 以来经历了奇数个 \(1\)（\(2K+1\) 个），故处于因果态 \(S_1\)。因此，在宏观状态 \(y^*\) 下，真实的一步预测分布是 \(\mu_0\) 与 \(\mu_1\) 的混合，其权重为两种过去在 \(y^*\) 内的相对测度（均正）。由于 \(\mu_0\neq\mu_1\)，该混合分布不同于任一纯态分布。给定 \(Y_t=y^*\) 的条件分布不等于给定完整过去 \(\overleftarrow{z}\) 的条件分布（即纯因果态分布）。由条件互信息的定义，
\[
\iota_1(C) = I(Z_{t+1};\overleftarrow{Z}_t\mid Y_t)
= \mathbb{E}\big[D_{\mathrm{KL}}(\mu(\cdot\mid\overleftarrow{Z}_t)\,\|\,\mu(\cdot\mid Y_t))\big] > 0,
\]
因为至少在 \(y^*\) 对应的过去集合上，KL 散度严格正。故对任意有限 \(L\) 及任意 \(C\)，\(\iota_1(C)>0\)。\(\square\)

**推论 7.D.2**  
对 Even Process，\(\mathcal{F}_L^*(2) > 0\) 对所有 \(L<\infty\) 成立，且当 \(L\to\infty\) 时趋于 \(0\) 但永不等于零。这证实了尺度非饱和定理（对比定理 8.1 要求的 \(L_0<\infty\)）。

### 补充 7.E  关于“相变”与“梯度流”术语的严格界定

为确保理论严谨性，我们明确区分以下概念：

- **资源诱导的阈值转变**：在有限模型类 \(\mathcal{M}_{R,L}\) 中，最优闭包缺陷 \(\mathcal{F}_L^*(R)\) 在整数 \(R\) 跨越过程固有因果态数 \(N\) 时发生从正值到零的刚性跳跃。这是有限组合结构的严格性质，无需热力学极限。本文中称为“闭包阈值转变”。
- **热力学相变**：指在系统规模趋于无穷时，自由能或序参量出现非解析行为的现象。当前理论未引入此极限，故暂不宣称“相变”。第七部分 7.4 将其列为开放方向。
- **离散下降流**：指变分算子 \(\mathcal{V}_{R,L}\) 生成的严格单调序列。在当前有限离散空间中，它是一个确定性的离散动力系统。我们避免使用“梯度流”一词，除非未来在连续极限或赋予度量结构后建立了真正的梯度下降结构。
- **下降指数**：为免与代数拓扑中的 Morse 指数混淆，本文采用“下降指数”术语，其定义为下降方向数目，而非常规的 Hessian 指标。

### 补充 7.F  二维资源相图的形式化基础

基于实例的启发，我们为二维资源空间给出初步的形式刻画，为未来工作奠定基础。

**定义 7.F.1（相区域）**  
对于给定的过程族，定义下列区域（以序参量 \(\mathcal{F}_L^*(R)\) 的正性与否为主要标识）：
- **完美闭包区**：\(R \ge N\) 且 \(L \ge L_0\)。此时 \(\mathcal{F}^*_{L}(R)=0\)。
- **容量受限区**：\(R < N\)，\(L \ge L_0\)。此时 \(\mathcal{F}^*_{L}(R) \ge \Delta_0(R,L) > 0\)。
- **深度受限区**：\(L < L_0\)（或 \(L_0=\infty\)），无论 \(R\) 多大，\(\mathcal{F}^*_L(R) > 0\)，且缺陷随 \(L\) 增大而递减。
- **严重欠资源区**：\(R\) 小且 \(L\) 浅，缺陷最大。

序参量 \(\mathcal{F}_L^*(R)\) 在 \((R,L)\) 平面上的等值线以及跨过 \(R=N\) 或 \(L=L_0\) 时的跃变行为，将构成二维相图的严格数学基础。进一步的研究可考察 \(\partial_R \mathcal{F}\) 与 \(\partial_L \mathcal{F}\) 的奇异性。

### 补充 7.G  核心定义与结构公理

为使整个理论的定义系统完全一致并固化其公理基础，本节统一列出贯穿全文的核心对象、操作与公理。

#### 7.G.1 基本对象

1. **生成过程**：平稳遍历概率测度 \(\mu\) 满足假设1（有限 unifilar 因果态、有限记忆 \(L_0\)、一步可辨识性）。
2. **资源参数**：最大状态数 \(R\in\mathbb{N}\)，窗口长度 \(L\in\mathbb{N}\)。
3. **粗粒化映射类**：\(\mathscr{C}_{R,L} = \{C:\Sigma^L\to\mathcal{Y}\mid |\mathcal{Y}|\le R\}\)，其中 \(\mathcal{Y}\) 有限。
4. **模型类**：\(\mathcal{M}_{R,L} = \{(C,P)\mid C\in\mathscr{C}_{R,L},\;P \text{ 为可计算条件分布}\}\)。
5. **能量泛函**：闭包缺陷 \(\iota_1(C)=I(Z_{t+1};\overleftarrow{Z}_t\mid Y_t)\)，\(Y_t=C(Z_{t-L+1:t})\)。统一变分泛函 \(\mathcal{F}(C,P)=\iota_1(C)+\mathbb{E}D_{\mathrm{KL}}(\mu(\cdot|Y_t)\|P(\cdot|Y_t))\)。

#### 7.G.2 局部操作与图结构

6. **邻域结构** \(\mathcal{N}(C)\)：包含以下两种单步操作所可达的粗粒化映射：
   - **合并操作**：将 \(C\) 的任意两个状态合并为一个新状态，得到新映射 \(C'\)。若被合并的状态预测等价，则 \(\iota_1(C')=\iota_1(C)\)；否则 \(\iota_1(C')>\iota_1(C)\)。
   - **有效分裂操作**：仅当 \(C\) 为约化（\(M(C)=C\)）、\(\iota_1(C)>0\) 且 \(|Y_C|<R\) 时允许。选取一个状态及其内部两个非空子集使其条件分布不同，将其分裂为两个新状态，得到 \(C'\)，此时 \(\iota_1(C')<\iota_1(C)\) 且下降量至少为动力学谱间隙。
7. **合并算子 \(M\)**：将 \(C\) 的所有预测等价状态合并，得到唯一约化代表 \(M(C)\)，满足 \(\iota_1(M(C))=\iota_1(C)\) 且状态数最小。
8. **分裂算子 \(S\)**：在约化映射上定义，若满足分裂条件则返回一个有效分裂结果（按确定规则选择），否则返回原映射。
9. **变分闭包算子**：\(\mathcal{V}_{R,L} = S\circ M\)。

#### 7.G.3 动力学与不动点

10. **离散下降流**：由 \(C_{k+1}=\mathcal{V}_{R,L}(C_k)\) 定义的序列。
11. **不动点**：满足 \(\mathcal{V}_{R,L}(C)=C\) 的 \(C\)。
12. **下降指数**：\(\operatorname{desc}(C)=|\{C'\in\mathcal{N}(C):\iota_1(C')<\iota_1(C)\}|\)。
13. **局部极小**：\(\operatorname{desc}(C)=0\) 且 \(M(C)=C\)。全局极小对应 \(\iota_1=0\)（完美闭包）。

#### 7.G.4 能量景观与谱间隙

14. **闭包能量景观**：三元组 \(\mathfrak{L}_{R,L}=(\mathscr{C}_{R,L},\iota_1,\mathcal{N})\)。
15. **静态谱间隙**：\(\Delta_{\mathrm{spec}} = \min\{|a-b|:a,b\in\{\iota_1(C)\},a\neq b\}\)。
16. **动力学谱间隙**：\(\Delta_{\mathrm{dyn}} = \min_{C:\delta(C)<\infty}\delta(C)\)，其中 \(\delta(C)\) 见定义 7.A.1。
17. **越障高度**：从局部极小到更低能量极小所需路径上能量的最大增量。

#### 7.G.5 公理陈述

> **公理 1（变分压缩原理）**  
> 观察者在资源 \((R,L)\) 约束下形成的稳定宏观规律，恰为变分闭包算子 \(\mathcal{V}_{R,L}\) 在粗粒化映射格上的不动点。该不动点极小化统一泛函 \(\mathcal{F}\)（或在理想预测下极小化 \(\iota_1\)）。

> **公理 2（闭包结构普适性）**  
> 对于满足假设1的任意过程，其不动点结构仅依赖于过程的因果态数目 \(N\) 和有限记忆深度 \(L_0\)，而与字母表、具体转移概率等细节无关。当 \(R\ge N\) 且 \(L\ge L_0\) 时，存在唯一的零缺陷不动点（因果态划分）；否则所有不动点均具有正缺陷，或属于资源壁垒亚稳态。

> **公理 3（资源诱导阈值跃变）**  
> 在固定 \(L\ge L_0\) 下，序参量 \(\mathcal{F}_L^*(R)\) 作为 \(R\) 的函数，在 \(R=N\) 处发生刚性跃变：从 \(\ge\Delta_{\mathrm{dyn}}>0\) 降至 \(0\)。该跃变是有限模型类上的严格组合事实，并成为规律“涌现”的数学标志。

> **公理 4（局部下降与亚稳态）**  
> 变分流严格遵循能量单调下降原则。当资源不足以支撑完美闭包时，或当初始状态位于局部极小吸引域内时，动力学将陷于亚稳闭包结构（资源壁垒不动点）。从任意初始状态出发，存在一条可达完美闭包的有限步路径，但该路径可能需临时增加能量（越障），因此不被局部变分算子自主实现。

这些公理与定义构成了本文全部定理与实例的共同基础，确保了整个理论的内部自洽性与可扩展性。

## 第八部分：认知几何学——观测作为维度间的信息投影（框架太大，改不动了）

本部分将前述变分闭包理论从单一过程的规律发现，升华为一种关于**任意维度认知空间之间相互观测**的严格几何学框架。核心思想是：将"观测"重新定义为**一个有限资源认知架构对另一个生成过程的信息压缩投影**，将"维度"重新定义为**过程被完美闭包所需的最小认知资源**。

**重要声明**：本部分的核心对象（认知态射、认知代价、相空间）均已给出严格数学定义，其基本性质（非负性、零点条件、单调性、有限步收敛）均有严格证明。关于"认知距离"的度量性质、"认知曲率"的几何解释、"自我观测不可能性"的一般形式，目前属于有严格定义但部分性质待证明的**理论纲领**。凡未经严格证明的命题，均明确标注为"猜想"或"开放问题"。

所有定义与主体文档的前七部分保持逻辑闭合。

### 8.1 基本对象：生成空间与认知架构

**定义 8.1.1（生成空间）**  
一个**生成空间**是一个二元组 \(\mathscr{G} = (\mu, \mathbf{D})\)，其中：
- \(\mu\) 是满足假设1（有限 unifilar 因果态、有限记忆 \(L_0\)、一步可辨识性）的平稳遍历过程；
- \(\mathbf{D} = (N, L_0)\) 是该过程的**内在因果维度**，\(N = |\mathcal{S}|\) 为因果态数，\(L_0\) 为最小记忆深度。

**注 8.1.2**：\(\mathbf{D}\) 的两个分量分别度量过程的**横向复杂度**（需要多少宏观状态来区分因果态）和**纵向复杂度**（需要回溯多深的历史来精确定位因果态）。Even Process 是 \(N < \infty\) 但 \(L_0 = \infty\) 的典型实例（见7.2节），严格证明这两种复杂度在逻辑上相互独立。

**定义 8.1.3（认知架构）**  
一个**认知架构**是一个二元组 \(\mathscr{O} = (R, L)\)，其中 \(R, L \in \mathbb{N}\) 分别表示该架构可稳定操作的最大宏观状态数和最大回溯窗口长度。

**定义 8.1.4（认知架构的模型类）**  
认知架构 \(\mathscr{O} = (R, L)\) 作用于生成空间 \(\mathscr{G} = (\mu, \mathbf{D})\) 时，可构建的合法理论对全体记为
\[
\mathcal{M}_{\mathscr{O}}(\mathscr{G}) = \mathcal{M}_{R, L} = \{(C, P) \mid C \in \mathscr{C}_{R, L}, \; P \text{ 为 } C \text{ 上的可计算条件分布}\},
\]
其中粗粒化映射 \(C: \Sigma^L \to \mathcal{Y}\) 满足 \(|\mathcal{Y}| \le R\)。由引理1.2，\(\mathcal{M}_{\mathscr{O}}(\mathscr{G})\) 是有限集合。

---

### 8.2 观测作为信息投影：认知态射

**定义 8.2.1（认知态射）**  
设 \(\mathscr{O} = (R, L)\) 为一个认知架构，\(\mathscr{G} = (\mu, \mathbf{D})\) 为一个生成空间。一个从 \(\mathscr{G}\) 到 \(\mathscr{O}\) 的**认知态射**是一个粗粒化映射 \(C: \Sigma^L \to \mathcal{Y}\)，满足 \(|\mathcal{Y}| \le R\)，记为
\[
C: \mathscr{G} \to \mathscr{O}.
\]
\(C\) 的全体构成的集合记为 \(\operatorname{Hom}(\mathscr{G}, \mathscr{O}) = \mathscr{C}_{R, L}\)，为有限集。

**定义 8.2.2（认知态射的代价）**  
认知态射 \(C\) 的代价由其**闭包缺陷**度量：
\[
\iota_1(C) = I(Z_{t+1}; \overleftarrow{Z}_t \mid Y_t),
\]
其中 \(Y_t = C(Z_{t-L+1:t})\)。\(\iota_1(C) \ge 0\)，且 \(\iota_1(C) = 0\) 当且仅当 \(C\) 实现完美一步闭包。

**定义 8.2.3（最优认知态射与最小认知代价）**  
在给定认知架构 \(\mathscr{O}\) 下，对生成空间 \(\mathscr{G}\) 的**最小认知代价**定义为
\[
\mathcal{U}(\mathscr{O}, \mathscr{G}) = \inf_{C \in \operatorname{Hom}(\mathscr{G}, \mathscr{O})} \iota_1(C).
\]
由于 \(\operatorname{Hom}(\mathscr{G}, \mathscr{O})\) 是有限集，下确界实为最小值，即存在 \(C^*\) 使得 \(\mathcal{U}(\mathscr{O}, \mathscr{G}) = \iota_1(C^*)\)。

**命题 8.2.4（认知代价的基本性质）**  
1. **非负性**：\(\mathcal{U}(\mathscr{O}, \mathscr{G}) \ge 0\)。
2. **零点条件**：若 \(\mathscr{G}\) 的内在维度 \(\mathbf{D} = (N, L_0)\) 且 \(\mathscr{O} = (R, L)\) 满足 \(R \ge N\) 且 \(L \ge L_0\)，则 \(\mathcal{U}(\mathscr{O}, \mathscr{G}) = 0\)。反之，若 \(R < N\) 或 \(L < L_0\)，则 \(\mathcal{U}(\mathscr{O}, \mathscr{G}) \ge \Delta_{\mathrm{dyn}} > 0\)，其中 \(\Delta_{\mathrm{dyn}}\) 为模型类上的动力学谱间隙。
3. **非对称性**：一般情况下，\(\mathcal{U}(\mathscr{O}_1, \mathscr{G}_2) \neq \mathcal{U}(\mathscr{O}_2, \mathscr{G}_1)\)。

*证明*：
1. 由互信息的非负性直接得证。
2. 零点条件是定理7.1（刚性闭包阈值）的直接推论：当 \(R \ge N\) 且 \(L \ge L_0\) 时，因果态划分 \(\epsilon \in \mathscr{C}_{R,L}\) 满足 \(\iota_1(\epsilon)=0\)，故下确界为零。反之，若 \(R < N\)，则任何 \(C\) 必然合并至少两个不同因果态，由一步可辨识性得 \(\iota_1(C) > 0\)，再由动力学谱间隙的正性（命题7.A.2）得下界 \(\Delta_{\mathrm{dyn}}\)。若 \(L < L_0\)，类似论证适用。
3. 非对称性来源于 \(\mathscr{O}_1, \mathscr{G}_2\) 与 \(\mathscr{O}_2, \mathscr{G}_1\) 涉及不同的过程和不同的认知资源，无理由对称。具体反例可由 Golden Mean Process 与 Even Process 构造。\(\square\)

**命题 8.2.5（认知代价对资源的单调性）**  
若 \(\mathscr{O}_1 = (R_1, L_1)\) 与 \(\mathscr{O}_2 = (R_2, L_2)\) 满足 \(R_1 \le R_2\) 且 \(L_1 \le L_2\)，则对任意生成空间 \(\mathscr{G}\)，
\[
\mathcal{U}(\mathscr{O}_2, \mathscr{G}) \le \mathcal{U}(\mathscr{O}_1, \mathscr{G}).
\]
即：增加认知资源不会增加最小认知代价。

*证明*：当 \(R_1 \le R_2\) 且 \(L_1 \le L_2\) 时，任何 \(C_1: \Sigma^{L_1} \to \mathcal{Y}\) 满足 \(|\mathcal{Y}| \le R_1\)，可通过扩展窗口定义 \(C_1' \in \mathscr{C}_{R_2, L_2}\)（令 \(C_1'\) 仅依赖最近 \(L_1\) 个符号，忽略多余符号，且状态数不超过 \(R_1 \le R_2\)）。因此 \(\operatorname{Hom}(\mathscr{G}, \mathscr{O}_1)\) 可嵌入 \(\operatorname{Hom}(\mathscr{G}, \mathscr{O}_2)\)。由于 \(\mathcal{U}\) 取最小值，搜索空间更大时最小值只能更小或相等。\(\square\)

**注 8.2.6**：单调性是认知几何的"偏序结构"基础。它严格保证了"更多资源不会更差"这一直觉，为后续相空间和几何结构的定义提供了逻辑前提。

### 8.3 观测的三种基本模式

**定义 8.3.1（观测模式分类）**  
设生成空间 \(\mathscr{G}_{\text{obj}}\) 的内在维度为 \(\mathbf{D}_{\text{obj}} = (N_{\text{obj}}, L_{0,\text{obj}})\)，认知架构 \(\mathscr{O}_{\text{obs}}\) 的资源为 \((R, L)\)。观测模式按维度关系分为三类：

1. **低维观测高维**：\(R < N_{\text{obj}}\) 或 \(L < L_{0,\text{obj}}\)（或两者均不满足）。
2. **高维观测低维**：\(R \ge N_{\text{obj}}\) 且 \(L \ge L_{0,\text{obj}}\)。
3. **同维观测**：\(\mathbf{D}_{\text{obs}} \approx \mathbf{D}_{\text{obj}}\)，即认知架构的资源和过程的维度处于同一量级（指 \(R\) 与 \(N_{\text{obj}}\)、\(L\) 与 \(L_{0,\text{obj}}\) 的数值接近）。

**命题 8.3.2（低维观测的必然亚稳态）**  
若 \(\mathscr{O}_{\text{obs}}\) 对 \(\mathscr{G}_{\text{obj}}\) 的观测属于低维观测高维模式，则最小认知代价严格为正：
\[
\mathcal{U}(\mathscr{O}_{\text{obs}}, \mathscr{G}_{\text{obj}}) \ge \Delta_{\mathrm{dyn}} > 0.
\]
此外，变分闭包算子 \(\mathcal{V}_{R, L}\) 从任何初态出发，均收敛到资源壁垒不动点（\(\iota_1 > 0\)），无法达到完美闭包。

*证明*：由命题8.2.4的零点条件反向部分直接导出正下界。收敛到资源壁垒不动点由定理4.2保证。\(\square\)

**命题 8.3.3（高维观测的完美闭包）**  
若 \(\mathscr{O}_{\text{obs}}\) 对 \(\mathscr{G}_{\text{obj}}\) 的观测属于高维观测低维模式，则 \(\mathcal{U}(\mathscr{O}_{\text{obs}}, \mathscr{G}_{\text{obj}}) = 0\)，完美闭包可达。继续增加 \(\mathscr{O}_{\text{obs}}\) 的资源（更大的 \(R\) 或 \(L\)）不会进一步降低认知代价（定理8.1尺度饱和）。

**命题 8.3.4（同维观测的临界性）**  
若 \(\mathscr{O}_{\text{obs}}\) 与 \(\mathscr{G}_{\text{obj}}\) 处于同维状态，则完美闭包的达成取决于细节：
- 若恰好 \(R \ge N_{\text{obj}}\) 且 \(L \ge L_{0,\text{obj}}\)，则 \(\mathcal{U} = 0\)，完美理解可达；
- 若略有不匹配（\(R\) 略小于 \(N_{\text{obj}}\) 或 \(L\) 略小于 \(L_{0,\text{obj}}\)），则 \(\mathcal{U} > 0\)，陷入亚稳态。

### 8.4 自我观测与自指闭包障碍

**定义 8.4.1（自我观测）**  
设生成空间 \(\mathscr{G} = (\mu, \mathbf{D})\) 的内在维度为 \(\mathbf{D} = (N, L_0)\)。**自我观测**是指 \(\mathscr{G}\) 同时扮演被观测对象和认知架构的角色——即存在一个认知架构 \(\mathscr{O}_{\text{self}} = (R, L)\)，它是 \(\mathscr{G}\) 的一个子系统，且 \(R \le N, L \le L_0\)。

**命题 8.4.2（自指闭包障碍的充分条件）**  
若在自我观测中，认知架构的资源严格小于过程的完整因果维度——即 \(R < N\) 或 \(L < L_0\)——则完美闭包不可达，即
\[
\mathcal{U}(\mathscr{O}_{\text{self}}, \mathscr{G}) \ge \Delta_{\mathrm{dyn}} > 0.
\]

*证明*：这是命题8.3.2的特例。\(\square\)

**注 8.4.3**：上述命题给出了自我完美闭包不可能的**一个充分条件**——认知资源不足。然而，即使 \(R \ge N\) 且 \(L \ge L_0\)，自我观测仍可能面临更深层的自指障碍：当粗粒化映射 \(C\) 本身是 \(\mathscr{G}\) 的因果态结构的一部分时，自我包含可能导致状态数需求的递归膨胀。这一现象的严格形式化需要建立"自建模自动机"（self-modeling automata）的数学理论。

**猜想 8.4.4（自指障碍的一般形式）**  
不存在一个生成空间 \(\mathscr{G}\) 和其内部可构造的认知架构 \(\mathscr{O}\)，使得 \(\mathscr{O}\) 能够完美闭包 \(\mathscr{G}\) 的**全部**状态——包括 \(\mathscr{O}\) 自身的内部状态。该猜想的严格证明是开放问题，其与哥德尔不完备定理和图灵停机问题的内在联系需要进一步阐明。

### 8.5 观测影响程度的统一量化

**定义 8.5.1（统一认知代价）**  
设认知架构 \(\mathscr{O}\) 对生成空间 \(\mathscr{G}\) 采用认知态射 \(C\) 及预测模型 \(P\)，形成理论对 \(\Pi = (C, P)\)。**统一认知代价**定义为
\[
\mathcal{F}_{\mathscr{O}, \mathscr{G}}(\Pi) = \iota_1(C) + \mathbb{E}_{Y_t}\big[D_{\mathrm{KL}}(\mu(\cdot|Y_t) \| P(\cdot|Y_t))\big].
\]
该泛函是主体文档公式 (1.1) 的直接应用。

**定义 8.5.2（代价分解的含义）**  
- **第一项** \(\iota_1(C)\)：**结构不可理解度**——认知架构由于状态容量 \(R\) 或记忆深度 \(L\) 不足，在最优粗粒化下仍无法消除的预测盲区。
- **第二项** \(\mathbb{E}[D_{\mathrm{KL}}]\)：**统计不可理解度**——即使在结构层面可达完美闭包，由于有限样本或优化不足导致的剩余偏差。

**命题 8.5.3（结构不可理解度的严格下界）**  
在低维观测高维模式下（\(R < N_{\text{obj}}\) 或 \(L < L_{0,\text{obj}}\)），即使预测模型完美（第二项为零），统一认知代价仍有严格正下界：
\[
\inf_{P} \mathcal{F}_{\mathscr{O}, \mathscr{G}}(C, P) = \iota_1(C) \ge \mathcal{U}(\mathscr{O}, \mathscr{G}) \ge \Delta_{\mathrm{dyn}} > 0.
\]

*证明*：由命题8.2.4和命题8.3.2直接导出。\(\square\)

### 8.6 认知相空间与相变

**定义 8.6.1（认知相空间）**  
**认知相空间**是认知架构资源参数 \((R, L)\) 构成的正整数格点 \(\mathbb{N}^2\)。每个点 \((R, L)\) 代表一类资源约束。

**定义 8.6.2（序参量）**  
对于固定的生成空间 \(\mathscr{G}\)（内在维度 \(\mathbf{D} = (N, L_0)\)），定义序参量
\[
\mathcal{F}_{\mathscr{G}}^*(R, L) = \inf_{\Pi \in \mathcal{M}_{R, L}} \mathcal{F}(\Pi).
\]
序参量度量了在资源 \((R, L)\) 下对 \(\mathscr{G}\) 的可理解程度：为零表示完美闭包可达，为正表示存在不可约化的认知盲区。

**命题 8.6.3（序参量的单调性）**  
若 \(R_1 \le R_2\) 且 \(L_1 \le L_2\)，则 \(\mathcal{F}_{\mathscr{G}}^*(R_2, L_2) \le \mathcal{F}_{\mathscr{G}}^*(R_1, L_1)\)。

*证明*：完全平行于命题8.2.5。\(\square\)

**命题 8.6.4（相边界与相区域）**  
认知相空间被以下临界线分割为不同的相区域：
- **临界容量线** \(R = N\)：横向相边界。
- **临界深度线** \(L = L_0\)：纵向相边界。

由此划分出四个相区域：
- **完美闭包相**（\(\mathcal{P}_{\text{perfect}}\)）：\(R \ge N\) 且 \(L \ge L_0\)，序参量为零。
- **容量受限相**（\(\mathcal{P}_{\text{capacity}}\)）：\(R < N\) 且 \(L \ge L_0\)，序参量由状态容量不足导致，为正。
- **深度受限相**（\(\mathcal{P}_{\text{depth}}\)）：\(R \ge N\) 且 \(L < L_0\)，序参量由记忆深度不足导致，为正。对于 \(L_0 = \infty\) 的过程（如 Even Process），该相覆盖所有有限 \(L\)。
- **严重欠资源相**（\(\mathcal{P}_{\text{under}}\)）：\(R < N\) 且 \(L < L_0\)，双重不足，序参量最大。

**注 8.6.5**：此处的"相变"是有限格上的刚性阈值跃变，而非热力学极限下的连续相变（已在7.E节严格界定）。该相图是严格成立的离散结构。

### 8.7 认知几何的初步结构（理论纲领）

本节给出认知几何的形式定义，但关于距离的三角不等式、曲率的几何解释等性质，目前属于待证明的**理论纲领**。

**定义 8.7.1（有向认知差异）**  
对两个生成空间 \(\mathscr{G}_A, \mathscr{G}_B\)，定义它们在各自"自然"认知架构下的**有向认知差异**：
\[
\delta(\mathscr{G}_A, \mathscr{G}_B) = \mathcal{U}(\mathscr{O}_A, \mathscr{G}_B),
\]
其中 \(\mathscr{O}_A = (N_A, L_{0,A})\) 是与 \(\mathscr{G}_A\) 自身维度匹配的认知架构，\(\mathscr{O}_B\) 同理。

**注 8.7.2**：\(\delta(\mathscr{G}_A, \mathscr{G}_B) = 0\) 意味着一个具有与 \(\mathscr{G}_A\) 相同认知能力的观察者，可以完美理解 \(\mathscr{G}_B\)。这不等价于 \(\mathscr{G}_A\) 与 \(\mathscr{G}_B\) 相同，因为可能 \(\mathscr{G}_A\) 的认知资源恰好足够覆盖 \(\mathscr{G}_B\) 的复杂度。

**定义 8.7.3（对称认知差异）**  
定义 \(\mathscr{G}_A\) 与 \(\mathscr{G}_B\) 之间的**对称认知差异**为
\[
D(\mathscr{G}_A, \mathscr{G}_B) = \delta(\mathscr{G}_A, \mathscr{G}_B) + \delta(\mathscr{G}_B, \mathscr{G}_A).
\]

**命题 8.7.4（对称认知差异的基本性质）**  
1. **非负性**：\(D(\mathscr{G}_A, \mathscr{G}_B) \ge 0\)。
2. **对称性**：\(D(\mathscr{G}_A, \mathscr{G}_B) = D(\mathscr{G}_B, \mathscr{G}_A)\)。
3. **自反性**：\(D(\mathscr{G}_A, \mathscr{G}_A) = 0\)（因为 \(\mathcal{U}(\mathscr{O}_A, \mathscr{G}_A) = 0\)）。

*证明*：1由 \(\mathcal{U}\) 的非负性得证。2由定义直接得证。3由命题8.2.4的零点条件得证（\(\mathscr{O}_A\) 的维度与 \(\mathscr{G}_A\) 自身匹配）。\(\square\)

**注 8.7.5（\(D\) 不是严格的距离）**  
\(D(\mathscr{G}_A, \mathscr{G}_B) = 0\) **不蕴含** \(\mathscr{G}_A = \mathscr{G}_B\)。例如，两个不同的过程可能恰好都能被对方的自然认知架构完美闭包。因此 \(D\) 至多是伪度量（pseudo-metric）。此外，三角不等式是否成立尚无证明。

**开放问题 8.7.6（认知差异的三角不等式）**  
对任意三个生成空间 \(\mathscr{G}_A, \mathscr{G}_B, \mathscr{G}_C\)，是否总有
\[
D(\mathscr{G}_A, \mathscr{G}_C) \le D(\mathscr{G}_A, \mathscr{G}_B) + D(\mathscr{G}_B, \mathscr{G}_C) \; ?
\]
若成立，\(D\) 构成认知空间的伪度量；若不成立，认知几何更接近有向度量或 Finsler 型结构。

**定义 8.7.7（认知响应函数）**  
在认知相空间的临界线 \(R = N\) 附近，定义**横向认知响应函数**为序参量对资源参数的离散二阶变化率：
\[
\chi_R(R, L) = -\Delta_R^2 \mathcal{F}_{\mathscr{G}}^*(R, L) = -\big[\mathcal{F}^*(R+1, L) - 2\mathcal{F}^*(R, L) + \mathcal{F}^*(R-1, L)\big],
\]
类似可定义**纵向认知响应函数** \(\chi_L\)。该量度量了"理解的边际收益递减"的剧烈程度。

**解释**：认知响应函数在涌现临界点 \(R = N\) 附近达到极大值（序参量发生刚性跌落），对应着认知景观的"高度弯曲"——微小的资源增加带来巨大的理解提升。

**注 8.7.8**：此处的 \(\chi_R\) 是离散响应函数，更接近统计力学中的磁化率或比热的离散类比，而非微分几何意义上的截面曲率。将其严格化为真正的几何曲率，需要首先建立认知空间的切空间与联络结构，这是开放问题。

### 8.8 总结：认知几何学的公理基础

本框架的核心可概括为三条公理：

> **公理 CG1（认知相对性）**  
> 规律不是对象自身的绝对属性，而是对象的内在因果维度 \(\mathbf{D}_{\text{obj}}\) 与观察者认知架构的资源 \((R, L)\) 之间闭包关系的产物。同一对象，在不同认知架构下呈现不同的"可理解程度"，由认知代价 \(\mathcal{U}\) 量化。

> **公理 CG2（认知维度的二维性）**  
> 生成空间的"维度"由完美闭包所需的最小认知资源定义：\(\mathbf{D} = (N, L_0)\)。横向维度 \(N\)（状态复杂度）与纵向维度 \(L_0\)（深度复杂度）在逻辑上相互独立，分别由因果态数目和最小记忆深度刻画。

> **公理 CG3（认知代价的变分单调性）**  
> 观测行为的本质是最小化统一认知代价 \(\mathcal{F}\)，其结果由变分闭包算子 \(\mathcal{V}_{R, L}\) 的动力学决定。认知代价对资源单调不增：增加认知资源不会增加最小认知代价。完美理解（\(\mathcal{F}=0\)）当且仅当认知架构的资源达到或超过对象的因果维度。

这三条公理将主体文档的变分闭包理论，从序列预测的具体问题，升华为一个关于**认知、生成、压缩三者关系的普遍几何学框架**。它为所有涉及"理解"的学科——物理学、认知科学、人工智能——提供了一套统一的数学语言和量化工具。

本框架中已严格证明的结论（认知代价的非负性、零点条件、单调性，相图结构，低维观测的必然亚稳态）构成其坚实核心。认知距离的三角不等式、认知曲率的几何解释、自我观测障碍的一般形式，则为未来的严格化推进指明了方向。

## 补充证明一：有效分裂的存在性（Layer A）

本命题保证在资源允许且存在闭包缺陷时，变分闭包算子必然能执行一次严格下降的分裂操作，从而避免动力学停滞。

### 1.1 约化性的精确化

在主体文档中，粗粒化映射 \(C\) 称为**约化**的，若任意两个不同宏观状态具有不同的一步预测分布。形式地：

\[
M(C) = C \quad \Longleftrightarrow \quad \bigl( \forall y_1 \neq y_2,\; \mu(\cdot \mid Y=y_1) \neq \mu(\cdot \mid Y=y_2) \bigr).
\]

本补充证明全文仅使用**一步预测约化**，因为核心泛函 \(\iota_1(C)\) 仅涉及给定 \(Y_t\) 时 \(Z_{t+1}\) 的条件分布，且一步可辨识性假设保证了完美闭包等价于因果态划分。因此，无需在此处引入全未来预测等价（即计算力学中的完全因果等价）。

### 1.2 命题及证明

**命题 S1（有效分裂存在性）**  
设过程满足假设 1（有限 unifilar 因果态、有限记忆 \(L_0\)、一步可辨识性），且取窗口长度 \(L \ge L_0\)。若粗粒化映射 \(C \in \mathscr{C}_{R,L}\) 是约化的，且 \(\iota_1(C) > 0\)，且当前状态数 \(|Y_C| < R\)，则存在一次有效分裂操作：将某个宏观状态 \(y_0 \in \mathcal{Y}\) 划分为两个非空子状态，得到新映射 \(C'\)，满足

\[
|Y_{C'}| = |Y_C| + 1 \le R,\qquad \iota_1(C') < \iota_1(C),
\]

且下降量至少为动力学谱间隙 \(\Delta_{\mathrm{dyn}} > 0\)。

**证明**：

1. **缺陷正性蕴含预测歧义**  
由 \(\iota_1(C) = H(Z_{t+1} \mid Y_t) - h_\mu > 0\) 知，宏观状态 \(Y_t\) 未完全捕获完整过去 \(\overleftarrow{Z}_t\) 中关于 \(Z_{t+1}\) 的全部信息。故存在状态 \(y_0 \in \mathcal{Y}\)，在其内部的条件互信息严格正：
\[
I(Z_{t+1}; \overleftarrow{Z}_t \mid Y_t = y_0) > 0.
\]
因此，在 \(C^{-1}(y_0)\) 中必然存在两个非空子集 \(A, B\)，使得
\[
\mu(\cdot \mid A) \neq \mu(\cdot \mid B),
\]
且 \(\mu(A) > 0, \mu(B) > 0\)。否则给定 \(y_0\) 后所有过去的预测分布相同，局部条件互信息为零。

2. **利用因果态构造可分裂子集**  
由假设 1，过程具有有限因果态集 \(\mathcal{S}\)。当 \(L \ge L_0\) 时，每个窗口 \(w \in \Sigma^L\) 唯一对应一个因果态 \(\sigma(w)\)。将 \(C^{-1}(y_0)\) 中的窗口按因果态分类。因 \(C\) 是约化的，\(y_0\) 内不可能所有窗口属于同一因果态：若如此，该状态预测分布即此因果态的分布，而其余宏观状态若也包含窗口，其预测分布也必须是该因果态的分布（否则必存在同一因果态窗口对应不同预测分布，但这与因果态的定义矛盾），最终会导致不同宏观状态预测分布相同，违反约化性。故存在至少两个不同的因果态 \(\sigma_a \neq \sigma_b\)，使得
\[
W_a := C^{-1}(y_0) \cap \sigma^{-1}(\sigma_a) \neq \emptyset,\quad
W_b := C^{-1}(y_0) \cap \sigma^{-1}(\sigma_b) \neq \emptyset.
\]
令 \(A = W_a\)，\(B = C^{-1}(y_0) \setminus A\)（非空，因至少包含 \(W_b\)）。由平稳性，\(\mu(A) > 0, \mu(B) > 0\)。一步可辨识性保证 \(\mu(\cdot \mid A) = \mu(\cdot \mid \sigma_a) \neq \mu(\cdot \mid \sigma_b) = \mu(\cdot \mid B)\)。

3. **分裂降低条件熵**  
定义新映射 \(C'\)：将 \(y_0\) 分裂为 \(y_{0A}\)（对应 \(A\)）与 \(y_{0B}\)（对应 \(B\)），其余状态不变。显然 \(|Y_{C'}| = |Y_C| + 1 \le R\)。  
记 \(\pi_0 = \mu(Y_t = y_0) > 0\)，\(w_A = \mu(A)/\pi_0 \in (0,1)\)，\(w_B = 1 - w_A\)。原状态 \(y_0\) 的局部预测分布为 \(p_{\text{mix}} = w_A p_A + w_B p_B\)，其中 \(p_A = \mu(\cdot \mid A), p_B = \mu(\cdot \mid B)\) 且 \(p_A \neq p_B\)。  
分裂后，两个新状态的局部条件熵贡献分别为 \(\pi_0 w_A H(p_A)\) 与 \(\pi_0 w_B H(p_B)\)。由熵的严格凹性，
\[
H(p_{\text{mix}}) > w_A H(p_A) + w_B H(p_B).
\]
两边乘以 \(\pi_0\)，得 \(H(Z_{t+1} \mid Y'_t) < H(Z_{t+1} \mid Y_t)\)，故 \(\iota_1(C') < \iota_1(C)\)。

4. **下降量的正下界**  
模型类 \(\mathscr{C}_{R,L}\) 有限，\(\iota_1\) 仅取有限个值。所有可能通过单次有效分裂实现的下降量构成有限正实数集，故其最小值 \(\Delta_{\mathrm{dyn}} > 0\) 存在（动力学谱间隙）。因此 \(\iota_1(C) - \iota_1(C') \ge \Delta_{\mathrm{dyn}}\)。

\(\square\)

## 补充证明二：非退化条件下的局部一致正间隙（Layer B）

原命题试图证明当 \(R < N\) 时所有粗粒化的闭包缺陷有一致正下界，但审查指出若某些因果态在宏观状态内占比可趋于零，则全局缺陷可以任意小。本修正版将命题退至**局部 Jensen 差**的一致正下界，并给出保证全局下界的附加条件。

### 2.1 局部 Jensen 差的一致正下界

**命题 S2（局部缺陷的一致正下界）**  
设过程满足假设 1，一步可辨识。对任意粗粒化 \(C \in \mathscr{C}_{R,L}\)，考虑任一宏观状态 \(y\)，若其内部包含至少两个不同因果态的窗口，且这两个因果态在 \(y\) 内的条件概率满足 \(\alpha_a \ge \eta, \alpha_b \ge \eta\)（\(0 < \eta \le 1/2\)），则该状态对闭包缺陷的局部贡献满足
\[
\Delta \iota(y) := H\bigl(\mu(\cdot \mid y)\bigr) - \mathbb{E}_{\overleftarrow{Z} \mid y} \bigl[ H(\mu(\cdot \mid \overleftarrow{Z})) \bigr]
\ge \frac{2}{\ln 2} \eta^2 \min_{\sigma \neq \sigma'} \|\mu(\cdot \mid \sigma) - \mu(\cdot \mid \sigma')\|_1^2 > 0.
\]

**证明**：  
给定 \(Y_t = y\)，设涉及的因果态集为 \(\{\sigma_i\}\)，其一步预测分布为 \(p_i = \mu(\cdot \mid \sigma_i)\)，条件概率为 \(\alpha_i\)。则混合分布为 \(\bar{p} = \sum_i \alpha_i p_i\)。利用 Pinsker 不等式和 KL 散度的凸性，有
\[
\Delta \iota(y) = \sum_i \alpha_i D_{\mathrm{KL}}(p_i \| \bar{p})
\ge \frac{2}{\ln 2} \sum_{i < j} \alpha_i \alpha_j \|p_i - p_j\|_1^2.
\]
（该不等式为标准结果，可通过 \(D_{\mathrm{KL}}(p \| q) \ge \frac{2}{\ln 2} \|p-q\|_1^2\) 及 Jensen 差的双重求和表示推出。）  
取 \(\sigma_a, \sigma_b\) 为满足 \(\alpha_a \ge \eta, \alpha_b \ge \eta\) 的两个因果态，并令 \(\kappa = \min_{\sigma \neq \sigma'} \|p_\sigma - p_{\sigma'}\|_1^2 > 0\)。则
\[
\Delta \iota(y) \ge \frac{2}{\ln 2} \alpha_a \alpha_b \kappa \ge \frac{2}{\ln 2} \eta^2 \kappa > 0.
\]
\(\square\)

### 2.2 全局正下界的附加条件

由命题 S2 直接可得：若存在一个合并不同因果态的宏观状态 \(y\)，且其概率 \(\mu(Y_t = y) \ge \gamma > 0\)，则该粗粒化的全局闭包缺陷 \(\iota_1(C) \ge \gamma \cdot \frac{2}{\ln 2} \eta^2 \kappa > 0\)。因此，在以下任一附加条件下，全局一致正下界成立：

- **(A1) 统一占据下界**：对所有合并了不同因果态的宏观状态 \(y\)，有 \(\mu(Y_t = y) \ge \gamma > 0\)。
- **(A2) 非退化合并**：整个粗粒化格上考虑的子族 \(\mathscr{C}^*\) 中，每个粗粒化均存在某个合并态同时满足 \(\alpha_a \ge \eta, \alpha_b \ge \eta\) 且该态概率 \(\ge \gamma\)，其中 \(\eta, \gamma\) 为仅依赖于过程和 \(R,L\) 的正常数。

若这些附加条件不成立，则不能排除缺陷趋于零的可能性。本命题属于 Layer B。

## 补充证明三：Even Process 的指数衰减缺陷（Layer C 启发式论证）

Even Process 是因果态有限但记忆深度无限的过程典型。我们给出其闭包缺陷随窗口深度 \(L\) 指数衰减的启发式论证，并指出严格证明的路径。

### 3.1 机制：Synchronization Ambiguity

Even Process 的因果态由“自上一个 0 以来出现的 1 的个数的奇偶性”决定。当观测窗口长度 \(L\) 有限时，以下两种情形将产生预测歧义：

- 窗口全由 1 组成：例如 \(11\dots1\)。此时窗口内部无 0，无法确定该全 1 串是否是更长 1-游程的后缀，从而无法确定奇偶性。**注意**：尽管窗口内 1 的个数 \(L\) 已知，但这并不直接给出奇偶性，因为真实游程可能为 \(L, L+2, L+4, \dots\)（若游程奇偶性与 \(L\) 相同）或 \(L+1, L+3, \dots\)（若相反），取决于窗口之前是否有未观测到的连续 1。全 1 窗口的概率测度为 \(\Theta(2^{-L})\)，且给定该窗口时因果态的后验分布并非退化。

- 窗口后缀为 \(0 1^k\) 且窗口前紧邻符号未知：如窗口截断了更长的 1-游程，则同样存在 parity ambiguity。此类窗口概率亦为 \(\Theta(2^{-L})\)。

在这些歧义窗口上，任何基于有限窗口的粗粒化必须将其与某些其他窗口合并（因状态数有限），导致局部预测分布为至少两个不同因果态分布的混合，从而产生正的局部缺陷。每步缺陷的期望值约为这些歧义窗口概率乘以一个常数（Jensen 差），因此整体闭包缺陷以至少 \(O(2^{-L})\) 的速率衰减。

### 3.2 严格证明路径

要得到严格的指数下界，需使用计算力学中的**混合态表示（mixed-state presentation）**或**同步理论（synchronization theory）**。在 \(L\) 步有限记忆下，观测者维持一个关于因果态的信念分布。混合态演化的马尔可夫链具有唯一的平稳分布，且从任意初始信念出发的同步速率由该链的谱间隙决定。对于 Even Process，当 \(p=1/2\) 时，混合态空间是概率单纯形的一维子集，其同步速率恰好为 \(\ln 2\)，闭包缺陷（即平均预测失真）正比于 \(2^{-L}\)。详细的构造可参见 Travers & Crutchfield (2011) 等文献。

### 3.3 当前定位

本节给出的是**启发式论证**（Layer C），其物理图像正确，但尚未在本框架内完成完整的严格证明。我们将其列为下列正式猜想：

> **猜想 S3**：对于 Even Process（\(p=1/2\)），存在常数 \(c > 0\) 使得对所有 \(L \ge 1\)，有 \(\mathcal{F}_L^*(2) \ge c \, 2^{-L}\)。

该猜想的前向（指数衰减）已被数值模拟和混合态分析强烈支持，其在连续极限中的 RG 解释亦高度可信。

## 补充证明四：认知预序与有向差异（Layer A / Layer B）

审查意见正确指出，对称认知差异一般不满足三角不等式，强行构造“认知度量”会引入严重障碍。本补充证明退而建立更稳固的结构：**认知预序**（自反、传递），并在此基础上定义有向差异。

### 4.1 认知预序

**定义 S4.1（认知预序）**  
设 \(\mathscr{G}_A, \mathscr{G}_B\) 为生成空间，其内在维度分别为 \(\mathbf{D}_A = (N_A, L_A), \mathbf{D}_B = (N_B, L_B)\)。定义
\[
\mathscr{G}_A \preceq \mathscr{G}_B \quad \Longleftrightarrow \quad \mathcal{U}(\mathscr{O}_B, \mathscr{G}_A) = 0,
\]
其中 \(\mathscr{O}_B = (N_B, L_B)\) 是与 \(\mathscr{G}_B\) 自身维度匹配的认知架构。直观上，\(\mathscr{G}_A \preceq \mathscr{G}_B\) 意味着“以 \(\mathscr{G}_B\) 的认知能力，足以完美闭包（完全理解）\(\mathscr{G}_A\)”。

**命题 S4.2（预序性质）**  
1. 自反性：对任意 \(\mathscr{G}\)，有 \(\mathscr{G} \preceq \mathscr{G}\)。  
2. 传递性：若 \(\mathscr{G}_A \preceq \mathscr{G}_B\) 且 \(\mathscr{G}_B \preceq \mathscr{G}_C\)，则 \(\mathscr{G}_A \preceq \mathscr{G}_C\)。

**证明**：  
自反性：由因果态划分属于 \(\mathscr{C}_{N, L_0}\) 且 \(\iota_1=0\)，得到 \(\mathcal{U}(\mathscr{O}, \mathscr{G})=0\)。  
传递性：\(\mathscr{G}_A \preceq \mathscr{G}_B\) 推出 \(\mathcal{U}(\mathscr{O}_B, \mathscr{G}_A)=0\)，即 \((N_B, L_B)\) 资源可完美闭包 \(A\)。\(\mathscr{G}_B \preceq \mathscr{G}_C\) 推出 \((N_C, L_C)\) 资源可完美闭包 \(B\)，特别地，由命题 8.2.4 零点条件的反向，必须有 \(N_C \ge N_B\) 且 \(L_C \ge L_B\)（否则无法完美闭包维度为 \((N_B, L_B)\) 的过程）。因此 \((N_C, L_C) \ge (N_B, L_B)\)，资源单调性（命题 8.2.5）保证该架构也能完美闭包 \(A\)，即 \(\mathcal{U}(\mathscr{O}_C, \mathscr{G}_A)=0\)。\(\square\)

认知预序天然规避了最优粗粒化的组合问题，因为它只关心“是否能完美闭包”这一零/壹判定。

### 4.2 有向认知差异

在预序基础上，可定义量化差异。由于三角不等式一般不存在，我们谨慎地仅称其为**有向差异**（directed divergence）。

**定义 S4.3（有向认知差异）**  
对于任意两个生成空间 \(\mathscr{G}_A, \mathscr{G}_B\)，定义
\[
\Delta(\mathscr{G}_A \rightarrow \mathscr{G}_B) = \mathcal{U}(\mathscr{O}_A, \mathscr{G}_B),
\]
即用 \(\mathscr{G}_A\) 的资源去预测 \(\mathscr{G}_B\) 时的最小闭包缺陷。

**性质**：\(\Delta(\mathscr{G}_A \rightarrow \mathscr{G}_B) = 0\) 当且仅当 \(\mathscr{G}_B \preceq \mathscr{G}_A\)。该量非负，且满足关于被观测过程的某种单调性，但不声称对称性或三角不等式。它刻画的是“A 对 B 的理解盲区”。

### 4.3 同维过程类上的对称化（Layer B）

考虑所有满足假设 1 且内在维度均为 \((N, L_0)\) 的过程族。此时认知架构资源均相同，记为 \(\mathscr{O}\)。定义
\[
D_{\text{sym}}(\mathscr{G}_A, \mathscr{G}_B) = \mathcal{U}(\mathscr{O}, \mathscr{G}_B) + \mathcal{U}(\mathscr{O}, \mathscr{G}_A).
\]
此量非负、对称，且 \(D_{\text{sym}}(\mathscr{G}_A, \mathscr{G}_A) = 0\)。然而，由于 \(\mathcal{U}\) 涉及极小化，我们**不宣称**它满足三角不等式。一般 Bregman 散度（如 KL 散度）不满足三角不等式，且此处涉及最优粗粒化，结构更复杂。因此，\(D_{\text{sym}}\) 应被视为一种对称化差异度量，而非严格的距离函数。它是否构成伪度量，留作开放问题。

## 连续尺度极限与重整化群（Layer C 与已证性质）

我们梳理从离散闭包动力学到连续重整化群的自然路径，严格区分已证明的性质与启发式解释。

### 5.1 已严格证明的尺度性质（Layer A / B）

以下命题已在主文档或上述补充证明中严格成立：

1. **尺度饱和**（定理 8.1）：若 \(L_0 < \infty\)，则对所有 \(L \ge L_0\) 有 \(\mathcal{F}_L^*(R) = 0\)（当 \(R \ge N\)）。
2. **刚性阈值**（定理 7.1）：在固定 \(L \ge L_0\) 下，\(\mathcal{F}_L^*(R)\) 在 \(R = N\) 处从正跳跃至零。
3. **单调下降**（命题 4.1）：变分闭包算子使 \(\iota_1\) 单调不增。
4. **有限步收敛**（定理 4.2）：迭代在有限步内终止于不动点（零缺陷或资源壁垒）。
5. **指数衰减启发**（猜想 S3）：Even Process 等无限记忆过程，缺陷随 \(L\) 指数衰减。

### 5.2 连续极限的启发式构造（Layer C）

当我们将窗口长度 \(L\) 视为连续尺度变量并考察 \(L \to \infty\) 时的行为时，可获得如下启发式图像：

- **β 函数**：定义离散差分 \(\Delta \mathcal{F}(L) = \mathcal{F}(L+1) - \mathcal{F}(L)\)。对于指数衰减情形，\(\Delta \mathcal{F} \approx -(\ln 2) \mathcal{F}\)，暗示微分方程 \(\frac{d\mathcal{F}}{d\ell} = -\gamma \mathcal{F}\)（\(\ell = \ln L\)）。这类似于重整化群流向稳定不动点。
- **关联长度与临界点**：有限记忆过程的饱和长度 \(L_0\) 扮演了关联长度的角色。若存在一族过程其 \(L_0 \to \infty\)（例如通过参数调节），则 \(\mathcal{F}(L)\) 在临界点附近的标度行为可能展现幂律衰减和普适指数，这正是统计物理中连续相变的标志。
- **闭包自由能与梯度流**：在连续极限下，粗粒化映射空间成为函数空间，闭包自由能 \(\Phi_\beta\) 作为泛函，其梯度流 \(\partial_t C_t = -\nabla \Phi_\beta(C_t)\) 为变分算子迭代的连续版本。稳定点对应原不动点，线性化谱分解给出相关/无关算符。

### 5.3 地位声明

上述连续极限及重整化群论述均属于 **Layer C**，它们建立在已严格证明的离散骨架之上，为未来工作提供了物理洞察和数学猜想，但本身尚未在本框架内作为定理确立。具体而言，下列问题是公开的：

- 证明一般条件下 β 函数的存在性及光滑性。
- 从 Even Process 的严格下界推导临界指数。
- 在合适的 Banach 流形上建立梯度流的适定性并刻画其长时间行为。
