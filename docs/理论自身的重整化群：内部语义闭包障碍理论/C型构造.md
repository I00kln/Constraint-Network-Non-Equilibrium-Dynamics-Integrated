## C型弱反射结构的显式构造方案（基于通用解释器与Kleene递归定理）

### 1. 参数空间与概率模型

设 \(\mathcal{U} = \mathbb{N} \times \mathbb{N}\)，其中第一个分量是程序代码（哥德尔数），第二个分量是输入。记 \(\Theta = \mathcal{U}\)。

定义 **通用解释器** \(U: \mathbb{N} \times \mathbb{N} \to \mathbb{N}\) 是一个部分递归函数，满足：
- 对任意代码 \(c \in \mathbb{N}\) 和输入 \(u \in \mathbb{N}\)，\(U(c, u)\) 定义为第 \(c\) 个程序在输入 \(u\) 下的输出（若停机）。
- 我们考虑概率版本：对每个 \((c, u)\)，定义一个概率分布 \(p_{(c,u)}\) 在 \(\mathbb{N}\) 上：
  \[
  p_{(c,u)}(y) = \Pr\bigl( \text{执行 } c \text{ 输入 } u \text{ 在 } t \text{ 步内停机且输出 } y \bigr)
  \]
  其中 \(t\) 服从某个无偏的截断分布（例如几何分布），以保证总概率为1且所有计算可测。更简单的构造：取 \(p_{(c,u)}(y) = 2^{-T(c,u,y)}\)，其中 \(T(c,u,y)\) 是使 \(U(c,u)\) 输出 \(y\) 的最短步数（若不停机则 \(p\) 为0）。但这种定义需要归一化。标准做法是使用**通用半测度**（universal semimeasure）\(m(c,u)\)，但为了概率归一化，我们可以限制代码和输入长度，或者使用**概率图灵机**：每个程序 \(c\) 在输入 \(u\) 上以概率方式输出（例如允许随机数生成器）。为简单起见，我们采用**确定性程序 + 噪声**模型：设 \(f(c,u)\) 为程序 \(c\) 在输入 \(u\) 上的确定性输出（若停机），否则令 \(f(c,u)=\bot\)。定义
\[
p_{(c,u)}(y) = (1-\varepsilon) \cdot \mathbf{1}_{f(c,u)=y} + \varepsilon \cdot \rho(y)
\]
其中 \(\rho\) 是某个固定的分布（如均匀分布），\(\varepsilon>0\) 很小。这样保证概率分布良定义且处处为正（避免除零），同时保留计算性。

### 2. 自编码映射 \(E\) 的定义

利用 **smn 定理**（参数化定理）和 **Kleene 递归定理**。

定义自编码映射 \(E: \Theta \to \mathcal{M}(\Theta)\)。对于 \(\theta = (c, u)\)，我们希望 \(E(\theta)\) 是参数空间上的一个分布，使得实现一致性 \(p_\theta = p_{E(\theta)}\) 成立。构造如下：

首先，存在一个递归函数 \(\mathsf{encode}: \mathbb{N} \to \mathbb{N}\)，使得对任意代码 \(c\)，\(\mathsf{encode}(c)\) 是**一个模拟器**的代码：该模拟器以输入 \(u\) 运行，首先模拟 \(U(c, u)\) 的计算，然后输出相同结果。形式化地，根据 smn 定理，存在递归函数 \(S\) 满足：
\[
U(S(c), u) \simeq U(c, u) \quad \text{对所有 } u.
\]
实际上，\(S(c)\) 就是“以 \(c\) 为参数构造一个模拟器”的代码。递归定理（Kleene 第二递归定理）说：存在一个代码 \(c_0\) 使得
\[
U(c_0, u) \simeq U(S(c_0), u) \quad \text{对所有 } u.
\]
即 \(c_0\) 和它的模拟器行为相同。现在定义
\[
E(c, u) = \delta_{(S(c), u)}
\]
即点质量在参数 \((S(c), u)\) 上。那么对于任意输入 \(u\)，有
\[
p_{(c,u)}(y) = p_{(S(c), u)}(y)
\]
因为两个程序对所有输入产生相同的输出分布（在确定情形下甚至逐点相同）。特别地，对 \(c_0\) 有 \(p_{(c_0,u)} = p_{(S(c_0), u)} = p_{E(c_0,u)}\)。对于一般的 \(c\)，等式也成立，因为 \(S(c)\) 是模拟器，它直接调用 \(U(c, \cdot)\)，故输出分布相同。

**注意**：通常 \(S(c) \neq c\)，因为模拟器比原程序多了一层解释开销。因此 \(E\) 非平凡且不是由群作用产生的（因为 \(S\) 是递归函数，不是可逆的群运算；不存在有限维李群或离散群使得 \(S(c)=g\cdot c\) 对所有 \(c\) 成立，因为 \(S\) 不是同胚而是递归映射，且轨道无限多）。

### 3. 理论范畴中的两个不同态射

定义理论 \(T\) 为马尔可夫核：样本空间 \(\mathcal{D}=\mathbb{N}\)，参数空间 \(\Theta\)，条件分布 \(p_{\theta}(y)\) 如上。

构造 \(T\) 到自身的两个自同态 \(f, g: T \to T\)：

- **恒等态射 \(f\)**：
  \[
  f_{\mathcal{U}} = \mathrm{id}_{\Theta},\quad f_{\mathcal{D}} = \mathrm{id}_{\mathbb{N}}.
  \]
  显然 \(\mathfrak{R}(f) = \mathrm{id}_{\mathfrak{R}(T)}\)。

- **非平凡态射 \(g\)**：
  定义 \(g_{\mathcal{U}}: \Theta \to \Theta\) 为 \(g_{\mathcal{U}}(c, u) = (S(c), u)\)，即对每个参数应用模拟器代码变换。
  定义 \(g_{\mathcal{D}}: \mathbb{N} \to \mathbb{N}\) 为恒等映射 \(\mathrm{id}_{\mathbb{N}}\)。
  验证态射条件：对于任意 \(\theta = (c,u)\) 和任意 \(A \subseteq \mathbb{N}\)，
  \[
  T(A \mid g_{\mathcal{U}}(\theta)) = p_{(S(c), u)}(A) = p_{(c,u)}(A) = T(g_{\mathcal{D}}^{-1}(A) \mid f_{\mathcal{U}}(\theta)).
  \]
  由于 \(g_{\mathcal{D}}\) 是恒等，\(g_{\mathcal{D}}^{-1}(A)=A\)，且 \(f_{\mathcal{U}} = \mathrm{id}\)，故条件满足。因此 \(g\) 是合法的理论态射。

由于 \(S(c) \neq c\) 对无限多个 \(c\) 成立，\(g \neq f\)（因为它们在参数空间上的映射不同）。

### 4. 实现后的相等性

计算实现函子 \(\mathfrak{R}(f)\) 和 \(\mathfrak{R}(g)\)：
- \(\mathfrak{R}(f)\) 是恒等映射。
- \(\mathfrak{R}(g)\) 由 \((g_{\Theta}, g_{\mathcal{Y}}) = (g_{\mathcal{U}}, \mathrm{id})\) 给出。由构造，对任意 \(\theta\) 和任意 \(y\)，\(p_{\theta}(y) = p_{g_{\mathcal{U}}(\theta)}(y)\)，因此作为统计映射，\(\mathfrak{R}(g)\) 也等于恒等映射（因为参数映射后分布不变）。更精确地说，对于实现范畴中的对象 \(M = \mathfrak{R}(T)\)，态射 \(\mathfrak{R}(g): M \to M\) 将参数 \(\theta\) 映射到 \(\theta' = g_{\mathcal{U}}(\theta)\)，但保持样本空间不变。由于分布相等，这个态射实际上就是恒等态射吗？在实现范畴中，态射由一对映射定义，如果两个态射 \((g_{\Theta}, g_{\mathcal{Y}})\) 和 \((\mathrm{id}, \mathrm{id})\) 满足：对任意 \(\theta\)，\(p_{\theta}(A) = p_{g_{\Theta}(\theta)}(A)\) 且 \(g_{\mathcal{Y}}=\mathrm{id}\)，则它们被视为**不同的实现态射**吗？注意：在实现范畴的定义中，态射是具体的映射对，而不是分布相等下的等价类。因此 \(\mathfrak{R}(f) = (\mathrm{id}, \mathrm{id})\)，\(\mathfrak{R}(g) = (g_{\mathcal{U}}, \mathrm{id})\)。虽然它们作用在分布族上得到相同的联合分布（因为 \(p_\theta = p_{g_{\mathcal{U}}(\theta)}\)），但作为映射对它们**不相等**，因为 \(g_{\mathcal{U}} \neq \mathrm{id}\)。因此 \(\mathfrak{R}(f) \neq \mathfrak{R}(g)\) 作为**实现范畴中的态射**。

**关键点**：文档中期望的“\(\mathfrak{R}(f) = \mathfrak{R}(g)\)”是指**作为从 \(\Theta\) 到分布族的函数，它们诱导的映射是否相同**？实际上，范畴论中两个态射相等要求对应映射逐点相等。这里 \(f\) 和 \(g\) 在理论范畴中已经不同，但实现后仍然是不同的态射（因为参数映射不同）。然而，如果我们考虑**实现函子将理论映射为统计模型，并将理论态射映射为实现态射**，那么 \(\mathfrak{R}(f)\) 和 \(\mathfrak{R}(g)\) 是不同态射。它们唯一的“不可区分性”在于：对任何统计推断过程，仅通过观测数据（即样本 \(y\)）无法区分参数 \(\theta\) 和 \(g_{\mathcal{U}}(\theta)\)。这正是**参数不可辨识性**，而不是**态射层面相等**。

文档中的“忠实性失效”指的是：存在不同的理论态射，使得实现后它们成为**相同的实现态射**。上面的构造没有达到这一点，因为实现态射仍然不同。要达到“相同”，我们需要在实现范畴中额外商掉某种等价关系，例如将“对任意 \(\theta\) 有 \(p_\theta = p_{g_{\mathcal{U}}(\theta)}\)”视为同态。但标准范畴定义不允许这样做。

因此，更精确的表述应该是：**存在不同的理论态射 \(f, g\)，使得对于任意理论 \(T\) 及其实现，通过任何可观测的统计实验（即任何基于样本的决策规则），无法区分 \(\mathfrak{R}(f)\) 与 \(\mathfrak{R}(g)\)**。这等价于说，实现函子虽然对象上保持不同，但其像在**统计等价**意义下重合。为了捕捉这一现象，我们应当将实现范畴的态射视为**外延等价类**，即两个实现态射若对几乎处处参数产生相同的分布，则视为相等。这相当于取**实现范畴的商范畴**，商去“统计不可区分性”。我们称这个商范畴为 \(\mathbf{Realization}_{\mathrm{obs}}\)。

那么，在上述构造中，\(\mathfrak{R}(f)\) 和 \(\mathfrak{R}(g)\) 在 \(\mathbf{Realization}_{\mathrm{obs}}\) 中成为同一个态射。而C型条件要求这个等价不能被任何保持表达性的局部化消除，即不能通过提升到更细的范畴来分离它们。

### 5. 不可Morita平凡化论证（草图）

需要证明：不存在反射保留的栈等价 \(L\) 使得 \(L(T)\) 的实现函子严格忠实。等价地，无法通过任何保持算术结构的局部化将 \(g_{\mathcal{U}}\) 变成恒等映射。假设存在这样的 \(L\)，则 \(L\) 会将递归结构 \(S\) 映射为某个可逆变换，但 \(S\) 是递归函数且不是可逆的（实际上 \(S\) 是单射？需要检查）。更关键的是，递归定理给出的固定点 \(c_0\) 使得 \(S(c_0)\) 与 \(c_0\) 产生相同分布。在局部化下，如果要求保持所有可观测统计量，那么 \(c_0\) 和 \(S(c_0)\) 必须被等同。这会导致一个商对象，其中自指结构被压扁。但是，如果等同了它们，则根据递归定理，该理论将无法区分程序与其模拟器，从而失去通用解释能力（因为通用解释器的本质就是能够模拟任意程序，包括自身）。因此，任何保持通用计算能力的局部化必然保留这种不可区分性，从而无法实现严格忠实。这是一个非平凡的障碍，其严格证明需要引入**递归等价性**与**栈的惯性群**概念，目前属于猜想范围，但已有强烈直觉支持。

### 6. 总结：构造的完备性

| 组件 | 具体实现 |
|------|----------|
| 参数空间 \(\Theta\) | \(\mathbb{N} \times \mathbb{N}\)（程序代码，输入） |
| 分布 \(p_{(c,u)}\) | 由通用图灵机的输出加上小噪声定义 |
| 自编码映射 \(E\) | \(E(c,u) = \delta_{(S(c), u)}\)，其中 \(S\) 是模拟器构造函数 |
| 递归定理 | 保证存在 \(c_0\) 使 \(p_{(c_0,u)} = p_{(S(c_0),u)}\) |
| 非平凡态射 \(g\) | \(g_{\mathcal{U}}(c,u) = (S(c), u)\)，\(g_{\mathcal{D}}=\mathrm{id}\) |
| 恒等态射 \(f\) | \(\mathrm{id}\) |
| 实现后的可观测等价 | 对任意输入，\(p_\theta = p_{g_{\mathcal{U}}(\theta)}\)，故统计上不可区分 |
| 不可消除性（猜想） | 任何保持通用计算能力的局部化都会保留该等价，因此无法使实现函子严格忠实 |

这个构造满足v7.0中C型条件1（递归自编码且非群作用），条件3（弱反射：存在编码映射 \(\eta\) 将参数映射到其哥德尔编码，例如取 \(\eta(c,u) = (c,0)\) 等）。条件2（不可Morita平凡化）目前仍是猜想，但上述构造提供了验证该猜想的试验平台。
