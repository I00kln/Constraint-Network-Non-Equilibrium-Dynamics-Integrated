3. 主定理及其证明

定理（不可严格忠实化）。设 Theory\mathbf{Theory}Theory 为如上定义的理论范畴，R:Theory→PER\mathfrak{R}: \mathbf{Theory} \to \mathbf{PER}R:Theory→PER 为实现函子。则不存在细化范畴 Realization~\widetilde{\mathbf{Realization}}Realization（即不存在忠实嵌入 J:Realization~↪PERJ: \widetilde{\mathbf{Realization}} \hookrightarrow \mathbf{PER}J:Realization↪PER 与提升 R~:Theory→Realization~\widetilde{\mathfrak{R}}: \mathbf{Theory} \to \widetilde{\mathbf{Realization}}R:Theory→Realization 满足 J∘R~=RJ \circ \widetilde{\mathfrak{R}} = \mathfrak{R}J∘R=R）使得 R~\widetilde{\mathfrak{R}}R 是严格忠实的。

证明。假设存在这样的细化范畴，即存在忠实函子 JJJ 与函子 R~\widetilde{\mathfrak{R}}R 满足 J∘R~=RJ \circ \widetilde{\mathfrak{R}} = \mathfrak{R}J∘R=R，且 R~\widetilde{\mathfrak{R}}R 忠实（严格忠实）。考虑第 2 节中构造的理论 TTT 及其自同态 f≠gf \neq gf=g。由构造知
R(f)=R(g).\mathfrak{R}(f) = \mathfrak{R}(g).R(f)=R(g).
应用等式 J∘R~=RJ \circ \widetilde{\mathfrak{R}} = \mathfrak{R}J∘R=R 得
J(R~(f))=R(f)=R(g)=J(R~(g)).J\bigl(\widetilde{\mathfrak{R}}(f)\bigr) = \mathfrak{R}(f) = \mathfrak{R}(g) = J\bigl(\widetilde{\mathfrak{R}}(g)\bigr).J(R(f))=R(f)=R(g)=J(R(g)).
由于 JJJ 是忠实的，即对任意态射 α,β\alpha, \betaα,β，若 J(α)=J(β)J(\alpha) = J(\beta)J(α)=J(β) 则 α=β\alpha = \betaα=β，故
R~(f)=R~(g).\widetilde{\mathfrak{R}}(f) = \widetilde{\mathfrak{R}}(g).R(f)=R(g).
又因为 R~\widetilde{\mathfrak{R}}R 是忠实的（严格忠实），由此推出 f=gf = gf=g。这与 f≠gf \neq gf=g 矛盾。因此假设不成立，不存在这样的细化范畴。∎

注：上述证明中未使用“反射保留”条件，因此结论更强：即使允许任意（不保证反射保留的）细化，只要 JJJ 忠实且 R~\widetilde{\mathfrak{R}}R 忠实，就不可能使得 R~\widetilde{\mathfrak{R}}R 成为严格忠实的提升。特别地，任何反射保留的细化同样不可能。

4. 与 C 型结构的关系

在第 2 节中构造的理论 TTT 实际上满足 C 型弱反射结构（定义见原纲领 v7.0 第三部分）：

  • 递归自编码：映射 E(c,u)=(S(c),u)E(c,u) = (S(c), u)E(c,u)=(S(c),u) 是递归的且满足 pθ=pE(θ)p_\theta = p_{E(\theta)}pθ​=pE(θ)​。
  • 非群作用：gUg_{\mathcal{U}}gU​ 不是任何可逆群作用（因为 SSS 非单射且存在非平凡不动点结构）。
  • 弱反射稳定性：存在将参数映射到其哥德尔编码的可定义映射（例如取 η(c,u)=(c,0)\eta(c,u) = (c,0)η(c,u)=(c,0) 并利用递归定理调整分布，或直接使用自描述程序的技术）。

因此，本证明给出了一个具体的 C 型理论实例，并以此否定了任何忠实提升的存在性。C 型结构在此处的作用正是提供了两个不同的理论态射在实现函子下不可区分，从而构成反例。

5. 结论

我们已经严格证明了：对于包含充分表达性理论且具有 C 型弱反射结构的理论范畴，不存在任何细化范畴（无论是否反射保留）使得实现函子提升为一个严格忠实的函子。证明的核心是构造两个不同的理论态射，它们在 PER 中的像相等，然后通过函子性质导出矛盾。整个论证不依赖于选择公理、投射对象或幂等分裂等高级工具，因此是自洽且闭合的。
