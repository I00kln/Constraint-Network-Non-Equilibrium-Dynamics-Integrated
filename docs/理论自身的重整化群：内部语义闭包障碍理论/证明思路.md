# 不存在使实现函子变为严格忠实的细化范畴的严格证明（修订版）

## 1. 前置框架与定义

### 1.1 底层计算模型
- 自然数集 \(\mathbb{N}\)，通用部分递归函数 \(U: \mathbb{N} \times \mathbb{N} \to \mathbb{N}\)（Kleene 谓词 \(T\) 与结果函数 \(U\)）。
- 递归函数理论中的 **smn 定理** 与 **Kleene 递归定理**（第二递归定理）均可用。

### 1.2 范畴 \(\mathbf{PER}\)
- 对象：自然数集上的部分等价关系 (Partial Equivalence Relation) \(R \subseteq \mathbb{N} \times \mathbb{N}\)，满足对称性与传递性。
- 态射：从 \((A, R_A)\) 到 \((B, R_B)\) 的态射是一个部分递归函数 \(f: \mathbb{N} \rightharpoonup \mathbb{N}\)，使得：  
  - 若 \(n R_A m\) 且 \(f(n)\downarrow, f(m)\downarrow\)，则 \(f(n) R_B f(m)\)。  
  - 若 \(n R_A m\) 且 \(f(n)\downarrow\)，则 \(f(m)\downarrow\)（一致性）。
- 两个态射 \(f, g\) 相等，如果对任意 \(n \in \mathbb{N}\)，当 \(f(n)\downarrow\) 或 \(g(n)\downarrow\) 时均有 \(f(n) R_B g(n)\)。  
  特别地，若对每个 \(n\) 有 \(f(n)\downarrow\) 且 \(g(n)\downarrow\)，则条件简化为：\(\forall n,\; f(n) R_B g(n)\)。

### 1.3 范畴 \(\mathbf{Theory}\)
- 对象：马尔可夫核 \(T: \mathcal{U} \to \mathrm{Prob}(\mathcal{D})\)，其中：  
  - \(\mathcal{U} = \mathbb{N} \times \mathbb{N}\)（参数空间），  
  - \(\mathcal{D} = \mathbb{N}\)（描述空间），  
  - \(\mathrm{Prob}(\mathcal{D})\) 是 \(\mathbb{N}\) 上概率分布的集合。  
  每个 \(\theta = (c, u) \in \mathcal{U}\) 对应一个概率分布 \(p_\theta\) 于 \(\mathbb{N}\)。
- 态射：从 \(T\) 到 \(T'\) 的态射是一对可测映射 \((\phi_{\mathcal{U}}, \phi_{\mathcal{D}})\)，其中 \(\phi_{\mathcal{U}}: \mathcal{U} \to \mathcal{U}'\)，\(\phi_{\mathcal{D}}: \mathcal{D} \to \mathcal{D}'\)，使得对任意 \(\theta \in \mathcal{U}\) 与可测集 \(A \subseteq \mathcal{D}'\) 有  
  \[
  T'(A \mid \phi_{\mathcal{U}}(\theta)) = T(\phi_{\mathcal{D}}^{-1}(A) \mid \theta).
  \]  
  所有映射均为**可计算可定义的**（即由部分递归函数实现），且理论对象满足充分表达性（能编码一阶算术）。

### 1.4 实现函子 \(\mathfrak{R}: \mathbf{Theory} \to \mathbf{PER}\)
- 对理论 \(T\)，定义 PER 对象 \(\mathfrak{R}(T) = (\Theta, \sim_T)\)，其中 \(\Theta = \mathbb{N} \times \mathbb{N}\)，等价关系为：  
  \[
  \theta_1 \sim_T \theta_2 \quad\Longleftrightarrow\quad p_{\theta_1} = p_{\theta_2} \text{ 作为 } \mathbb{N} \text{ 上的概率分布}.
  \]
- 对理论态射 \((\phi_{\mathcal{U}}, \phi_{\mathcal{D}})\)，定义 PER 态射 \(\mathfrak{R}(\phi) = \phi_{\mathcal{U}}\)（视为部分递归函数）。它自动保持等价关系：若 \(\theta_1 \sim_T \theta_2\) 则 \(p_{\theta_1}=p_{\theta_2}\)，结合态射条件可得 \(p_{\phi_{\mathcal{U}}(\theta_1)} = p_{\phi_{\mathcal{U}}(\theta_2)}\)，即 \(\phi_{\mathcal{U}}(\theta_1) \sim_T \phi_{\mathcal{U}}(\theta_2)\)。

### 1.5 细化范畴
称一个范畴 \(\widetilde{\mathbf{Realization}}\) 为 \(\mathbf{PER}\) 的一个**细化**，如果：
- 存在忠实嵌入函子 \(J: \widetilde{\mathbf{Realization}} \hookrightarrow \mathbf{PER}\)（即 \(J\) 在态射上为单射）。
- 存在提升函子 \(\widetilde{\mathfrak{R}}: \mathbf{Theory} \to \widetilde{\mathbf{Realization}}\) 使得 \(J \circ \widetilde{\mathfrak{R}} = \mathfrak{R}\)。
- \(\widetilde{\mathfrak{R}}\) 称为**严格忠实**，如果 \(\widetilde{\mathfrak{R}}(f) = \widetilde{\mathfrak{R}}(g)\) 蕴含 \(f = g\)（即 \(\widetilde{\mathfrak{R}}\) 为忠实函子）。

**注**：后文的证明不依赖“反射保留”条件，因此即使不要求该条件，结论依然成立。

## 2. 构造一个具有 C 型弱反射结构的理论 \(T\) 及其自同态 \(f \neq g\) 使得 \(\mathfrak{R}(f) = \mathfrak{R}(g)\)

### 2.1 参数空间与概率分布族
固定常数 \(0 < \varepsilon < 1\) 以及 \(\mathbb{N}\) 上的一个概率分布 \(\rho\)（例如几何分布 \(\rho(y) = 2^{-y-1}\)）。对每个 \(\theta = (c, u) \in \mathbb{N} \times \mathbb{N}\) 定义分布  
\[
p_{(c,u)}(y) = (1-\varepsilon)\cdot \mathbf{1}_{U(c,u)=y} \;+\; \varepsilon \cdot \rho(y),
\]  
其中 \(\mathbf{1}_{U(c,u)=y}\) 在 \(U(c,u)\) 有定义且等于 \(y\) 时为 1，否则为 0。约定当 \(U(c,u)\uparrow\) 时该项为 0。显然 \(p_{(c,u)}\) 是 \(\mathbb{N}\) 上的概率分布，且对每个 \(y\) 有 \(p_{(c,u)}(y) > 0\)（因为 \(\varepsilon \rho(y) > 0\)）。

### 2.2 模拟器映射 \(S\)
由 smn 定理，存在递归函数 \(S: \mathbb{N} \to \mathbb{N}\) 使得对任意 \(c, u\)：  
\[
U(S(c), u) \simeq U(c, u).
\]  
即 \(S(c)\) 是模拟程序 \(c\) 的代码。定义 \(g_{\mathcal{U}}: \Theta \to \Theta\) 为  
\[
g_{\mathcal{U}}(c, u) = (S(c), u).
\]

**性质**：对任意 \(\theta = (c, u)\)，由于 \(U(S(c), u)\) 与 \(U(c, u)\) 有相同的收敛性及输出值，因此  
\[
p_{(c,u)} = p_{(S(c), u)}.
\]  
从而 \(\theta \sim_T g_{\mathcal{U}}(\theta)\)。

### 2.3 理论 \(T\) 及其自同态
定义理论 \(T\) 为马尔可夫核：  
\[
T(A \mid (c,u)) = \sum_{y \in A} p_{(c,u)}(y), \quad A \subseteq \mathbb{N}.
\]

定义自同态：
- \(f = \mathrm{id}_T\)，即 \(f_{\mathcal{U}} = \mathrm{id}_{\Theta}\)， \(f_{\mathcal{D}} = \mathrm{id}_{\mathbb{N}}\)。
- \(g\) 由 \(g_{\mathcal{U}}\) 如上， \(g_{\mathcal{D}} = \mathrm{id}_{\mathbb{N}}\)。

验证 \(g\) 是理论态射：对任意 \(\theta = (c,u)\) 与 \(A \subseteq \mathbb{N}\)，  
\[
T(A \mid g_{\mathcal{U}}(\theta)) = T(A \mid (S(c),u)) = \sum_{y \in A} p_{(S(c),u)}(y) = \sum_{y \in A} p_{(c,u)}(y) = T(g_{\mathcal{D}}^{-1}(A) \mid \theta),
\]  
因为 \(g_{\mathcal{D}}^{-1}(A) = A\)。故 \(g\) 为自同态。

由于存在 \(c\) 使得 \(S(c) \neq c\)（例如取任意程序代码，其模拟代码通常不同），因此 \(g_{\mathcal{U}} \neq \mathrm{id}_{\Theta}\)，从而 \(f \neq g\) 作为理论态射。

### 2.4 实现函子下的像
由性质 \(\theta \sim_T g_{\mathcal{U}}(\theta)\) 得：对每个 \(\theta \in \Theta\)，  
\[
\mathrm{id}_{\Theta}(\theta) = \theta \;\sim_T\; g_{\mathcal{U}}(\theta).
\]  
在 \(\mathbf{PER}\) 中，两个态射 \(\alpha, \beta: \mathfrak{R}(T) \to \mathfrak{R}(T)\) 相等当且仅当对每个输入 \(\theta\)（定义域为整个 \(\Theta\)）有 \(\alpha(\theta) \sim_T \beta(\theta)\)（因为 \(\mathrm{id}_{\Theta}\) 与 \(g_{\mathcal{U}}\) 均为处处定义的部分递归函数）。因此  
\[
\mathrm{id}_{\Theta} = g_{\mathcal{U}} \quad \text{在 } \mathbf{PER} \text{ 中作为态射}.
\]  
于是  
\[
\mathfrak{R}(f) = \mathrm{id}_{\Theta} = g_{\mathcal{U}} = \mathfrak{R}(g).
\]

至此，我们构造了具体的理论 \(T\) 及其两个不同的自同态 \(f \neq g\)，满足 \(\mathfrak{R}(f) = \mathfrak{R}(g)\)。

## 3. 主定理及其证明

**定理**（不可严格忠实化）。设 \(\mathbf{Theory}\) 为如上定义的理论范畴，\(\mathfrak{R}: \mathbf{Theory} \to \mathbf{PER}\) 为实现函子。则不存在细化范畴 \(\widetilde{\mathbf{Realization}}\)（即不存在忠实嵌入 \(J: \widetilde{\mathbf{Realization}} \hookrightarrow \mathbf{PER}\) 与提升 \(\widetilde{\mathfrak{R}}: \mathbf{Theory} \to \widetilde{\mathbf{Realization}}\) 满足 \(J \circ \widetilde{\mathfrak{R}} = \mathfrak{R}\)）使得 \(\widetilde{\mathfrak{R}}\) 是严格忠实的。

**证明**。假设存在这样的细化范畴，即存在忠实函子 \(J\) 与函子 \(\widetilde{\mathfrak{R}}\) 满足 \(J \circ \widetilde{\mathfrak{R}} = \mathfrak{R}\)，且 \(\widetilde{\mathfrak{R}}\) 忠实（严格忠实）。考虑第 2 节中构造的理论 \(T\) 及其自同态 \(f \neq g\)。由构造知  
\[
\mathfrak{R}(f) = \mathfrak{R}(g).
\]  
应用等式 \(J \circ \widetilde{\mathfrak{R}} = \mathfrak{R}\) 得  
\[
J\bigl(\widetilde{\mathfrak{R}}(f)\bigr) = \mathfrak{R}(f) = \mathfrak{R}(g) = J\bigl(\widetilde{\mathfrak{R}}(g)\bigr).
\]  
由于 \(J\) 是忠实的，即对任意态射 \(\alpha, \beta\)，若 \(J(\alpha) = J(\beta)\) 则 \(\alpha = \beta\)，故  
\[
\widetilde{\mathfrak{R}}(f) = \widetilde{\mathfrak{R}}(g).
\]  
又因为 \(\widetilde{\mathfrak{R}}\) 是忠实的（严格忠实），由此推出 \(f = g\)。这与 \(f \neq g\) 矛盾。因此假设不成立，不存在这样的细化范畴。∎

**注**：上述证明中未使用“反射保留”条件，因此结论更强：即使允许任意（不保证反射保留的）细化，只要 \(J\) 忠实且 \(\widetilde{\mathfrak{R}}\) 忠实，就不可能使得 \(\widetilde{\mathfrak{R}}\) 成为严格忠实的提升。特别地，任何反射保留的细化同样不可能。

## 4. 与 C 型结构的关系

在第 2 节中构造的理论 \(T\) 实际上满足 **C 型弱反射结构**（定义见原纲领 v7.0 第三部分）：
- **递归自编码**：映射 \(E(c,u) = (S(c), u)\) 是递归的且满足 \(p_\theta = p_{E(\theta)}\)。
- **非群作用**：\(g_{\mathcal{U}}\) 不是任何可逆群作用（因为 \(S\) 非单射且存在非平凡不动点结构）。
- **弱反射稳定性**：存在将参数映射到其哥德尔编码的可定义映射（例如取 \(\eta(c,u) = (c,0)\) 并利用递归定理调整分布，或直接使用自描述程序的技术）。

因此，本证明给出了一个具体的 C 型理论实例，并以此否定了任何忠实提升的存在性。C 型结构在此处的作用正是提供了两个不同的理论态射在实现函子下不可区分，从而构成反例。

## 5. 结论

我们已经严格证明了：对于包含充分表达性理论且具有 C 型弱反射结构的理论范畴，不存在任何细化范畴（无论是否反射保留）使得实现函子提升为一个严格忠实的函子。证明的核心是构造两个不同的理论态射，它们在 PER 中的像相等，然后通过函子性质导出矛盾。整个论证不依赖于选择公理、投射对象或幂等分裂等高级工具，因此是自洽且闭合的。
