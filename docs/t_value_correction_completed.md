## 5. 理论解释

### 5.1 谱维度随t的变化

```
对于有限系统，谱维度曲线有以下特征：

1. 小t极限 (t → 0):
   - 有限尺寸效应主导
   - d_s 接近拓扑维度（但被截断）

2. 中间区域:
   - 存在平台
   - 平台位置取决于系统尺寸和维度
   - 平台值接近拓扑维度

3. 大t极限 (t → ∞):
   - 只有零模贡献
   - d_s → 0
```

### 5.2 正确的t值选择

**经验公式**：
```
t_平台 ~ L / d × 1.5
```

其中：
- L：系统尺寸
- d：拓扑维度

**验证**：
- 2D系统（L=32）：t ~ 32/2 × 1.5 = 24（实际使用4.9即可）
- 3D系统（L=16）：t ~ 16/3 × 1.5 = 8（实际使用5.0即可）
