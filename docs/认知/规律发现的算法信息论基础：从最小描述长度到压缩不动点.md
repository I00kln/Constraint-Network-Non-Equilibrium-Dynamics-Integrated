# 规律发现的算法信息论基础：从最小描述长度到压缩不动点

**版本 4.2（严格修订）**

**摘要**  
以算法信息论与计算力学为基座，本文严格建立观察者约束下规律发现的变分压缩理论。核心对象是观测数据、受限粗粒化映射与宏观预测模型构成的多层三元组。规律选择的目标被精炼为**理论压缩代价**，统一了描述长度与预测编码长度。我们首先在平稳遍历信源下证明一般超额编码分解，从而将局部闭包缺陷与全局闭包缺陷自然嵌入编码低效。随后在模型空间上构造**局部变分压缩算子**并证明其单调下降，强压缩不动点对应于自洽且预测高效的有效理论。因果态被严格证为完美闭包约束下的极小充分统计量；在标准显式编码方案下，因果态模型是该类中的全局压缩极小元。引入尺度依赖的压缩算子族，严格证明有限因果态过程的尺度饱和定理，并对嵌套 MDL 粗粒化流与闭包相变提出精确猜想。核心聚焦于提取不同尺度共有的**闭包结构**——规律发现的真正不变量——同时保留同一尺度下局部放缩的开放路线。所有关键定理均给出完整证明，严格区分已证结果与猜想。

## 第一部分：严格基础

### 1. 基本定义与符号

#### 1.1 概率空间与过程
设 \((\Omega,\mathcal{F},\mu)\) 为概率空间，\(\mu\) 是平稳遍历可计算概率测度。取双向无限序列空间 \(\Sigma^{\mathbb{Z}}\)，\(\Sigma=\{0,1\}\) 为有限字母表。坐标过程 \(Z_t(\omega)=\omega_t\)。记过去半无限序列 \(\overleftarrow{Z}_t = (\dots, Z_{t-1}, Z_t)\)，未来 \(\overrightarrow{Z}_{t} = (Z_{t+1}, Z_{t+2}, \dots)\)。平稳性保证对任意可测集 \(A\)，\(\mu(T^{-1}A)=\mu(A)\)（\(T\) 为左移算子）。遍历性要求尾σ-代数 \(\bigcap_{n\ge 0}\sigma(Z_{n+1},Z_{n+2},\dots)\) 平凡。假设所有条件概率 \(\mu(Z_{t+1}\mid \overleftarrow{Z}_t)\) 为可计算函数（见后）。

**熵率** 定义为
\[
h_\mu = \lim_{n\to\infty} \frac{1}{n} H(Z_{1:n}) = \lim_{n\to\infty} H(Z_n\mid Z_{1:n-1}),
\]
平稳性保证极限存在。引入关键恒等式：
\[
H(Z_{t+1}\mid \overleftarrow{Z}_t) = h_\mu. \tag{1.1}
\]
证明基于倒向鞅收敛定理与平稳遍历性，远端过去σ-代数平凡。

#### 1.2 观测、计算资源与可计算性
观察者直接获得有限观测序列 \(D = z_{1:T}\in\Sigma^T\)，由真实测度 \(\mu\) 生成。

**柯尔莫哥洛夫复杂度**  
固定通用前缀图灵机 \(U\)。有限串 \(x\) 的前缀复杂度 \(K(x)=\min\{|p|:U(p)=x\}\)；条件复杂度 \(K(x\mid y)=\min\{|p|:U(p,y)=x\}\)。

**可计算模型**  
概率分布 \(P\) 在 \(\Sigma^{\mathbb{N}}\) 上称**可计算**，若存在图灵机对任意有限历史 \(z_{1:n}\) 及有理数 \(\varepsilon>0\)，可在有限步输出有理数 \(q\) 满足 \(|P(z_{n+1}\mid z_{1:n})-q|<\varepsilon\)。程序自身长度为其前缀码长，记为 \(K(H)\)。若 \(H\) 为此类程序，以 \(P(\cdot\mid H)\) 记其分布。

#### 1.3 观察者约束与合法理论类
**定义 1.1（观察者）**  
观察者 \(\mathcal{O}\) 由观测映射与计算资源类 \(\mathcal{C}\) 构成。\(\mathcal{C}\) 描述观察者可动用的最大时间、空间或逻辑深度。本文直接以观测序列作为基点，故只论及 \(\mathcal{C}\) 对粗粒化与模型构造的约束。

**定义 1.2（受约束粗粒化映射与合法模型类）**  
粗粒化映射 \(C:\Sigma^*\to \mathcal{Y}\)（\(\mathcal{Y}\) 可数）将任意有限历史映射至宏观状态，须满足存在在 \(\mathcal{C}\) 内可计算的函数 \(\phi\) 使 \(Y_t = C(z_{1:t}) = \phi(z_{1:t})\)。所有此类映射集合记为 \(\mathcal{C}_{\mathcal{O}}\)。一个**理论对** \(\Pi = (C, H)\) 由粗粒化映射 \(C\) 和宏观预测模型 \(H\) 组成，且 \(H\) 的实现程序必须在资源 \(\mathcal{C}\) 之内。合法理论对全体记为 \(\mathcal{M}_{\mathcal{O}}\)。

#### 1.4 动力学闭包条件
为了区分一步预测与完全预测，我们定义两级闭包缺陷。

**定义 1.3（局部闭包缺陷）**  
对于粗粒化映射 \(C\)，定义其**一步闭包缺陷**
\[
\iota_1(C) = I(Z_{t+1}; \overleftarrow{Z}_t \mid Y_t),\quad Y_t = C(\overleftarrow{Z}_t),
\]
其中互信息在 \(\mu\) 下计算。平稳遍历保证与 \(t\) 无关。

**定义 1.4（全局闭包缺陷与完美闭包）**  
定义**完全预测闭包缺陷**
\[
\iota_{\infty}(C) = I(\overrightarrow{Z}_t; \overleftarrow{Z}_t \mid Y_t).
\]
若 \(\iota_{\infty}(C)=0\)，称 \(C\) 具有**完美动力学闭包**（预测充分性），此时几乎处处成立
\[
\mu(\overrightarrow{Z}_t \mid \overleftarrow{Z}_t) = \mu(\overrightarrow{Z}_t \mid Y_t).
\]
显然 \(\iota_1(C)\le \iota_{\infty}(C)\)。需要特别指出：\(\iota_1(C)=0\) 一般不蕴含 \(\iota_{\infty}(C)=0\)；仅当过程满足特定结构化条件（如 unifilar 有限状态因果态）时才等价。本文在论及完美闭包时，均显式要求 \(\iota_{\infty}(C)=0\)。

#### 1.5 观测等价与可辨识性
**定义 1.5（观测等价）**  
两个理论对 \(\Pi_1, \Pi_2\) 称为**观测等价**，若它们对所有有限序列赋予相同的分布，即 \(P_{\Pi_1}(z_{1:T}) = P_{\Pi_2}(z_{1:T})\) 对所有 \(T\) 成立。记等价类为 \([\Pi]\)。

**定义 1.6（可辨识性与严格分离）**  
设真实测度为 \(\mu\)。理论类 \(\mathcal{M}\) 称为**观测可辨识**，若存在一个函数 \(\Delta: \mathcal{M}\to [0,\infty)\) 使得对于真实等价类 \([\Pi_0]\) 及任意 \(\Pi\notin[\Pi_0]\)，每步期望编码代价之差满足
\[
\ell(\Pi) - \ell(\Pi_0) \ge \Delta(\Pi) > 0,
\]
其中 \(\ell(\Pi) = \mathbb{E}_\mu[-\log P_\Pi(Z_{t+1}\mid Y_t)]\)，且当模型类有限时 \(\inf_{\Pi\notin[\Pi_0]}\Delta(\Pi) > 0\)。这意味着真实等价类的期望损失严格最小且与其他类分离。等价地，KL 散度 \(D_{KL}(\mu \parallel P_{\Pi})\) 在等价类上达到唯一最小值。

#### 1.6 显式模型编码方案与复杂度层次
实际编码中区分三种相关但不同的复杂度概念：

* **统计复杂度** \(C_\mu = H(\mathcal{S})\) 是纯信息论量，依赖于过程分布。
* **柯尔莫哥洛夫复杂度** \(K(\Pi)\) 是理论程序的理想描述长度，不可直接计算。
* **显式编码长度** \(L_E(\Pi)\) 是观察者在特定资源约束下对理论的可计算编码长度，为实际使用的代理复杂度。

**定义 1.7（标准有限状态显式编码方案 \(E\)）**  
对于有限状态完美闭包模型 \(\Pi\)，状态空间大小 \(m = |\mathcal{Y}|\)，对每个状态 \(y\) 和输出符号 \(a\in\Sigma\)，条件概率 \(\mu(a\mid y)\) 以固定精度 \(\delta>0\) 量化。编码方案 \(E\) 将 \(\Pi\) 编为二元串：以自定界码存储 \(m\)，长度 \(\le \log m + 2\log\log m + O(1)\)；对每个 \(y,a\) 以 \(\log(1/\delta)+O(1)\) 比特存储量化概率值。故存在常数 \(c_1,c_2\) 使
\[
L_E(\Pi) \le \log m + m|\Sigma|(\log(1/\delta) + c_1) + c_2.
\]
由通用图灵机性质，\(K(\Pi) \le L_E(\Pi) + c\)。\(L_E(\Pi)\) 随 \(m\) 严格单调递增（当 \(m\ge 1\) 且 \(\delta\le 1/2\)）。该方案提供可计算复杂度代理。

#### 1.7 理论压缩代价
给定合法理论对 \(\Pi = (C, H)\)，定义**理论压缩代价**
\[
\boxed{\mathcal{L}(\Pi) = K(\Pi) - \log P(D\mid \Pi)}. \tag{1.2}
\]
其中 \(P(D\mid \Pi)\) 递归分解为
\[
P_{\Pi}(z_{1:T}) = \prod_{t=1}^{T} P_{\Pi}(z_t \mid z_{1:t-1}),\quad
P_{\Pi}(z_t \mid z_{1:t-1}) = H(z_t \mid C(z_{1:t-1})).
\]
实际计算中常采用经验代理 \(\mathcal{L}_{\text{emp}}(\Pi) = \mathcal{K}(\Pi) - \log P_{\Pi}(D)\)，其中 \(\mathcal{K}(\Pi)\) 取 \(L_E\) 或类似可计算上界。

### 2. 超额编码率与闭包缺陷：一般分解

**命题 2.1（每步期望编码代价的一般分解）**  
设真实测度 \(\mu\) 平稳遍历，理论对 \(\Pi = (C,H)\) 给出预测分布 \(P_{\Pi}(\cdot\mid Y_t)\)，其中 \(Y_t = C(\overleftarrow{Z}_t)\)。则每步期望负对数似然满足
\[
\mathbb{E}_\mu[-\log P_{\Pi}(Z_{t+1}\mid Y_t)] = h_\mu + \iota_1(C) + D_{KL}\big(\mu(\cdot\mid Y_t) \,\|\, P_{\Pi}(\cdot\mid Y_t)\big). \tag{2.1}
\]

*证明*：  
由条件熵定义，
\[
\begin{aligned}
\mathbb{E}_\mu[-\log P_{\Pi}(Z_{t+1}\mid Y_t)] 
&= H(Z_{t+1}\mid Y_t) + D_{KL}\big(\mu(\cdot\mid Y_t) \,\|\, P_{\Pi}(\cdot\mid Y_t)\big).
\end{aligned}
\]
由平稳性与互信息关系，
\[
\iota_1(C) = I(Z_{t+1};\overleftarrow{Z}_t\mid Y_t) = H(Z_{t+1}\mid Y_t) - H(Z_{t+1}\mid Y_t,\overleftarrow{Z}_t).
\]
因为 \(Y_t\) 是 \(\overleftarrow{Z}_t\) 的函数，\(H(Z_{t+1}\mid Y_t,\overleftarrow{Z}_t) = H(Z_{t+1}\mid \overleftarrow{Z}_t) = h_\mu\)（由(1.1)）。于是 \(H(Z_{t+1}\mid Y_t) = h_\mu + \iota_1(C)\)。代入即得。\(\square\)

**定理 2.1（完美匹配下超额编码率等于局部闭包缺陷）**  
在命题 2.1 的设定下，若理论对中的预测分布完美匹配由粗粒化导出的条件分布，即几乎处处有 \(P_{\Pi}(\cdot\mid Y_t) = \mu(\cdot\mid Y_t)\)，则
\[
\lim_{T\to\infty} \frac{1}{T}\mathbb{E}_\mu[-\log P_{\Pi}(z_{1:T})] = h_\mu + \iota_1(C).
\tag{2.2}
\]

*证明*：此时 KL 散度项为零，由遍历定理及平稳性，期望时间平均收敛到 \(H(Z_{t+1}\mid Y_t)=h_\mu+\iota_1(C)\)。\(\square\)

**推论 2.1（MDL 自然惩罚闭包缺陷）**  
最小化 \(\mathcal{L}(\Pi)\) 等价于折衷模型复杂度与预测性能；超额编码直接由 \(\iota_1(C)\) 和模型误设 KL 散度共同贡献。因此最小描述长度自动偏好低闭包缺陷且拟合良好的模型，无需外部惩罚项。

### 3. 因果态、统计复杂度与描述长度桥接

#### 3.1 完美闭包与极小充分统计量
**定义 3.1（因果等价与因果态）**  
两个过去 \(\overleftarrow{z}, \overleftarrow{z}'\) 称为因果等价，记为 \(\overleftarrow{z} \sim_\epsilon \overleftarrow{z}'\)，若它们对未来具有相同的条件分布：
\[
\mu(\overrightarrow{Z} \in \cdot \mid \overleftarrow{z}) = \mu(\overrightarrow{Z} \in \cdot \mid \overleftarrow{z}').
\]
等价类称为**因果态**，映射 \(\epsilon: \overleftarrow{z} \mapsto \sigma\) 为因果态划分。因果态集合记为 \(\mathcal{S}\)。由定义，因果态满足完美闭包 \(\iota_{\infty}(\epsilon)=0\)，从而 \(\iota_1(\epsilon)=0\)。统计复杂度定义为 \(C_\mu = H(\mathcal{S})\)。

**定理 3.1（因果态的极小充分性）**  
在所有满足完美闭包 (\(\iota_{\infty}=0\)) 的确定性粗粒化映射 \(Y = C(\overleftarrow{Z})\) 中，因果态映射 \(\epsilon\) 具有最小的熵 \(H(\mathcal{S})\)。

*证明*：设 \(Y = f(\overleftarrow{Z})\) 满足 \(\iota_{\infty}(f)=0\)，则给定 \(Y\)，\(\overleftarrow{Z}\) 与 \(\overrightarrow{Z}\) 条件独立。若两个过去 \(\overleftarrow{z}_1,\overleftarrow{z}_2\) 映射到同一 \(y\) 值，则对所有可测集 \(A\)，
\[
\mu(\overrightarrow{Z}\in A\mid \overleftarrow{z}_1) = \mu(\overrightarrow{Z}\in A\mid Y=y) = \mu(\overrightarrow{Z}\in A\mid \overleftarrow{z}_2),
\]
故 \(\overleftarrow{z}_1\sim_\epsilon \overleftarrow{z}_2\)。因此 \(f\) 的每个水平集包含于某个因果态内，因果态划分是 \(f\) 划分的精炼化，即存在 \(g\) 使 \(\epsilon(\overleftarrow{Z}) = g(Y)\) 几乎处处成立。由熵的单调性，\(H(\epsilon)\le H(Y)\)，即 \(C_\mu\le H(Y)\)。\(\square\)

#### 3.2 预测充分性层次与有限记忆收敛
对于资源受限的观察者，粗粒化映射只能使用有限长度的历史。定义基于过去 \(L\) 个符号的粗粒化映射类 \(\mathcal{C}_L\)，并考虑相应的最优闭包缺陷。

**定义 3.2（有限记忆粗粒化类）**  
给定记忆长度 \(L\)，令 \(\mathcal{C}_L\) 为所有形如 \(Y_t = C(z_{t-L+1:t})\) 的可计算映射集合。相应的理论对类为 \(\mathcal{M}_L\)。

**命题 3.1（预测充分性层次收敛）**  
设真实过程具有有限因果态，且存在 \(L_0\) 使得因果态划分可由长度 \(L_0\) 的历史完全确定（即 \(\epsilon(\overleftarrow{z})\) 是 \(z_{-L_0+1:0}\) 的函数）。则对每个 \(L\)，定义可达最小局部闭包缺陷
\[
\iota_1^*(L) = \inf_{\Pi \in \mathcal{M}_L} \iota_1(C_\Pi).
\]
那么序列 \(\iota_1^*(L)\) 单调非增，且当 \(L\ge L_0\) 时 \(\iota_1^*(L) = 0\)。

*证明*：\(\mathcal{C}_L\) 随 \(L\) 增加而扩大，故最小缺陷不增。当 \(L\ge L_0\) 时，\(\mathcal{M}_L\) 包含实现因果态划分的理论，其闭包缺陷为零，故下确界为零。\(\square\)

此层次结构为因果态的可逼近性提供了有限记忆基础。

#### 3.3 在显式编码下的描述长度最优性
**定理 3.2（因果态机描述复杂度上界）**  
设因果态机具有 \(N=|\mathcal{S}|\) 个状态，条件概率均为可计算实数。对任意精度 \(\delta>0\)，其程序长度满足
\[
K(\Pi_\epsilon) \le N |\Sigma| \big( \log(1/\delta) + O(1) \big) + O(N \log N).
\]
*证明*：使用定义 1.7 的标准编码方案 \(E\) 即得。

**定理 3.3（因果态模型在显式编码方案下的压缩极小性）**  
设真实过程的因果态数 \(N\) 有限。令 \(\mathcal{M}_{\text{perf}}\) 为所有有限状态、完美闭包 (\(\iota_{\infty}=0\)) 的理论对集合。在显式状态表编码方案 \(E\) 下，因果态模型 \(\Pi_\epsilon\) 具有最小的描述长度：
\[
L_E(\Pi_\epsilon) \le L_E(\Pi), \quad \forall \Pi \in \mathcal{M}_{\text{perf}}.
\]
进而，若所有此类模型均能完美预测（期望负对数似然达到 \(T h_\mu\)），则总压缩代价 \(\mathcal{L}_E(\Pi):= L_E(\Pi) - \log P_\Pi(D)\) 在 \(\mathcal{M}_{\text{perf}}\) 中也是全局极小。

*证明*：对任意 \(\Pi\in\mathcal{M}_{\text{perf}}\)，记其状态数为 \(m\)。由定理 3.1，因果态是任意完美闭包划分的精炼，故 \(m\ge N\)。\(L_E\) 对状态数严格单调递增，因此 \(L_E(\Pi_\epsilon)\le L_E(\Pi)\)。完美闭包下期望编码代价相同，差异仅取决于描述长度，故总压缩代价最小。\(\square\)

**注**：该极小性严格依赖于显式状态表编码 \(E\)。更通用的柯尔莫哥洛夫复杂度因不可计算性及可能的“短程序异常”，不能断言绝对极小性；但 \(L_E\) 作为实际 MDL 准则的复杂度代理已足够。

## 第二部分：压缩不动点与局部变分动态

### 4. 模型空间上的局部变分压缩算子

#### 4.1 有限状态理论空间与邻居构造
固定最大状态数 \(M\)（由资源约束 \(R\) 决定）及概率参数量化精度 \(\delta_0>0\)。考虑所有 \(|\mathcal{Y}|\le M\)、状态转移与发射概率以精度 \(\delta_0\) 量化的可计算理论对组成的有限模型类 \(\mathcal{M}_{M,\delta_0}\)。定义可计算复杂度代理
\[
\mathcal{K}(\Pi) = c_1 |\mathcal{Y}| \log |\mathcal{Y}| + c_2 |\mathcal{Y}| |\Sigma| \log(1/\delta_0),
\]
常数与编码方案 \(E\) 一致，保证 \(\mathcal{K}(\Pi)\) 是 \(L_E(\Pi)\) 的上界同阶量且随状态数严格单调递增。

**定义 4.1（合法邻居集）**  
对 \(\Pi\in\mathcal{M}_{M,\delta_0}\)，其邻居集 \(\mathcal{N}(\Pi)\) 包含所有可通过以下操作之一获得且仍属于 \(\mathcal{M}_{M,\delta_0}\) 的理论对 \(\Pi'\)：
- **合并**：将两个宏观状态 \(y_1, y_2\) 合并为一个新状态，重新估计转移和发射概率（基于数据频率加权平均），并量化至精度 \(\delta_0\)；
- **分裂**：将一个宏观状态 \(y\) 依照某个基于历史上下文的准则（例如最近一个符号的取值）拆分为两个子状态，并重新估计概率，同样量化。
邻居集中的理论允许暂时不满足完美闭包，但需记录其闭包缺陷估计值。

#### 4.2 经验压缩代价与局部变分算子
定义经验压缩代价
\[
\mathcal{L}_{\text{emp}}(\Pi) = \mathcal{K}(\Pi) - \log P_{\Pi}(D). \tag{4.1}
\]

**定义 4.2（局部变分压缩算子）**  
定义算子 \(\widetilde{\mathcal{V}} : \mathcal{M}_{M,\delta_0} \to \mathcal{M}_{M,\delta_0}\) 为
\[
\widetilde{\mathcal{V}}(\Pi) = \operatorname{argmin}_{\Pi' \in \mathcal{N}(\Pi) \cup \{\Pi\}} \mathcal{L}_{\text{emp}}(\Pi'),
\]
若极小元不唯一则按固定规则选取。该算子本质是贪婪 MDL 下降步骤。

**引理 4.1（单调下降性）**  
\[
\mathcal{L}_{\text{emp}}(\widetilde{\mathcal{V}}(\Pi)) \le \mathcal{L}_{\text{emp}}(\Pi).
\]

*证明*：由定义，\(\widetilde{\mathcal{V}}(\Pi)\) 在包含 \(\Pi\) 自身的集合上极小化 \(\mathcal{L}_{\text{emp}}\)。\(\square\)

**定义 4.3（压缩不动点）**  
\(\Pi^*\) 称为**强压缩不动点**，若 \(\widetilde{\mathcal{V}}(\Pi^*) = \Pi^*\) 且对所有 \(\Pi' \in \mathcal{N}(\Pi^*)\) 有 \(\mathcal{L}_{\text{emp}}(\Pi') > \mathcal{L}_{\text{emp}}(\Pi^*)\)。  
若存在 \(\epsilon>0\) 使得所有邻居不优于 \(\Pi^*\) 超过 \(\epsilon\)，则称 \(\Pi^*\) 为 \(\epsilon\)**-弱压缩不动点**。

**命题 4.1（因果态模型的渐近局部压缩稳定）**  
设真实过程可由有限因果态模型 \(\Pi_0\) 精确表达，且 \(\Pi_0\in\mathcal{M}_{M,\delta_0}\)。存在样本量 \(T_0\)，使得当 \(T > T_0\) 时，对任意 \(\Pi' \in \mathcal{N}(\Pi_0)\)，有
\[
\mathbb{P}\big( \mathcal{L}_{\text{emp}}(\Pi') > \mathcal{L}_{\text{emp}}(\Pi_0) \big) \to 1 \quad \text{as } T\to\infty,
\]
即 \(\Pi_0\) 依概率成为强压缩不动点。该结论基于以下事实：在 \(\Pi_0\) 处闭包缺陷为零，预测编码接近熵率下界；任何局部改变要么增大状态数而受复杂度惩罚，要么破坏闭包从而增加预测编码代价；大样本下似然比优势压倒涨落。严格的有限样本论证需借助指数混合过程的集中不等式，此处仅给出启发式思路，完整证明留作技术细节。

### 5. 贪婪压缩下降过程与收敛性

**定义 5.1（贪婪压缩下降过程）**  
给定观察者预算序列 \(\{M_n\}\)（状态数上限逐步提高，量化精度 \(\delta_0\) 不变），从初始理论 \(\Pi_0\in\mathcal{M}_{M_0,\delta_0}\) 出发，迭代
\[
\Pi_{n+1} = \widetilde{\mathcal{V}}_{M_n}(\Pi_n),
\]
其中 \(\widetilde{\mathcal{V}}_{M_n}\) 是可行域为 \(\mathcal{M}_{M_n,\delta_0}\) 时的局部变分压缩算子。若 \(\widetilde{\mathcal{V}}_{M_n}(\Pi_n) = \Pi_n\) 且当前预算无法接纳更优邻居，则过程停止。

**定理 5.1（单调下降定理）**  
经验压缩代价序列 \(\{\mathcal{L}_{\text{emp}}(\Pi_n)\}\) 单调非增。

*证明*：由算子定义直接得出。\(\square\)

**定理 5.2（有限状态空间的经验收敛定理）**  
设最大状态数 \(M_{\max}\) 有界，概率参数量化至固定精度 \(\delta_0\)。则压缩下降过程在有限步内收敛至一个**经验压缩不动点** \(\Pi^*\)，满足 \(\widetilde{\mathcal{V}}_{M_{\max}}(\Pi^*) = \Pi^*\) 且 \(\mathcal{L}_{\text{emp}}(\Pi^*)\) 为局部极小。

*证明*：在约束下模型空间有限（状态划分有限、量化概率取值有限）。\(\mathcal{L}_{\text{emp}}\) 取值于有限离散集，单调下降序列必在有限步停止，停止点即为所求不动点。\(\square\)

#### 5.3 全局经验压缩代价最小化的一致性
为了建立统计一致性，我们转而考虑**全局最小化器**而非局部搜索轨迹。定义
\[
\Pi_T^* = \underset{\Pi \in \mathcal{M}_{M_{\max},\delta_0}}{\operatorname{argmin}} \; \mathcal{L}_{\text{emp}}(\Pi).
\]
下列定理表明，在适当条件下，该全局最优理论渐近收敛到真实因果态模型的等价类。

**定理 5.3（全局经验压缩代价最小化器的一致性）**  
假设：  
1. 真实过程 \(\mu\) 平稳遍历，满足 \(\alpha\)-混合系数 \(\alpha(k)\le C e^{-ck}\)（指数混合），且具有有限状态数 \(N\) 的因果态模型 \(\Pi_0\)；  
2. 模型类 \(\mathcal{M}\equiv \mathcal{M}_{M_{\max},\delta_0}\) 为固定有限集合（状态数 \(\le M_{\max}\)，概率参数量化至固定精度 \(\delta_0\)），且包含 \(\Pi_0\)；  
3. \(\mathcal{M}\) 是观测可辨识的（定义 1.6）：真实等价类具有严格最小的每步期望编码代价，且与其他等价类分离。  
则全局经验压缩代价最小化器 \(\Pi_T^*\) 几乎必然满足：
\[
\lim_{T\to\infty} D_{KL}(\mu \parallel P_{\Pi_T^*}) = 0,\qquad
\lim_{T\to\infty} \iota_{\infty}(C_T^*) = 0,\qquad
\lim_{T\to\infty} \iota_{1}(C_T^*) = 0.
\]
若进一步假设模型类的结构满足最小 unifilar 同步条件与状态可辨识性，则 \(\Pi_T^*\) 的状态划分在重标记下几乎必然收敛到因果态划分。

*证明*：详见附录 A。核心思路：固定有限模型类下，利用指数 \(\alpha\)-混合过程的强大数律与一致收敛性质，证明经验压缩代价在模型类上几乎必然一致收敛到总体期望代价。观测可辨识性保证真实等价类具有严格最小的期望代价，从而最小化器必然落入该等价类。由于因果态模型完美闭包，收敛到其等价类自动确保 \(\iota_{\infty}=0\) 及 \(\iota_1=0\)。状态划分收敛需额外结构条件，作为推论给出。\(\square\)

**注 5.4（局部压缩下降的全局收敛条件）**  
定理 5.2 保证局部算子收敛到某个局部极小，但不保证达到全局最优。然而，若邻域结构设计为可连通所有状态划分（例如允许任意合并与受控分裂），且配合模拟退火或多次随机重启，则压缩下降过程亦可以概率趋于全局最优。本文将此留作开放算法问题，并在猜想中提出更广泛的收敛性质（见 8.1 节）。

### 6. Even Process 的压缩下降示例

Even Process 生成平稳二元序列，“1”必成对出现。其最小因果态机具有两状态：\(S_0\)（输出 0，以 1/2 概率自旋，1/2 输出 1 转至 \(S_1\)）和 \(S_1\)（必然输出 1 并返回 \(S_0\)）。\(C_\mu=1\) bit，闭包完美。

假设观察者初始采用一阶马尔可夫模型 \(\Pi_0\)（状态仅由前一符号决定）。由无限样本估计得 \(P(0|0)=2/3, P(1|0)=1/3, P(0|1)=1, P(1|1)=0\)。该模型不满足完美闭包（\(\iota_1>0\)），预测编码代价高于 \(h_\mu\)。

在局部压缩下降过程中，预算逐步放开，操作包括分裂状态。经过有限步合并与分裂，过程发现最优划分是将历史按“最近一个 1 之后是否已出现 1”划分，即记忆最近两个符号。最终收敛至两状态因果态模型 \(\Pi^*\)，闭包为零。此时 \(\mathcal{K}(\Pi^*)\) 略增，但预测编码代价大幅降低，总压缩代价 \(\mathcal{L}_{\text{emp}}(\Pi^*) < \mathcal{L}_{\text{emp}}(\Pi_0)\)。该模型为强压缩不动点，任何局部操作均导致代价上升。

### 7. 算法实例

#### 7.1 算法一：最优因果态发现（CSSR 动态规划）
适用于有限符号序列，目标为找到完美闭包且统计复杂度最小的宏观划分。

**算法（CSSR）**  
输入序列 \(z_{1:T}\)，最大历史长度 \(L_{\max}\)。  
1. 以长度为 \(L_{\max}\) 的历史串为初始状态；  
2. 估计每个状态的下一符号经验分布；  
3. 若某状态的经验分布与统计检验下显著不同于其他，则分裂；  
4. 若两状态分布不可区分，则合并；  
5. 重复分裂/合并直至收敛，得因果态划分；  
6. 计算统计复杂度 \(C_\mu = H(\mathcal{S})\)。

**定理 7.1（CSSR 收敛性）**  
若真实过程平稳且具有有限因果态，则当 \(T\to\infty\) 时，CSSR 几乎必然收敛到真实因果态映射，状态划分在所有闭包映射中统计复杂度最小。（证明参见 Shalizi & Crutchfield, 2001。）

#### 7.2 算法二：受约束变分神经压缩
适用于高维连续观测。学习随机编码器 \(q_\phi(y_t \mid y_{t-1}, z_t)\)、宏观转移先验 \(p_\theta(y_t \mid y_{t-1})\)、预测解码器 \(p_\psi(z_{t+1} \mid y_{t+1})\)。损失函数对应理论压缩代价的松弛：
\[
\begin{aligned}
\mathcal{L}_{\text{neural}} = \frac{1}{T}\sum_t \Big[ &-\mathbb{E}_{q_\phi} \log p_\psi(z_{t+1}\mid y_{t+1}) \\
&+ \beta \, \mathrm{KL}(q_\phi(y_t\mid\cdot) \| p_\theta(y_t\mid y_{t-1})) \Big]
+ \alpha \, \widehat{\iota}_1(C) + \gamma \|\Theta\|^2,
\end{aligned}
\]
其中 \(\widehat{\iota}_1(C)\) 为局部闭包缺陷的神经互信息估计，\(\|\Theta\|^2\) 为复杂度代理。梯度下降动态平衡预测、压缩与闭包。

## 第三部分：闭包相变与嵌套 MDL 粗粒化流

### 8. 压缩不动点与理论相变

#### 8.1 主线 A：压缩不动点与涌现实在
关键问题：无限模型类中强压缩不动点的存在唯一性；压缩下降过程的吸引域；预算变化时最优不动点统计复杂度的非连续跳变（理论相变）；不同微观过程涌现相同宏观闭包结构的普适类。

**猜想 1（无限模型类的弱收敛）**  
在 Solomonoff 先验下，无界压缩下降过程以概率 1 收敛到某个弱压缩不动点，其预测分布与 \(\mu\) 的 KL 散度趋于零。

#### 8.2 主线 B：资源约束下的有效理论与闭包相变
观察者资源约束 \(\mathcal{M}_{\mathcal{O},R}\) 决定可达理论类。当资源跨越临界值，最优理论可能发生结构性突变：从非闭包跳到完美闭包，统计复杂度跳跃。我们以闭包缺陷 \(\iota_1\) 或 \(\iota_{\infty}\) 作为序参量刻画相变。

**定义 8.1（资源约束下的最优闭包缺陷）**  
给定资源参数 \(R\)（例如最大状态数），定义可达理论类 \(\mathcal{M}_R\)（固定量化精度），最优可达局部闭包缺陷为
\[
\iota_1^*(R) = \inf_{\Pi \in \mathcal{M}_R} \iota_1(C_\Pi),
\]
其中下限取为使经验压缩代价最小化的理论类（在渐近期望意义下）。若存在临界资源 \(R_c\) 使得
\[
\lim_{R\to R_c^-} \iota_1^*(R) > 0,\qquad \lim_{R\to R_c^+} \iota_1^*(R) = 0,
\]
则称在 \(R_c\) 处发生了**闭包相变**。

**例 8.1（隐马尔可夫过程的闭包相变）**  
考虑两隐藏状态 \(s_1,s_2\) 的平稳过程，转移矩阵：每个状态以概率 0.5 自旋，0.5 跳转。发射概率：\(P(1\mid s_1)=\alpha, P(1\mid s_2)=1-\alpha\)，输出 \(\{0,1\}\)。平稳分布两状态等概。  
资源约束以最大状态数 \(R\) 表示。  
- 当 \(\alpha=0.5\)，两状态发射分布相同，序列为独立同分布伯努利(1/2)。因果态退化为单状态，\(N=1\)，\(C_\mu=0, \iota_1=0\)。若 \(R=1\)，可达理论类包含该单态完美闭包模型，因此 \(\iota_1^*(1)=0\)。  
- 当 \(\alpha\neq 0.5\)，真实因果态数为 \(N=2\)。若资源上限 \(R=1\)，所有单状态模型的闭包缺陷严格为正：单状态模型预测仅依赖于平稳分布，其一步预测互信息 \(\iota_1^*(1)=I(Z_{t+1};\overleftarrow{Z}_t) >0\)。一旦资源增加到 \(R\ge 2\)，可达理论类包含真实的双状态完美闭包模型，故 \(\iota_1^*(2)=0\)。因此在 \(R_c=2\) 处发生闭包相变：最优可达闭包缺陷由正突降为零，统计复杂度从 0 跳至 1 bit。此模型为资源诱导的相变提供可解实例。

**猜想 2（资源诱导的理论相变）**  
存在真实测度和观察者族，在预算临界值 \(R_c\) 处最优有效理论的 \(\iota_1(C)\) 由正变零，统计复杂度发生非连续下降。临界行为具有普适性，仅依赖于约束类型与过程统计结构。

#### 8.3 主线 C：嵌套 MDL 粗粒化流
引入尺度参数 \(\ell\)：粗粒化映射所能使用的最大历史长度，即 \(Y_t = C(z_{t-\ell+1:t})\)。令 \(\mathcal{M}_\ell\) 为所有此类有限记忆理论对集合（状态数及量化精度固定），定义尺度依赖的最优压缩代价
\[
\mathcal{L}^*(\ell) = \min_{\Pi \in \mathcal{M}_\ell} \mathcal{L}(\Pi),
\]
此处 \(\mathcal{L}\) 取渐近期望形式。

**定理 8.2（有限因果态过程的尺度饱和）**  
设真实过程具有有限因果态，且存在有限记忆长度 \(L_0\) 使得因果态划分可由长度为 \(L_0\) 的历史完全确定。则对任意 \(\ell \ge L_0\)，有 \(\mathcal{L}^*(\ell) = \mathcal{L}^*(L_0)\)。进而，若定义尺度压缩算子 \(\mathfrak{R}_{\ell_2,\ell_1}\)（定义见后），则对所有 \(\ell_2 > \ell_1 \ge L_0\)，有 \(\mathfrak{R}_{\ell_2,\ell_1}(\Pi^*) = \Pi^*\)，其中 \(\Pi^*\) 为因果态模型。

*证明*：当记忆长度达到 \(L_0\) 后，模型类已包含因果态表示，其压缩代价达到全局极小，且不能因记忆增长而进一步降低。因此最优压缩代价恒定。\(\square\)

**定义 8.2（尺度依赖压缩算子）**  
对于 \(\ell_1 < \ell_2\)，定义映射 \(\mathfrak{R}_{\ell_2,\ell_1} : \mathcal{M}_{\ell_1} \to \mathcal{M}_{\ell_2}\) 为：给定 \(\Pi \in \mathcal{M}_{\ell_1}\)，通过将其自然嵌入 \(\mathcal{M}_{\ell_2}\)（记忆扩展但初始宏观状态继承原有划分），随后在 \(\mathcal{M}_{\ell_2}\) 中执行局部变分压缩下降至收敛，得到的一个压缩不动点。若存在多个不动点，按确定规则选择或考虑所有可能。该算子定义了不同尺度间的粗粒化流。

**注 8.3（与严格 RG 的区别）**  
由于该算子依赖于优化路径和局部极小选择，通常不满足严格的半群性质：
\[
\mathfrak{R}_{\ell_3,\ell_2} \circ \mathfrak{R}_{\ell_2,\ell_1} \neq \mathfrak{R}_{\ell_3,\ell_1},
\]
这与统计物理中的重整化群变换有本质区别。我们仅在渐近意义下探求其近似半群性，并将其作为研究计划而非已证定理。

**猜想 3（嵌套 MDL 粗粒化流的渐近半群性与标度律）**  
在适当极限下（\(T\to\infty\) 且考虑期望压缩代价），算子族 \(\{\mathfrak{R}_{\ell_2,\ell_1}\}\) 可能在特定不动点邻域内近似满足半群性质：
\[
\mathfrak{R}_{\ell_3,\ell_2} \circ \mathfrak{R}_{\ell_2,\ell_1} \simeq \mathfrak{R}_{\ell_3,\ell_1}.
\]
进而可形式地定义离散压缩 \(\beta\)-函数 \(\beta_\ell = \mathcal{L}^*(\ell+1) - \mathcal{L}^*(\ell)\) 或差分形式 \(\Delta_{\log \ell}\mathcal{L}^*\)。压缩不动点对应 \(\beta_\ell = 0\)，在其附近线性化可区分相关与无关方向，形成涌现规律的临界理论。特别地，可能存在非平凡不动点（如临界过程）使得压缩代价呈现标度律。严格刻画此类不动点结构及其在模型类中的普适性，是未来的核心挑战。

#### 8.4 局部放缩与开放路线
本文框架允许在同一尺度下调整资源预算进行局部放缩，即观察者在微观精确与宏观粗糙间连续过渡，是研究理论相变临界行为的理想探针。例如，改变编码精度 \(\delta\) 或搜索深度，可观测压缩不动点的分岔与消失。未来开放路线包括：开发数值算法跟踪局部放缩下的不动点移动；在随机偏微分方程或场论模型中实例化该框架，提取连续系统的闭包结构。

## 附录 A：定理 5.3 的详细证明

**设定**：真实测度 \(\mu\) 是平稳遍历的，且满足指数 \(\alpha\)-混合：存在常数 \(C,c>0\) 使混合系数 \(\alpha(k)\le C e^{-ck}\)。真实因果态模型 \(\Pi_0\) 具有有限状态数 \(N\)，属于固定有限模型类 \(\mathcal{M}\equiv\mathcal{M}_{M_{\max},\delta_0}\)（最大状态数 \(M_{\max}<\infty\)，概率参数量化至固定精度 \(\delta_0\)，故 \(|\mathcal{M}|<\infty\)）。观测序列 \(z_{1:T}\) 由 \(\mu\) 生成。

定义总体压缩代价（每步期望）：
\[
\mathcal{L}^0(\Pi) = \frac{1}{T}\mathbb{E}[-\log P_\Pi(z_{1:T})] + \frac{\mathcal{K}(\Pi)}{T}.
\]
由平稳性与 \(\alpha\)-混合，\(\mathbb{E}[-\log P_\Pi(z_{1:T})] = T\cdot \ell(\Pi) + o(T)\)，其中 \(\ell(\Pi)=\mathbb{E}_\mu[-\log P_\Pi(Z_{t+1}\mid Y_t)] = h_\mu + \iota_1(C_\Pi) + D_{KL}(\mu(\cdot\mid Y_t) \parallel P_\Pi(\cdot\mid Y_t))\)（命题 2.1）。对于真实模型 \(\Pi_0\)，闭包缺陷和 KL 散度均为零，故 \(\ell(\Pi_0)=h_\mu\)。由可辨识性（定义 1.6），存在 \(\Delta>0\) 使得对所有 \(\Pi\notin[\Pi_0]\)，\(\ell(\Pi) - h_\mu \ge \Delta\)，或者当预测项接近时，复杂度项使总代价分离。

经验压缩代价除以 \(T\) 为：
\[
\frac{1}{T}\mathcal{L}_{\text{emp}}(\Pi) = -\frac{1}{T}\log P_\Pi(z_{1:T}) + \frac{\mathcal{K}(\Pi)}{T}.
\]

**步骤 1：几乎必然一致收敛**  
定义 \(f_t(\Pi) = -\log P_\Pi(z_t\mid z_{1:t-1})\)。过程 \(\{z_t\}\) 是平稳指数 \(\alpha\)-混合的，且 \(f_t(\Pi)\) 是其确定性函数，故 \(\{f_t(\Pi)\}\) 也具有指数 \(\alpha\)-混合。由于模型类有限，我们只需对每个 \(\Pi\) 证明 \((1/T)\sum_{t=1}^T f_t(\Pi) \to \ell(\Pi)\) 几乎必然成立。指数混合性保证了强大数律（参见 Rio, 1999, 定理 6.3 或 Bosq, 1998）。具体而言，\(\{f_t\}\) 的协方差几何衰减，Kronecker 引理给出几乎必然收敛。因为 \(|\mathcal{M}|\) 有限，取所有模型并集上的可数交，得到一致收敛：
\[
\sup_{\Pi\in\mathcal{M}} \left| -\frac{1}{T}\log P_\Pi(z_{1:T}) - \ell(\Pi) \right| \to 0 \quad a.s.
\]
此外，\(\mathcal{K}(\Pi)/T \to 0\) 因为 \(T\to\infty\) 时 \(\mathcal{K}(\Pi)\) 为常量（固定精度 \(\delta_0\) 导致固定复杂度量级）。

**步骤 2：全局极小化器的收敛**  
由于一致收敛，对于足够大的 \(T\)，经验代价 \(\frac{1}{T}\mathcal{L}_{\text{emp}}(\Pi)\) 与总体代价 \(\mathcal{L}^0(\Pi)\) 的排序几乎必然一致。真实等价类 \([\Pi_0]\) 具有严格最小的总体代价（最小预测项，复杂度项不高于任何非等价类模型，且由可辨识性条件保证与其他类的代价间隙）。因此所有经验全局最小化器 \(\Pi_T^*\) 几乎必然落入真实等价类 \([\Pi_0]\)。即存在重标记下的同构，使得其预测分布与 \(\mu\) 一致。

**步骤 3：KL 散度与闭包缺陷**  
\(\Pi_T^*\) 落入 \([\Pi_0]\) 意味着其预测分布与 \(\mu\) 的 KL 散度趋于零（因为等价类中模型均可完美预测）。因此 \(D_{KL}(\mu \parallel P_{\Pi_T^*})\to 0\)。由于真实因果态满足完美闭包 (\(\iota_{\infty}(\Pi_0)=0\))，且观测等价意味着未来预测分布完全一致，因此等价类中任何理论的粗粒化映射同样满足完美闭包，故 \(\iota_{\infty}(C_T^*) = 0\) 且显然 \(\iota_1(C_T^*)=0\) 最终成立。此处并未从 \(\iota_1\to0\) 推导 \(\iota_{\infty}\to0\)，而是直接由等价类性质得到完美闭包。

**步骤 4：状态划分收敛（额外条件下）**  
若进一步假设模型类中的理论均具有最小 unifilar 表示且状态可同步可辨识，则预测分布等价直接蕴含底层状态转移结构的等价，从而状态划分在重标记下收敛。证毕。\(\blacksquare\)
