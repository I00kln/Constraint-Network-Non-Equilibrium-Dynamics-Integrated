## 摘要

本理论以**生成结构**与**有限资源预测系统**为唯二基础对象，以**闭包缺陷** \(\Phi\) 为唯一核心量。理论由四条第一性方程闭合：

1. **闭包缺陷的 Pythagorean 分解**：\(\Phi = I_{\mathrm{cl}} + D_{\mathrm{mis}}\)，区分资源不足（\(I_{\mathrm{cl}}\)）与模型误设（\(D_{\mathrm{mis}}\)）。
2. **保守–耗散动力学与缺陷单调下降**：在固定生成结构下，若系统服从梯度公理，则 \(\frac{d\Phi}{dt} = -\|\operatorname{grad}_g \Phi\|^2 \le 0\)。
3. **闭包缺陷–生存风险不等式**：\(\mathbb{E}[\mathrm{TV}(P_t,Q_t)] \le \sqrt{\Phi/2}\)，且对任意事件 \(\mathbb{E}|p_E - q_E| \le \sqrt{\Phi/2}\)。
4. **资源–缺陷定量关系**：\(\Phi \ge I_{\mathrm{pred}}^{\mathrm{total}} - C_{\mathrm{cap}}(R,L) + D_{\mathrm{mis}}\)，其中 \(C_{\mathrm{cap}}\) 为有限资源下的预测信息容量。

v22.2 针对 v22.1 的分析建议完成以下闭合性修正：

1. **动力学直接基于真实闭包缺陷**：公理以真实缺陷 \(\Phi\) 的自然梯度下降为核心，严格证明 \(\frac{d\Phi}{dt} \le 0\)，消除经验缺陷与真实缺陷之间未证明的收敛间隙。
2. **经验闭包缺陷的统计收敛**：将经验缺陷 \(\hat{\Phi}_T\) 作为 \(\Phi\) 的强相合估计，基于 Birkhoff 遍历定理严格证明收敛性，不再作为动力学定义的基础。
3. **风险推论的期望化完备表述**：保持期望界的严格形式，并补充充分条件。
4. **严格分离定理与猜想**：已被前文定理覆盖的猜想（原猜想 6.2）移除；容量增长与相变保留为严格限定的开放问题。

其余核心结构保持与 v22.1 一致，整体框架形成数学闭合。

## 第〇部分：生成结构与因果态

### 0.1 生成结构

**定义 0.1（生成结构）**  
一个**生成结构** \(\mathcal{G}\) 是平稳遍历概率空间 \((\Omega,\mathcal{F},\mathbb{P})\) 上的双向符号序列过程 \(\{X_t\}_{t\in\mathbb{Z}}\)，取值于有限字母表 \(\mathcal{X}\)。记 \(P\) 为其分布，\(P(\cdot \mid x_{<t})\) 为正则条件概率。熵率定义为
\[
h(\mathcal{G}) = \lim_{n\to\infty} \frac{1}{n} H(X_{1:n}) = H(X_1 \mid X_{<1}).
\]

**公理 0.1（观察者接口）**  
预测系统仅接收有限观测序列 \(x_{1:T}\)，无法直接访问 \(P\)。

### 0.2 因果态与预测信息总量

**定义 0.2（因果态）**  
令 \(S_t = \varepsilon(x_{<t})\) 为满足以下条件的映射：

* (i) **预测充分性**：\(P(X_{t+1} \mid x_{<t}) = P(X_{t+1} \mid S_t)\) a.s.；
* (ii) **极小性**：若 \(V = v(x_{<t})\) 亦满足 \(P(X_{t+1} \mid x_{<t}) = P(X_{t+1} \mid V)\) a.s.，则存在可测函数 \(\psi\) 使得 \(S_t = \psi(V)\) a.s.

\(S_t\) 取值于至多可数的因果态空间 \(\mathcal{S}\)。记 \(N = |\mathcal{S}|\)。一步预测信息总量定义为
\[
I_{\mathrm{pred}}^{\mathrm{total}} = I(X_{t+1}; S_t) = H(X_{t+1}) - h(\mathcal{G}).
\]

**假设 0.1（有限因果态与最小同步深度）**  
\(N < \infty\)，且存在有限**最小同步深度** \(L_0 < \infty\)，定义为
\[
L_0 = \min\{ L \in \mathbb{N} \mid \exists \varepsilon_L : \mathcal{X}^L \to \mathcal{S},\; \text{使得 } S_t = \varepsilon_L(x_{t-L+1}^{t}) \text{ a.s.} \}.
\]
换言之，\(L_0\) 是能够从有限长度历史几乎必然唯一确定因果态的最小窗口长度。任何 \(L < L_0\) 均无法保证 \(S_t\) 可由长度为 \(L\) 的历史确定。

## 第一部分：有限资源预测系统与闭包缺陷

### 1.1 资源与编码

**定义 1.1（资源参数）**  
资源参数 \(\rho = (R,L)\)，其中 \(R \in \mathbb{N}\) 为状态预算，\(L \in \mathbb{N}\) 为回溯窗口。

**定义 1.2（编码集）**  
给定 \((R,L)\)，**编码集**定义为全体映射
\[
\Phi_{R,L} = \{ \phi : \mathcal{X}^L \to \{1,\dots,R\} \}.
\]
因 \(\mathcal{X}\) 有限，\(|\Phi_{R,L}| = R^{|\mathcal{X}|^L} < \infty\)。赋予离散拓扑，\(\Phi_{R,L}\) 为紧致空间。

**定义 1.3（预测系统与模型类）**  
一个**预测系统** \(\mathcal{S}\) 是三元组 \((\phi, Q, \rho)\)：

* \(\phi \in \Phi_{R,L}\)，给出状态 \(h_t = \phi(x_{t-L+1}^{t})\)；
* \(Q : \{1,\dots,R\} \times \mathcal{X} \to [0,1]\) 为预测核，满足对每个 \(h\)，\(Q(\cdot \mid h) \in \Delta(\mathcal{X})\)。

**模型类** \(\mathcal{F}_\rho\) 为所有此类系统的集合，即
\[
\mathcal{F}_\rho = \Phi_{R,L} \times (\Delta(\mathcal{X}))^R.
\]
\(\Phi_{R,L}\) 为有限离散空间（紧致），\((\Delta(\mathcal{X}))^R\) 为概率单纯形的紧致积空间，故 \(\mathcal{F}_\rho\) 为紧致集。

**定义 1.4（最优预测核）**  
对给定编码 \(\phi\)，定义最优预测核
\[
Q_\phi^*(x \mid h) = \mathbb{E}[ P(x \mid S_t) \mid h_t = h ] = P(x \mid h).
\]

### 1.2 闭包缺陷的 Pythagorean 分解

**定义 1.5（信息闭包缺陷）**  
\[
I_{\mathrm{cl}}(\phi) = I(X_{t+1}; S_t \mid h_t) = I_{\mathrm{pred}}^{\mathrm{total}} - I(X_{t+1}; h_t).
\]

**定义 1.6（模型误设缺陷）**  
对系统 \(\mathcal{S} = (\phi, Q, \rho)\)，
\[
D_{\mathrm{mis}}(Q,\phi) = \mathbb{E}_{h_t}\big[ D_{\mathrm{KL}}( Q_\phi^*(\cdot \mid h_t) \,\|\, Q(\cdot \mid h_t) ) \big].
\]

**定理 1.1（闭包缺陷的 Pythagorean 分解）** ■  
\[
\Phi(\mathcal{S}) := \mathbb{E}_{S_t, h_t}\big[ D_{\mathrm{KL}}( P(\cdot \mid S_t) \,\|\, Q(\cdot \mid h_t) ) \big] = I_{\mathrm{cl}}(\phi) + D_{\mathrm{mis}}(Q,\phi).
\]
*证明*：
\[
\begin{aligned}
&D_{\mathrm{KL}}(P(\cdot \mid S_t) \,\|\, Q(\cdot \mid h_t)) \\
&= \sum_x P(x\mid S_t) \log\frac{P(x\mid S_t)}{Q_\phi^*(x\mid h_t)} 
   + \sum_x P(x\mid S_t) \log\frac{Q_\phi^*(x\mid h_t)}{Q(x\mid h_t)} .
\end{aligned}
\]
第一项对 \((S_t,h_t)\) 取期望，利用条件概率迭代得 \(I(X_{t+1}; S_t \mid h_t) = I_{\mathrm{cl}}(\phi)\)。第二项中 \(\log(Q_\phi^*/Q)\) 仅依赖于 \(h_t\)，其条件期望 \(\mathbb{E}[\cdot \mid h_t]\) 即为 \(D_{\mathrm{KL}}(Q_\phi^* \| Q)\)，再取全期望得 \(D_{\mathrm{mis}}(Q,\phi)\)。■

**推论 1.1（最优可达缺陷）**  
对固定 \(\phi\)，取 \(Q = Q_\phi^*\)，则 \(D_{\mathrm{mis}} = 0\)，\(\Phi = I_{\mathrm{cl}}(\phi)\)。最小可达闭包缺陷（在资源 \(\rho\) 下）为
\[
\Phi_\rho^* = \min_{\mathcal{S} \in \mathcal{F}_\rho} \Phi(\mathcal{S}) = I_{\mathrm{pred}}^{\mathrm{total}} - C_{\mathrm{cap}}(R,L),
\]
其中 \(C_{\mathrm{cap}}\) 如下定义。

## 第二部分：预测信息容量与资源–缺陷定量关系

### 2.1 预测信息容量

**定义 2.1（预测信息容量）**  
给定资源 \(\rho = (R,L)\)，**预测信息容量**为
\[
C_{\mathrm{cap}}(R,L) = \max_{\phi \in \Phi_{R,L}} I(X_{t+1}; h_t), \quad h_t = \phi(x_{t-L+1}^{t}).
\]
因 \(\Phi_{R,L}\) 为有限集，最大值必达到。

### 2.2 充分性等价定理

**定理 2.1（容量饱和的充分必要条件）** ■  
设生成结构满足假设 0.1，且 \(L_0\) 为最小同步深度。令 \(h_t = \phi(x_{t-L+1}^{t})\)。则
\[
I(X_{t+1}; h_t) = I_{\mathrm{pred}}^{\mathrm{total}}
\]
当且仅当 \(h_t\) 是 \(S_t\) 的充分统计量，即存在可测函数 \(\psi\) 使 \(S_t = \psi(h_t)\) a.s.。此时必有 \(|\mathrm{range}(h_t)| \ge N\) 且 \(L \ge L_0\)。

*证明*：  
（\(\Rightarrow\)）假设 \(I(X_{t+1}; h_t) = I_{\mathrm{pred}}^{\mathrm{total}}\)。由定义，\(I_{\mathrm{pred}}^{\mathrm{total}} = I(X_{t+1}; S_t)\)，且 \(S_t\) 满足 \(H(X_{t+1}\mid S_t) = h(\mathcal{G})\)。因 \(h_t\) 是 \(x_{<t}\) 的函数，有
\[
H(X_{t+1}\mid h_t) \ge H(X_{t+1}\mid h_t, S_t) = H(X_{t+1}\mid S_t) = h(\mathcal{G}),
\]
其中等式 \(H(X_{t+1}\mid h_t, S_t) = H(X_{t+1}\mid S_t)\) 成立是因为 \(S_t\) 是预测充分的：对任何由 \(x_{<t}\) 决定的函数 \(h_t\)，有 \(P(X_{t+1}\mid h_t, S_t) = P(X_{t+1}\mid S_t)\) a.s.。

另一方面，\(I(X_{t+1}; h_t) = I_{\mathrm{pred}}^{\mathrm{total}}\) 给出 \(H(X_{t+1}\mid h_t) = h(\mathcal{G})\)。因此
\[
H(X_{t+1}\mid h_t) = H(X_{t+1}\mid h_t, S_t),
\]
故条件互信息 \(I(X_{t+1}; S_t \mid h_t) = 0\)。于是 \(X_{t+1}\) 与 \(S_t\) 在给定 \(h_t\) 时条件独立。结合 \(S_t\) 的预测充分性，对几乎处处 \(h_t\)，有
\[
P(X_{t+1} \mid h_t) = P(X_{t+1} \mid h_t, S_t) = P(X_{t+1} \mid S_t).
\]
因此 \(h_t\) 亦是预测充分统计量。根据因果态的极小性（定义 0.2(ii)），存在可测函数 \(\psi\) 使 \(S_t = \psi(h_t)\) a.s.。此时必然有 \(|\mathrm{range}(h_t)| \ge |\mathrm{range}(S_t)| = N\)，并且由于 \(h_t\) 是长度为 \(L\) 的窗口的函数，由 \(S_t = \psi(h_t) = (\psi\circ\phi)(x_{t-L+1}^{t})\) 可知 \(S_t\) 可由长度 \(L\) 的历史决定，结合最小同步深度定义，必有 \(L \ge L_0\)（否则与 \(L_0\) 的最小性矛盾）。

（\(\Leftarrow\)）若 \(S_t = \psi(h_t)\) a.s.，则由数据处理不等式，\(I(X_{t+1}; h_t) \ge I(X_{t+1}; S_t) = I_{\mathrm{pred}}^{\mathrm{total}}\)；同时 \(I(X_{t+1}; h_t) \le I_{\mathrm{pred}}^{\mathrm{total}}\) 恒成立，故等号成立。■

### 2.3 容量间隙定理与资源扩张定理

**定理 2.2（容量间隙定理）** ■  
若 \(R < N\) 或 \(L < L_0\)，则 \(C_{\mathrm{cap}}(R,L) < I_{\mathrm{pred}}^{\mathrm{total}}\)。间隙
\[
\Delta(R,L) := I_{\mathrm{pred}}^{\mathrm{total}} - C_{\mathrm{cap}}(R,L) > 0.
\]

*证明*：  
假设存在 \(\phi \in \Phi_{R,L}\) 使得 \(I(X_{t+1}; h_t) = I_{\mathrm{pred}}^{\mathrm{total}}\)。由定理 2.1，必存在 \(\psi\) 使 \(S_t = \psi(h_t)\) a.s.。

* 若 \(R < N\)，则 \(|\mathrm{range}(h_t)| \le R < N\)，但定理 2.1 要求 \(|\mathrm{range}(h_t)| \ge N\)，矛盾。
* 若 \(L < L_0\)，则由 \(h_t = \phi(x_{t-L+1}^{t})\) 得 \(S_t = (\psi\circ\phi)(x_{t-L+1}^{t})\) a.s.，即 \(S_t\) 由长度 \(L\) 的历史确定，这与 \(L_0\) 是最小同步深度（定义 0.2 与假设 0.1）矛盾。

因此，在 \(R < N\) 或 \(L < L_0\) 时，任意 \(\phi \in \Phi_{R,L}\) 均不可能使互信息达到 \(I_{\mathrm{pred}}^{\mathrm{total}}\)。由于 \(\Phi_{R,L}\) 是有限集，最大值 \(C_{\mathrm{cap}}(R,L)\) 存在且必严格小于该上确界，故 \(\Delta(R,L) > 0\)。■

**注 2.1（正间隙对有限编码类的依赖性）**  
上述证明中，\(\Delta(R,L) > 0\) 的结论依赖于 \(\Phi_{R,L}\) 的有限性（从而最大值可达且严格小于上确界）。若未来将框架扩展至连续编码器空间，则可能出现 \(\sup C_{\mathrm{cap}} = I_{\mathrm{pred}}^{\mathrm{total}}\) 但最大值不可达的情形，此时 \(\Delta\) 仅代表可达间隙的下界，需另行讨论。

**定理 2.3（资源扩张定理）** ■  
1. **单调性**：若 \(R_1 \le R_2\) 且 \(L_1 \le L_2\)，则 \(C_{\mathrm{cap}}(R_1,L_1) \le C_{\mathrm{cap}}(R_2,L_2)\)。  
2. **饱和性**：当 \(R \ge N\) 且 \(L \ge L_0\) 时，\(C_{\mathrm{cap}}(R,L) = I_{\mathrm{pred}}^{\mathrm{total}}\)。  
3. **临界性**：\(C_{\mathrm{cap}}(R,L) = I_{\mathrm{pred}}^{\mathrm{total}}\) 当且仅当 \(R \ge N\) 且 \(L \ge L_0\)。

*证明*：  
(1) 对任意 \(\phi_1 \in \Phi_{R_1,L_1}\)，构造 \(\phi_2 \in \Phi_{R_2,L_2}\) 如下：对 \(x_{t-L_2+1}^{t}\)，取其末尾 \(L_1\) 个符号，应用 \(\phi_1\)，所得编码值在 \(\{1,\dots,R_1\} \subseteq \{1,\dots,R_2\}\) 中，故 \(\phi_2 \in \Phi_{R_2,L_2}\)。由平稳性，\(h_t^{(2)} = \phi_2(\cdot)\) 与 \(h_t^{(1)} = \phi_1(\cdot)\) 同分布，故 \(I(X_{t+1}; h_t^{(2)}) = I(X_{t+1}; h_t^{(1)})\)。因此
\[
C_{\mathrm{cap}}(R_2,L_2) \ge \sup_{\phi_1} I(X_{t+1}; h_t^{(1)}) = C_{\mathrm{cap}}(R_1,L_1).
\]
(2) 设 \(R \ge N\)，\(L \ge L_0\)。由最小同步深度定义，存在映射 \(\varepsilon_{L_0} : \mathcal{X}^{L_0} \to \mathcal{S}\) 使得 \(S_t = \varepsilon_{L_0}(x_{t-L_0+1}^{t})\) a.s.。定义编码 \(\phi^* : \mathcal{X}^L \to \{1,\dots,R\}\) 为：取观测后缀 \(x_{t-L_0+1}^{t}\)，计算 \(S_t = \varepsilon_{L_0}(x_{t-L_0+1}^{t})\)，再将 \(S_t\) 单射嵌入 \(\{1,\dots,R\}\)（因 \(N \le R\)）。则 \(h_t = \phi^*(x_{t-L+1}^{t})\) 与 \(S_t\) 一一对应，故 \(I(X_{t+1}; h_t) = I_{\mathrm{pred}}^{\mathrm{total}}\)。由定义，\(C_{\mathrm{cap}}(R,L) = I_{\mathrm{pred}}^{\mathrm{total}}\)。  
(3) 充分性即 (2)；必要性由定理 2.2 直接推出。■

**推论 2.2（最小不可消除缺陷的充要条件）**  
\(\Phi_\rho^* = 0\) 当且仅当 \(R \ge N\) 且 \(L \ge L_0\)；否则 \(\Phi_\rho^* \ge \Delta(R,L) > 0\)。

**注 2.2（经验容量）**  
从有限样本 \(x_{1:T}\) 可得经验容量 \(\hat{C}_{\mathrm{cap}}(R,L)\)，在平稳遍历条件下 \(\hat{C}_{\mathrm{cap}} \to C_{\mathrm{cap}}\) a.s.。任何具体算法（如 CSSR）给出 \(C_{\mathrm{cap}}\) 的下界。

## 第三部分：保守–耗散动力学（公理体系）

本部分建立预测系统参数演化的动力学，直接以真实闭包缺陷 \(\Phi\) 的梯度流为核心，并证明缺陷的单调下降。随后引入经验闭包缺陷作为统计可估计量，证明其强相合性。

### 3.1 参数化与 Fisher 度量

固定编码映射 \(\phi\)（由资源 \(\rho\) 选定），考虑预测核 \(Q\) 的参数化。设 \(Q = Q_\theta\)，其中 \(\theta \in \Theta_\rho \subset \mathbb{R}^d\)，且映射 \(\theta \mapsto Q_\theta\) 是光滑的。对每个隐状态 \(h \in \{1,\dots,R\}\)，令
\[
\ell_h(x;\theta) = \ln Q_\theta(x \mid h).
\]
**Fisher 信息度量**（在参数空间上）定义为
\[
g_{ij}(\theta) = \mathbb{E}_{h}\mathbb{E}_{x \sim Q_\theta(\cdot \mid h)}\big[ \partial_i \ell_h(x;\theta) \, \partial_j \ell_h(x;\theta) \big],
\]
其中 \(\mathbb{E}_h\) 是编码状态 \(h_t\) 的稳态分布下的期望。假设 \(g(\theta)\) 在 \(\Theta_\rho\) 上正定，则 \((\Theta_\rho, g)\) 构成 Riemann 流形。

### 3.2 真实闭包缺陷的梯度流

对给定的编码 \(\phi\) 与参数 \(\theta\)，真实闭包缺陷为
\[
\Phi(\theta) = I_{\mathrm{cl}}(\phi) + \mathbb{E}_{h}\big[ D_{\mathrm{KL}}( Q_\phi^*(\cdot \mid h) \,\|\, Q_\theta(\cdot \mid h) ) \big],
\]
其中 \(Q_\phi^*\) 为定义 1.4 的最优预测核。记 \(\nabla_\theta \Phi\) 为 Euclid 梯度，\(\operatorname{grad}_g \Phi = g^{-1} \nabla_\theta \Phi\) 为 Riemann 梯度。

**公理 3.1（保守–耗散动力学）**  
在固定生成结构 \(\mathcal{G}\) 下，预测核参数 \(\theta\) 的连续时间演化服从
\[
\dot{\theta} = -\eta \, g^{-1}(\theta) \nabla_\theta \Phi(\theta) + F_{\mathrm{cons}}(\theta),
\]
其中 \(\eta > 0\) 为学习率常数（可吸收进时间尺度），\(F_{\mathrm{cons}}\) 满足 \(\langle \nabla_\theta \Phi, F_{\mathrm{cons}} \rangle_g = 0\)（守恒流），即守恒部分不做信息功。

**定理 3.1（真实缺陷单调下降）** ■  
在公理 3.1 下，
\[
\frac{d\Phi}{dt} = -\eta \|\operatorname{grad}_g \Phi\|^2_g \le 0,
\]
等号成立当且仅当 \(\operatorname{grad}_g \Phi = 0\)。

*证明*：
\[
\frac{d\Phi}{dt} = \langle \nabla_\theta \Phi, \dot{\theta} \rangle
= \langle \nabla_\theta \Phi, -\eta g^{-1}\nabla_\theta \Phi + F_{\mathrm{cons}} \rangle
= -\eta \langle \nabla_\theta \Phi, g^{-1}\nabla_\theta \Phi \rangle + \langle \nabla_\theta \Phi, F_{\mathrm{cons}} \rangle.
\]
由 Riemann 度量定义，\(\langle \nabla_\theta \Phi, g^{-1}\nabla_\theta \Phi \rangle = \|\operatorname{grad}_g \Phi\|^2_g\)；由守恒流条件，第二项为零。故得证。■

此定理给出了闭包缺陷在理想学习动力学下的严格单调性，构成理论的第二个核心方程。

### 3.3 经验闭包缺陷及其强相合性

实际系统中真实缺陷 \(\Phi(\theta)\) 不可直接计算，需通过有限观测序列估计。引入经验闭包缺陷作为可操作的统计量。

**定义 3.1（经验闭包缺陷）**  
给定来自生成结构 \(\mathcal{G}\) 的样本 \(x_{1:T}\)、编码 \(\phi\) 与参数 \(\theta\)，**经验闭包缺陷** \(\hat{\Phi}_T(\theta)\) 定义为
\[
\hat{\Phi}_T(\theta) = \frac{1}{T-L}\sum_{t=L}^{T-1} \Big( -\ln Q_\theta(x_{t+1} \mid h_t) \Big) - h(\mathcal{G}),
\]
其中 \(h_t = \phi(x_{t-L+1}^{t})\)，\(h(\mathcal{G})\) 为生成结构的熵率（假设已知或已一致估计）。

**定理 3.2（经验闭包缺陷的强相合性）** ■  
若 \(\mathbb{E}\big[ |\ln Q_\theta(X_{t+1} \mid h_t)| \big] < \infty\)，则在平稳遍历条件下，
\[
\lim_{T\to\infty} \hat{\Phi}_T(\theta) = \Phi(\theta) \quad \text{a.s.}
\]

*证明*：  
令 \(f_t(\theta) = -\ln Q_\theta(X_{t+1} \mid h_t)\)。过程 \(\{f_t\}\) 是平稳遍历的，且由假设一阶矩有限。由 Birkhoff 遍历定理，
\[
\frac{1}{T-L}\sum_{t=L}^{T-1} f_t(\theta) \to \mathbb{E}[f_1(\theta)] \quad \text{a.s.}
\]
而
\[
\mathbb{E}[f_1(\theta)] = \mathbb{E}[-\ln Q_\theta(X_{t+1} \mid h_t)] = H(X_{t+1} \mid h_t) + \mathbb{E}[D_{\mathrm{KL}}(P(\cdot|h_t) \| Q_\theta(\cdot|h_t))],
\]
其中 \(P(\cdot|h_t) = Q_\phi^*(\cdot|h_t)\)。注意到 \(H(X_{t+1} \mid h_t) = h(\mathcal{G}) + I_{\mathrm{cl}}(\phi)\)（因为 \(H(X_{t+1}) - H(X_{t+1}|h_t) = I(X_{t+1};h_t)\) 且 \(I_{\mathrm{pred}}^{\mathrm{total}} - I(X_{t+1};h_t) = I_{\mathrm{cl}}\)）。因此
\[
\mathbb{E}[f_1(\theta)] = h(\mathcal{G}) + I_{\mathrm{cl}}(\phi) + \mathbb{E}[D_{\mathrm{KL}}(Q_\phi^* \| Q_\theta)] = h(\mathcal{G}) + \Phi(\theta).
\]
故 \(\hat{\Phi}_T(\theta) \to \Phi(\theta)\) a.s.。■

此定理将公理 3.1 的动力学建立在统计可估计的基础之上：经验缺陷是真实缺陷的强相合估计，因而梯度流方向可通过样本近似实现。

## 第四部分：规律涌现与生存风险

### 4.1 规律作为闭包极小盆地

**定义 4.1（均匀基准）**  
\[
\Phi_{\mathrm{rand}} = \ln|\mathcal{X}| - h(\mathcal{G}).
\]

**定义 4.2（弱规律）**  
弱规律是 \(\Phi\) 在 \(\Theta_\rho\) 上的局部极小盆地 \(\mathcal{R}\)，满足 \(\forall \theta \in \mathcal{R}, \Phi(\theta) < \Phi_{\mathrm{rand}}\)。

**定义 4.3（强规律）**  
强规律是弱规律盆地满足 \(\Phi_\rho^* = 0\)。

**定理 4.1（涌现阈值）** ■  
强规律存在当且仅当 \(R \ge N\) 且 \(L \ge L_0\)（且 \(D_{\mathrm{mis}}\) 可达零）。

*证明*：由推论 2.2 直接得出。■

**猜想 4.1（资源扩张的稳定化）** ◇  
增加 \(R\) 或 \(L\) 可将 Hessian \(\nabla_g^2 \Phi\) 的部分零特征值方向转化为正曲率方向，增强稳定性。

### 4.2 生存风险闭合

**定理 4.2（闭包缺陷–风险不等式）** ■  
令 \(P_t = P(\cdot \mid S_t)\)，\(Q_t = Q(\cdot \mid h_t)\)。则有
\[
\mathrm{TV}(P_t, Q_t) \le \sqrt{ \frac{D_{\mathrm{KL}}(P_t \,\|\, Q_t)}{2} },
\]
取期望得全局形式：
\[
\mathbb{E}\big[ \mathrm{TV}(P_t, Q_t) \big] \le \sqrt{\frac{\Phi}{2}}.
\]
特别地，对任意由 \(X_{t+1}\) 决定的事件 \(E\)，设 \(p_E = P(E \mid S_t)\)，\(q_E = Q(E \mid h_t)\)，则
\[
|p_E - q_E| \le \mathrm{TV}(P_t, Q_t) \le \sqrt{ \frac{D_{\mathrm{KL}}(P_t \,\|\, Q_t)}{2} },
\]
进而
\[
\mathbb{E}|p_E - q_E| \le \sqrt{\frac{\Phi}{2}}.
\]

*证明*：由 Pinsker 不等式直接应用于条件分布，再对 \((S_t, h_t)\) 取期望，利用 Jensen 不等式 \(\mathbb{E}\sqrt{D} \le \sqrt{\mathbb{E}D}\) 即得。■

**推论 4.1（安全充分条件）**  
若 \(\sup_{h_t} q_E(h_t) + \sqrt{\Phi/2} \le \varepsilon_c\)，则
\[
\mathbb{E}[p_E] \le \varepsilon_c.
\]
*证明*：由定理 4.2，\(\mathbb{E}|p_E - q_E| \le \sqrt{\Phi/2}\)，故
\[
\mathbb{E}[p_E] \le \mathbb{E}[q_E] + \sqrt{\Phi/2} \le \sup_{h_t} q_E(h_t) + \sqrt{\Phi/2} \le \varepsilon_c.
\]

（注：此处给出的是期望控制；若需高概率保证，须额外引用浓度不等式。）

## 第五部分：核心方程总结

\[
\begin{aligned}
&\Phi = I_{\mathrm{cl}} + D_{\mathrm{mis}} \\
&\frac{d\Phi}{dt} = -\eta \|\operatorname{grad}_g \Phi\|^2 \quad (\text{公理 3.1 下}) \\
&\mathbb{E}\,\mathrm{TV}(P,Q) \le \sqrt{\Phi/2} \;\Longrightarrow\; \mathbb{E}|p_E - q_E| \le \sqrt{\Phi/2} \\
&\Phi \ge I_{\mathrm{pred}}^{\mathrm{total}} - C_{\mathrm{cap}}(R,L) + D_{\mathrm{mis}} \\
&C_{\mathrm{cap}}(R,L) \nearrow I_{\mathrm{pred}}^{\mathrm{total}},\; \text{达到饱和} \iff R \ge N \text{ 且 } L \ge L_0 \\
&\text{否则 } \Phi_\rho^* \ge \Delta(R,L) > 0
\end{aligned}
\]

## 第六部分：开放问题——容量增长与标度律

现有理论确定了容量 \(C_{\mathrm{cap}}(R,L)\) 的存在性、单调性及饱和条件，但未刻画其在有限资源下逼近 \(I_{\mathrm{pred}}^{\mathrm{total}}\) 的**具体速率**。这一标度关系决定了闭包缺陷 \(\Phi_\rho^*\) 随资源扩张的衰减律，是从结构理论走向定量预测的关键。

**定义 6.1（容量间隙与标度函数）**  
令资源参数 \(\rho = (R,L)\)，容量间隙为 \(\Delta(R,L) = I_{\mathrm{pred}}^{\mathrm{total}} - C_{\mathrm{cap}}(R,L)\)。对于给定的生成结构 \(\mathcal{G}\)，定义**标度函数** \(f: \mathbb{N} \to [0, I_{\mathrm{pred}}^{\mathrm{total}}]\) 满足 \(\Delta(R,L) = f(R,L; \mathcal{G})\)。

**猜想 6.1（容量增长律的一般形式）** ◇  
存在与 \(\mathcal{G}\) 的统计复杂度相关的指数 \(\alpha,\beta,\gamma > 0\)，使得当 \(R\) 或 \(L\) 增加时，容量间隙满足
\[
\Delta(R,L) \sim O(R^{-\alpha}) \quad (\text{固定 } L \ge L_0 \text{ 时}),
\]
\[
\Delta(R,L) \sim O(e^{-\beta L}) \quad (\text{固定 } R \text{ 充分大时，若同步深度无限则可能呈指数衰减}),
\]
或在联合扩张下呈现更复杂的标度行为 \(\Delta(R,L) \sim O(R^{-\alpha} L^{-\gamma})\)。具体指数由生成结构的混合性质与因果态拓扑决定。

**猜想 6.2（最优编码的相变行为）** ◇  
当资源跨越临界值 \((N, L_0)\) 时，系统从“缺陷不可消除相”突变为“零缺陷相”。若考虑因果态数 \(N \to \infty\) 或引入温度参数，则 \(\Phi_\rho^*\) 可能展现出非连续下降（一阶相变）或 Hessian 特征值的非解析变化。当前框架仅在离散有限资源下证实了严格的阈值现象；连续极限下的相变描述是开放问题。

上述猜想为下一阶段的核心研究方向。若能建立 \(C_{\mathrm{cap}}\) 的可计算理论或变分表征，整个框架将具备完整的定量预测能力。
