# 从因果信息结构到有效物理动力学：数值验证路线设计（修订版）

**版本**：2.0  
**依据**：原理论框架 + 修正意见（零模型、谱粗化、谱图理论、有限尺度分析、规范相位、响应函数光锥、离散爱因斯坦方程、相图、普适性）

---

## 一、设计原则

1. **允许理论失败**：明确区分“结构涌现”与“复杂系统通用伪影”。
2. **强制零模型**：任何观测信号必须与三种对照组比较。
3. **避免激活函数伪影**：用图拉普拉斯谱截断替代 sigmoid 作为首选宏微观分解。
4. **谱图理论优先**：路径模稳定性通过拉普拉斯本征模分析，而非 DFS 枚举环。
5. **有限尺度标度**：所有结论需外推到 \(N\to\infty\)。
6. **规范结构严格化**：边权升级为 \(U(1)\) 或 \(SU(2)\) 相位，Wilson loop 测量 holonomy。
7. **几何涌现可操作定义**：光锥用响应函数，引力用离散 Ricci 曲率-能量密度关系。
8. **相图与普适性**：扫描参数空间，寻找不依赖于参数细节的普适区。

## 二、核心模型定义（修订）

### 2.1 图与变量

- 有向图 \(\Gamma=(V,E)\)，\(N=|V|\)。
- 节点信息密度 \(\rho_v\in\mathbb{R}^+\)，满足 \(\sum_v\rho_v = C=1\)。
- **边权升级**：每条边携带 \(U_{uv} \in U(1)\)（复相位 \(e^{i\theta_{uv}}\)）或 \(SU(2)\) 矩阵（用于检验规范群候选）。初始相位均匀随机 \([0,2\pi)\)。

### 2.2 动力学步骤（无 sigmoid 版本，采用谱粗化）

#### 步骤 1：因果传播（保持实数密度）
\[
\rho_v^{t+1/3} = (1-\eta)\rho_v^t + \eta \sum_{u\to v} \rho_u^t \cdot |U_{uv}|^2
\]
其中 \(|U_{uv}|=1\)（相位不影响密度传播，仅影响 holonomy）。

#### 步骤 2：谱宏微观分解（代替 sigmoid）
- 计算图的对称归一化拉普拉斯：\(L = I - D^{-1/2} A D^{-1/2}\)（有向图可用入度/出度对称化）。
- 计算本征值 \(\lambda_0=0 \le \lambda_1 \le \dots \le \lambda_{N-1}\)。
- 选择低通截止 \(k_c\)（例如本征值 \(<0.1\) 对应的特征模数目）。
- 宏观场：
\[
\Phi = P_{k<k_c} \,\rho \quad (\text{投影到低本征模子空间})
\]
- 微观场：
\[
\Psi = \rho - \Phi
\]
- **无 sigmoid**，无激活函数。分解是线性的、正交的。

#### 步骤 3：总量重分配（同前）
\[
\sum_v \Phi_v + \lambda \sum_v \Psi_v = C
\]
缩放 \(\Psi\) 实现。

#### 步骤 4：拓扑与相位重连
- **边的删除**：若边上的 holonomy 累积变化超过阈值，或传播效率过低（见步骤1），删除概率 \(P_{\text{del}} \propto (1 - |\frac{1}{L}\sum_{\text{cycles}} U_{\text{cycle}}|)\)。
- **新边的创建**：对节点对 \((u,v)\)，概率
\[
P_{\text{new}} \propto \exp\left( \alpha\, \rho_u \rho_v + \beta\, \frac{1}{|\mathcal{N}(u,v)|}\sum_{\text{paths}} \text{Re}(\text{Tr}(U_{\text{path}})) \right)
\]
- **相位更新**：每条边相位随机游走 \(\theta_{uv} \leftarrow \theta_{uv} + \xi\)，\(\xi\sim\mathcal{N}(0,\sigma_\theta^2)\)，但受 holonomy 反馈调整（倾向于使小环 holonomy 接近单位元）。

#### 步骤 5：监测与输出
每步存储：\(\rho, \Phi, \Psi\)，边相位，拉普拉斯本征谱前20个本征值，Wilson loop 统计量，响应函数。

## 三、零模型（Null Models）设计

必须与以下三种对照比较，否则任何结构都不能归因于因果自指机制。

| 零模型 | 修改 | 预期 |
|--------|------|------|
| **NM-A：无宏微观分解** | 去掉谱分解，\(\Phi=\rho, \Psi=0\)，仅保留传播+重连（无 \(\lambda\) 约束） | 若仍出现三代谱或规范残余，说明机制非必要 |
| **NM-B：随机重连** | 新边概率与 \(\rho\) 和 holonomy 无关，均匀随机 | 若仍出现 Wilson 峰，说明规范信号纯属统计伪影 |
| **NM-C：静态拓扑** | 不进行边删除/创建，仅传播和分解 | 检验谱结构是否来自初始图而非动力学 |

**判定**：理论机制必须产生 NM 中不存在的显著结构差异（例如，NM 中三代峰不出现或峰间距明显不同）。

## 四、观测量与分析方法（修订）

### 4.1 谱图理论分析（取代 DFS 环枚举）

- 计算图拉普拉斯本征值 \(\lambda_i\) 和本征向量 \(\psi_i\)。
- **稳定模定义**：本征值位于间隙（gap）中的模式，且参与比（inverse participation ratio）高 → 局域稳定结构。
- 统计本征值分布，寻找**分立谱簇**（三代候选）。预期在低能区（小 \(\lambda\)）出现三个分离的簇，其间距随 \(N\) 增大趋于常数（有限尺度标度检验）。

### 4.2 规范场观测：Wilson loop 分布

- 对每个长度为 \(L=3,4,5\) 的环 \(\gamma\)，计算 holonomy：
\[
W(\gamma) = \text{Tr}\left( \prod_{e\in\gamma} U_e \right)
\]
（对 \(U(1)\)，迹即乘积；对 \(SU(2)\)，取迹归一化）
- 统计所有环的 \(W\) 在复平面上的分布。若有效规范群为紧致李群，则分布应集中于 \(U(1)\) 的单位圆上的若干点（离散相位），或 \(SU(2)\) 的某类特征值集合。
- **对比零模型**：随机相位下 \(W\) 分布应均匀或接近零均值。

### 4.3 几何涌现：光锥响应函数

定义时空点 \((v,t)\)，选择一节点 \(v_0\) 作为原点，计算平均响应：
\[
G(d, t) = \left\langle \rho_v(t) \,\rho_{v_0}(0) \right\rangle_{\text{over } v: \text{dist}(v,v_0)\approx d}
\]
其中图距离 \(d\) 为最短路径长度（动态更新）。**光锥存在**意味着存在 \(v_{\max}\) 使得当 \(d > v_{\max} t\) 时 \(G\approx 0\)（在统计误差内）。通过拟合提取 \(v_{\max}\)，检验各向同性。

### 4.4 离散爱因斯坦方程

使用 **Ollivier-Ricci 曲率** 或 **Forman-Ricci 曲率** 计算每个节点的曲率 \(R_v\)。同时计算节点上的有效能量密度 \(T_v = \rho_v\)（或 \(\Phi_v\)）。检验线性关系：
\[
R_v = a + b \, T_v + \text{噪声}
\]
在涌现几何的区域，期望 \(b\) 为正，且随 \(N\) 增大收敛到常数（与引力常数相关）。

### 4.5 有限尺度标度分析

对 \(N = 500, 1000, 2000, 5000, 10000\) 分别运行稳定态（达到不动点后平均）。标度假设：
- 谱簇数量 \(n_c(N)\) 应趋于常数（3）。
- 簇间本征值间隔 \(\Delta \lambda(N)\) 应 \(\sim N^0\)（尺寸无关）。
- 若 \(n_c(N)\) 随 \(N\) 增加而增长，则三代是有限尺寸伪影。

## 五、参数扫描与相图

### 5.1 主要扫描参数

| 参数 | 范围 | 步数 | 说明 |
|------|------|------|------|
| \(\eta\) | 0.01 – 0.6 | 8 | 传播速率 |
| \(\lambda\) | 0.1 – 3.0 | 8 | 微观压缩因子 |
| 截止 \(k_c\) | 低本征模数量占 \(N\) 的 1%, 5%, 10% | 3 | 粗化尺度 |
| \(\alpha\) | 0 – 2.0 | 5 | 重连密度依赖强度 |
| \(\sigma_\theta\) | 0.01 – 0.5 | 4 | 相位噪声 |

总计约 \(8\times8\times3\times5\times4 = 3840\) 组合。每个组合运行至稳态或 5000 步，采用早停（连续 200 步各观测量变化 <1e-4）。

### 5.2 相图输出

对 \((\eta, \lambda)\) 平面（固定 \(k_c, \alpha, \sigma_\theta\) 在参考值），标记以下区域：
- **不动点区**：\(\|\mathcal{I}_{t+1}-\mathcal{I}_t\|<10^{-5}\)
- **周期/混沌区**：非收敛
- **三代谱区**：拉普拉斯本征值出现 ≥3 个可分离簇
- **几何区**：光锥涌现且爱因斯坦关系斜率显著

目标：寻找**普适区**（大范围参数下行为不变）。

## 六、对照组与理论机制的判定逻辑

每个观测信号必须经过以下检验流程：

1. **信号存在性**：在目标模型中观测到（如三峰、Wilson 峰、光锥）。
2. **零模型消融**：在 NM-A、NM-B、NM-C 中该信号显著减弱或消失。
3. **参数稳定性**：在普适区内改变参数，信号定性不变。
4. **有限尺度外推**：\(N\to\infty\) 时信号保持（数量收敛、间隔有限）。
5. **非平凡性**：信号无法简化为已知网络通用性质（如随机图必然出现的幂律）。

只有通过全部 5 项，才能声称“因果自指机制产生了物理结构”。

## 七、计算实施计划

### 阶段 0（2周）：实现与调试
- 实现带 \(U(1)\) 相位的因果图模拟器（C++/Julia）。
- 实现谱粗化（调用 ARPACK 或 Julia 的 Arpack.jl 计算拉普拉斯本征模）。
- 实现三种零模型开关。
- 单元测试：守恒律、相位传播。

### 阶段 1（3周）：小规模探索与参数筛选
- \(N=500\)，粗扫参数，剔除明显发散组合。
- 确定普适区候选（例如 \(\eta\in[0.05,0.2], \lambda\in[0.5,1.5]\)）。
- 确认零模型不产生虚假三代峰。

### 阶段 2（4周）：主扫描与相图绘制
- \(N=1000\)，全参数扫描，每组合 5 个随机种子。
- 计算相图，标记普适区。
- 存储稳定态的快照（图、相位、本征谱）。

### 阶段 3（3周）：大尺度验证与标度分析
- 在普适区固定一组参数（如 \(\eta=0.1, \lambda=1.0, k_c=5\%\)）。
- 运行 \(N=2000,5000,10000\)（10000 可能需要分布式内存）。
- 收集有限尺度标度数据：谱簇数量、间隙、Wilson 峰分布宽度。

### 阶段 4（2周）：几何与引力分析
- 计算响应函数 \(G(d,t)\)，拟合光锥速度。
- 计算 Ollivier-Ricci 曲率，拟合爱因斯坦关系。
- 检验曲率-密度关系在 \(N\) 增大时是否收敛。

### 阶段 5（2周）：报告撰写
- 包含相图、标度图、零模型对比表。
- 明确回答：是否涌现三代谱？是否涌现 \(SU(3)\times SU(2)\times U(1)\) 候选？是否涌现类光锥和爱因斯坦方程？
- 列出无法通过检验的假设，提出理论修正方向。

## 八、可能结果与理论命运

| 模拟结果 | 对理论的意义 |
|----------|----------------|
| 在三代谱区，零模型无此结构，且 \(N\to\infty\) 三代稳定 | **强支持**：三代机制合理 |
| Wilson loop 分布出现离散峰（非均匀），且零模型无 | **支持**：规范群涌现候选 |
| 响应函数显示光锥，且爱因斯坦关系成立 | **支持**：几何涌现机制成立 |
| 谱簇数量随 \(N\) 增长（如 4,5,…） | **证伪**：三代猜想不成立 |
| 任何观测信号在零模型中也出现 | **无效**：需重新设计机制 |
| 无法找到任何普适区（行为依赖参数细节） | **理论过拟合**：不可接受 |

## 九、总结

本修订版路线设计严格遵循了修正意见的核心要求：
- 用谱粗化替代 sigmoid，避免激活函数伪影；
- 用拉普拉斯本征模分析替代 DFS 环枚举；
- 加入三种零模型作为对照；
- 边权升级为 \(U(1)/SU(2)\) 相位，Wilson loop 测量 holonomy；
- 用响应函数定义光锥，用 Ollivier-Ricci 曲率检验爱因斯坦方程；
- 执行有限尺度标度分析，绘制相图，寻找普适区。

**核心哲学**：允许理论被证伪，区分“复杂系统通用自组织”与“物理结构涌现”。模拟结果将直接决定该因果信息框架是否值得进一步发展为量子引力与粒子物理的统一理论。
