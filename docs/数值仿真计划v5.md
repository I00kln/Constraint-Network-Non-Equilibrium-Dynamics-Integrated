## 一、验证目标总览

本数值验证旨在检验上述理论框架的核心可检验预言，区分已数学成立的部分与结构性猜想，并为后续的理论修正提供数据基础。验证分为三个层级：

| 层级 | 目标 | 对应理论部分 |
|------|------|--------------|
| **L1** | 离散因果图动力系统的基本行为（不动点、守恒律、粗化分解） | 公理1–4，定理1–3 |
| **L2** | 稳定路径模的自发涌现与谱结构（三代、质量比） | 猜想3–4 |
| **L3** | 有效规范对称残余与几何涌现的统计信号 | 猜想2、定理6、预测1 |

**核心问题**：在无任何预设物理定律的纯因果图演化中，是否自发出现：
- 三代稳定的粒子谱？
- 类似 \(SU(3)\times SU(2)\times U(1)\) 的对称性残余？
- 长程几何结构与洛伦兹锥？

## 二、数值模型定义

### 2.1 图结构与初始化

- **节点数**：\(N \in \{500, 1000, 2000, 5000, 10000\}\)（逐步增加以检验有限尺度效应）
- **初始图类型**：
  - 随机有向 Erdős–Rényi 图（入度/出度可调），概率 \(p = 2/N\) 确保稀疏
  - 一维环 + 随机长程边（小世界初始结构）
  - 完全空图（仅节点，无边，动力学会生成边）
- **初始信息密度**：\(\rho_v^0 = C / N\)（均匀分布），\(C = 1\)（归一化）
- **边权初始化**：\(w_{uv} = 1\)（若边存在）

### 2.2 动力学期望参数范围

| 参数 | 符号 | 初始范围 | 说明 |
|------|------|----------|------|
| 传播速率 | \(\eta\) | 0.05 – 0.5 | 稳定性上限 \(\eta < 1/(\max\text{入度})\) |
| 微观压缩因子 | \(\lambda\) | 0.1 – 2.0 | 公理4中的权重 |
| 激活函数斜率 | \(\sigma\) | sigmoid：\(\sigma(x)=1/(1+e^{-kx})\)，\(k=1,5,10\) | 宏观/微观分解 |
| 边重连系数 | \(\alpha, \beta\) | \(\alpha=0.5, \beta=0.1\) | 重连概率公式 |
| 作用量耦合 | \(\gamma\)（可忽略） | 暂不主动调整 | 用于后处理分析 |

### 2.3 更新规则（离散时间，每步为一次全局更新）

**步骤 A – 因果传播**  
\[
\rho_v^{t+1/3} = (1-\eta)\rho_v^t + \eta \sum_{u \in \text{Pa}(v)} \rho_u^t \cdot w_{uv}
\]  
边界处理：无入边节点保留原值。

**步骤 B – 宏微观分解**  
计算邻居集 \(\mathcal{N}(v)\)（包括入边与出边），局部均值 \(\bar{\rho}_v = \frac{1}{|\mathcal{N}(v)|}\sum_{u\in\mathcal{N}(v)}\rho_u^{t+1/3}\)。  
\[
\rho_v^{\text{macro}} = \sigma(\rho_v^{t+1/3} - \bar{\rho}_v),\quad
\rho_v^{\text{micro}} = \rho_v^{t+1/3} - \rho_v^{\text{macro}}
\]  
使用同一 \(\sigma\) 函数。

**步骤 C – 总量重分配**  
计算当前宏观和 \(M = \sum_v \rho_v^{\text{macro}}\)，微观和 \(m = \sum_v \rho_v^{\text{micro}}\)。  
若 \(M + \lambda m \neq C\)，则缩放所有微观值：  
\[
\rho_v^{\text{micro}} \leftarrow \rho_v^{\text{micro}} \times \frac{C - M}{\lambda m}
\]  
（若 \(m=0\)，则均匀增加微小噪声）

**步骤 D – 拓扑重连**  
逐条边计算权重衰减：\(w_{uv}^{t+1} = \max(0, w_{uv}^t - \delta)\)，\(\delta=0.01\)。  
删除 \(w_{uv} < \epsilon_{\text{del}}=0.01\) 的边。  
对每对节点 \((u,v)\)（无现有边者），以概率  
\[
P_{\text{new}}(u,v) = \text{softmax}\left( \alpha \rho_u^{\text{macro}} \rho_v^{\text{macro}} + \beta |w_{uv}^{\text{avg}} - \bar{w}| \right)
\]  
创建新边，初始权 \(w_{uv}=1\)。为控制计算量，每步最多添加 \(0.1N\) 条新边。

**步骤 E – 宏观场的低通滤波**（每 \(T_{\text{filter}}=10\) 步执行一次）  
计算 \(\Phi_v = \rho_v^{\text{macro}}\)，对图进行局部平均（步长2）得到粗化版本，用于诊断尺度行为。

### 2.4 守恒量监测

每步计算：
- 总信息量 \(\sum \rho_v\)（应恒为 \(C\)）
- 信息泛函 \(\mathcal{I}[\rho]\)（定义3），验证漂移 \(<10^{-6}\) 每步

## 三、观测量与分析工具

### 3.1 图统计量（每步存储，后期时均）

| 观测项 | 计算方法 | 预期信号 |
|--------|----------|----------|
| 节点信息熵 | \(H = -\sum (\rho_v/\text{sum}) \log(\rho_v/\text{sum})\) | 收敛到稳定值 |
| 度分布 | 入度/出度分布 | 出现幂律或双峰？ |
| 闭环统计 | 枚举长度 \(L=3,4,5\) 环的数量 | 稳定闭环数上升 |
| 路径作用量分布 | 对每个节点计算最短返回路径的 \(S[\gamma]\) | 形成分立峰（三代候选） |

### 3.2 谱分析（粒子模识别）

**目标**：识别“稳定路径模”——即图中具有极小作用量 \(S[\gamma] = \oint (\rho \cdot d\ell)\) 的闭合路径类。

**方法**：
1. 对每个节点 \(v\)，运行受限DFS（最大深度 \(L_{\max}=20\)），枚举所有简单环。
2. 计算每个环的归一化作用量 \(S = \sum_{e\in\gamma} (\rho_u + \rho_v)/2\)（边权待定）。
3. 聚类：按作用量 \(S\) 与环的拓扑（长度、转向比）分组。
4. 统计各组在稳定状态下的出现频率。预期出现 **3 个分离的簇**（三代），高阶簇指数衰减。

### 3.3 对称性残余检测（规范群候选）

由于无法直接观测连续李群，我们检测**低能有效理论中对称性留下的不变量统计**。

**检测方法**：
- 定义“规范不变量”候选：每个节点的局部 \(U(n)\) 相位可从邻接矩阵的循环流获取。计算**Wilson loop**统计：对每个小环，计算 \(\text{Tr}(\prod_e w_e)\) 的分布。
- 若出现近似 \(SU(3)\) 信号，则三个颜色的闭环（长度为3的环）上的乘积应具有一致模长。
- 统计各长度环上的相位累积分布，寻找紧致李群特征的尖峰。

**定性判断标准**：环积分的模长分布出现**分离的窄峰**，而非均匀分布，则表明有效对称性涌现。

### 3.4 几何涌现诊断

- **光锥检验**：选取高信息密度区域，计算因果方向上的“最快传播路径”速度。若速度趋近于常数（各向同性），则洛伦兹对称性自发产生。
- **曲率估计**：对每个节点，统计邻居的 \({\rho}\) 二阶差，计算离散Ricci曲率（Forman–Ollivier），观察是否形成平滑变化。

### 3.5 质量比计算（从作用量间接得）

在稳定态，对每个路径模类 \(\gamma^{(n)}\)，计算平均作用量 \(\langle S_n \rangle\)。预测质量比：
\[
\frac{m_n}{m_0} = \exp\left( \alpha (\langle S_n \rangle - \langle S_0 \rangle) \right)
\]
其中 \(\alpha\) 由第一代和第二代比值反推（若二代出现）。检验是否 \(m_1 : m_2 : m_3 \approx 1 : e^{a} : e^{2a}\)。

## 五、预期结果与判定标准

| 预言 | 判定标准 | 如果失败 |
|------|----------|----------|
| **不动点存在** | 信息泛函变化率 \(<10^{-5}\) / 步 | 调整参数或检查守恒律实现错误 |
| **三代稳定路径模** | 出现至少3个分离的 \(S\) 峰值，且第4峰峰值 < 第3峰的10% | 若仅2代或1代：理论需修正；若第4代明显：猜想3被证伪 |
| **质量比指数规律** | \(\log(m_i/m_{i-1})\) 近似常数 | 若无规律：候选机制4不成立 |
| **规范残余信号** | Wilson环模长分布有 >2 个窄峰 | 若无：猜想2不成立或尺度不足 |
| **光锥涌现** | 最大传播速度在空间方向上各向异性 < 10% | 连续极限假设可能无效 |
| **引力弱化标度** | 有效耦合 \(\kappa_{\text{eff}} \propto 1/\sqrt{N}\) 在 \(N\) 变化时成立 | 定理10需重审 |

## 七、风险与应对

| 风险 | 概率 | 应对 |
|------|------|------|
| 系统不收敛，持续震荡 | 中 | 调整 \(\eta\) 更小，或加入自适应扩散项 |
| 三代谱不出现 | 高（候选机制未证实） | 如实报告，尝试修改分解函数（如改变 \(\sigma\) 形式） |
| 计算开销过大（N=10000） | 中 | 采用采样重连而非全对，以及稀疏图优化 |
| 对称残余信号太弱 | 高 | 增加拓扑重连的对称性偏好（作为对照实验） |
