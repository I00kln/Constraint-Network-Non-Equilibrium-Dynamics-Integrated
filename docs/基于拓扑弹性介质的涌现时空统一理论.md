# 基于拓扑弹性介质的涌现时空统一理论  v1.2

## 1. 基本场与涌现度规

设 \(M\) 为四维光滑微分流形，不预设任何度规结构。引入基本场

\[
\varphi^{A}: M \longrightarrow \mathbb{R}^{1,5}, \qquad A=0,1,\dots ,5,
\]
\[
n^{a}: M \longrightarrow S^{2}\hookrightarrow \mathbb{R}^{3}, \qquad a=1,2,3,\quad \delta_{ab}n^{a}n^{b}=1 .
\]

目标空间 \(\mathbb{R}^{1,5}\) 配以平直度规 \(\eta_{AB}= \operatorname{diag}(-1,1,1,1,1,1)\)，\(\mathbb{R}^{3}\) 配以标准欧几里得度规 \(\delta_{ab}\)。

**定义 1.1（涌现度规）** 时空度规张量 \(g_{\mu\nu}\) 由嵌入场 \(\varphi^{A}\) 的拉回定义：

\[
\boxed{g_{\mu\nu} := \eta_{AB}\,\partial_{\mu}\varphi^{A}\,\partial_{\nu}\varphi^{B}} \tag{1}
\]

**命题 1.1（号差条件）** 若映射 \(\varphi: M\to \mathbb{R}^{1,5}\) 的切映射 \(d\varphi\) 在每点秩为 \(4\)，则 \(g_{\mu\nu}\) 非退化且具有恒定洛伦兹号差 \((-,+,+,+)\)。

*证明*：对任意非零切向量 \(v^{\mu}\in T_{x}M\)，
\[
g_{\mu\nu}v^{\mu}v^{\nu}= \eta_{AB}(v^{\mu}\partial_{\mu}\varphi^{A})(v^{\nu}\partial_{\nu}\varphi^{B}) .
\]
因 \(d\varphi\) 为单射，向量 \(V^{A}=v^{\mu}\partial_{\mu}\varphi^{A}\) 非零，故二次型非退化。号差恒定性由 \(M\) 的连通性及 \(d\varphi\) 的连续性保证。 \(\square\)

余维数 \(2\)（目标空间为 \(6\) 维）是平衡点：过低则过度约束允许的度规空间，过高则引入非物理自由度。关于嵌入的充分性分析见第 2.4 节。

## 2. 真空构型、对称性破缺与嵌入自由度

### 2.1 真空构型

**定义 2.1（真空背景）** 选取如下场位形为真空解：

\[
\langle\varphi^{\mu}\rangle = x^{\mu}\quad (\mu=0,1,2,3),\qquad
\langle\varphi^{4}\rangle = 0,\qquad
\langle\varphi^{5}\rangle = 0, \tag{2a}
\]
\[
\langle n^{a}\rangle = (0,0,1). \tag{2b}
\]

在此背景上，涌现度规 (1) 退化为闵可夫斯基度规 \(\eta_{\mu\nu}\)。场 \(n^{a}\) 的真空方向自发破缺 \(SO(3)\to SO(2)\cong U(1)\).

### 2.2 全局对称性与 Goldstone 模式分析

基本作用量（第 3 节）的全局对称性群为

\[
G_{\text{global}} = T^{4} \ltimes SO(1,5) \times SO(3)_{\text{internal}},
\]

其中 \(T^{4}\) 为 \(\varphi^{\mu}\) 切向平移对称性（平直目标空间特有）。

**平移对称性破缺**：真空 (2a) 自发破缺 \(T^{4}\)，产生 4 个 Goldstone 标量。微分同胚不变性 \(\text{Diff}(M)\) 作为规范对称性将这些模式吸收：在规范 \(\varphi^{\mu}(x)=x^{\mu}\) 下它们被消除。

**内部转动破缺**：\(SO(3)_{\text{internal}}\) 由 (2b) 破缺至 \(SO(2)\)，产生 2 个 Goldstone 模式。若作用量不含 \(SO(3)\) 显式破缺项，它们无质量；引入势能 \(V(n)\) 后获得质量（第 8 节）。

**法向场**：\(\varphi^{4},\varphi^{5}\) 方向不具有平移对称性（势能 \(V_{\perp}\) 的存在），天生具有质量，不产生 Goldstone 模式。

### 2.3 度规涨落的线性分析

将 \(\varphi^{A}\) 分解为真空背景与涨落：

\[
\varphi^{\mu}(x) = x^{\mu} + \pi^{\mu}(x), \qquad
\varphi^{i}(x) = \phi^{i}(x), \quad i\in\{4,5\}.
\]

代入 (1) 得

\[
g_{\mu\nu} = \eta_{\mu\nu} + \partial_{\mu}\pi_{\nu} + \partial_{\nu}\pi_{\mu}
            + \eta_{\alpha\beta}\partial_{\mu}\pi^{\alpha}\partial_{\nu}\pi^{\beta}
            + \partial_{\mu}\phi^{i}\partial_{\nu}\phi^{i}. \tag{3}
\]

在高斯法向规范 \(\pi^{\mu}=0\) 下

\[
g_{\mu\nu} = \eta_{\mu\nu} + \partial_{\mu}\phi^{i}\partial_{\nu}\phi^{i}. \tag{4}
\]

**关键观察**：式 (4) 表明度规涨落是法向场梯度的**二次型**，因此线性化理论中 \(\delta g_{\mu\nu}=0\)。法向场的线性激发不产生线性度规波动；引力波作为度规的线性传播模式不能直接从 \(\phi^{i}\) 的一阶涨落读出。引力子作为量子集体激发涌现（第 4 节）。

### 2.4 Nash 嵌入定理与物理度规子空间

**定理 2.1（Nash 嵌入定理，Lorentz 版本）** 设 \((M,g)\) 为 \(m\) 维光滑 Lorentz 流形。对于 \(m=4\)，号差 \((-,+,+,+)\) 的 Lorentz 流形可局部等距嵌入 \(\mathbb{R}^{1,d}\) 的充分条件为余维数

\[
\mathrm{codim} = d+1-4 \ge \frac{4\times5}{2} - 4 = 6,
\]

即 \(d\ge 9\)，总维数 \(D\ge 10\)。此为局部嵌入的充分条件，非必要。

本框架选取目标空间 \(\mathbb{R}^{1,5}\)（总维数 \(6\)，余维数 \(2\)），远小于一般充分维数。故本理论并不声称**所有**四维 Lorentz 度规均可涌现，而是研究由该嵌入约束定义的**物理度规子空间**

\[
\mathcal{G}_{\text{phys}} = \{ g_{\mu\nu}\in \mathrm{Lor}(M) \mid \exists\, \varphi: M\to \mathbb{R}^{1,5} \text{ 使得 } g_{\mu\nu} = \eta_{AB}\partial_{\mu}\varphi^{A}\partial_{\nu}\varphi^{B} \}.
\]

**命题 2.1（物理度规子空间）** \(\mathcal{G}_{\text{phys}}\) 是全空间 \(\mathcal{G}_{\text{all}}\) 的无限维真子集，包含闵可夫斯基度规及弱场扰动。在 FLRW 对称性或静态球对称等假设下足以容纳已知解。一般爱因斯坦方程解是否全部属于 \(\mathcal{G}_{\text{phys}}\) 为开放问题。该限制构成可证伪的**预言**。

在经典层面，嵌入约束使度规变分受限于 \(\mathcal{G}_{\text{phys}}\)，导致运动方程比爱因斯坦方程多出额外约束。如后文所示，量子效应可在低能解除这些约束，使广义相对论作为有效理论涌现。

## 3. 弹性膜作用量与爱因斯坦-希尔伯特项的几何等价

### 3.1 弹性膜作用量

将场 \(\varphi^{A}\) 解释为四维弹性膜在平坦 \(\mathbb{R}^{1,5}\) 背景中的嵌入坐标。膜的动力学包含张力与弯曲能。

**定义 3.1（外曲率）** 令 \(\nabla_{\mu}\) 为与涌现度规 \(g_{\mu\nu}\) 相容的 Levi-Civita 联络。完整外曲率张量

\[
K^{A}_{\mu\nu} := \nabla_{\mu}\partial_{\nu}\varphi^{A} 
= \partial_{\mu}\partial_{\nu}\varphi^{A} - \Gamma^{\rho}_{\mu\nu}\partial_{\rho}\varphi^{A}, \tag{5}
\]

其切向分量为零，法向投影给出非平凡部分。下文 \(K^{A}_{\mu\nu}\) 均指其法向投影。

**公设 3.1（基本作用量）** 弹性膜的基本作用量取为

\[
S_{\varphi} = -\tau \int_{M} d^{4}x \sqrt{-g} 
+ \frac{1}{2\kappa} \int_{M} d^{4}x \sqrt{-g} \left( K^{A}_{\mu\nu}K_{A}^{\mu\nu} - K^{A}K_{A} \right) 
+ S_{\perp}[\varphi^{\perp}], \tag{6}
\]

其中 \(K^{A}:=g^{\mu\nu}K^{A}_{\mu\nu}\)，\(\kappa = 8\pi G\)，\(\tau\) 为膜张力。法向场部分

\[
S_{\perp} = \int_{M} d^{4}x \sqrt{-g} \Big[ -\frac12 g^{\mu\nu} \delta_{ij} \partial_{\mu}\varphi^{i}\partial_{\nu}\varphi^{j} - V_{\perp}(\varphi^{\perp}) \Big], \quad i,j\in\{4,5\},
\]

势能 \(V_{\perp}\) 在 \(\varphi^{\perp}=0\) 处取极小值 \(V_{\perp}(0)=0\)，Hessian \(\partial_{i}\partial_{j}V_{\perp}|_{0} = m_{ij}^{2}\) 正定。

### 3.2 高斯方程与标量曲率等价性

**定理 3.1（高斯方程）** 设 \(\varphi: M\hookrightarrow \mathbb{R}^{1,5}\) 为等距嵌入（以涌现度规 \(g_{\mu\nu}\) 为诱导度规）。内蕴黎曼曲率张量与外曲率满足

\[
R_{\mu\nu\rho\sigma} = \eta_{AB}\left( K^{A}_{\mu\rho}K^{B}_{\nu\sigma} - K^{A}_{\mu\sigma}K^{B}_{\nu\rho} \right). \tag{7}
\]

*证明*：对平坦背景的联络应用 Ricci 恒等式至切向量 \(\partial_{\mu}\varphi^{A}\)，利用 Gauss-Weingarten 分解即得（详见附录 A）。 \(\square\)

**推论 3.1（标量曲率）** 缩并 \(g^{\mu\rho}g^{\nu\sigma}\) 得

\[
R = \eta_{AB}\left( K^{A}K^{B} - K^{A}_{\mu\nu}K^{B\mu\nu} \right)
    = K^{A}K_{A} - K^{A}_{\mu\nu}K_{A}^{\mu\nu}. \tag{8}
\]

**定理 3.2（弯曲能与爱因斯坦-希尔伯特项等价）** 弯曲能密度满足恒等式

\[
K^{A}_{\mu\nu}K_{A}^{\mu\nu} - K^{A}K_{A} = -R .
\]

因此作用量 (6) 可重写为

\[
\boxed{S_{\varphi} = -\tau \int_{M} d^{4}x \sqrt{-g} 
- \frac{1}{2\kappa} \int_{M} d^{4}x \sqrt{-g} \, R 
+ S_{\perp}[\varphi^{\perp}] + \text{边界项}.} \tag{9}
\]

该推导表明：爱因斯坦-希尔伯特项**不是独立假设**，而是弹性膜弯曲能通过高斯恒等式的等价重写。这是本理论的核心构建原则。

### 3.3 经典运动方程与嵌入约束

若将涌现度规 \(g_{\mu\nu}\) 视为由 \(\varphi^{A}\) 通过 (1) 定义的复合场，则对基本场 \(\varphi^{A}\) 的变分给出经典运动方程。将方程投影到切向与法向：

- **切向方程**等价于协变守恒律 \(\nabla_{\mu}T^{\mu\nu}=0\)。
- **法向方程**构成对度规的额外微分约束（广义 Codazzi 方程）。

因此，经典层面度规 \(g_{\mu\nu}\) 并不满足完整的爱因斯坦方程，而是满足爱因斯坦方程**加上**嵌入约束。这些约束限制了物理度规子空间 \(\mathcal{G}_{\text{phys}}\)。

## 4. 量子涌现与引力子自由度的分析

**目标**：在低能极限下，证明涌现度规 \(g_{\mu\nu}\) 的有效动力学由爱因斯坦-希尔伯特作用量描述，并存在无质量自旋-2 激发。以下给出微扰论下的构造性论证，并明确其未严格证明的假设。

### 4.1 辅助度规与路径积分表示

引入辅助度规张量场 \(\gamma_{\mu\nu}\) 及对称张量拉格朗日乘子场 \(\lambda^{\mu\nu}\)，将基本作用量等价写为

\[
S[\varphi,\gamma,\lambda] = S_{\varphi}[\gamma] 
+ \int d^{4}x \sqrt{-\gamma}\, \lambda^{\mu\nu} \big( \gamma_{\mu\nu} - \eta_{AB}\partial_{\mu}\varphi^{A}\partial_{\nu}\varphi^{B} \big), \tag{10}
\]

其中 \(S_{\varphi}[\gamma]\) 由 (9) 式将 \(g_{\mu\nu}\) 替换为 \(\gamma_{\mu\nu}\) 得到（包含张力、曲率及法向场部分）。对 \(\lambda^{\mu\nu}\) 的泛函积分产生 \(\delta\)-泛函，强制约束

\[
\gamma_{\mu\nu} = \eta_{AB}\partial_{\mu}\varphi^{A}\partial_{\nu}\varphi^{B},
\]

从而等价于原始理论。量子配分函数为

\[
Z = \int \mathcal{D}\varphi\,\mathcal{D}n\,\mathcal{D}\gamma\,\mathcal{D}\lambda\,
e^{-S[\varphi,\gamma,\lambda] - S_{n}[n,\gamma]}.
\]

选取规范 \(\varphi^{\mu}=x^{\mu}\)（高斯法向规范），切向涨落被消除。剩余积分变量为 \(\phi^{i}\;(i=4,5)\)，\(n^{a}\)，\(\gamma_{\mu\nu}\)，\(\lambda^{\mu\nu}\)。

### 4.2 法向场积分与低能有效作用量

法向场作用量

\[
S_{\perp}[\phi,\gamma] = \int d^{4}x \sqrt{-\gamma} \Big[ -\frac12 \gamma^{\mu\nu}\delta_{ij}\partial_{\mu}\phi^{i}\partial_{\nu}\phi^{j} 
- \frac12 m_{\perp}^{2}\delta_{ij}\phi^{i}\phi^{j} - V_{\text{int}}(\phi) \Big].
\]

在低能极限 \(E\ll m_{\perp}\) 下，对 \(\phi^{i}\) 进行高斯积分（一圈近似），得到有效作用量修正：

\[
\Delta\Gamma[\gamma,\lambda] = \frac12 \operatorname{Tr}\ln\big( -\Box_{\gamma} + m_{\perp}^{2} \big) 
+ \mathcal{O}(V_{\text{int}}).
\]

利用 Schwinger 固有时间展开（热核展开），得

\[
\Delta\Gamma = \int d^{4}x \sqrt{-\gamma} \Big[ c_{0}m_{\perp}^{4} + c_{1}m_{\perp}^{2}R(\gamma) + c_{2}R(\gamma)^{2} + \cdots \Big],
\]

其中 \(c_{i}\) 为数值系数。\(c_{1}m_{\perp}^{2}R\) 项提供对引力常数 \(\kappa\) 的量子修正。

### 4.3 乘子场动力学与退耦

乘子场 \(\lambda^{\mu\nu}\) 在作用量 (10) 中与约束线性耦合。将 \(\varphi^{A}\) 的涨落消去后，\(\lambda\) 的二次有效作用量可通过计算单圈图获得。在 \(E\ll m_{\perp}\) 的低能极限下，其主导项为

\[
\Gamma_{\lambda}^{(2)} \sim \int d^{4}x \sqrt{-\gamma} \left[ \frac{1}{2} \lambda^{\mu\nu} \mathcal{O}_{\mu\nu,\rho\sigma} \lambda^{\rho\sigma} + m_{\lambda}^{2} \lambda^{\mu\nu}\lambda_{\mu\nu} + \cdots \right],
\]

其中质量标度 \(m_{\lambda} \sim m_{\perp}\)（由带有法向场内线的图生成）。因此，\(\lambda^{\mu\nu}\) 在红外具有大质量，其传播子在低能下被压低，不产生长程动力学自由度。

**工作假设 4.1（乘子退耦）** 在完整的非微扰量子理论中，乘子场 \(\lambda^{\mu\nu}\) 在低能极限下完全退耦，不产生零质量模式，且不改变度规场的低能有效自由度。该假设已有单圈微扰论支持，但其非微扰严格性仍为开放问题。

### 4.4 低能度规有效作用量与引力子

在积分掉 \(\phi^{i}\) 和 \(\lambda^{\mu\nu}\) 后，低能有效作用量仅为 \(\gamma_{\mu\nu}\) 的泛函。在单圈近似及工作假设 4.1 下，得到

\[
\Gamma_{\text{eff}}[\gamma] = -\frac{1}{2\kappa_{\text{ren}}} \int d^{4}x \sqrt{-\gamma}\, R(\gamma) 
+ 2\Lambda_{\text{ren}} + \mathcal{O}\big( \tfrac{R^{2}}{m_{\perp}^{2}}, \tfrac{\nabla^{4}}{m_{\perp}^{4}} \big), \tag{11}
\]

其中 \(\kappa_{\text{ren}}^{-1} = \kappa^{-1} + 2c_{1}m_{\perp}^{2}\)，\(\Lambda_{\text{ren}}\) 为重整化宇宙学常数。

**命题 4.1（引力子涌现，微扰论）** 在有效作用量 (11) 中，以平坦背景 \(\gamma_{\mu\nu} = \eta_{\mu\nu} + h_{\mu\nu}\) 展开至二阶，在谐和规范下得到 Pauli-Fierz 作用量

\[
\Gamma^{(2)}[h] = \frac{1}{4\kappa_{\text{ren}}} \int d^{4}x \Big[ 
-\frac12 \partial_{\rho}h_{\mu\nu}\partial^{\rho}h^{\mu\nu} 
+ \frac12 \partial_{\rho}h \partial^{\rho}h \Big].
\]

动量空间传播子为

\[
\Delta_{\mu\nu,\rho\sigma}(k) = \frac{1}{k^{2} - i\varepsilon}
\left( \eta_{\mu\rho}\eta_{\nu\sigma} + \eta_{\mu\sigma}\eta_{\nu\rho} - \eta_{\mu\nu}\eta_{\rho\sigma} \right)
\frac{\kappa_{\text{ren}}}{2}. \tag{12}
\]

该极点结构对应无质量自旋-2 粒子。因此，在单圈近似及工作假设 4.1 成立的条件下，**引力子作为涌现自由度存在**。

*证明概要*：上述步骤 4.2–4.4 提供了构造性推导。完整严格性依赖于乘子场退耦的非微扰证明，目前列为工作假设。 \(\square\)

**注**：引力子不是基本场 \(\varphi^{A}\) 的经典线性激发，而是有效度规场的低能集体激发。经典嵌入约束在量子层面因大质量乘子场退耦而被解除，使得低能度规获得完整的广义协变动力学。

### 4.5 与 Weinberg-Witten 定理的一致性分析

Weinberg-Witten 定理指出：在具有洛伦兹协变守恒能量-动量张量的量子场论中，不能存在带有非零能量-动量的无质量自旋-2 复合粒子。本理论在该定理下的情况分析如下：

- 基本场 \(\varphi^{A}, n^{a}\) 定义在平直目标空间上，具有全局 \(SO(1,5)\) 对称性，存在守恒的能量-动量张量 \(T^{\mu\nu}_{\text{fund}}\)。
- 然而，涌现引力子是有效度规场 \(\gamma_{\mu\nu}\) 的量子涨落，其低能有效理论具有微分同胚不变性。
- 在广义相对论中，引力场的能量-动量无法定义为协变守恒的局域张量（爱因斯坦赝张量依赖坐标）。因此有效理论**不满足** Weinberg-Witten 定理的前提，该定理不构成直接约束。
- **开放问题**：需严格证明底层 Hilbert 空间中不存在包含引力子的表示，以完成定理条件的规避。当前论证为合理分析，未达严格证明标准。

## 5. 时空密度、因果结构与统计熵

### 5.1 时间与空间密度

**定义 5.1（时间密度）** 局部时间密度标量

\[
\rho_{t}(x) := \sqrt{-g_{00}(x)} = \sqrt{-\eta_{AB}\partial_{0}\varphi^{A}\partial_{0}\varphi^{B}}.
\]

**定义 5.2（空间密度）** 令 \(\gamma_{ij} := g_{ij}\)（\(i,j=1,2,3\)）为诱导空间度规，空间体积密度为 \(\sqrt{\det\gamma}\).

### 5.2 路径测度与因果结构

**定义 5.3（因果路径）** 路径 \(\Gamma: [0,1]\to M\) 称为因果的，若其切向量处处类时或类光：\(g_{\mu\nu}\dot{\gamma}^{\mu}\dot{\gamma}^{\nu}\le 0\)。从 \(a\) 到 \(b\) 的因果路径集记为 \(\Gamma(a,b)\).

**定义 5.4（路径积分测度）** 对泛函 \(F[\gamma]\) 定义加权路径和

\[
\langle F \rangle_{a\to b} = \int_{\Gamma(a,b)} \mathcal{D}\gamma\, F[\gamma]\, e^{-S_{\text{path}}[\gamma]},
\]

其中 \(S_{\text{path}}[\gamma]\) 为固有时间泛函，测度由涌现度规诱导。

## 14. 与 Weinberg-Witten 定理的一致性（论证）

Weinberg-Witten 定理（1980）断言：在具有洛伦兹协变守恒能量-动量张量 \(T^{\mu\nu}\) 的量子场论中，不能存在带有非零 \(T^{\mu\nu}\) 的无质量自旋-2 粒子。本理论与该定理的关系如下：

1. 基本场 \(\varphi^{A}, n^{a}\) 定义在平坦目标空间，具有守恒的能量-动量张量，但它们不包含无质量自旋-2 态。
2. 涌现引力子是有效度规场 \(\gamma_{\mu\nu}\) 的量子涨落。其低能有效作用量为爱因斯坦-希尔伯特作用量，具有微分同胚不变性。
3. 在广义相对论中，引力场的能量-动量无法定义为一个协变守恒的局域张量（爱因斯坦赝张量依赖于坐标）。因此，有效理论**不满足** Weinberg-Witten 定理的前提（不存在全局洛伦兹协变守恒的 \(T^{\mu\nu}\) 包含引力子）。
4. 定理的条件在本模型的低能有效理论中被微分同胚不变性规避，故不构成约束。严格规避证明尚需底层 Hilbert 空间的进一步分析，当前为合理自洽论证。

**结论**：Weinberg-Witten 定理不禁止本模型，反而与引力作为规范相互作用涌现的图景自洽。
