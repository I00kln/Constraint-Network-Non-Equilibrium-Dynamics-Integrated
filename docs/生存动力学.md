# 生存动力学：基于 Clifford 环面的弹性正规双曲吸引结构、谱分析与接触 Hamilton 形式化

**摘要**  
将一类非线性系统的“生存”概念严格化为相空间中**法向吸引、切向中性的正规双曲不变流形**。对复二维二阶耗散‑陀螺系统，构造全局吸引的不变环面——给定半径的 Clifford 环面 \(\mathcal{M}\cong\mathbb{T}^2\)。在该环面上，无耗散的流构成 Hopf 纤维化在 \(\mathcal{M}\) 上的限制：一族相互链环数为 \(1\) 的周期轨道。引入速度误差与陀螺恢复项后，生存环面成为相空间中的**吸引型正规双曲不变流形**，其上承载共振的周期流。利用严格 Lyapunov–LaSalle 方法证明从几乎所有初值出发的轨线均以阻尼振荡方式收敛到该环面；通过 Fenichel 理论建立该结构在 \(C^1\) 扰动下的持久性，并证明其上的稳定叶状结构。在线性化层面给出法向谱与切向零模的完整分解，将状态偏差诠释为可恢复的激发态。进一步，在三个方向上推进谱几何动力学：(i) 对弱对称破缺扰动导出有效相位方程；(ii) 将模型形式推广到空间延伸场，引入波动算子，在严格性边界内讨论拓扑缺陷与拟谱；(iii) 在旋转坐标系下揭示系统具有严格的接触 Hamilton 结构，耗散编码于 Reeb 场。全文严格区分定理、命题与猜想，给出完整证明。

---

## 1. 引言

经典非线性动力学往往追求轨线向孤立平衡点或极限环的收敛。这种刚性稳定在扰动超出吸引域时会导致不可逆失稳。许多复杂适应系统展现出另一种稳定模式：状态可暂时偏离某种“相容关系”，但通过内部弹性恢复力与耗散，轨线最终回归到由相容状态构成的不变流形。本文将这一原则提炼为一个自洽的数学框架——**弹性生存动力学**。

状态由一对复振子 \((z_1,z_2)\in\mathbb{C}^2\) 描述。给定正数 \(r_1,r_2>0\)，生存状态的集合为 **Clifford 环面**
\[
\mathcal{M}:=\{(z_1,z_2)\in\mathbb{C}^2 : |z_1|=r_1,\;|z_2|=r_2\}.
\]
系统不依赖外部控制，而是通过陀螺旋转与弹性势构建内部恢复力，使 \(\mathcal{M}\) 成为相空间中的吸引型正规双曲不变流形，其上内禀流恰为 Hopf 流在 \(\mathcal{M}\) 上的限制——共振情形下的一族周期轨道。引入速度误差后直接构造耗散‑陀螺系统，使得任何偏离均激发法向恢复模，并通过阻尼耗散能量，最终回归到环面上的某条轨道。该系统在连续对称群 \(U(1)\times U(1)\) 下存在切向零模，构成可存储相位信息的中性自由度。

本文主要贡献包括：
1. 严格分析 Clifford 环面上的受限 Hopf 纤维结构，计算其链环数；
2. 证明原点为双曲鞍点，并准确给出稳定流形与不稳定流形的维数；
3. 完整刻画除原点外两类部分退化不变集的稳定流形，确定几乎全局吸引性的零测排除集；
4. 给出线性化谱的严格分解，证明切向中性模是孤立零特征值，法向谱具有严格负实部；
5. 在上述正规双曲结构上建立稳定叶状结构，描绘吸引域的纤维化几何；
6. 在三个方向推进：导出不变环面上的有效相位方程；形式推广至空间延伸场并引入波动算子；建立旋转坐标系下的接触 Hamilton 等价性，并严格区分猜想与已证明结果。

## 2. 弹性生存系统的基本数学构造

### 2.1 状态空间与生存环面

位形空间取 \(\mathbb{C}^2\cong\mathbb{R}^4\)，赋予标准 Hermite 内积。给定参数 \(r_1,r_2>0\)，定义位形空间中的 **Clifford 环面**
\[
\mathcal{M}:= \{ (z_1,z_2)\in\mathbb{C}^2 : |z_1|=r_1,\ |z_2|=r_2 \}.
\]
\(\mathcal{M}\) 微分同胚于二维环面 \(\mathbb{T}^2\)，且可光滑嵌入三维球面
\[
S^3_R:=\{(z_1,z_2)\in\mathbb{C}^2 : |z_1|^2+|z_2|^2=R^2\},\quad R^2=r_1^2+r_2^2.
\]

相空间为切丛 \(T\mathbb{C}^2\cong \mathbb{C}^2\times\mathbb{C}^2\)，坐标为 \((z_1,z_2,\dot z_1,\dot z_2)\)。我们将在其中构造一个包含生存环面且与速度约束相容的不变子流形。

### 2.2 弹性势能

定义实解析函数 \(V:\mathbb{C}^2\to\mathbb{R}_{\ge 0}\)，
\[
\boxed{V(z_1,z_2) := \frac{1}{2}\sum_{k=1}^{2} \big(|z_{k}|^{2} - r_{k}^{2}\big)^{2}}. \tag{1}
\]
该势能具有性质：
- \(V(z)=0 \iff z\in\mathcal{M}\);
- \(V\) 是强制的：\(\|z\|\to\infty \Rightarrow V(z)\to\infty\);
- 在 \(\mathcal{M}\) 上，\(V\) 取得全局极小值 \(0\)，且法向 Hessian 非退化：对 \(k=1,2\)，\(\nabla_{z_k}V = 2(|z_k|^2-r_k^2)z_k\)，在 \(|z_k|=r_k\) 处其二阶法向导数为 \(4r_k^2>0\).

### 2.3 耗散‑陀螺动力学

设目标旋转角频率为 \(\omega>0\)。对每个 \(k=1,2\)，定义**速度误差**
\[
v_k := \dot z_k - i\omega z_k,
\]
该误差衡量实际速度相对于刚体旋转的偏离。系统动力学取为
\[
\boxed{\begin{aligned}
\dot z_k &= i\omega z_k + v_k,\\
\dot v_k &= -\gamma v_k - \nabla_{z_k} V(z),
\end{aligned}} \tag{2}
\]
其中 \(\gamma>0\) 为阻尼系数，\(\nabla_{z_k}V\) 是关于标准实内积的梯度（Wirtinger 微积分给出 \(\nabla_{z_k}V = 2\frac{\partial V}{\partial \bar z_k}\)）。方程组 (2) 是自治的：若 \(z\) 位于 \(\mathcal{M}\) 且以角频率 \(\omega\) 匀速旋转，则 \(v_k=0\)，\(\nabla V=0\)，系统保持该运动；否则弹性恢复力与阻尼共同驱使轨线回归。

消去 \(v_k\) 可得显式二阶形式：
\[
\ddot z_k + (\gamma - i\omega)\dot z_k - i\omega\gamma z_k + 2(|z_k|^2 - r_k^2)z_k = 0. \tag{3}
\]

### 2.4 能量函数与耗散律

定义系统的**机械能**（动能与弹性势能之和）：
\[
E(z,v) := \frac12\sum_{k=1}^{2} |v_k|^2 + V(z). \tag{4}
\]

**定理1** (能量单调衰减)  
沿 (2) 的任意解，有
\[
\frac{dE}{dt} = -\gamma\sum_{k=1}^{2} |v_k|^2 \le 0. \tag{5}
\]
*证明*：计算
\[
\dot E = \sum_{k} \operatorname{Re}(\bar v_k \dot v_k) + \sum_{k}\Big( \frac{\partial V}{\partial z_k}\dot z_k + \frac{\partial V}{\partial \bar z_k}\dot{\bar z}_k \Big).
\]
代入 \(\dot v_k = -\gamma v_k - \nabla_{z_k}V\) 及 \(\dot z_k = i\omega z_k + v_k\)，得
\[
\begin{aligned}
\dot E &= \sum_{k} \operatorname{Re}\!\big[\bar v_k(-\gamma v_k - \nabla_{z_k}V)\big] + \operatorname{Re}\!\sum_{k} \overline{\nabla_{z_k}V}\,(i\omega z_k+v_k)\\
&= -\gamma\sum_{k} |v_k|^2 - \operatorname{Re}\!\sum_{k} \bar v_k\nabla_{z_k}V + \operatorname{Re}\!\sum_{k} \overline{\nabla_{z_k}V}\,i\omega z_k + \operatorname{Re}\!\sum_{k} \overline{\nabla_{z_k}V}\,v_k.
\end{aligned}
\]
因 \(\operatorname{Re}(\bar v_k \nabla_{z_k}V)=\operatorname{Re}(\overline{\nabla_{z_k}V}v_k)\)，第二项与第四项相消。剩余项为 \(\operatorname{Re}\sum_{k} \overline{\nabla_{z_k}V}\, i\omega z_k\)。代入 \(\nabla_{z_k}V = 2(|z_k|^2-r_k^2)z_k\)，有
\[
\overline{\nabla_{z_k}V}\, i\omega z_k = 2(|z_k|^2-r_k^2)\bar z_k\cdot i\omega z_k = 2i\omega (|z_k|^2-r_k^2)|z_k|^2,
\]
其实部为零。故 \(\dot E = -\gamma\sum_{k} |v_k|^2\)。∎

**引理1** (全局存在性与预紧性)  
对任意初值，系统 (2) 的解对所有 \(t\ge 0\) 存在且一致有界。进一步，正轨线是预紧的。
*证明*：由于 \(\dot E\le 0\)，有 \(E(t)\le E(0)\)。\(E\) 是强制的，故子水平集 \(\{E\le E(0)\}\) 为 \(\mathbb{R}^8\) 中的紧集。状态 \((z,v)\) 始终停留其中，因此轨线有界且不会在有限时间逃逸至无穷，从而解全局存在。紧子水平集的有界闭性还保证了轨线的预紧性。∎

## 3. 收敛性：生存环面的几乎全局吸引性

### 3.1 不变集与最大不变集

在相空间 \(\mathbb{C}^2\times\mathbb{C}^2\) 中定义**生存相空间流形**
\[
\mathcal{S}:= \{ (z,v) \mid |z_k|=r_k,\; v_k=0,\ k=1,2\}.
\]
\(\mathcal{S}\) 微分同胚于 \(\mathbb{T}^2\)，且在系统 (2) 下为不变集，其上的动力学约化为 \(\dot z_k = i\omega z_k\)，\(\dot v_k=0\)，即常角速度旋转。

原点 \(z=0,v=0\) 是一个平衡点。此外还存在部分零化的不变圆：
\[
\begin{aligned}
\Gamma_{1} &:= \{ (z,v) \mid z_{1}=0,\ |z_{2}|=r_{2},\ v=0 \},\\
\Gamma_{2} &:= \{ (z,v) \mid |z_{1}|=r_{1},\ z_{2}=0,\ v=0 \}.
\end{aligned}
\]
这些集合在流下不变，其上动力学为相应非零分量的匀速旋转。由 \(\dot E=0\) 的条件及梯度为零，可知 LaSalle 意义下的最大不变集为
\[
\mathcal{I} = \mathcal{S} \cup \Gamma_1 \cup \Gamma_2 \cup \{0\}. \tag{6}
\]

### 3.2 原点的双曲不稳定性

**定理2** (原点的双曲鞍点结构)  
设 \(r_k>0\)，\(\gamma>0\)，\(\omega\in\mathbb{R}\)。则原点为系统 (2) 的双曲平衡点。其稳定流形 \(W^s(0)\) 实维数为 \(4\)，不稳定流形实维数亦为 \(4\)，从而 \(W^s(0)\) 在相空间 \(\mathbb{R}^8\) 中为 Lebesgue 零测集。

*证明*：在原点附近，\(\nabla_{z_k}V = 2(|z_k|^2-r_k^2)z_k = -2r_k^2 z_k + \mathcal{O}(|z|^2)\)。线性化系统为
\[
\dot z_k = i\omega z_k + v_k,\qquad \dot v_k = -\gamma v_k + 2r_k^2 z_k.
\]
作旋转坐标变换 \(u_k = z_k e^{-i\omega t}\)，\(w_k = v_k e^{-i\omega t}\)，得自治线性系统
\[
\dot u_k = w_k,\qquad \dot w_k = -(\gamma + i\omega) w_k + 2r_k^2 u_k. \tag{7}
\]
将 \((u_k,w_k)\) 视为复变量，其系数矩阵为
\[
A_{\mathbb{C}} = \begin{pmatrix} 0 & 1 \\ 2r_k^2 & -(\gamma+i\omega) \end{pmatrix}.
\]
特征方程为 \(\lambda^2 + (\gamma+i\omega)\lambda - 2r_k^2 = 0\)。两根为
\[
\lambda_{\pm} = \frac{-(\gamma+i\omega) \pm \sqrt{(\gamma+i\omega)^2 + 8r_k^2}}{2}.
\]
记 \(\Delta = (\gamma+i\omega)^2 + 8r_k^2 = \gamma^2-\omega^2+8r_k^2 + 2i\gamma\omega\)，并令 \(\sqrt{\Delta} = a+ib\)，\(a>0\)。直接计算可知 \(a>\gamma\)（见附录），故
\[
\operatorname{Re}(\lambda_+) = \frac{-\gamma+a}{2} > 0,\qquad
\operatorname{Re}(\lambda_-) = \frac{-\gamma-a}{2} < 0.
\]
两个特征值均非纯虚数。因此对每个 \(k\)，线性化系统在实四维空间中有二维稳定子空间和二维不稳定子空间。整个系统是 \(k=1,2\) 两个独立子系统的直和，故原点线性化有实四维稳定子空间和实四维不稳定子空间。原系统 (2) 在原点处的线性化与 (7) 通过可逆线性变换联系，双曲性得以保持。因此原点为双曲鞍点，其全局稳定流形 \(W^s(0)\) 为四维浸入子流形，在 \(\mathbb{R}^8\) 中 Lebesgue 测度为零。∎

### 3.3 部分退化不变集的正规双曲性

**引理2** (\(\Gamma_1,\Gamma_2\) 的鞍型正规双曲性)  
\(\Gamma_1\)（同理 \(\Gamma_2\)）是系统 (2) 的鞍型正规双曲不变圆。具体地，在 \(\Gamma_1\) 的每一点，切空间有不变直和分解
\[
T\mathbb{R}^8|_{\Gamma_1} = T\Gamma_1 \oplus E^s \oplus E^u,
\]
其中 \(\dim T\Gamma_1=1\)，\(\dim E^s=5\)，\(\dim E^u=2\)，且法向压缩/扩张率满足正规双曲性估计。

*证明*：在 \(\Gamma_1\) 上取点 \(p=(0,z_2^0,0,0)\)，\(|z_2^0|=r_2\)。线性化系统解耦为 \((z_1,v_1)\) 与 \((z_2,v_2)\) 两个独立块。
- \((z_1,v_1)\) 部分：因为 \(z_1=0\) 处 \(\nabla_{z_1}V = -2r_1^2 z_1 + \mathcal{O}(|z_1|^2)\)，线性化与定理 2 相同，故具有二维稳定和二维不稳定子空间。
- \((z_2,v_2)\) 部分：在 \(|z_2|=r_2,\,v_2=0\) 处线性化。设 \(z_2 = r_2 e^{i\phi_2}+\delta z_2\)，\(v_2=\delta v_2\)，通过旋转坐标可得到一个三阶块（径向、速度实虚部）与一个一阶相位方向。计算三阶块的特征多项式可利用后续定理 5 的方法，用 Routh–Hurwitz 判据得出所有特征值实部为负。因此该部分提供 1 维中心方向（相位）和 3 维稳定方向，无正实部法向模式。

总切空间分解为：1 维中心方向、5 维稳定方向、2 维不稳定方向。法向收缩率与扩张率均严格分离于中心方向的零增长率，满足正规双曲性定义。故 \(\Gamma_1\) 为鞍型正规双曲不变圆。\(\Gamma_2\) 分析类似。∎

### 3.4 几乎全局吸引性

**定理3** (生存环面的几乎全局吸引性)  
存在相空间中的一个 Lebesgue 零测集 \(\mathcal{B}\)，使得对任意初值 \((z(0),v(0))\notin \mathcal{B}\)，轨线的 \(\omega\)-极限集包含于 \(\mathcal{S}\)，且
\[
\lim_{t\to\infty} \operatorname{dist}\big((z(t),v(t)),\,\mathcal{S}\big) = 0.
\]
具体地，\(\mathcal{B} = W^s(0) \cup W^s(\Gamma_1) \cup W^s(\Gamma_2)\)。

*证明*：由引理 1，任何正轨线包含于紧集，因而预紧。根据 LaSalle 不变原理，其正向极限集 \(\omega(x_0)\) 非空、紧致且包含于 \(\{\dot E=0\}\) 的最大不变集中。由 (5)，\(\dot E=0\) 当且仅当所有 \(v_k=0\)。在该不变集上，\(\dot v_k=0\) 同时要求 \(\nabla_{z_k}V=0\)，即对每个 \(k\) 有 \(|z_k|=r_k\) 或 \(z_k=0\)。因此最大不变集为 \(\mathcal{I}=\mathcal{S}\cup\Gamma_1\cup\Gamma_2\cup\{0\}\)。

若轨线的 \(\omega\)-极限集与 \(\Gamma_1,\Gamma_2\) 或 \(\{0\}\) 相交，则该轨线必位于这些集合的全局稳定流形中。定理 2 给出 \(\dim W^s(0)=4<8\)。引理 2 表明 \(\Gamma_1,\Gamma_2\) 是鞍型正规双曲不变圆，由 Fenichel 稳定流形定理，其稳定流形 \(W^s(\Gamma_i)\) 的维数为
\[
\dim \Gamma_i + \dim E^s = 1 + 5 = 6 < 8.
\]
故 \(\mathcal{B} = W^s(0)\cup W^s(\Gamma_1)\cup W^s(\Gamma_2)\) 是有限个维数至多为 6 的浸入子流形之并，在 \(\mathbb{R}^8\) 中 Lebesgue 测度为零。对任意初值 \(x_0\notin\mathcal{B}\)，其 \(\omega\)-极限集必不与 \(\{0\}\cup\Gamma_1\cup\Gamma_2\) 相交，因而完全包含于 \(\mathcal{S}\)。由 \(\mathcal{S}\) 的闭性及轨线的预紧性，可得距离收敛至零。∎

**注记1** (Morse–Conley 型层级结构)  
系统的不变集 \(\{0\}\)（双曲平衡点）、\(\Gamma_1,\Gamma_2\)（鞍型 NHIC）和 \(\mathcal{S}\)（吸引型 NHIM）形成了标准的 Morse 分解。可进一步研究其 Conley 指标与连接映射，构建完整的全局相图。

## 4. 共振 Hopf 流与拓扑链接

### 4.1 内禀周期流与受限 Hopf 纤维结构

在不变环面 \(\mathcal{S}\) 上，动力学简化为
\[
\dot z_k = i\omega z_k,\quad k=1,2,
\]
通解为 \(z_k(t)=r_k e^{i(\omega t + \phi_k)}\)。此时两自由度的角频率完全相等（共振情形），故任一轨线是周期为 \(2\pi/\omega\) 的闭曲线，可表示为
\[
t \mapsto e^{i\omega t}(r_1 e^{i\phi_1}, r_2 e^{i\phi_2}).
\]
这正是 Hopf 纤维化 \(\pi:S^3_R\to S^2\cong\mathbb{C}P^1\) 中经过点 \((r_1 e^{i\phi_1}, r_2 e^{i\phi_2})\) 的纤维在 Clifford 环面 \(\mathcal{M}\) 上的完全限制。所有轨线全体构成 Hopf 纤维化中完全包含于 \(\mathcal{M}\) 的一族圆，称为**受限 Hopf 流**。各纤维由相位差 \(\Delta\phi:=\phi_2-\phi_1 \bmod 2\pi\) 参数化。

（若两频率出现失调，环面上的运动一般变为拟周期流，此时讨论纤维的链环数失去意义，故本章的拓扑论述严格限定于完全共振情形。）

### 4.2 链环数的计算

**定义4.1** (链环数)  
设 \(\gamma_1,\gamma_2\) 为 \(S^3\) 中两条不相交的可定向闭曲线。链环数 \(\operatorname{Lk}(\gamma_1,\gamma_2)\) 定义为 \(\gamma_1\) 与以 \(\gamma_2\) 为边界的任一 Seifert 曲面 \(\Sigma_2\) 的交截数（计及定向）。此定义为整系数同痕不变量。

**定理4** (Hopf 链环)  
在 \(\mathcal{S}\) 上，任意两条不同轨线（对应 \(\Delta\phi_1\neq\Delta\phi_2\)）作为 \(S^3_R\) 中的闭曲线，其链环数为
\[
\operatorname{Lk} = \pm 1,
\]
符号取决于轨道定向。

*证明*：考虑 Hopf 纤维化 \(\pi:S^3_R\to\mathbb{C}P^1\cong S^2\)，\(\pi(z_1,z_2)=(z_1:z_2)\)。每条轨线是 \(\pi\) 的一条纤维。两不同纤维对应 \(\mathbb{C}P^1\) 上两个不同点。标准 Hopf 不变量 \(\pm 1\) 恰等于任意两条不同纤维的链环数（Milnor, Arnold–Khesin 等）。亦可通过构造一条纤维的 Seifert 曲面（取连接两点路径的 \(\pi^{-1}\) 提升）直接计算得到。∎

### 4.3 关于拓扑保护的注记

正规双曲吸引不变环面 \(\mathcal{S}\) 在 \(C^1\) 小扰动下仍持续存在（Fenichel 定理），但环面上的流一般不再保持闭轨族：共振条件极易被破坏，导致流变为 Kronecker 型拟周期流。在此通有情形下，环面上不再有良定义的闭轨链环数。因此，**Hopf 链接结构并非结构稳定**，而是一种在特殊共振参数下的精致性质。若扰动额外保留适当的对称性（例如强制保持 \(S^1\times S^1\) 对称），则闭轨族得以保存，此时链环数 \(\pm 1\) 作为同痕不变量保持不变。这一点揭示了拓扑性质对参数敏感的特征，而非通常意义下的“拓扑保护”。

## 5. 正规双曲吸引性与谱动力学

### 5.1 旋转坐标与线性化解耦

引入旋转坐标 \(u_k = z_k e^{-i\omega t}\)，并记 \(w_k := \dot u_k = v_k e^{-i\omega t}\)。则系统 (2) 化为自治系统
\[
\dot u_k = w_k,\quad \dot w_k = -(\gamma + i\omega) w_k - 2(|u_k|^2 - r_k^2)u_k. \tag{8}
\]
此时生存环面变为
\[
\mathcal{S}_u:=\{|u_k|=r_k,\; w_k=0\},
\]
其上所有点皆为平衡点。取其中一点 \(u_k^0 = r_k\)（利用旋转对称性），引入实扰动变量：
\[
u_k = r_k (1 + a_k + i b_k),\qquad w_k = r_k (x_k + i y_k),
\]
其中 \(a_k,b_k,x_k,y_k\in\mathbb{R}\) 为小量。代入 (8) 并保留线性项，得到实线性系统（两个 \(k\) 解耦）：
\[
\begin{aligned}
\dot a_k &= x_k,\\
\dot x_k &= -4r_k^2 a_k - \gamma x_k + \omega y_k,\\
\dot y_k &= -\omega x_k - \gamma y_k,\\
\dot b_k &= y_k.
\end{aligned} \tag{9}
\]
注意 \(b_k\) 不反馈至前三个方程。因此系统矩阵（变量顺序 \(a_k,x_k,y_k,b_k\)）为
\[
A_k = \begin{pmatrix}
0 & 1 & 0 & 0\\
-4r_k^2 & -\gamma & \omega & 0\\
0 & -\omega & -\gamma & 0\\
0 & 0 & 1 & 0
\end{pmatrix}.
\]
其特征多项式按最后一列展开得 \(\det(\lambda I - A_k) = \lambda \cdot \det(\lambda I_3 - B_k)\)，其中
\[
B_k = \begin{pmatrix}
0 & 1 & 0\\
-4r_k^2 & -\gamma & \omega\\
0 & -\omega & -\gamma
\end{pmatrix}.
\]

### 5.2 谱分解与正规双曲吸引性

**定理5** (线性化谱与正规双曲吸引性)  
在 \(\mathcal{S}_u\) 的每一点，系统 (8) 的线性化具有如下谱特性：
- 两个孤立零特征值，对应切向相位平移坐标 \((b_1,b_2)\)，几何与代数重数均为 2，与连续对称群 \(U(1)\times U(1)\) 相对应；
- 六个具有严格负实部的特征值，由径向与速度扰动分量 \((a_k,x_k,y_k)_{k=1,2}\) 决定，形成法向双曲收缩。

因此，\(\mathcal{S}_u\) 是一个**吸引型正规双曲不变流形** (normally hyperbolic attracting invariant manifold)，即其切丛与稳定丛满足 \(T\mathbb{R}^8|_{\mathcal{S}_u} = T\mathcal{S}_u \oplus E^s\)，且存在常数 \(C,\alpha>0\) 使得对一切 \(p\in\mathcal{S}_u\)，有
\[
\|D\phi_t|_{E^s}\| \le C e^{-\alpha t}\;(t\ge 0),\qquad
\|D\phi_t|_{T\mathcal{S}_u}\| \le C\;(t\in\mathbb{R}).
\]

*证明*：由上述解耦，仅需分析单个 \(k\) 的 \(A_k\)。零特征值已由因子 \(\lambda\) 给出，其代数重数为 1，对应特征向量 \((0,0,0,1)^{\mathsf{T}}\)，无广义零模态。两个 \(k\) 直和后得到两个孤立零特征值，张成切空间 \(T\mathcal{S}_u\)。

\(B_k\) 的特征多项式为
\[
\det(\lambda I_3 - B_k) = \lambda\big[(\lambda+\gamma)^2 + \omega^2\big] + 4r_k^2(\lambda+\gamma) = 0,
\]
展开得
\[
\lambda^3 + 2\gamma\lambda^2 + (\gamma^2+\omega^2+4r_k^2)\lambda + 4r_k^2\gamma = 0.
\]
所有系数为正，且 Routh–Hurwitz 条件：
\[
a_1=2\gamma>0,\; a_2=\gamma^2+\omega^2+4r_k^2>0,\; a_3=4r_k^2\gamma>0,
\]
\[
a_1a_2 - a_3 = 2\gamma(\gamma^2+\omega^2) + 4\gamma r_k^2 > 0.
\]
故 \(B_k\) 的所有特征值实部为负。因此，法向线性化由两个解耦的三维稳定子块构成，总稳定维数为 6，不存在零实部或正实部法向特征值。\(\mathcal{S}_u\) 是平衡点构成的二维环面，满足正规双曲吸引性的谱条件。∎

**系1** (持久性)  
对于系统 (2) 的任何 \(C^1\) 小扰动（保持耗散结构），皆存在 \(C^1\) 邻近的吸引型正规双曲不变环面，其法向收缩谱及中心维数保持不变。
*证明*：直接应用 Fenichel 正规双曲不变流形持久性定理。∎

### 5.3 谱动力学诠释

线性化谱给出了完整的**恢复–中性模**结构：
- **法向恢复谱**：特征值 \(\lambda_{\perp}\in\mathbb{C}\) 满足 \(\operatorname{Re}(\lambda_{\perp})<0\)，控制径向振幅与速度误差的演化。其实部给出恢复率，虚部给出回弹频率。这类模式对应偏离生存环面后的阻尼振荡激发态，其能量指数衰减。
- **切向中性模**：两个孤立零特征值对应联合相位平移 \((b_1,b_2)\)，构成与连续对称性相伴的**相位零模**。它们在相空间切丛上描述维持生存环面但改变相对相位的运动，对应于可储存信息的中性自由度。

因此，任意状态的偏差可分解为法向激发模与切向漂移的叠加：法向模持续衰减，切向模无耗散维持，最终轨线收敛至生存环面上的某条纤维。此即“误差=可恢复激发态”的谱论含义。

### 5.4 稳定叶状结构与吸引域几何

正规双曲不变流形理论保证了稳定丛在非线性层面可积，从而形成稳定叶状结构，给出 basin of attraction 的纤维化描述。

**定理6** (稳定叶状结构与吸引域)  
系统 (8) 在 \(\mathcal{S}_u\) 的整个吸引域内存在一个连续的叶状结构 \(\mathcal{F}^s\)，其叶片均为 \(C^1\) 浸入子流形，且
1. \(\mathcal{S}_u\) 本身是其中一个叶片（中心叶片）；
2. 每个叶片 \(W^s(p)\) 与 \(\mathcal{S}_u\) 横截相交于唯一一点 \(p\in\mathcal{S}_u\)，且叶片上的轨线以指数速率收敛于 \(p\)；
3. 叶片均同胚于 \(\mathbb{R}^6\)，吸引域可表征为
   \[
   \mathrm{Basin}(\mathcal{S}_u) = \mathbb{R}^8 \setminus \bigl( W^s(0) \cup W^s(\Gamma_1) \cup W^s(\Gamma_2) \bigr).
   \]

*证明*：由定理 5，\(\mathcal{S}_u\) 是紧致吸引型 NHIM。根据 Fenichel 的叶状结构定理，在其充分小邻域 \(U\) 内存在唯一的局部 \(C^0\) 叶状结构 \(\mathcal{F}^s_{\text{loc}}\)，叶片为 \(C^1\) 嵌入子流形，且 \(\mathcal{S}_u\) 为其中一个叶片。借助渐近相位映射（asymptotic phase），可定义连续投影 \(P:U\to\mathcal{S}_u\) 使每个 \(x\in U\) 属于 \(W^s_{\text{loc}}(P(x))\)。

对 basin 中任意点 \(x_0\)，由定理 3 知其正轨线最终进入 \(U\)。利用流映射的可逆性将局部叶片向后延拓，可覆盖整个 basin，并且不同叶片互不相交。因为 basin 内每点的极限点唯一确定其在 \(\mathcal{S}_u\) 上的渐近相位，故整体叶状结构是良定义的。每个叶片为稳定流形，微分同胚于法向稳定丛纤维 \(\mathbb{R}^6\)。三个排除的低维稳定流形各自属于其他不变集的叶状结构，不进入 \(\mathcal{S}_u\) 的 basin。∎

**注记2** (叶状结构的拓扑)  
此处的稳定叶状结构一般不是全局乘积 \(\mathcal{S}_u\times\mathbb{R}^6\)，而是以 \(\mathcal{S}_u\) 为底空间的纤维丛（叶片为纤维）。由于纤维可缩，该丛必为拓扑平凡，但未必有微分同胚的整体直积分解。这一结构细化了 basin 的几何层次。

## 6. 谱几何动力学推进

在上述严格分析的基础上，我们将理论推进至三个更深层次的方向，初步构建**谱几何动力学**框架。本章中明确区分已严格证明的命题与基于合理推测的猜想。

### 6.1 不变环面上的有效相位动力学

未扰动的切向动力学过于平凡（均匀旋转）。考虑小幅度破坏连续对称性的扰动，使相位产生非平凡耦合。设受扰系统为
\[
\dot z_k = i\omega_k z_k + v_k + \varepsilon f_k(z,\bar z),\quad
\dot v_k = -\gamma v_k - \nabla_{z_k}V(z) + \varepsilon g_k(z,\bar z,v),
\]
其中 \(\omega_k = \omega + \varepsilon \delta_k\)，\(f_k,g_k\) 光滑，\(\varepsilon\ll1\)，且扰动不破坏全局耗散结构。

**命题1** (有效相位方程的存在性)  
对充分小的 \(\varepsilon\)，系统存在一个 \(C^1\) 邻近的不变环面 \(\mathcal{S}_{\varepsilon}\cong\mathbb{T}^2\)，且其上的动力学可通过引入角坐标 \((\theta_1,\theta_2)\in\mathbb{T}^2\) 约化为二维相位方程：
\[
\dot\theta_i = \Omega_i + \varepsilon F_i(\theta_1,\theta_2;\varepsilon) + \mathcal{O}(\varepsilon^2),\quad i=1,2,
\]
其中 \(\Omega_i\) 为扰动后的平均频率，\(F_i\) 为 \(2\pi\)-周期光滑函数。

*证明思路*：由系 1，未扰 NHIM 持久，故存在 \(\varepsilon\) 小扰下的不变环面 \(\mathcal{S}_\varepsilon\)。在该环面上利用标准相位约化技术（Fenichel 平均化），将流投影到切丛并引入角变量使未扰流为线性，然后通过平均扰动项即得上述方程。详细计算将另文给出。∎

下列陈述为**猜想**：

**猜想1** (复杂相位动力学)  
适当选取扰动 \(f_k,g_k\) 与参数，并在通有非共振条件下，有效相位方程可展现出：
- 频率锁定与同步区域（Arnold 舌头）；
- 多数拟周期运动的持久性，类似于耗散 KAM 现象；
- 在强扰动下可能发生环面破裂并导致混沌相位动力学。
目前针对此类耗散‑陀螺系统的一般化耗散 KAM 定理尚待建立。

### 6.2 连续场论的形式推广与波动算子

将有限维系统推广至空间延伸连续介质。设空间域为 \(\mathbb{R}^n\) 或周期域 \(\mathbb{T}^n\)，场变量为 \(z_k(x,t)\)。势能泛函推广为
\[
\mathcal{V}[z] = \int_{\mathbb{R}^n} \Big[ \frac12\sum_{k=1}^{2} (|z_k|^2 - r_k^2)^2 + \frac{\kappa}{2}\sum_{k=1}^{2} |\nabla z_k|^2 \Big] d^{n}x,
\]
其中 \(\kappa>0\) 为刚度系数。动力学取耗散‑陀螺波方程
\[
\partial_t z_k = i\omega z_k + v_k,\quad
\partial_t v_k = -\gamma v_k - \frac{\delta\mathcal{V}}{\delta \bar z_k} + D\nabla^2 v_k,
\]
其中 \(D>0\) 为粘性扩散系数。相空间取为合适的 Sobolev 空间，例如 \(H^1\times L^2\)。

为分析恢复模的传播与散射，引入**波动算子**。在空间均匀背景解 \(\{|z_k|\equiv r_k, v_k\equiv 0\}\) 附近，令
\[
z_k = r_k e^{i\omega t}(1+\zeta_k),\quad v_k = r_k e^{i\omega t}\eta_k,
\]
并线性化，得到
\[
\partial_t \begin{pmatrix} \boldsymbol{\zeta} \\ \boldsymbol{\eta} \end{pmatrix}
= \mathcal{L} \begin{pmatrix} \boldsymbol{\zeta} \\ \boldsymbol{\eta} \end{pmatrix},
\]
其中 \(\mathcal{L}\) 是含有 Laplace 算子的矩阵微分算子。经 Fourier 变换 \(x\mapsto q\)，其符号 \(\hat{\mathcal{L}}(q)\) 的法向块特征值决定波数为 \(q\) 的恢复模的衰减率与振荡频率。形式上有
\[
\max \operatorname{Re}\sigma(\hat{\mathcal{L}}_{\perp}(q)) \le -\delta_0 < 0,\quad \forall q\in\mathbb{R}^n,
\]
其中 \(\delta_0\) 为一致耗散间隙。切向块在 \(q=0\) 处有零特征值，对应全局相位旋转对称性；当 \(q\neq0\) 时，由于色散与耗散，其实部亦为负，但可能随 \(|q|\to0\) 趋向于零，致使本质谱触及虚轴。这破坏了无穷维正规双曲性的严格谱间隙条件，但仍提示某种弱稳定叶状结构可能存在。

**猜想2** (场论中的拓扑缺陷与束缚态)  
在上述连续模型中，若允许相位缠绕，可能产生涡旋线或点缺陷。此类缺陷受拓扑保护，并与背景 Hopf 链接相互作用。在适当条件下，波动算子 \(\mathcal{L}\) 在缺陷背景上的谱可能包含局域化的稳定本征模（拓扑保护束缚态）。其严格存在性与稳定性依赖于对无穷维 NHIM 理论在非自伴算子情形的推广，目前仍为开放问题。

（以上为形式分析与推测，尚未获得严格证明。）

### 6.3 接触 Hamilton 形式化

旋转坐标系下的系统 (8) 可完全纳入接触几何框架，耗散体现为对额外变量 \(S\) 的线性依赖。

**定义6.1** (接触 Hamilton 系统)  
在扩展相空间 \(\mathcal{C}=\mathbb{R}^{4}\times\mathbb{R}^{4}\times\mathbb{R}\) 上取标准接触 1‑形式 \(\eta = dS - \mathbf{p}\cdot d\mathbf{q}\)，其中 \(\mathbf{q}\in\mathbb{R}^4,\mathbf{p}\in\mathbb{R}^4,S\in\mathbb{R}\)。Reeb 向量场为 \(R=\partial_S\)。给定接触 Hamilton 函数 \(H_c:\mathcal{C}\to\mathbb{R}\)，接触向量场 \(X_{H_c}\) 由
\[
\eta(X_{H_c}) = H_c,\qquad \iota_{X_{H_c}}d\eta = dH_c - (R H_c)\,\eta
\]
唯一确定。在局部坐标下，
\[
\dot{\mathbf{q}} = \frac{\partial H_c}{\partial \mathbf{p}},\quad
\dot{\mathbf{p}} = -\frac{\partial H_c}{\partial \mathbf{q}} - \mathbf{p}\,\frac{\partial H_c}{\partial S},\quad
\dot S = \mathbf{p}\cdot\frac{\partial H_c}{\partial \mathbf{p}} - H_c.
\]

**定理7** (旋转坐标系下的接触等价性)  
将实变量取为 \(\mathbf{q}=(\operatorname{Re}u_1,\operatorname{Im}u_1,\operatorname{Re}u_2,\operatorname{Im}u_2)\)，\(\mathbf{p}=\dot{\mathbf{q}}\)。取接触 Hamilton 函数
\[
\boxed{H_c(\mathbf{q},\mathbf{p},S) = \frac{1}{2}\|\mathbf{p}\|^2 + V(\mathbf{q}) + \gamma S}, \tag{10}
\]
其中 \(V(\mathbf{q}) = \frac12\sum_{k} (|u_k|^2 - r_k^2)^2\)。则由 \(H_c\) 生成的接触向量场在 \((\mathbf{q},\mathbf{p})\) 子空间上的投影与旋转坐标系下的阻尼振子系统 (8) 严格等价。

*证明*：直接计算得
\[
\dot{\mathbf{q}} = \mathbf{p},\qquad
\dot{\mathbf{p}} = -\nabla_{\mathbf{q}}V - \gamma \mathbf{p},\qquad
\dot S = \|\mathbf{p}\|^2 - H_c.
\]
前两个方程正是 \(\ddot{\mathbf{q}} + \gamma\dot{\mathbf{q}} + \nabla_{\mathbf{q}}V = 0\)，即 (8) 的实形式。∎

由于系统 (8) 是由原始系统 (2) 经过时变旋转变换 \(u_k = z_k e^{-i\omega t}\) 得来的，因此原始系统**在旋转坐标系下等价于一个接触 Hamilton 系统**。换言之，耗散陀螺动力学的本质是接触流形上由 \(H_c\) 生成的接触流在常数 \(S\) 切片上的投影。生存环面 \(\mathcal{S}_u\) 可解释为一族接触 Legendre 子流形（在 \(S\) 为某常数时），法向吸引则对应接触流的梯度性质。该等价性为利用接触拓扑与耗散全纯方法研究系统铺平了道路。

|\Delta| > \gamma^2+\omega^2-8r_k^2.
