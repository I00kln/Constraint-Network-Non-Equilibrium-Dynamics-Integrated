# 有限资源目标捕获的变分原理

**版本 11.1 —— 值信息对齐的几何基础 · 横截判据与解耦结构 · 一般凸值函数框架**

---

## 摘要

在有限交互预算 \(B\) 下，决策者需最大化首次命中目标集 \(\mathcal{G}\) 的概率。本文在贝叶斯自适应控制框架下建立严格变分理论。核心对象为定义在信念状态空间 \(\mathcal{B} = \mathcal{S} \times \Delta(\mathcal{P})\) 上的贝叶斯最优值函数 \(V^*\)。引入 KL 正则化后得到软贝尔曼方程，统一了贝叶斯规划、最大熵控制与信息几何上的自然梯度流。为分离探索的纯信息价值与物理状态转移效应，我们在一个解耦的参考状态设定下，严格定义了**信息价值 (VoI)** 与**预期信息增益 (EIG)**，并证明了**值信息对齐定理**：信息价值被预期信息增益的平方根从上方无条件界定，且通过精细的 Lipschitz 分析将常数改进为 \(\sqrt{1/2}\)（定理 3.1）；在信念满足先验正下界且满足**决策边界横截条件**的区域内，信息价值被预期信息增益从下方线性界定（定理 3.3）。进一步，我们将上述不等式提升为**一般凸值函数定理**（§3.0），使 VoI–EIG 关系适用于任何一族线性函数的上包络。横截判据定理（定理 3.2, 3.6）将横截条件归约为后验分布于决策边界次梯度锥上的可验证几何条件。上述结果在多步信息收集情形下借助值函数的凸性与 Lipschitz 连续性递归成立（定理 3.4 与 3.5）。所有定理给出完整证明，严格区分已证结果（\(\blacksquare\)）与开放猜想（\(\Diamond\)）。

## 第〇部分：概率空间与任务环境

### 0.1 基础概率空间

**定义 0.1（基础概率空间）**  
设 \((\Omega, \mathcal{F}, \mathbb{P})\) 为完备概率空间。状态空间 \(\mathcal{S}\) 为有限集（\(|\mathcal{S}|<\infty\)），动作空间 \(\mathcal{A}\) 为有限集（\(|\mathcal{A}|<\infty\)），目标集 \(\mathcal{G} \subset \mathcal{S}\) 非空。预算 \(B \in \mathbb{N}\) 为最大交互步数。初始状态 \(s_0 \in \mathcal{S}\setminus\mathcal{G}\)。

**定义 0.2（转移核空间）**  
真实环境由未知转移核 \(P^*\in\mathcal{P}\) 描述。本文在有限模型假设下工作：设 \(\mathcal{P}=\{P_1,\dots,P_K\}\)，\(K\in\mathbb{N}\)。该假设使信念状态空间为有限维概率单纯形 \(\Delta^{K-1}\)，从而使随后的微分与距离运算严格成立。

**定义 0.3（先验与后验）**  
先验信念 \(\mu_0 \in \Delta(\mathcal{P})\)。给定历史 \(h_t = (s_0,a_0,\dots,s_t)\)，后验信念 \(\mu_t\in\Delta(\mathcal{P})\) 由贝叶斯定理确定：
\[
\mu_t(k) = \frac{\mu_0(k)\,P_k(h_t)}{\sum_{j=1}^{K} \mu_0(j)\,P_j(h_t)},\qquad k=1,\dots,K,
\]
其中 \(P_k(h_t) = \prod_{\tau=0}^{t-1} P_k(s_{\tau+1}\mid s_\tau,a_\tau)\) 为模型 \(k\) 下的似然。

### 0.2 策略与目标泛函

**定义 0.4（策略类）**  
随机历史依赖策略 \(\pi = (\pi_0,\dots,\pi_{B-1})\)，其中 \(\pi_t: \mathcal{H}_t \to \Delta(\mathcal{A})\)。全体策略空间记为 \(\Pi\)。

**定义 0.5（命中概率）**  
在转移核 \(P\) 下，策略 \(\pi\) 的预算 \(B\) 命中概率为
\[
V_\pi^P(s_0,B) = \mathbb{E}_P^\pi\left[ \mathbb{I}\left(\bigvee_{t=1}^{B}\{S_t\in\mathcal{G}\}\right) \;\Bigg|\; S_0=s_0\right].
\]

**定义 0.6（贝叶斯期望命中概率）**  
\[
J(\pi) = \mathbb{E}_{P\sim\mu_0}\!\left[\,V_\pi^P(s_0,B)\,\right].
\]

### 0.3 解耦假设：参考状态与纯信息收集

为将探索的信息价值从物理状态转移效应中分离，我们引入一个解耦结构。设存在一个特殊的参考状态 \(s_{\mathrm{ref}}\in\mathcal{S}\setminus\mathcal{G}\) 及一族信息收集动作 \(\mathcal{A}_{\mathrm{info}}\subseteq\mathcal{A}\)，满足：

* 对任意 \(a\in\mathcal{A}_{\mathrm{info}}\) 和任意模型 \(P_k\)，若 \(s_{\mathrm{ref}}\) 非目标，则 \(P_k(s_{\mathrm{ref}}\mid s_{\mathrm{ref}},a)=1\)（自环），而观测仍可产生信息以更新信念；
* 利用阶段的价值评估均从 \(s_{\mathrm{ref}}\) 出发，采用动作集 \(\mathcal{A}_{\mathrm{exploit}}\subseteq\mathcal{A}\)。

该假设在最优实验设计、传感器调度等问题中自然成立，并使值信息对齐定理的表述与证明摆脱对物理状态转移的依赖。后续第三部分将在该假设下进行。

## 第一部分：贝叶斯自适应马尔可夫决策过程 (BAMDP)

### 1.1 信念状态与信念 MDP

**定义 1.1（信念状态）**  
信念状态 \(b_t = (s_t,\mu_t) \in \mathcal{B} := \mathcal{S}\times\Delta(\mathcal{P})\)。\(\mathcal{B}\) 为紧致可度量空间。

**定义 1.2（信念转移核）**  
对当前 \(b=(s,\mu)\) 和动作 \(a\)，下一信念状态 \(b'=(s',\mu')\) 的分布由后验预测分布和贝叶斯更新完全确定。具体地，给定 \(b=(s,\mu)\) 与 \(a\in\mathcal{A}\)，下一状态 \(s'\) 的预测分布为 \(P_\mu(s'\mid s,a) = \sum_{k=1}^{K} \mu(k) P_k(s'\mid s,a)\)；观测到 \(s'\) 后，后验 \(\mu'\) 的分量为
\[
\mu'(k) = \frac{\mu(k) P_k(s'\mid s,a)}{P_\mu(s'\mid s,a)}.
\]

**定理 1.1（信念 MDP 的马尔可夫性）**  
过程 \(\{b_t\}\) 是 \(\mathcal{B}\) 上的完美马尔可夫决策过程。\(\blacksquare\)

*证明*：由 \(b_t\) 的定义及贝叶斯更新性质，给定 \(b_t\) 与 \(a_t\)，\((s_{t+1},\mu_{t+1})\) 的条件分布不依赖于更早的历史，故 \(\{b_t\}\) 是马尔可夫过程。其转移核由定义 1.2 给出。\(\blacksquare\)

### 1.2 贝叶斯最优值函数

**定义 1.3（扩展目标集）**  
\(\mathcal{G}_{\mathcal{B}} = \{(s,\mu)\in\mathcal{B}: s\in\mathcal{G}\}\)。

**定理 1.2（有限时域 HJB 方程）**  
定义 \(V^*:\mathcal{B}\times\{0,\dots,B\}\to[0,1]\) 为
\[
V^*(b,\tau) =
\begin{cases}
1, & b\in\mathcal{G}_{\mathcal{B}},\\[4pt]
\displaystyle\max_{a\in\mathcal{A}} \mathbb{E}_{b'\mid b,a}\!\big[V^*(b',\tau-1)\big], & b\notin\mathcal{G}_{\mathcal{B}},\;\tau>0,\\[4pt]
0, & b\notin\mathcal{G}_{\mathcal{B}},\;\tau=0.
\end{cases}
\]
则：
1. 方程有唯一解；
2. \(V^*(b_0,B) = \sup_{\pi\in\Pi} J(\pi)\)；
3. 贪婪策略 \(\pi^*(a\mid b,\tau)=\arg\max_a \mathbb{E}_{b'}[V^*(b',\tau-1)]\) 是最优的。\(\blacksquare\)

*证明*：对剩余步数 \(\tau\) 归纳。\(\tau=0\) 时，在目标集外价值为零，与定义一致。假设 \(\tau-1\) 时方程给出最优值。对任意策略 \(\pi\)，第一步在 \(b\) 下选择 \(a\sim\pi_0(\cdot\mid b,\tau)\)，则
\[
J_\tau^\pi(b) = \mathbb{E}_{a\sim\pi_0}\mathbb{E}_{b'\mid b,a}[V^{\pi_{\text{rest}}}(b',\tau-1)] \le \mathbb{E}_{a\sim\pi_0}\mathbb{E}_{b'\mid b,a}[V^*(b',\tau-1)] \le \max_a \mathbb{E}_{b'\mid b,a}[V^*(b',\tau-1)].
\]
右边恰为动态规划方程右端，且贪婪策略达到该上界。对 \(\tau\) 取上确界即得最优性。唯一性由向后归纳构造直接保证。\(\blacksquare\)

## 第二部分：正则化变分泛函与软最优性

### 2.1 策略分布的 KL 正则化

**定义 2.1（轨迹测度与策略 KL）**  
给定信念转移核 \(P_{\mathcal{B}}\)，策略 \(\pi\) 诱导信念空间上的轨迹分布 \(P_{\mathcal{B}}^\pi\)。对于两个策略 \(\pi,\pi_0\)，定义
\[
D_{\mathrm{KL}}(\pi\|\pi_0) = \mathbb{E}_{b_{0:B-1}\sim P_{\mathcal{B}}^\pi}\!\left[\sum_{t=0}^{B-1} D_{\mathrm{KL}}\big(\pi(\cdot\mid b_t,\tau_t)\,\|\,\pi_0(\cdot\mid b_t,\tau_t)\big)\right].
\]

**定义 2.2（正则化泛函）**  
对 \(\lambda\ge 0\)，定义
\[
\mathcal{F}_\lambda[\pi] = J(\pi) - \lambda\,D_{\mathrm{KL}}(\pi\|\pi_0),
\]
其中 \(\pi_0\) 为均匀随机基准策略：\(\pi_0(a\mid b,\tau) = 1/|\mathcal{A}|\)。

### 2.2 软贝尔曼方程

**定理 2.1（软贝尔曼最优性）**  
定义软值函数 \(V_\lambda:\mathcal{B}\times\{0,\dots,B\}\to\mathbb{R}\) 为
\[
V_\lambda(b,\tau) =
\begin{cases}
1, & b\in\mathcal{G}_{\mathcal{B}},\\[6pt]
\lambda\log\sum_{a}\pi_0(a\mid b)\exp\!\Big(\frac{1}{\lambda}\,\mathbb{E}_{b'\mid b,a}[V_\lambda(b',\tau-1)]\Big), & b\notin\mathcal{G}_{\mathcal{B}},\;\tau>0,\\[6pt]
0, & b\notin\mathcal{G}_{\mathcal{B}},\;\tau=0.
\end{cases}
\]
则：
1. 方程有唯一解；
2. 最大化 \(\mathcal{F}_\lambda\) 的策略为 Boltzmann 分布
\[
\pi_\lambda^*(a\mid b,\tau) \propto \pi_0(a\mid b)\exp\!\Big(\frac{1}{\lambda}\,\mathbb{E}_{b'}[V_\lambda(b',\tau-1)]\Big);
\]
3. \(\max_\pi\mathcal{F}_\lambda[\pi] = V_\lambda(b_0,B)\)；
4. 当 \(\lambda\to 0^+\) 时，\(V_\lambda\to V^*\) 点态，\(\pi_\lambda^*\) 趋向硬最大化策略。\(\blacksquare\)

*证明概要*：对 \(\tau\) 归纳。\(\tau=0\) 平凡。设 \(\tau-1\) 成立。对固定 \(b\) 与 \(\tau\)，将正则化期望回报写为策略的函数：
\[
F(\pi(\cdot\mid b,\tau)) = \sum_a \pi(a\mid b,\tau) Q_\lambda(b,a,\tau) - \lambda D_{\mathrm{KL}}(\pi(\cdot\mid b,\tau)\|\pi_0(\cdot\mid b,\tau)),
\]
其中 \(Q_\lambda(b,a,\tau) = \mathbb{E}_{b'\mid b,a}[V_\lambda(b',\tau-1)]\)。对 \(\pi(\cdot\mid b,\tau)\) 求约束最大化（概率单纯形上的凸优化），KL 散度作为 Bregman 散度，最优解为 Boltzmann 分布 \(\pi_\lambda^*(a) \propto \pi_0(a)\exp(Q_\lambda(b,a,\tau)/\lambda)\)。代入目标函数即得对数–和–指数形式的递归关系。\(\lambda\to 0^+\) 时，Boltzmann 分布退化为 \(\arg\max_a Q(b,a,\tau)\) 上的等概率分布，软值函数收敛到硬值函数。\(\blacksquare\)

## 第三部分：值信息对齐定理——一般凸值函数框架与横截结构

本节在解耦假设（§0.3）下建立信息价值与信息增益之间的严格定量关系。物理状态固定为 \(s_{\mathrm{ref}}\)，探索动作仅改变信念。我们先发展一个抽象的一般凸值函数框架（§3.0），再将具体模型作为特例嵌入（§3.1–§3.2）。所有定理均在此框架下获得。

### 3.0 一般凸值函数的 VoI–EIG 不等式

设模型空间有限，信念 \(\mu\in\Delta^{K-1}\)。令 \(\mathcal{C}\subseteq[0,1]^K\) 为紧集，定义值函数 \(V:\Delta^{K-1}\to[0,1]\) 为
\[
V(\mu) = \sup_{c\in\mathcal{C}} \langle \mu, c\rangle .
\]
显然 \(V\) 是凸函数，且 \(V(\mu)\in[0,1]\)。对信念 \(\mu\) 及信息收集动作 \(a\)（诱导后验分布 \(P(\cdot|\mu,a)\)，满足 \(\mathbb{E}_{\mu'\sim P}[\mu']=\mu\)），定义

* **信息价值**：\(\operatorname{VoI}_V(\mu,a) = \mathbb{E}_{\mu'}[V(\mu')] - V(\mu)\).
* **预期信息增益**：\(\operatorname{EIG}(\mu,a) = \mathbb{E}_{\mu'}[D_{\mathrm{KL}}(\mu'\|\mu)]\).

以下给出不依赖于特定 \(\mathcal{C}\) 的一般结果。

**引理 3.0.1（Lipschitz 性）**  
对任意 \(\mu,\mu'\in\Delta^{K-1}\)，有 \(|V(\mu)-V(\mu')| \le D_{\mathrm{TV}}(\mu,\mu')\).

*证明*：对任意 \(c\in\mathcal{C}\)，因 \(c_k\in[0,1]\) 且 \(\sum_k(\mu_k-\mu'_k)=0\)，
\[
|\langle \mu-\mu',c\rangle| \le \tfrac12 \|\mu-\mu'\|_1 = D_{\mathrm{TV}}(\mu,\mu').
\]
取上确界即得 \(|V(\mu)-V(\mu')| \le D_{\mathrm{TV}}(\mu,\mu')\). \(\blacksquare\)

**定理 3.0.1（上界，一般形式）**  
对任意 \(\mu,a\)，有
\[
\operatorname{VoI}_V(\mu,a) \le \sqrt{\frac12\,\operatorname{EIG}(\mu,a)}.
\]

*证明*：由引理 3.0.1，
\[
\operatorname{VoI}_V \le \mathbb{E}_{\mu'}\big[|V(\mu')-V(\mu)|\big] \le \mathbb{E}_{\mu'}\big[D_{\mathrm{TV}}(\mu',\mu)\big].
\]
由 Pinsker 不等式 \(D_{\mathrm{TV}}\le \sqrt{\frac12 D_{\mathrm{KL}}}\)，再用 Jensen 不等式得
\[
\mathbb{E}[D_{\mathrm{TV}}] \le \mathbb{E}\Big[\sqrt{\tfrac12 D_{\mathrm{KL}}}\Big] \le \sqrt{\tfrac12}\,\sqrt{\mathbb{E}[D_{\mathrm{KL}}]} = \sqrt{\tfrac12\operatorname{EIG}}.
\]
\(\blacksquare\)

**定义 3.0.1（决策边界与次梯度锥）**  
令 \(\mathcal{C}^*(\mu)=\{c\in\mathcal{C}:\langle\mu,c\rangle=V(\mu)\}\)。若存在 \(c_1,c_2\in\mathcal{C}^*(\mu)\) 且 \(c_1\neq c_2\)，则称 \(\mu\) 位于 \(V\) 的**决策边界**上。此时次梯度锥为
\[
\partial V(\mu) = \operatorname{conv}\,\mathcal{C}^*(\mu).
\]

**定义 3.0.2（横截条件）**  
称信息收集动作 \(a\) 关于 \(\mu\) 满足横截条件，若存在常数 \(m>0\)，使得对每个可能的后验 \(\mu'\)（\(P(\mu'|\mu,a)>0\)）均有
\[
V(\mu') - V(\mu) \ge m\,D_{\mathrm{TV}}(\mu',\mu).
\]

**定理 3.0.2（下界，一般形式）**  
设模型数 \(K<\infty\)，信念 \(\mu\) 满足 \(\min_k\mu_k\ge\delta>0\)，且动作 \(a\) 关于 \(\mu\) 满足横截条件（常数 \(m>0\)）。则
\[
\operatorname{VoI}_V(\mu,a) \ge \frac{m\sqrt{\delta}}{2\sqrt{\log(1/\delta)}}\,\operatorname{EIG}(\mu,a).
\]

*证明*：由横截条件，
\[
\operatorname{VoI}_V \ge m\,\mathbb{E}[D_{\mathrm{TV}}(\mu',\mu)]. \tag{1}
\]
因 \(\mu_k\ge\delta\)，对任意后验 \(\mu'\)，利用 \(\chi^2\)-散度有
\[
D_{\mathrm{KL}}(\mu'\|\mu) \le \chi^2(\mu'\|\mu) \le \frac{1}{\delta}\|\mu'-\mu\|_1^2 = \frac{4}{\delta}D_{\mathrm{TV}}^2,
\]
故
\[
D_{\mathrm{TV}} \ge \frac{\sqrt{\delta}}{2}\sqrt{D_{\mathrm{KL}}(\mu'\|\mu)}.
\]
取期望得
\[
\mathbb{E}[D_{\mathrm{TV}}] \ge \frac{\sqrt{\delta}}{2}\,\mathbb{E}\Big[\sqrt{D_{\mathrm{KL}}}\Big]. \tag{2}
\]
由于 \(\mu_k\ge\delta\)，有 \(D_{\mathrm{KL}}\le \log(1/\delta)\)。记 \(B=\log(1/\delta)\)，对 \(0\le X\le B\) 使用 \(\sqrt{X}\ge X/\sqrt{B}\)，令 \(X=D_{\mathrm{KL}}\) 并取期望，
\[
\mathbb{E}\big[\sqrt{D_{\mathrm{KL}}}\big] \ge \frac{\mathbb{E}[D_{\mathrm{KL}}]}{\sqrt{B}} = \frac{\operatorname{EIG}}{\sqrt{\log(1/\delta)}}. \tag{3}
\]
将 (3) 代入 (2) 再代入 (1) 即得结论。\(\blacksquare\)

**定理 3.0.3（横截判据，一般形式）**  
设 \(\mu\) 位于决策边界，且存在常数 \(\eta>0\)，使得对每个可能的后验 \(\mu'\)，存在次梯度 \(g\in\partial V(\mu)\) 满足
\[
\langle g, \mu'-\mu\rangle \ge \eta\,D_{\mathrm{TV}}(\mu',\mu),
\]
则横截条件成立，且可取 \(m=\eta\).

*证明*：由次梯度不等式，对每个 \(\mu'\) 及相应的 \(g\in\partial V(\mu)\)，
\[
V(\mu') \ge V(\mu) + \langle g, \mu'-\mu\rangle \ge V(\mu) + \eta D_{\mathrm{TV}}(\mu',\mu).
\]
因此 \(V(\mu')-V(\mu)\ge \eta D_{\mathrm{TV}}\)，对所有 \(\mu'\) 一致成立。\(\blacksquare\)

以上三个定理构成了从凸值函数的 Lipschitz 性质与横截几何出发的完整 VoI–EIG 不等式链。下面将它们具体化到两阶段及多步动态规划中。

### 3.1 两阶段信息–利用模型中的特化

在解耦假设下，利用阶段从 \(s_{\mathrm{ref}}\) 出发，选择动作 \(a'\in\mathcal{A}_{\mathrm{exploit}}\)，其命中概率向量为 \(v(a') = (v_1(a'),\dots,v_K(a'))\in[0,1]^K\)。值函数
\[
U(\mu) = \max_{a'\in\mathcal{A}_{\mathrm{exploit}}} \langle \mu, v(a')\rangle
\]
是 §3.0 中取 \(\mathcal{C}_{\mathrm{exploit}} = \{v(a'): a'\in\mathcal{A}_{\mathrm{exploit}}\}\) 的特例。

**定义 3.1（信息价值 VoI）**  
\[
\operatorname{VoI}(\mu,a) = \mathbb{E}_{\mu'}[U(\mu')] - U(\mu).
\]

**定理 3.1（上界）**  
\(\operatorname{VoI}(\mu,a) \le \sqrt{\tfrac12\operatorname{EIG}(\mu,a)}\). \(\blacksquare\)  
*证明*：直接由定理 3.0.1 得出。

**定义 3.2（单步决策边界）**  
信念 \(\mu\) 位于决策边界当且仅当存在 \(a_1,a_2\in A^*(\mu)=\arg\max_{a'}\langle\mu,v(a')\rangle\) 使得 \(v(a_1)\neq v(a_2)\)。此时 \(\partial U(\mu) = \operatorname{conv}\{v(a):a\in A^*(\mu)\}\).

**定义 3.3（横截条件）**  
称动作 \(a\in\mathcal{A}_{\mathrm{info}}\) 关于 \(\mu\) 满足横截条件，若存在 \(m>0\) 使对所有可能后验 \(\mu'\)，\(U(\mu')-U(\mu)\ge m D_{\mathrm{TV}}(\mu',\mu)\).

**定理 3.2（横截判据）**  
若存在 \(\eta>0\) 使对每个后验 \(\mu'\)，存在 \(g\in\partial U(\mu)\) 满足 \(\langle g,\mu'-\mu\rangle \ge \eta D_{\mathrm{TV}}(\mu',\mu)\)，则横截条件成立，\(m=\eta\). \(\blacksquare\)  
*证明*：定理 3.0.3 的特例。

**定理 3.3（下界）**  
设 \(|\mathcal{P}|=K<\infty\)，\(\mu_k\ge\delta>0\) 对所有 \(k\) 成立，且动作 \(a\) 满足横截条件（常数 \(m>0\)）。则
\[
\operatorname{VoI}(\mu,a) \ge \frac{m\sqrt{\delta}}{2\sqrt{\log(1/\delta)}}\,\operatorname{EIG}(\mu,a).
\]
\(\blacksquare\)  
*证明*：由定理 3.0.2 直接给出。

### 3.2 多步信息价值的递归推广

在解耦假设下，物理状态固定于 \(s_{\mathrm{ref}}\)，贝叶斯最优值函数仅依赖于信念与剩余步数 \(\tau\)：
\[
V_\tau(\mu) := V^*((s_{\mathrm{ref}},\mu),\tau).
\]
满足递归式
\[
V_\tau(\mu) = \max_{a\in\mathcal{A}} \mathbb{E}_{\mu'\sim P(\cdot|\mu,a)}[V_{\tau-1}(\mu')],\quad V_0(\mu)=0,
\]
且 \(V_1(\mu)=U(\mu)\).

**引理 3.1（值函数表示与凸性）**  
对任意 \(\tau\ge 1\)，存在紧集 \(\mathcal{C}_\tau\subseteq[0,1]^K\) 使得
\[
V_\tau(\mu) = \sup_{c\in\mathcal{C}_\tau}\langle\mu,c\rangle .
\]
具体地，\(\mathcal{C}_\tau = \{c^\pi : \pi\in\Pi\}\)，其中 \(c^\pi_k = V_\pi^{P_k}(s_{\mathrm{ref}},\tau)\) 为模型 \(k\) 下策略 \(\pi\) 的 \(\tau\) 步命中概率。因此 \(V_\tau\) 是 Lipschitz-1 凸函数，且满足定理 3.0.1–3.0.3 的所有条件。

*证明*：\(V_\tau(\mu) = \sup_\pi \sum_k \mu_k V_\pi^{P_k}(s_{\mathrm{ref}},\tau)\). 每个 \(c^\pi_k\in[0,1]\)，故 \(\mathcal{C}_\tau\subset[0,1]^K\)，且 Lipschitz 性由引理 3.0.1 保证。\(\blacksquare\)

**定义 3.4（多步信息价值）**  
\[
\operatorname{VoI}_\tau(\mu,a) = \mathbb{E}_{\mu'}[V_{\tau-1}(\mu')] - V_{\tau-1}(\mu).
\]

**定理 3.4（多步上界）**  
\(\operatorname{VoI}_\tau(\mu,a) \le \sqrt{\tfrac12\operatorname{EIG}(\mu,a)}\). \(\blacksquare\)  
*证明*：由定理 3.0.1 与引理 3.1 直接得到。

**定义 3.5（多步决策边界）**  
\(\mu\) 位于 \(V_{\tau-1}\) 的决策边界当且仅当存在 \(\pi_1,\pi_2\in\Pi_{\tau-1}^*(\mu)\)（达到最优的策略集合）使得 \(c^{\pi_1}\neq c^{\pi_2}\)。此时次梯度锥为 \(\partial V_{\tau-1}(\mu) = \operatorname{conv}\{c^\pi : \pi\in\Pi_{\tau-1}^*(\mu)\}\).

**定义 3.6（多步横截条件）**  
称动作 \(a\) 关于 \(\mu\) 满足 \(\tau\)-步横截条件，若存在 \(m_\tau>0\) 使对所有可能后验 \(\mu'\)，有
\[
V_{\tau-1}(\mu') - V_{\tau-1}(\mu) \ge m_\tau D_{\mathrm{TV}}(\mu',\mu).
\]

**定理 3.5（多步下界）**  
在与定理 3.3 相同的 \(\delta\)-先验条件下，若 \(a\) 满足 \(\tau\)-步横截条件（常数 \(m_\tau>0\)），则
\[
\operatorname{VoI}_\tau(\mu,a) \ge \frac{m_\tau\sqrt{\delta}}{2\sqrt{\log(1/\delta)}}\,\operatorname{EIG}(\mu,a).
\]
\(\blacksquare\)  
*证明*：由定理 3.0.2 与引理 3.1 得到。

**定理 3.6（多步横截判据）**  
设 \(\mu\) 位于 \(V_{\tau-1}\) 的决策边界，且存在 \(\eta_\tau>0\)，使得对每个可能后验 \(\mu'\)，存在次梯度 \(g\in\partial V_{\tau-1}(\mu)\) 满足
\[
\langle g, \mu'-\mu\rangle \ge \eta_\tau D_{\mathrm{TV}}(\mu',\mu),
\]
则 \(\tau\)-步横截条件成立，且 \(m_\tau=\eta_\tau\). \(\blacksquare\)  
*证明*：定理 3.0.3 的特例。

## 第四部分：信息几何梯度流与算法涌现

### 4.1 策略流形的自然梯度

**定义 4.1（参数化策略与 Fisher 度量）**  
设策略由参数 \(\theta \in \Theta \subseteq \mathbb{R}^d\) 参数化：\(\pi_\theta(a \mid b, \tau)\)。在信念状态 \(b\) 和剩余步数 \(\tau\) 下，局部 Fisher 信息矩阵为
\[
\mathcal{I}_{b,\tau}(\theta) = \mathbb{E}_{a \sim \pi_\theta(\cdot \mid b, \tau)}\left[ \nabla_\theta \log \pi_\theta(a\mid b,\tau) \nabla_\theta \log \pi_\theta(a\mid b,\tau)^\top \right].
\]
该矩阵为策略流形 \(\{\pi_\theta\}\) 赋予黎曼度量，即 Fisher–Rao 度量。

**定理 4.1（正则化策略的自然梯度）**  
最大化 \(\mathcal{F}_\lambda(\theta)\) 的自然梯度流为
\[
\tilde{\nabla}_\theta \mathcal{F}_\lambda(\theta) = \mathcal{I}(\theta)^{-1} \nabla_\theta \mathcal{F}_\lambda(\theta).
\]
由策略梯度定理，
\[
\nabla_\theta \mathcal{F}_\lambda(\theta) = \mathbb{E}_{P_{\mathcal{B}}^{\pi_\theta}}\left[ \sum_{t=0}^{B-1} \nabla_\theta \log \pi_\theta(a_t \mid b_t, \tau_t) \cdot A^{\pi_\theta}_\lambda(b_t, a_t, \tau_t) \right],
\]
其中 \(A^{\pi_\theta}_\lambda\) 为软优势函数：
\[
A^{\pi_\theta}_\lambda(b,a,\tau) = \mathbb{E}_{b'\mid b,a}[V_\lambda(b',\tau-1)] - V_\lambda(b,\tau).
\]
自然梯度流是策略分布流形上最大化正则化期望回报的最陡上升动力学。\(\blacksquare\)

### 4.2 乘性权重作为镜像下降

**定理 4.2（乘性权重的变分解释）**  
设候选策略集 \(\{\pi_i\}_{i=1}^{N}\)，权重向量 \(w \in \Delta^{N-1}\)。在单纯形上定义泛函
\[
\mathcal{F}_\lambda[w] = \sum_{i=1}^{N} w_i J(\pi_i) - \lambda \sum_{i=1}^{N} w_i \log \frac{w_i}{1/N}.
\]
乘性权重更新 \(w_i^{(t+1)} \propto w_i^{(t)} \exp(\eta J(\pi_i))\) 是使用 KL 散度作为 Bregman 散度，在单纯形上最大化 \(\mathcal{F}_{1/\eta}[w]\) 的镜像下降步骤。当步长趋于零时，其连续极限为复制子方程
\[
\dot{w}_i = w_i\Big(J(\pi_i) - \sum_{j=1}^{N} w_j J(\pi_j)\Big),
\]
该方程是概率单纯形上的自然梯度流。\(\blacksquare\)

### 4.3 严格结果与启发式联系的分界

上述定理 4.1 和 4.2 为严格证明的结果。以下内容进入启发式联系与开放问题。

**命题 4.1（Thompson 采样的策略分布）**  
Thompson 采样 (TS) 的动作选择机制为从后验采样模型，再根据采样模型选择贪婪动作。这等价于从策略分布
\[
\pi_{\text{TS}}(a\mid b) = \mu\big(\{P_k : a = \arg\max_{a'} Q_{P_k}^*(s, a')\}\big)
\]
中采样。其中 \(Q_{P_k}^*\) 为模型 \(P_k\) 下的最优动作价值函数。

**猜想 4.1（Thompson 采样与自然梯度的条件等价）**  
对于表格 MDP 且先验为 Dirichlet 分布的独立乘积形式，TS 的期望更新方向可能与特定的自然梯度方向一致。在一般 BAMDP 中，TS 可视为一种随机化的自然梯度估计，但其噪声特性与最优方差缩减的梯度估计器不同。严格等价性及有限样本悔恨下界仍待证明。\(\Diamond\)

## 第五部分：数学物理联系

1. **随机最优控制与 HJB 方程**：定理 1.2 的 HJB 方程是离散时间、离散状态空间下最优控制的标准形式。当状态空间连续时，它自然地过渡到偏微分方程（Hamilton-Jacobi-Bellman 方程），对应着经典力学中的 Hamilton-Jacobi 理论。

2. **信息几何与统计推断**：软贝尔曼策略的 Boltzmann 形式是指数族的典范表达。策略空间在 Fisher–Rao 度量下构成统计流形。自然梯度流是这一几何上的最陡上升曲线，其结构在 Otto 变分演算与 JKO 格式中有对应表达。

3. **统计力学与热化过程**：引入温度参数 \(\lambda\) 的软贝尔曼方程，通过变换 \(Z_\lambda = \exp(V_\lambda / \lambda)\) 可线性化。\(\lambda\) 控制着“利用”（\(\lambda \to 0\)，基态冻结）与“探索”（\(\lambda \to \infty\)，等概率热态）之间的平衡，与统计力学中的自由能最小化和模拟退火过程有形式上的同构。

4. **机器学习中的泛化界**：定理 3.0.1–3.0.3 建立的价值–信息不等式，与 PAC-Bayes 理论中由 KL 散度界定的泛化误差上界在结构上一致。它揭示了认知不确定性减小（信息增益）如何通过严格的数学桥梁转化为决策性能的保障。

## 结论

本理论在 BAMDP 框架下，建立了有限预算目标命中问题的严格变分原理。核心贡献包括：

1. 给出了信念 MDP 及其软正则化版本的完整最优性方程（定理 1.2, 2.1）。

2. 发展了一般凸值函数的 VoI–EIG 不等式框架（§3.0），在解耦的参考状态设定下证明了**值信息对齐定理**：
   * 信息价值上界由预期信息增益的平方根无条件界定，常数精确为 \(\sqrt{1/2}\)（定理 3.0.1/3.1/3.4）；
   * 在有限模型空间、先验正下界且满足**决策边界横截条件**的区域内，信息价值下界由预期信息增益线性界定（定理 3.0.2/3.3/3.5）；
   * 建立了统一的**横截判据定理**（定理 3.0.3/3.2/3.6），将横截条件归约为后验分布在决策边界次梯度锥上的几何投影条件。单步与多步情形在凸分析的次梯度锥框架下完全统一。
   * 决策边界的定义严格化至达到最优的向量差异，而非策略个数（定义 3.0.1/3.2/3.5）。

3. 阐明了乘性权重、自然梯度流与信息几何之间的严格数学等价关系（定理 4.2）。

4. 严格结果结束于定理 4.2，之后指出了 Thompson 采样与精确自然梯度算法之间的鸿沟，并将其列为开放猜想（猜想 4.1）。
