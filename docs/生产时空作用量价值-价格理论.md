# 生产时空占用作用量 价值—价格理论

## 一个基于物理约束与竞争均衡的公理化框架（修订版）


## 1 物理基础与占用作用量

### 1.1 基本物理量纲与功率约束

**定义 1**（基本物理量纲）  
- 能量 \(E\)，量纲焦耳（\(\mathrm{J}\)）；  
- 时间 \(t\)，量纲秒（\(\mathrm{s}\)）；  
- 瞬时有效功率 \(P(t)\)，量纲瓦特（\(\mathrm{W}=\mathrm{J}/\mathrm{s}\)），表示时刻 \(t\) 实际投入生产转换的有效功率。

**公理 1**（物理稀缺与排他性公理）  
经济系统在任意时刻 \(t\)，所有生产活动占用的瞬时总有效功率 \(P(t)\) 受外生物理上界 \(\bar{P}\in\mathbb{R}_{++}\) 约束：
\[
0 \le P(t) \le \bar{P}, \quad \forall t.
\]
功率占用具有排他性：一旦某部分功率被某一活动占用，其同时段不可用于其他活动。时间本身不可储存、不可逆转。

### 1.2 生产路径与占用作用量

**定义 2**（生产路径）  
一单位商品的生产路径由三元组 \((T, \phi, \eta)\) 刻画：
- 生产周期 \(T\in\mathbb{R}_{++}\)（从开始到完成的时间长度）；
- 瞬时输入功率剖面 \(\phi:[0,T]\to\mathbb{R}_+\)，分段连续；
- 瞬时技术效率 \(\eta:[0,T]\to(0,1]\)。
有效功率剖面为 \(\phi_{\text{eff}}(t)=\phi(t)\eta(t)\)。

**定义 3**（占用作用量）  
生产路径的**占用作用量** \(\mathcal{A}\) 定义为
\[
\boxed{\mathcal{A} := \int_0^T (T-s)\,\phi_{\text{eff}}(s)\,\mathrm{d}s},
\]
量纲为 \(\mathrm{J\cdot s}\)。

**注 1**（占用作用量的直观含义与经济解释）  
占用作用量度量"未来生产能力被锁定的累积面积"。权重 \((T-s)\) 反映：在时刻 \(s\) 投入的功率，将在剩余的 \(T-s\) 时间内持续占用系统功率上界的一部分，从而排斥其他生产活动的可能性。机会成本随剩余时间线性累积。  
等价形式：令 \(E^{\text{rem}}(t) := \int_t^T \phi_{\text{eff}}(s)\,\mathrm{d}s\) 为时刻 \(t\) 剩余待投入的有效能量，则
\[
\mathcal{A} = \int_0^T E^{\text{rem}}(t)\,\mathrm{d}t.
\]
若在 \(t=0\) 瞬间投入全部有效能量（Dirac 脉冲），则对所有 \(t>0\) 有 \(E^{\text{rem}}(t)=E\)，故 \(\mathcal{A} = \int_0^T E\,\mathrm{d}t = ET\)，达到最大占用作用量。这反映了"过早锁定生产能力"的经济代价。

**定理 1**（占用作用量泛函形式的唯一性）  
设 \(\Phi\) 为所有可能的生产路径 \((\phi_{\text{eff}}, T)\) 组成的空间，其中 \(\phi_{\text{eff}}\in L^1[0,T]\) 且 \(T>0\)。若泛函 \(\mathcal{A}:\Phi\to\mathbb{R}_+\) 满足：  
1. **连续性**：对固定 \(T\)，\(\mathcal{A}\) 关于 \(\phi_{\text{eff}}\) 在 \(L^1[0,T]\) 中连续；  
2. **线性**：对固定 \(T\)，\(\mathcal{A}[\alpha\phi_1+\beta\phi_2]=\alpha\mathcal{A}[\phi_1]+\beta\mathcal{A}[\phi_2]\)；  
3. **时间平移不变性**：若 \(\phi_{\text{eff}}'(t)=\phi_{\text{eff}}(t-\tau)\) 定义在 \([\tau,\tau+T]\)，则 \(\mathcal{A}[\phi']=\mathcal{A}[\phi]\)；  
4. **量纲协变性**：对 \(\lambda>0\)，令 \(\phi^{(\lambda)}(t)=\lambda^{-1}\phi_{\text{eff}}(t/\lambda),\,t\in[0,\lambda T]\)，则 \(\mathcal{A}[\phi^{(\lambda)}]=\lambda \mathcal{A}[\phi]\)。  

则存在常数 \(c>0\) 使得
\[
\mathcal{A}[\phi] = c\int_0^T (T-t)\,\phi_{\text{eff}}(t)\,\mathrm{d}t.
\]
选取量纲使 \(c=1\)，即得定义 3。

*证明*：  
固定 \(T\)。由条件 1、2，根据 Riesz 表示定理，\((L^1[0,T])^* \cong L^\infty[0,T]\)，存在 \(k_T\in L^\infty[0,T]\) 使
\[
\mathcal{A}[\phi] = \int_0^T k_T(t)\,\phi_{\text{eff}}(t)\,\mathrm{d}t.
\]

由条件 3，对任意 \(\tau>0\) 及定义在 \([0,T]\) 上的 \(\phi\)，令 \(\phi'(t)=\phi(t-\tau)\) 定义在 \([\tau,\tau+T]\)，则 \(\mathcal{A}[\phi']=\mathcal{A}[\phi]\)。即
\[
\int_\tau^{\tau+T} k_{\tau+T}(t)\,\phi(t-\tau)\,\mathrm{d}t = \int_0^T k_T(s)\,\phi(s)\,\mathrm{d}s, \quad s=t-\tau.
\]
左边化为 \(\int_0^T k_{\tau+T}(s+\tau)\,\phi(s)\,\mathrm{d}s\)。由 \(\phi\in L^1[0,T]\) 的任意性，核函数作为 \(L^\infty\) 中的等价类几乎处处相等。选取代表元使 \(k_{\tau+T}(s+\tau)=k_T(s)\) 对几乎所有 \(s\) 成立。令 \(g(T):=k_T(0)\)，则对任意 \(T>0\) 和几乎处处的 \(t\in[0,T]\)，有 \(k_T(t)=k_{T-t}(0)=g(T-t)\)。故存在函数 \(g:\mathbb{R}_+\to\mathbb{R}\) 使得 \(k_T(t)=g(T-t)\)。于是
\[
\mathcal{A} = \int_0^T g(T-t)\,\phi_{\text{eff}}(t)\,\mathrm{d}t.
\]

由条件 4，对常值功率 \(P\) 及缩放路径 \(\phi^{(\lambda)}(t)=P/\lambda,\; t\in[0,\lambda T]\)，有
\[
\mathcal{A}[\phi^{(\lambda)}] = \int_0^{\lambda T} g(\lambda T-t)\frac{P}{\lambda}\,\mathrm{d}t = \frac{P}{\lambda}\int_0^{\lambda T} g(s)\,\mathrm{d}s.
\]
\[
\mathcal{A}[\phi] = \int_0^T g(T-t)P\,\mathrm{d}t = P\int_0^T g(s)\,\mathrm{d}s.
\]
尺度协变性要求 \(\frac{1}{\lambda}G(\lambda T) = \lambda G(T)\)，其中 \(G(x):=\int_0^x g(s)\,\mathrm{d}s\)。即 \(G(\lambda T) = \lambda^2 G(T)\)。\(G\) 可测且满足该函数方程，故存在常数 \(c>0\) 使 \(G(x)=c x^2\)。几乎处处微分得 \(g(x)=2c x\)。代入得
\[
\mathcal{A} = 2c\int_0^T (T-t)\,\phi_{\text{eff}}(t)\,\mathrm{d}t.
\]
重标常数吸收因子 \(2c\) 为 \(1\)，即得定义 3 形式。\(\square\)

### 1.3 总占用作用量的物理上界

假定每一时期长度为 \(\tau>0\)，所有生产活动均在该时期内开始并完成（生产周期 \(T\le \tau\)），不存在跨期未完成任务。

**定理 2**（总占用作用量的物理上界）  
在公理 1 及上述时期内完成假设下，任意长度为 \(\tau\) 的时期内所有生产活动的占用作用量之和 \(\mathcal{A}_{\text{total}}\) 满足
\[
\mathcal{A}_{\text{total}} \le \frac{1}{2}\bar{P}\tau^2.
\]
等号当且仅当瞬时总功率恒为 \(\bar{P}\) 且所有活动恰在时期初启动、时期末完成。

*证明*：  
设活动 \(j\) 在区间 \([a_j,b_j]\subseteq[0,\tau]\) 进行，有效功率剖面为 \(\phi_j(t)\)。系统总占用有效功率 \(P(t)=\sum_j \phi_j(t)\mathbf{1}_{[a_j,b_j]}(t)\) 满足 \(0\le P(t)\le\bar{P}\)。占用作用量之和：
\[
\sum_j \mathcal{A}_j = \sum_j \int_{a_j}^{b_j} (b_j-t)\phi_j(t)\,\mathrm{d}t
\le \sum_j \int_{a_j}^{b_j} (\tau-t)\phi_j(t)\,\mathrm{d}t
\le \int_0^\tau (\tau-t) P(t)\,\mathrm{d}t
\le \bar{P}\int_0^\tau (\tau-t)\,\mathrm{d}t = \frac{1}{2}\bar{P}\tau^2.
\]
第一个不等号因 \(b_j\le\tau\) 且 \(\phi_j\ge0\)，第二个不等号将各活动积分合并并放大到全区间，第三个因 \(P(t)\le\bar{P}\)。等号条件：\(P(t)\equiv\bar{P}\) 对所有 \(t\) 且所有 \(b_j=\tau\)（即所有活动在时期末完成）。为同时满足 \(P(t)\equiv\bar{P}\) 和 \(b_j=\tau\)，所有活动的起始时间必须恰为 \(a_j=0\)。\(\square\)

**定义 4**（时期可用作用量禀赋）  
每一时期，经济系统可支配的最大作用量禀赋 \(\bar{\mathcal{A}}\in\mathbb{R}_{++}\) 由外生物理环境给定，不可储存、不可跨期转移。由定理 2，\(\bar{\mathcal{A}}\le\frac{1}{2}\bar{P}\tau^2\)。

### 1.4 最小占用作用量原理与能量度量的不充分性

**命题 1**（最小占用作用量原理——最优控制形式）  
设生产某商品必需的最小累积有效能量为 \(E>0\)，即 \(\int_0^T \phi_{\text{eff}}(t)\,\mathrm{d}t = E\)，且约束 \(0\le \phi_{\text{eff}}(t)\le \bar{P}\)。则占用作用量 \(\mathcal{A}\) 的**下确界**为
\[
\mathcal{A}_{\min} = \frac{E^2}{2\bar{P}},
\]
且唯一由最优控制 \(\phi_{\text{eff}}^*(t)=\bar{P},\, t\in[0, E/\bar{P}]\)（即将功率全部集中在生产周期起始段投入的 bang-bang 控制）取得。任何偏离该控制的路径严格产生更大的 \(\mathcal{A}\)。

*证明*：  
定义状态变量 \(x(t)=E^{\text{rem}}(t)=\int_t^T \phi_{\text{eff}}(s)\,\mathrm{d}s\)，满足 \(x(T)=0,\;x(0)=E\)，动力学 \(\dot{x}(t)=-\phi_{\text{eff}}(t)\)，控制约束 \(\phi_{\text{eff}}(t)\in[0,\bar{P}]\)。由注 1，目标泛函
\[
\mathcal{A} = \int_0^T x(t)\,\mathrm{d}t.
\]
这是一类 Lagrange 型最优控制问题。哈密顿量定义为
\[
\mathcal{H} = -x + \lambda(-\phi_{\text{eff}}) = -x - \lambda\phi_{\text{eff}}.
\]
（选取该符号约定以直接极小化 \(\mathcal{A}\)。）协态方程：
\[
\dot{\lambda} = -\frac{\partial\mathcal{H}}{\partial x} = 1,
\]
由横截条件 \(\lambda(T)=0\) 解得 \(\lambda(t) = t-T \le 0\) 对所有 \(t\in[0,T]\)。极大化哈密顿量（即极小化 \(\mathcal{A}\)）要求：当 \(\lambda(t)<0\) 时，\(\phi_{\text{eff}}^*=\bar{P}\)；当 \(\lambda(t)>0\) 时，\(\phi_{\text{eff}}^*=0\)。因 \(\lambda(t)\le 0\) 在 \([0,T]\) 上恒成立且仅于 \(t=T\) 处为零，故对几乎所有 \(t\) 有 \(\phi_{\text{eff}}^*(t)=\bar{P}\)，直至能量耗尽。最优轨迹为
\[
\phi_{\text{eff}}^*(t) = \begin{cases}
\bar{P}, & 0 \le t \le E/\bar{P},\\
0, & E/\bar{P} < t \le T.
\end{cases}
\]
此时 \(T_{\min} = E/\bar{P}\)，对于任意 \(T \ge T_{\min}\)，最优路径在前段以 \(\bar{P}\) 投入，随后停机等待直至 \(T\)。剩余能量为
\[
E^{\text{rem}}(t) = \begin{cases}
E - \bar{P}t, & 0 \le t \le E/\bar{P},\\
0, & E/\bar{P} < t \le T.
\end{cases}
\]
代入目标泛函：
\[
\mathcal{A}_{\min} = \int_0^{E/\bar{P}} (E - \bar{P}t)\,\mathrm{d}t + \int_{E/\bar{P}}^T 0\,\mathrm{d}t = \frac{E^2}{2\bar{P}}.
\]
此值与 \(T\ge T_{\min}\) 的具体取值无关（只要不小于 \(E/\bar{P}\)）。Pontryagin 极大值原理保证该 bang-bang 控制为全局最优。\(\square\)

**推论 1**（最优生产周期）  
给定 \(E\) 和 \(\bar{P}\)，实现 \(\mathcal{A}_{\min}\) 所需的最小生产周期为 \(T_{\min} = E/\bar{P}\)。选择 \(T > T_{\min}\) 不改变最小作用量值（只需在能量投入完成后闲置至 \(T\)），但任何需要在全程摊薄功率投入的路径（如受工艺约束无法提前完成能量投入）将严格增大 \(\mathcal{A}\)。

**命题 2**（能量度量的不充分性）  
存在生产路径序列，累积有效能量相同，但占用作用量 \(\mathcal{A}_n\to\infty\)。仅按累积能量无法区分此类路径的稀缺性差异。

*证明*：取 \(\phi_n(t)\equiv \bar{P}/n\) 定义在 \([0, T_n]\)，其中 \(T_n = nE/\bar{P}\) 以保持总能量 \(E\) 不变。则
\[
\mathcal{A}_n = \int_0^{T_n} (T_n-t)\frac{\bar{P}}{n}\,\mathrm{d}t = \frac{\bar{P}}{n}\cdot\frac{T_n^2}{2} = \frac{\bar{P}}{2n}\cdot\frac{n^2 E^2}{\bar{P}^2} = \frac{E^2}{2\bar{P}}\cdot n \to \infty.
\]
占用作用量按 \(\Theta(n)\) 线性发散，表明时间占用的延长通过线性权重 \((T-t)\) 转化为占用成本的严格单调递增。能量度量 \(E\) 为守恒量，不包含时间结构信息，故无法捕获该稀缺性差异。\(\square\)

## 2 价值实体：全占用作用量

### 2.1 价值与价格的严格区分

**定义 5**（价值与价格的范畴区分）  
- **价值**（value）：商品内含的物理占用成本，由技术条件 \((A, h)\) 唯一确定，独立于市场制度、需求结构与稀缺性租金。价值是商品固有的物理属性之经济表达，量纲为 \(\mathrm{J\cdot s}\)。  
- **价格**（price）：商品在市场中实现的交换比例，受价值、稀缺租金、市场势力、制度摩擦与需求结构共同决定。价格是价值的市场表现，可系统性地偏离价值。

**公理 2**（价值优先性原则）  
价值先于价格产生：生产活动创造价值（物理占用成本的凝结），市场交换发现价格。价值是价格的引力中心，但价格可因租金与摩擦因素发生短期乃至长期偏离。

### 2.2 生产网络与完全再生产性

考虑 \(n\) 种可无限复制商品的经济。所有生产过程构成有向图 \(\mathcal{G}=(\mathcal{V},\mathcal{E})\)，顶点 \(v_i\) 对应商品 \(i\)，有向边表示商品用于生产其他商品。

**定义 6**（完全再生产性）  
经济满足完全再生产性，若存在商品子集 \(\mathcal{I}_R\subseteq\{1,\dots,n\}\)，使得对任意 \(i\in\mathcal{I}_R\)，存在有限长度生产链，每步仅消耗中间品与占用作用量（不依赖任何外生非生产性要素），且链终端为可直接消费的商品。所有非最终品均可通过该系统完全再生产。

**公理 3**（可复制部门原始要素假设）  
对于所有可无限复制商品组成的经济部门，其生产仅需两类投入：①其他可复制商品作为中间投入；②占用作用量作为原始要素投入。该部门满足完全再生产性：任何可再生要素服务（劳动、资本品服务等）均可通过有限生产链完全还原为中间品消耗与作用量消耗。供给完全无弹性的非生产性自然资源（土地、矿藏、频谱等）不属于可复制商品，其稀缺性租金需单独引入（见 §8）。占用作用量是可复制部门唯一外生的、不可扩大的稀缺资源。

### 2.3 要素可约性与全作用量还原

**定义 7**（要素可约性）  
设经济中存在可再生的要素服务 \(f\)，生产一单位商品 \(j\) 需投入 \(l_{fj}\ge0\) 单位 \(f\)。若一单位 \(f\) 本身可由可复制商品束 \(a_f\in\mathbb{R}_+^n\) 与占用作用量 \(h_f\in\mathbb{R}_{++}\) 再生产，则称 \(f\) 关于作用量**可约**。定义 \(f\) 的**全作用量**（即其价值）为
\[
H_f := h_f + a_f^{\mathsf{T}} H,
\]
其中 \(H\) 为可复制商品全作用量向量（见第 3 章）。

**定理 3**（劳动的可约性）  
若劳动服务 \(L\) 由消费品束 \(c_L\in\mathbb{R}_+^n\) 与直接占用作用量 \(\mathcal{A}_L\) 再生产，且生产函数一阶齐次，则 \(L\) 关于作用量可约，其价值为 \(H_L = h_L + c_L^{\mathsf{T}} H\)。

*证明*：一阶齐次性允许将单位劳动服务的投入标准化为 \((c_L, h_L)\)，直接占用作用量 \(h_L\) 与中间投入 \(c_L\) 的全作用量之和即 \(H_L\)。\(\square\)

**定理 4**（资本服务的可约性）  
设资本品 \(K\) 由可复制投入束 \(a_K\) 与占用作用量 \(h_K\) 一期建成，经济寿命为 \(\tau_K\) 期，每期提供一单位资本服务，折旧率 \(\delta\in(0,1]\)。则资本服务关于作用量可约，其价值为
\[
H_{K\text{-service}} = \frac{h_K + a_K^{\mathsf{T}} H}{\tau_K}.
\]

*证明*：资本品全作用量 \(H_K = h_K + a_K^{\mathsf{T}} H\) 在其寿命期内均匀摊销即得。\(\square\)

### 2.4 要素完备性的无循环条件

**假设 1**（可约性无循环条件）  
任一要素服务的再生产网络不包含自身节点，增广后的技术矩阵满足 Hawkins–Simon 条件。

**定理 5**（原始要素的完备性）  
设经济中存在可再生的要素服务 \(f\)，投入系数向量为 \(l_f\)，且 \(f\) 可由 \((a_f, h_f)\) 再生产并满足假设 1。将 \(f\) 作为第 \(n+1\) 种商品增广，则增广价值向量 \(\tilde{H}=(H, H_f)\) 满足 \(\tilde{H} = \tilde{h} + \tilde{A}^{\mathsf{T}}\tilde{H}\)，其中 \(\tilde{h}=(h, h_f)\)，\(\tilde{A}\) 为增广投入矩阵。系统仍仅有占用作用量作为可复制部门的原始价值实体。

*证明*：增广系统 \(\tilde{A} = \begin{pmatrix} A & a_f \\ l_f^{\mathsf{T}} & 0 \end{pmatrix},\; \tilde{h} = \begin{pmatrix} h \\ h_f \end{pmatrix}\)。由假设 1，\((I-\tilde{A}^{\mathsf{T}})^{-1}\) 存在且非负。全作用量方程 \(\tilde{H} = \tilde{h} + \tilde{A}^{\mathsf{T}}\tilde{H}\) 的解 \(\tilde{H} = (I-\tilde{A}^{\mathsf{T}})^{-1}\tilde{h}\)，其中第 \(n+1\) 分量为 \(H_f = h_f + a_f^{\mathsf{T}} H\)，与定义 7 一致。\(\square\)

## 3 生产技术的形式化

### 3.1 直接占用作用量与投入产出结构

**定义 8**（直接占用作用量）  
可复制商品 \(i\) 的直接占用作用量 \(h_i\) 为其最优生产路径的占用作用量：
\[
h_i := \min_{(\phi_i,\eta_i,T_i)} \int_0^{T_i} (T_i-s)\,\phi_i(s)\eta_i(s)\,\mathrm{d}s,
\]
取在给定技术与物理约束下可实现的最小值（命题 1 给出下确界）。\(h_i\) 是独立于价格体系的可测量物理技术常数，量纲 \(\mathrm{J\cdot s}\)。

**注 2**（\(h_i\) 的测算）  
\(h_i\) 由工程数据独立测算：记录生产周期 \(T_i\)、功率剖面 \(\phi_i(t)\) 和效率 \(\eta_i(t)\)，数值积分即得。若功率剖面基本恒定，近似公式为 \(h_i \approx \frac{1}{2} \bar{\phi}_{\text{eff},i} T_i^2\)。

**定义 9**（技术系数矩阵）  
- 中间投入系数矩阵 \(A\in\mathbb{R}_{\ge 0}^{n\times n}\)，元素 \(a_{ij}\) 为生产一单位商品 \(j\) 所需消耗商品 \(i\) 的物理数量。  
- 直接作用量向量 \(h\in\mathbb{R}_{++}^n\)，由定义 8 给出。

**假设 2**（生产可行性）  
\(A\) 满足 Hawkins–Simon 条件，即 \((I-A)^{-1}\) 存在且非负。

### 3.2 全占用作用量（价值向量）

**定义 10**（全占用作用量向量——价值向量）  
可复制商品的全占用作用量向量 \(H\in\mathbb{R}_{++}^n\) 由线性系统唯一确定：
\[
H = h + A^{\mathsf{T}} H,
\]
或
\[
\boxed{H = (I - A^{\mathsf{T}})^{-1} h}.
\tag{1}
\]
\(H_j\) 为商品 \(j\) 的**价值**——凝结在其中的全部直接与间接占用作用量之和，量纲 \(\mathrm{J\cdot s}\)。

**注 3**（与斯拉法体系的同构与本质区别）  
方程 (1) 在代数结构上与斯拉法价格方程 \(v = l + A^{\mathsf{T}} v\) 同构。本质区别在于：斯拉法体系以劳动时间 \(l\) 为价值实体，本文以物理占用作用量 \(h\) 为价值实体。两者均采用相同的投入产出核算框架，但价值基底不同。本文的创新在于将价值锚定于物理占用而非特定生产要素，从而为劳动价值论与能值理论提供了统一的物理基础。

**定理 6**（价值的存在性与展开）  
在假设 2 下，\(H\) 存在、唯一且严格正，并可展开为绝对收敛级数：
\[
H_j = \sum_{m=0}^\infty [h^{\mathsf{T}} A^m]_j.
\]

*证明*：由 Hawkins–Simon 条件，\(\rho(A)<1\)，故 \((I-A^{\mathsf{T}})^{-1} = \sum_{m=0}^\infty (A^{\mathsf{T}})^m\) 绝对收敛且非负；\(h\gg0\) 保证 \(H\gg0\)。\(\square\)

## 4 竞争均衡与价值—价格关系

### 4.1 经济环境

考虑封闭经济，时期长度 \(\tau>0\)，每期初经济拥有作用量禀赋 \(\bar{\mathcal{A}}\)（定义 4），不可储存。完全信息，价格接受，无外部性。\(n\) 种可复制商品技术满足假设 2，规模报酬不变，自由进入退出。代表性消费者效用 \(U:\mathbb{R}_+^n\to\mathbb{R}\) 连续、严格递增、拟凹，拥有全部禀赋 \(\bar{\mathcal{A}}\)。

### 4.2 社会计划者问题

**定义 11**（可行配置）  
非负向量组 \((x,c)\in\mathbb{R}^{2n}_+\) 可行，若
\[
x \ge A x + c, \qquad h^{\mathsf{T}} x \le \bar{\mathcal{A}}.
\tag{2}
\]

**定理 7**（计划者最优的影子价格与价值的关系）  
设 \((x^*, c^*)\) 为 \(\max_{x,c} U(c)\) s.t. (2) 的解，且 \(x^*\gg0,\;c^*\gg0\)。则存在乘子 \(p\in\mathbb{R}^n_+\)（对应物料平衡）和 \(w>0\)（对应作用量约束）满足
\[
p = A^{\mathsf{T}} p + w h,
\]
从而
\[
p = w H.
\]
影子价格正比于价值向量 \(H\)。

*证明*：拉格朗日函数 \(\mathcal{L}=U(c)+p^{\mathsf{T}}(x-Ax-c)+w(\bar{\mathcal{A}}-h^{\mathsf{T}}x)\)。由 \(c^*\gg0\) 得 \(\nabla U(c^*)=p\gg0\)。由 \(x^*\gg0\)，一阶条件给出 \(p^{\mathsf{T}}-p^{\mathsf{T}}A - w h^{\mathsf{T}}=0\)，即 \(p = A^{\mathsf{T}} p + w h\)，解得 \(p = w (I-A^{\mathsf{T}})^{-1} h = w H\)。\(\square\)

### 4.3 竞争均衡与自然价格

**定义 12**（竞争均衡）  
竞争均衡为价格体系 \((p,w)\in\mathbb{R}^{n+1}_+\)、活动水平 \(x\in\mathbb{R}^n_+\) 和消费 \(c\in\mathbb{R}^n_+\)，满足：  
1. 利润最大化：对所有 \(j\)，\(p_j \le \sum_i a_{ij} p_i + w h_j\)，若 \(x_j>0\) 则等号成立；  
2. 效用最大化：\(c\) 为 \(\max U(c)\) s.t. \(p^{\mathsf{T}}c \le w\bar{\mathcal{A}}\) 的解；  
3. 商品市场出清：\(x = A x + c\)；  
4. 作用量市场出清：\(h^{\mathsf{T}} x = \bar{\mathcal{A}}\)（因 \(w>0\)）。

**定理 8**（自然价格定理——价值的价格表现）  
设 \((p,w,x,c)\) 为满足定义 12 的竞争均衡。若所有可复制商品均被严格正生产（\(x\gg0\)），且经济中不存在供给刚性的非生产性资源（§8.3），则均衡价格向量正比于全占用作用量向量：
\[
p = w H,
\]
其中 \(H=(I-A^{\mathsf{T}})^{-1}h\)。相对价格 \(p_j/p_k = H_j/H_k\) 由物理价值 \((A,h)\) 唯一确定，与需求结构无关。

*证明*：由 \(x\gg0\) 及零利润条件得 \(p = A^{\mathsf{T}} p + w h\)，解得 \(p = w H\)。因 \(\bar{\mathcal{A}}>0\) 且 \(x\gg0\)，作用量约束束紧，\(w>0\)。相对价格由 \(H\) 决定，需求仅通过 \(w\) 影响绝对价格水平。\(\square\)

**推论 2**（价值与价格的区分在该定理中的体现）  
定理 8 表明：在纯竞争性可复制经济中，**价格** \(p\) 严格正比于**价值** \(H\)，即市场交换比例完全由物理占用成本决定。此即为"自然价格"状态——价值直接表现为价格，两者无偏离。一旦引入不可复制资源、市场势力或制度摩擦（§8），价格将偏离价值。

**推论 3**（计价物不敏感性）  
以商品 \(k\) 为计价物，则 \(w = 1/H_k\)，\(p_j = H_j/H_k\)。

**推论 4**（劳动价值论与能值理论在该框架中的位置）  
劳动的全作用量 \(H_L\) 决定工资率 \(w_L = w H_L\)；单纯能量分析忽略时间维度，仅捕捉 \(h\) 中的能量项，在全作用量核算中信息不全。占用作用量统一了焦耳与秒，为两者提供了物理基础。

## 5 跨期经济、利率与贴现价值

### 5.1 物理价值与贴现价值的区分

跨期经济中引入恒定实际利率 \(r\ge 0\)。物理占用与金融评价需严格区分。

**定义 13**（物理价值）  
商品 \(j\) 的**物理价值** \(H_j^{\text{phys}}\) 即为其全占用作用量 \(H_j\)，由定义 10 给出，与利率无关。物理价值是商品固有的、不随时间偏好的改变而改变的客观属性。

**定义 14**（贴现作用量函数）  
同一生产路径在利率 \(r\) 下的贴现作用量为
\[
h_j(r) := \int_0^{T_j} \frac{1-e^{-r(T_j-s)}}{r} \,\phi_j(s)\eta_j(s)\,\mathrm{d}s.
\]
当 \(r\to0\)，\(h_j(r)\to h_j^{\text{phys}}\)。对 \(r>0\)，\(h_j(r) < h_j^{\text{phys}}\)。

**定义 15**（贴现价值向量）  
\[
H^{\text{PV}}(r) := (I - A^{\mathsf{T}})^{-1} h(r),
\]
其中 \(h(r)=(h_1(r),\dots,h_n(r))^{\mathsf{T}}\)。\(H^{\text{PV}}(0)=H^{\text{phys}}\)。

### 5.2 跨期竞争均衡中的价格

**定理 9**（利率情形下的价格—贴现价值关系）  
在跨期竞争均衡中，若定理 8 的前提成立且所有商品均被生产，则价格向量满足
\[
p = w H^{\text{PV}}(r),
\]
即价格正比于贴现价值。

*证明*：零利润条件变为 \(p_j = \sum_i a_{ij} p_i + w h_j(r)\)，矩阵形式 \(p = A^{\mathsf{T}} p + w h(r)\)，解得 \(p = w (I-A^{\mathsf{T}})^{-1} h(r)\)。\(\square\)

### 5.3 利率引起的价格对物理价值的偏离

**命题 3**（利率对价格结构的扭曲）  
令物理价值比为 \(H_j^{\text{phys}}/H_k^{\text{phys}}\)，定义偏离度
\[
\Delta_{jk}(r) := \frac{H_j^{\text{PV}}(r)/H_k^{\text{PV}}(r)}{H_j^{\text{phys}}/H_k^{\text{phys}}} - 1.
\]
若 \(T_j > T_k\) 且功率剖面相似，则 \(\Delta_{jk}(r) < 0\)：长周期商品的相对价格随利率上升而下降。价格偏离物理价值，偏离方向由生产的时间结构决定。

*证明*：\(h_j(r)\) 为 \(T_j\) 的递减函数，且递减幅度随 \(r\) 增大而加剧。对 \(T_j > T_k\)，有 \(h_j(r)/h_j^{\text{phys}} < h_k(r)/h_k^{\text{phys}}\)，经 Leontief 逆传播后，\(H_j^{\text{PV}}(r)/H_j^{\text{phys}} < H_k^{\text{PV}}(r)/H_k^{\text{phys}}\)，从而 \(\Delta_{jk}(r)<0\)。\(\square\)

**推论 5**（零利率基准）  
\(r=0\) 时的价格反映纯物理价值 \(H^{\text{phys}}\)，是客观价值基准。\(r>0\) 的均衡价格偏离该基准，但物理价值 \(h_j^{\text{phys}}\) 仍是独立于利率的技术常数。

## 6 动态稳定性

**定义 16**（马歇尔价格动态）  
令 \(Z(p,w)\) 为商品超额需求函数。价格调整服从
\[
\dot{p}_j = \kappa_j Z_j(p,w), \quad \kappa_j>0,
\]
作用量价格 \(w\) 使作用量市场即时出清。

**条件 1**（局部稳定性条件）  
在均衡点 \((p^*,w^*)\) 的邻域内，雅可比矩阵 \(D_p Z(p^*,w^*)\) 负定。

**定理 10**（局部渐近稳定性）  
若条件 1 成立，则竞争均衡对上述动态局部渐近稳定。

*证明*：取李雅普诺夫候选 \(V(p)=\frac12\sum \kappa_j^{-1}(p_j-p_j^*)^2\)，则 \(\dot{V} = (p-p^*)^{\mathsf{T}} Z(p,w) \approx (p-p^*)^{\mathsf{T}} D_p Z(p^*,w^*) (p-p^*) < 0\) 对 \(p\neq p^*\)。\(\square\)

**命题 4**（竞争性可复制部门中的价值引力性质）  
在满足定理 10 局部稳定性条件且 \(Z\) 在紧集上满足 Lipschitz 条件的竞争性可复制部门经济中，对于均衡的任意小邻域，存在正测度初始价格集合使得价格路径收敛于 \(p^* = w^* H^{\text{phys}}\)，其中 \(w^*\) 为均衡作用量影子价格。物理价值 \(H^{\text{phys}}\) 是竞争性可复制部门价格的局部吸引子。

*证明*：定理 10 给出局部渐近稳定性，即存在 \(\delta>0\) 使得对满足 \(\|p_0-p^*\|<\delta\) 的所有 \(p_0\)，\(p_t\to p^*\)。由定理 8，\(p^* = w^* H^{\text{phys}}\)。该邻域具有正 Lebesgue 测度，结论成立。\(\square\)

**猜想 1**（全局引力性质）  
若马歇尔动态全局稳定且需求面临遍历性随机冲击，则竞争性可复制部门的价格向量的长期平均收敛于 \(w^* H^{\text{phys}}\)。

*评注*：全局稳定性需额外条件（如总替代性），遍历性下不变分布集中于均衡点，Birkhoff 定理保证长期平均收敛。完整证明有待后续研究。

## 7 可计算性与经验含义

### 7.1 物理价值的独立测算方案

**步骤 1：直接作用量 \(h\)**  
对每种商品 \(i\)：  
- 采集工程数据：生产周期 \(T_i\)，功率剖面 \(\phi_i(t)\) 及效率 \(\eta_i(t)\)。数据来源包括工业能源审计、LCA 数据库、传感器记录。  
- 若可得连续剖面，数值积分 \(h_i = \int_0^{T_i} (T_i-s)\phi_i(s)\eta_i(s)\,\mathrm{d}s\)。  
- 若仅有平均有效功率 \(\bar{\phi}_{\text{eff},i}\)，使用近似 \(h_i \approx \frac{1}{2}\bar{\phi}_{\text{eff},i} T_i^2\)，并通过不确定性分析评估误差。

**步骤 2：中间投入矩阵 \(A\)**  
- 构建混合单位投入产出表，行列以实物单位表示。  
- 利用物料平衡和能量平衡估算 \(a_{ij}\)。

**步骤 3：价值向量 \(H\)**  
- 计算 \(H = (I - A^{\mathsf{T}})^{-1} h\)。由于 \(A\) 和 \(h\) 完全由物理数据确定，\(H\) 外生于价格体系，构成独立的价值测度。

### 7.2 可检验的核心假说

**假说 1**（长期协整——竞争性部门）  
在可复制竞争性部门中，\(\ln p_{j,t}\) 与 \(\ln H_{j,t}\) 协整，协整向量为 \((1,-1)\)。

**假说 2**（增量解释力）  
在控制其他成本测度（劳动投入 \(L_j\)、直接能耗 \(E_j\)、资本服务 \(K_j\)）后，\(\ln H_j\) 保持显著解释力。

**假说 3**（误差修正）  
\(\Delta \ln p_{j,t}\) 对均衡偏差 \(d_{j,t-1} = \ln p_{j,t-1} - \ln H_{j,t-1}\) 的回归系数显著为负。

### 7.3 市场价格的分解模型

市场价格可表达为价值项与偏离项之和：
\[
p_j^{\text{market}} = w H_j + R_j + N_j + \mu_j,
\]
其中 \(R_j\) 为不可复制资源租金，\(N_j\) 为网络外部性租金，\(\mu_j\) 为市场势力加成。取对数线性近似：
\[
\ln p_j^{\text{market}} \approx \ln w + \ln H_j + s_j,
\]
其中 \(s_j\) 为各偏离项的对数近似。若偏离项与 \(H_j\) 相关，横截面 OLS 斜率不会严格等于 1，故检验应聚焦于协整与误差修正而非简单截面回归。

## 8 价格偏离价值的来源：租金与摩擦

### 8.1 非完全竞争与加成

设商品 \(j\) 面临市场势力，价格设为边际成本的 \(1+\mu_j\) 倍。均衡价格方程变为
\[
p_j = (1+\mu_j)\Bigl(\sum_i a_{ij}p_i + w h_j\Bigr) = (1+\mu_j) w H_j.
\]
加成异质性导致价格偏离价值比例。若 \(\mu_j\equiv\mu\) 恒定，比例关系仍保持，但绝对水平偏离。

### 8.2 功率峰值约束与峰荷租金

若瞬时总功率上界束紧，需引入峰荷租金。设 \(\max_t P(t)\le \bar{P}_{\text{peak}}\)，价格方程为
\[
p = A^{\mathsf{T}} p + w_h h + w_{\text{peak}} h^{\text{peak}},
\]
其中 \(h^{\text{peak}}_j = \max_s \phi_j(s)\eta_j(s)\)，\(w_{\text{peak}}\) 为峰荷租金率。此时相对价格不再严格正比于单一 \(H\) 向量，新增偏离项来源于峰值功率的稀缺性。

### 8.3 不可复制商品与稀缺租金

土地、矿藏、频谱等供给完全无弹性的非生产性商品，不满足公理 3。设不可复制资源集合 \(\mathcal{R}\)，每单位商品 \(j\) 需消耗资源 \(r\) 的系数为 \(l_{rj}\ge0\)。资源总存量固定为 \(\bar{R}_r\)。

**定义 17**（稀缺租金）  
不可复制资源的影子价格为稀缺租金率 \(R_r\ge 0\)，由资源市场出清条件 \(\sum_j l_{rj} x_j \le \bar{R}_r\)（束紧时取等号）内生决定。

**定理 11**（资源租金下的价格—价值关系）  
若不可复制资源约束束紧，可复制商品的市场价格满足
\[
p = A^{\mathsf{T}} p + w h + \sum_{r} R_r l_r,
\]
即
\[
p = w H + \sum_{r} R_r (I-A^{\mathsf{T}})^{-1} l_r.
\]
价格包含价值项 \(wH\) 与资源全租金项。相对价格不再纯由价值 \(H\) 决定。

*证明*：引入资源约束的拉格朗日乘子即得。\(\square\)

**注 4**（价值与价格的区分在本框架中的体现）  
在此框架中，商品的全作用量 \(H_j\) 是其**价值**——仅取决于可复制技术 \((A,h)\)，不依赖资源禀赋。市场**价格**则包含了稀缺租金 \(R_r\) 的影响。当 \(\mathcal{R}=\varnothing\) 或租金为零时，价格等于价值（正比于 \(H\)）。这种区分与古典经济学中"自然价格"和"市场价格"的辨析一脉相承，但在物理基础上给出了更清晰的界定。

### 8.4 网络外部性与垄断租金

某些商品或服务（平台、网络）的价值源于用户网络的互补性，而非单纯的生产成本。

**定义 18**（网络外部性租金）  
若商品 \(j\) 的网络规模 \(N_j\) 带来额外支付意愿，在非完全竞争市场中可形成高于价值成本的网络租金 \(\nu_j\ge0\)，使得价格
\[
p_j = w H_j + \nu_j + \sum_{r} R_r L_{rj},
\]
其中 \(L_{rj}\) 为全资源需求系数。

此类租金的严格内生化需引入网络形成博弈，此处仅指出其在价格方程中的位置。

### 8.5 普遍价格方程

综合 §8.1–§8.4，可复制商品的市场价格可写为
\[
\boxed{p_j = (1+\mu_j)\bigl(w H_j + \sum_{r} R_r L_{rj}\bigr) + \nu_j},
\]
其中：
- \(w H_j\) 为**价值项**（物理占用成本在完全竞争下的货币表现）；
- \(\sum_{r} R_r L_{rj}\) 为**不可复制资源全租金**；
- \(\mu_j\) 为**市场势力加成率**；
- \(\nu_j\) 为**网络外部性租金**。

该方程将物理价值、稀缺资源租金、网络租金和市场加成统一于同一框架内。物理价值 \(w H_j\) 构成可复制部门价格的基底——租金与加成是对价值的偏离，而非价值本身的组成部分。

### 8.6 知识、信息与零边际成本商品

**命题 5**（知识的物理表征）  
在完全竞争均衡中，任何可提高产出或降低投入的知识、算法、组织改进，其对均衡的影响完全通过技术系数 \((A, h)\) 的变化体现，即至少一个 \(a_{ij}\) 或 \(h_j\) 下降。不存在独立于 \((A, h)\) 的"知识贡献"需要单独定价。

**命题 6**（零边际成本商品的价值与价格）  
若某商品的复制边际占用作用量为零（如软件、数据、AI 模型副本），则在竞争均衡中其价值 \(H_j\) 仅由首次生产成本决定，其边际价值（即增量价值）为零。竞争性市场价格趋于零。现实中正价格来源于知识产权制度或自然垄断，属于制度性租金 \(\nu_j\) 的范畴，应归入普遍价格方程中的租金项而非价值项。

**定理 12**（专利价值的上界）  
若某项可专利知识将商品 \(j\) 的直接作用量系数从 \(h_j\) 永久降至 \(h_j'<h_j\)（中间投入系数亦可随之改变），导致价值从 \(H_j\) 降至 \(H_j'\)，且新技术受专利保护。在竞争性要素市场中，该专利的经济价值不超过其在寿命 \(T_p\) 内所节约价值成本的贴现值：
\[
V_{\text{patent}} \le \sum_{t=0}^{T_p} \frac{w_t \cdot (H_j - H_j') \cdot x_{j,t}}{(1+r_t)^t},
\]
其中 \(w_t\) 为第 \(t\) 期作用量价格，\(x_{j,t}\) 为适用产出规模，\(r_t\) 为贴现率。实际价值为 \(V_{\text{patent}} = \lambda\) 乘以上述上界，\(\lambda\in(0,1]\) 取决于保护强度与谈判能力。

*证明*：专利租金为成本节约的资本化价值，竞争性要素市场下成本节约完全由作用量节约捕获。\(\square\)

**开放问题 1**（知识生产的内生化）  
信息处理受 Landauer 原理约束，知识生产原则上可纳入作用量核算。但知识作为非竞争性商品（复制边际作用量为零）在竞争均衡中价格为零，现实中的正价格源于制度安排。其严格内生化——包括知识如何改变 \((A,h)\) 的动态过程——列为开放问题。

### 8.7 多原始要素扩展

若劳动供给因人口约束不能完全内生，可引入第二原始要素，价格体系变为
\[
p = A^{\mathsf{T}} p + w_h h + w_l l,
\]
其中 \(l\) 为外生劳动投入系数。此时价值 \(H\) 仍由 \(h\) 定义，但价格不再严格正比于 \(H\)——劳动稀缺性引入额外的偏离项 \(w_l (I-A^{\mathsf{T}})^{-1} l\)。本理论可视为劳动供给完全内生时的极限情形。

## 9 结论

本文构建的"生产时空占用作用量价值—价格理论"核心逻辑为：

1. **物理稀缺的重新定义**：经济系统受总功率上界与时间不可逆性支配。占用作用量——有效功率按剩余占用时长线性加权的积分——被确立为基本稀缺度量。其泛函形式由线性、时间平移不变性与量纲协变性唯一确定（定理 1），能区分相同能量、不同时间占用的路径（命题 2），且在所有可行路径中存在由最小能量和最大功率决定的下确界（命题 1）。时期总作用量禀赋由物理上界严格限定（定理 2）。

2. **价值实体的确立**：占用作用量构成商品**价值**的物理实体——价值是凝结在商品中的全部直接与间接占用作用量之和，由技术条件 \((A, h)\) 唯一确定（定义 10，定理 6）。价值先于价格、独立于价格（公理 2）。

3. **可复制部门的唯一原始价值来源**：在完全再生产性下，劳动、资本等可再生要素均可还原为中间品与占用作用量消耗。占用作用量被确立为可复制部门唯一原始价值实体（公理 3，定理 3–5）。

4. **自然价格定理**：在无不可复制资源、无联合生产、完全竞争条件下，可复制商品的市场价格正比于其价值 \(H=(I-A^{\mathsf{T}})^{-1}h\)（定理 8）。相对价格由物理技术唯一决定。此时价格等于价值。

5. **价格偏离价值的系统分析**：引入不可复制资源产生稀缺租金 \(R\)，非完全竞争产生加成 \(\mu\)，网络外部性产生网络租金 \(\nu\)，形成普遍价格方程 \(p_j = (1+\mu_j)(w H_j + \sum_{r} R_r L_{rj}) + \nu_j\)。价值 \(H\) 提供价格中独立于需求与制度的客观物理基底，租金与加成是对价值的偏离。

6. **跨期扩展**：明确区分物理价值 \(H^{\text{phys}}\) 与贴现价值 \(H^{\text{PV}}(r)\)，利率通过改变贴现权重使价格偏离物理价值基准，但物理价值本身保持不变（定理 9，命题 3）。

7. **动态稳定性**：在竞争性可复制部门中，物理价值是价格的局部吸引子（命题 4）。全局引力性质列为有待进一步研究的猜想（猜想 1）。

8. **经验可操作性**：所有物理变量均可通过工程数据和实物投入产出表独立于价格体系测算，构成严格外生的价值测度，可通过协整与误差修正模型进行检验。

该理论将劳动价值论与能值理论统一于更具物理一般性的框架内：劳动是占用作用量的一种特殊表现形式，单纯能量度量因忽略时间占用而信息不全。占用作用量——有效功率与占用时长的联合度量——是可复制竞争经济中价值的物理实体，是长期价格的客观引力中心。

### B.1 随机生产路径与占用作用量的概率描述

**定义 B.1**（随机生产路径）  
给定完备概率空间 \((\Omega, \mathcal{F}, \mathbb{P})\) 和滤子 \(\{\mathcal{F}_t\}_{t\ge 0}\)。一单位商品的生产路径由 \((\tau, \phi, \eta)\) 刻画：
- 生产周期 \(\tau\) 为一停时（可为确定值 \(T\)）；
- 瞬时输入功率过程 \(\phi_t\) 为 \(\mathcal{F}_t\)-适应、取值 \([0,\bar{P}]\) 的控制过程；
- 瞬时技术效率扰动过程 \(\eta_t\) 为取值于 \((0,1]\) 的 \(\mathcal{F}_t\)-适应过程，其路径外生于生产者决策。
有效功率实现为 \(\phi^{\text{eff}}_t = \phi_t \eta_t\)。

生产者选择控制 \(\phi_t\) 最小化期望占用作用量，但受制于物理约束：\(\phi_t \le \bar{P}\) 几乎必然（a.s.），且总有效能量期望满足 \(\mathbb{E}[\int_0^\tau \phi_t \eta_t\,\mathrm{d}t] \ge E\)。

**定义 B.2**（随机占用作用量）  
对于一条随机生产路径，**实际占用作用量**为随机变量
\[
\mathcal{A}(\omega) := \int_0^\tau (\tau - s)\,\phi_s(\omega)\eta_s(\omega)\,\mathrm{d}s,
\]
其量纲仍为 \(\mathrm{J\cdot s}\)。其期望 \(\mathbb{E}[\mathcal{A}]\) 为事前物理成本度量。

### B.2 风险中性下的最小期望作用量原理

**命题 B.1**（随机最小占用作用量——风险中性情形）  
设生产周期为确定值 \(T\)，控制 \(\phi_t\) 取值于 \([0,\bar{P}]\)，效率过程 \(\eta_t\) 满足：
(i) \(\eta_t\) 为有界 \(\mathcal{F}_t\)-适应过程且与 \(\phi_t\) 独立（或更弱：\(\eta_t\) 是鞅，且控制不依赖未来信息）；
(ii) 总有效能量约束为 \(\mathbb{E}[\int_0^T \phi_t\eta_t\,\mathrm{d}t] = E\)。
若生产者目标为最小化期望占用作用量 \(\mathbb{E}[\mathcal{A}]\)，则在信息结构容许下，最优控制 \(\phi_t^*\) 为确定性的前端 bang-bang 控制：
\[
\phi_t^* = \bar{P}\quad \text{对 } t \le t_0,\qquad \phi_t^* = 0\quad \text{对 } t > t_0,
\]
其中 \(t_0\) 满足 \(\bar{P}\,\mathbb{E}[\int_0^{t_0} \eta_t\,\mathrm{d}t] = E\)。相应的最小期望占用作用量为
\[
\mathcal{A}_{\min}^{\text{exp}} = \frac{E^2}{2\bar{P}} \cdot \frac{1}{\bar{\eta}},\quad \text{若 } \mathbb{E}[\eta_t] \equiv \bar{\eta} \text{ 为常数},
\]
而在效率为独立同分布鞅且均值为 \(\bar{\eta}\) 的一般情形下，最小期望作用量上确界为 \(E^2/(2\bar{P}\bar{\eta})\)，确定性 benchmark 下确界为 \(E^2/(2\bar{P})\)。但关键结论是：**最优控制的确定性与集中性结构不变，期望作用量由物理参数 \((E, \bar{P}, \bar{\eta})\) 唯一确定，不依赖于主观偏好**。

*证明思路*：由注 1，\(\mathcal{A} = \int_0^T E^{\text{rem}}_t\,\mathrm{d}t\)，其中 \(E^{\text{rem}}_t = \int_t^T \phi_s\eta_s\,\mathrm{d}s\)。取期望并由 Fubini 定理：
\[
\mathbb{E}[\mathcal{A}] = \mathbb{E}\left[ \int_0^T (T-s) \phi_s\eta_s\,\mathrm{d}s \right] = \int_0^T (T-s)\,\mathbb{E}[\phi_s\eta_s]\,\mathrm{d}s.
\]
若控制 \(\phi_s\) 是 \(\mathcal{F}_s\)-适应的，且 \(\eta_s\) 与 \(\phi_s\) 独立（或控制不利用效率的未来信息），则 \(\mathbb{E}[\phi_s\eta_s] = \mathbb{E}[\phi_s]\,\mathbb{E}[\eta_s]\)。最小化 \(\int (T-s)\mathbb{E}[\phi_s]\mathbb{E}[\eta_s]\,\mathrm{d}s\) 等价于将 \(\mathbb{E}[\phi_s]\) 集中在权重 \((T-s)\mathbb{E}[\eta_s]\) 最大的时刻。若 \(\mathbb{E}[\eta_s]\) 恒定，权重严格随 \(s\) 递减，故 \(\mathbb{E}[\phi_s]\) 应全部集中在 \(s=0\) 附近，即 \(\phi_s = \bar{P}\) 在初始段。若 \(\mathbb{E}[\eta_s]\) 为常数 \(\bar{\eta}\)，则最优解为确定性 bang-bang，最小期望作用量恰为 \(E^2/(2\bar{P}\bar{\eta})\)。若 \(\mathbb{E}[\eta_s]\) 时变，最优前端投入位置需微调，但仍为 bang-bang 型，期望作用量由技术分布客观决定。\(\square\)

**注 B.1**（风险中性假定的合理性）  
占用作用量为纯粹物理成本，在完全竞争、自由进入且无资本约束的理想情形下，企业以最小化期望物理消耗为目标。风险规避或融资约束产生的风险溢价属于市场摩擦，应归入价格方程中的加成项 \(\mu_j\) 或租金项，不改变商品的物理价值。

### B.3 随机环境中的可逆价值结构

**定义 B.3**（随机直接作用量）  
商品 \(j\) 的**直接占用作用量** \(h_j\) 定义为其在最优随机控制下的最小期望占用作用量：
\[
h_j := \min_{(\phi,\tau)} \mathbb{E}\left[ \int_0^\tau (\tau-s)\,\phi_s\eta_s\,\mathrm{d}s \right],
\]
受制于期望有效能量约束与功率上界，取在给定技术分布下的最小值。该值由技术分布的物理参数唯一确定，是客观常数（量纲 \(\mathrm{J\cdot s}\)）。

**推论 B.1**（价值向量的保持）  
若所有商品的生产满足命题 B.1 的条件，且中间投入系数 \(A\) 为确定性矩阵，则全占用作用量向量（价值）\(H\) 仍由方程 \(H = (I - A^{\mathsf{T}})^{-1} h\) 唯一给定。价值结构完全可逆：\(h \mapsto H\) 是线性同构，且该映射独立于随机扰动的一阶以上矩及市场风险偏好。

*证明*：因 \(h\) 为确定值，Leontief 逆为确定线性算子，故 \(H\) 确定且可逆。价值仅经由一阶矩（期望）吸收物理不确定性，高阶不确定性被截断在价值定义之外。\(\square\)

**定理 B.1**（随机竞争均衡中的自然价格）  
设风险中性的完全竞争经济中，代表性企业面临定义 B.3 的随机技术，无不可复制资源，无联合生产。则均衡价格向量 \(p\) 满足
\[
p = w H,
\]
其中 \(w\) 为作用量的均衡影子价格。相对价格正比于随机环境下的期望价值，与需求结构无关。

*证明*：零利润条件要求价格等于最小期望成本。对商品 \(j\)，其最小期望总成本由递归结构给出：\(p_j = \sum_i a_{ij}p_i + w h_j\)。因 \(h_j\) 为最小期望直接作用量，该方程对所有正生产的商品成立。矩阵形式 \(p = A^{\mathsf{T}}p + w h\)，解即 \(p = w H\)。\(\square\)

### B.4 风险规避与价格偏离：保持可逆性的分离结构

若企业风险规避（如效用为 \(\mathbb{E}[u(\pi)]\)，\(u\) 凹），则均衡价格将包含风险溢价，偏离 \(wH\)。引入如下分离：

**定义 B.4**（确定性等价作用量与风险溢价）  
将风险规避下的零利润条件写为
\[
p_j = \sum_i a_{ij}p_i + w \big(h_j + \rho_j(\gamma)\big),
\]
其中 \(\rho_j(\gamma)\ge 0\) 为风险溢价作用量（货币化后表现为价格加成），依赖于风险厌恶系数 \(\gamma\) 与随机扰动的分布。全价格方程为
\[
p = w (I - A^{\mathsf{T}})^{-1} (h + \rho(\gamma)).
\]
此时，物理价值 \(H = (I-A^{\mathsf{T}})^{-1}h\) 仍为定义的客观核心，价格偏离由 \(\rho(\gamma)\) 经 Leontief 逆传播。价格向量不再正比于价值 \(H\)，但价值结构仍可逆且独立于 \(\gamma\)：无论风险偏好如何，\(h\) 与 \(H\) 之间的映射不变，且价格可表达为 \(p = w H + w (I-A^{\mathsf{T}})^{-1} \rho\)。偏离项完全分离在第二项中，与 §8 的普遍价格方程结构一致。

**结论 B.1**（可逆价值结构的保持）  
随机控制扩展**不破坏**原有体系的以下核心性质：
1. **价值的客观性**：\(h_j\) 由技术分布的一阶矩及功率约束客观决定，不依赖效用函数形式。
2. **价值的线性可逆性**：\(H = (I-A^{\mathsf{T}})^{-1}h\) 为双射，给定投入产出结构即可从直接成本唯一还原全成本，反之亦可。
3. **价值与价格的范畴分离**：风险溢价、随机租金等全部进入价格偏离项，价值保持为纯物理成本的固定点。
4. **引力中心性质**：在长期中，若风险溢价随竞争消失（\(\rho\to 0\)）或可被套保，则价格收敛于 \(wH\)，物理价值仍为吸引子。

因此，本框架可自然容纳生产侧的一阶矩随机性，且价值结构保持严格可逆。更高阶矩的风险效应构成价格偏离的一个新来源，丰富了普遍价格方程的构成，但不影响价值作为物理基石的独立性。

**附录 C：价值锚、宏观周期与价格波动最小化**

本附录论证一个核心命题：在本理论体系内，若价格形成机制趋近于完全竞争均衡（即价格正比于物理价值 \(H\)），则宏观经济中可复制部门的价格波动达到最小——相对价格恒定，总体价格水平波动仅源于计价物选择。任何偏离价值的定价规则（租金、加成、利率扭曲）均会引入额外波动。因此，物理价值构成了宏观周期中价格波动最小化的基准。


### C.1 宏观价格波动的度量

**定义 C.1**（价格波动度量）  
给定长度为 \(T\) 的时期，可复制商品价格向量 \(p_t\) 的波动由平均对数方差矩阵的迹度量：

\[
\mathcal{V}_T := \frac{1}{T} \sum_{t=1}^T \| \ln p_t - \overline{\ln p} \|^2,
\]

其中对数运算对分量分别定义，\(\overline{\ln p}\) 为样本均值。该度量捕捉了相对价格与绝对价格的综合波动。

**定义 C.2**（价值基准计价物）  
选取一篮子商品组合，其价值权重 \(b \in \mathbb{R}^n_{++}\) 满足 \(b^{\mathsf{T}} H = 1\)，定义**价值基准价格水平** \(w_t^* := b^{\mathsf{T}} p_t\)。以该组合为计价物时，商品 \(j\) 的**真实价格**为 \(p_{j,t}/w_t^*\)，其对数形式记为 \(q_{j,t}\)。


### C.2 完全竞争情形下的零相对价格波动

**命题 C.1**（自然价格下的波动最小性）  
设经济满足定理 8 的条件（无不可复制资源、无摩擦、完全竞争），且技术 \((A, h)\) 在样本期内不变。则：
1. 相对价格恒为常数：\(p_{j,t}/p_{k,t} = H_j/H_k\)，不随时间变化；
2. 若采用价值基准计价物，则 \(q_{j,t} = \ln H_j\) 完全恒定，因此 \(\mathcal{V}_T = 0\)，价格波动为零；
3. 若采用其他计价物（如货币单位），则价格波动完全由 \(w_t\) 的波动引起，且该波动与商品相对稀缺性无关，仅反映宏观总需求与禀赋的时变。

*证明*：由定理 8，\(p_t = w_t H\)，其中 \(w_t\) 为时期 \(t\) 的作用量影子价格。因此对所有 \(j,k\)，\(p_{j,t}/p_{k,t} = H_j/H_k\) 与 \(t\) 无关。取价值基准计价物 \(b^{\mathsf{T}} H = 1\)，则 \(w_t^* = b^{\mathsf{T}} (w_t H) = w_t\)，故 \(q_{j,t} = \ln(w_t H_j / w_t) = \ln H_j\) 恒定。\(\square\)

**推论 C.1**（无摩擦可复制经济的价格完全可预期性）  
在此理想情形下，市场参与者在期初即可完全预期所有商品的相对交换比例，唯一不确定性来自宏观价格水平 \(w_t\)，而该水平可由总作用量禀赋 \(\bar{\mathcal{A}}_t\) 与偏好参数通过一般均衡唯一确定。不存在相对价格层面的意外冲击，故企业定价、消费者决策均无信息摩擦，资源配置完全平滑。


### C.3 租金与摩擦引入的价格波动

当存在不可复制资源租金、市场势力加成或网络外部性时，价格方程为

\[
p_{j,t} = (1+\mu_{j,t}) \big( w_t H_j + \sum_r R_{r,t} L_{rj} \big) + \nu_{j,t}.
\]

将各偏离项记为 \(\varepsilon_{j,t}\)，使得 \(p_{j,t} = w_t H_j + \varepsilon_{j,t}\)。

**命题 C.2**（偏离项对波动的贡献）  
设 \(H\) 恒定，\(w_t\) 有界，偏离项 \(\varepsilon_{j,t}\) 为平稳过程且与 \(H_j\) 不完全相关。则整体价格波动 \(\mathcal{V}_T\) 可分解为：

\[
\mathcal{V}_T = \text{Var}(\ln w_t) \cdot \|\mathbf{1}\|^2
+ \frac{1}{T}\sum_t \| \ln(1+\varepsilon_{j,t}/(w_t H_j)) \|^2
+ \text{cross-term},
\]

其中第二项严格非负，且当且仅当所有 \(\varepsilon_{j,t}=0\) 时为零。因此，任何时变偏离项必然增大价格波动。

*证明*：取对数展开 \(\ln p_{j,t} = \ln w_t + \ln H_j + \ln(1 + \varepsilon_{j,t}/(w_t H_j))\)。代入波动度量即得分解，交叉项在 \(H\) 恒定且 \(\varepsilon\) 与 \(H\) 相关时不为零，但平方和项恒为正。零偏离时退化至命题 C.1 情形。\(\square\)

经济含义：不可复制资源的稀缺租金随需求周期变化（如土地租金在经济繁荣时上升），市场势力加成随竞争强度改变（反周期或顺周期），网络租金随用户规模内生波动。这些时变项通过两种渠道增加宏观价格波动——直接增加相对价格噪声，以及经由投入产出网络间接传播。


### C.4 价值收敛机制的内生波动衰减

命题 4 与猜想 1 表明，在马歇尔价格动态下，竞争性可复制部门的价格受引力作用趋于 \(w H\)。这意味着即使存在短期偏离，市场内生调整会压缩波动。

**命题 C.3**（误差修正与波动衰减）  
设对数价格遵循误差修正过程：

\[
\Delta \ln p_{j,t} = -\alpha_j (\ln p_{j,t-1} - \ln H_j - \ln w_{t-1}) + \xi_{j,t},
\]

其中 \(\alpha_j \in (0,1)\)，\(\xi_{j,t}\) 为均值为零的短期冲击。则长期平稳分布下，\(\ln p_{j,t}\) 围绕 \(\ln H_j + \mathbb{E}[\ln w_t]\) 的方差有界，且当 \(\alpha_j \to 1\) 时波动趋近于命题 C.1 的零相对波动情形。误差修正项的存在使得波动低于纯随机游走情形。

*证明*：将过程改写为 \(\ln p_{j,t} - \ln H_j = (1-\alpha_j)(\ln p_{j,t-1} - \ln H_j) + \alpha_j \ln w_{t-1} + \xi_{j,t}\)。若 \(\ln w_t\) 平稳，则这是一个平稳 AR(1) 过程，方差为 \(\text{Var}(\xi_{j,t}) / (1 - (1-\alpha_j)^2)\) 加上 \(w_t\) 的贡献。\(\alpha_j\) 越大，自相关越低，方差越小。\(\alpha_j=1\) 时，价格瞬时调整至价值基准，仅剩 \(w_t\) 波动。\(\square\)

该命题证明了价值锚的存在通过误差修正机制内生地降低了价格波动。市场竞争越激烈、价格调整越灵活（\(\alpha_j\) 越大），波动衰减越快。


### C.5 宏观稳定政策含义：以价值为锚

**定义 C.3**（价值锚定价格规则）  
假定货币当局或市场制度使价格满足：\(p_{j,t} = \bar{w} H_j + \delta_{j,t}\)，其中 \(\bar{w}\) 为恒定作用量价格水平，\(\delta_{j,t}\) 为容许的小幅随机扰动（如测量误差），且具有零均值。则称价格体系**锚定于物理价值**。

**定理 C.1**（价值锚定的波动最小化性质）  
在技术 \((A, h)\) 稳定的可复制部门中，锚定于价值的定价规则在所有可行的定价规则中，使对数价格波动的长期均值达到下确界：

\[
\min_{\{p_t\}} \lim_{T\to\infty} \mathcal{V}_T = 0,
\]

该下确界在价值锚定且 \(\bar{w}\) 为常数的条件下取得。任何使价格系统偏离 \(H\) 比例关系的规则均严格增大概率。

*证明*：由命题 C.2，偏离项 \(\varepsilon_{j,t}\) 的非零方差对 \(\mathcal{V}_T\) 有正向贡献。锚定规则令 \(\varepsilon_{j,t}\) 的长期均值为零且方差最小（仅容许不可避免的微扰），故达到下确界。若技术 \(A, h\) 改变，\(H\) 随之变化，但仍保持价格与当期价值的比例关系，则相对价格波动仍为零（只是水平可能跳变）。\(\square\]


### C.6 结论

综上，本理论的逻辑推导直接给出：**价格的定义若严格对应竞争均衡下的自然价格（即正比于价值），则整体波动趋于最低。** 这是物理价值作为引力中心的直接推论，并非额外引入的规范标准，而是完全竞争经济内在稳定性在价格度量上的体现。宏观周期中观察到的价格波动，其根源恰在于市场结构、资源稀缺租金、跨期利率等因素造成的价格对价值的偏离。因此，使价格转向价值（消除摩擦），就是使宏观波动趋于最小的物理-经济路径。


**附录 D：实证验证方案设计**

本方案基于理论核心假说——在可复制竞争性部门，市场价格与物理价值存在长期协整关系，且价值是价格的引力中心——设计一套可操作的验证流程。全部计算仅依赖公开可获取的投入产出表、能源统计与价格指数，无需依赖理论自身的循环推导。


### D.1 部门选择与数据来源

**部门选择标准**  
选取制造业中满足以下条件的部门（尽可能逼近“可复制、竞争性”假定）：
1. 产品同质化程度高，市场集中度低（HHI < 1000，或 CR4 < 40%）；
2. 生产要素可完全再生产，不高度依赖不可移动的自然资源（排除采矿、初级金属冶炼中严重依赖矿藏品位的子类）；
3. 技术相对稳定，无频繁颠覆性创新。

建议部门示例（ISIC Rev.4）：  
- C13 纺织品  
- C14 服装  
- C17 纸及纸制品  
- C22 橡胶及塑料制品  
- C25 金属制品（不含机械设备）  
- C28 机械设备（部分细分）  
这些部门中间投入链清晰，能源消耗数据可获取，且价格指数连续。

**数据来源**

| 数据类别 | 来源 | 说明 |
|---------|------|------|

### D.2 变量构造

#### D.2.1 直接作用量 \(h_j\) 的近似计算

方法一（精准路径，数据要求高）：  
从 LCA 数据库或工厂能源审计中获取典型产品单位产出的功率剖面 \(\phi_j(t)\) 和生产周期 \(T_j\)，数值积分
\[
h_j = \int_0^{T_j} (T_j - t)\,\phi_j(t)\,\mathrm{d}t.
\]
若无法获得连续剖面，用平均有效功率 \(\bar{P}_{\text{eff},j}\) 近似：
\[
h_j \approx \frac{1}{2} \bar{P}_{\text{eff},j} T_j^2.
\]

方法二（宏观估算，适用于部门层面）：  
令部门 \(j\) 的年能源消费总量为 \(E_j^{\text{year}}\)（焦耳），年产出量为 \(X_j\)（实物单位或不变价），假设生产过程在一年内均匀进行（相当于连续批量生产），则单位产品直接消耗的有效能量 \(e_j = E_j^{\text{year}} / X_j\)。若进一步假设典型生产周期为 \(T_j\)（天），年工作日为 \(D\) 天，则平均功率 \(\bar{P}_j = e_j / T_j\)（时间单位统一）。由此：
\[
h_j \approx \frac{1}{2} \bar{P}_j T_j^2 = \frac{1}{2} e_j T_j.
\]
此公式直观：直接作用量与单位能耗乘以生产周期成正比，量纲为 \(\mathrm{J\cdot s}\)。

注意：此方法忽略了生产周期内功率分布不均匀带来的误差，但在部门平均水平上具有可用性。\(T_j\) 可通过行业平均制造周期调研获取。

#### D.2.2 价值向量 \(H\) 的计算

利用投入产出表得到直接消耗系数矩阵 \(A\)（方阵，满足 Hawkins-Simon 条件）。计算 Leontief 逆：
\[
H = (I - A^{\mathsf{T}})^{-1} h.
\]
计算需注意：
- \(A\) 矩阵应尽可能为“产品×产品”表，避免产业部门异质性偏差。
- 若投入产出表只有价值型，需将其转化为实物型或假定价格比例已反映物理消耗比例（在检验中此假定会引入一定的循环偏差，须在稳健性检验中用能源实物消耗校正）。

#### D.2.3 价格变量

取部门产出价格指数的对数 \(p_{j,t} = \ln(\text{PPI}_{j,t})\)。基准年份对齐投入产出表年份，以便与 \(H_j\) 对应。

#### D.2.4 偏离项代理

根据普遍价格方程，我们需要控制可能使价格偏离价值的因素：
- 行业集中度 \( \text{CR4}_{j,t} \) 或 赫芬达尔指数 \(\text{HHI}_{j,t}\)（代理 \(\mu_j\)）；
- 土地价格指数或资源租金代理（针对不可复制资源，如能源价格指数，代理 \(R_r\)）；
- 研发强度或专利存量（代理知识租金，命题 6）；
- 利率或资本成本（代理时间偏好扭曲，命题 3）。

### D.3 检验步骤

#### 步骤 1：面板单位根检验

对 \(\ln \text{PPI}_{j,t}\) 和 \(\ln H_{j,t}\)（价值随时间变化吗？若技术(A,h)在样本期内变化，则 \(H_{j,t}\) 可逐年重算）进行面板单位根检验（LLC、IPS、Hadri），确认两者均为 I(1) 过程。若技术变化缓慢，可将基准年 \(H_j\) 视为时不变，或使用技术变化调整后的 \(H_{j,t}\)。

#### 步骤 2：协整检验与长期关系估计

采用 Pedroni 面板协整检验或 Westerlund 误差修正模型检验，检验 \(\ln p_{j,t}\) 与 \(\ln H_{j,t}\) 是否存在协整关系。估计协整方程：
\[
\ln p_{j,t} = \alpha_j + \beta \ln H_{j,t} + u_{j,t}.
\]
核心假说：\(\beta = 1\)。利用 FMOLS 或 DOLS 估计组平均系数，并做 Wald 检验 \(H_0: \beta=1\)。

#### 步骤 3：误差修正模型（ECM）

估计面板误差修正模型：
\[
\Delta \ln p_{j,t} = \gamma_j + \lambda_j (\ln p_{j,t-1} - \hat{\beta} \ln H_{j,t-1}) + \sum_k \theta_k \Delta \ln p_{j,t-k} + \varepsilon_{j,t}.
\]
核心假说：\(\lambda_j < 0\)，表明价格向价值回归。可进一步检验若控制偏离项代理（集中度、资源租金）后，\(\lambda_j\) 是否更显著（即价值仍为吸引子）。

#### 步骤 4：偏离项的影响分析

在 ECM 或协整方程中加入时变偏离项：
\[
\ln p_{j,t} = \alpha_j + \beta \ln H_j + \delta_1 \text{HHI}_{j,t} + \delta_2 \text{ResourceRent}_{j,t} + \delta_3 \text{RDintensity}_{j,t} + u_{j,t}.
\]
预期：加入控制后，\(\beta\) 仍接近 1，且这些偏离项系数显著，调整的 \(R^2\) 提高。这验证了价格 = 价值 + 偏离的结构。

#### 步骤 5：波动性比较（附录 C 命题）

将部门按其竞争程度或租金强度分组，计算每组内相对价格波动率（对数价格方差）。预期：高竞争、低租金部门相对价格波动率显著低于高集中度、高资源依赖部门。进一步，以价值比 \(H_j/H_k\) 为基准，计算实际相对价格与价值基准的偏离方差，该方差应与租金代理正相关。

### D.4 基于已有数据的快速示范计算（示例）

**选取美国制造业数据作为示范：**

1. **投入产出表**：使用 BEA 2012 基准表（405 部门），聚合至约 20 个制造业部门，提取直接消耗系数矩阵 \(A\)。
2. **能源消耗**：使用 EIA Manufacturing Energy Consumption Survey (MECS) 2014，获取各部门总能源消耗（Trillion Btu），转为焦耳。
3. **产出**：使用 BEA 总产出（Gross Output）对应部门。
4. **生产周期**：使用美国人口普查局 Annual Survey of Manufactures 中的“inventory turnover”或典型制造周期（如纺织品 30 天，机械设备 60 天）作为近似。
5. **计算 \(h\)**：\(e_j\) = 总能耗 / 总产出，\(h_j \approx 0.5 \times e_j \times T_j\)。
6. **计算 \(H\)**：使用 Python 或 R 求逆。
7. **价格**：BLS PPI 行业数据，取 2012=100 的对数。
8. **初步检验**：画出 \(\ln p_{j,2019}\) 对 \(\ln H_j\) 的散点图，观察斜率；进一步可做时间序列协整。

即使仅用一年横截面，也可检验 \(\ln p_j\) 与 \(\ln H_j\) 的相关性是否强于与直接能耗 \(e_j\) 或劳动投入的相关性（假说 2）。

### D.5 方案可行性总结

| 要素 | 数据可获得性 | 计算复杂度 |
|------|-------------|-----------|
| 投入产出表 A | 公开、定期更新 | 低（矩阵求逆） |
| 直接作用量 h | 需能源统计 + 周期估算 | 中（积分近似） |
| 价值 H | 由 A, h 计算 | 低 |
| 价格指数 | 公开、月度/年度 | 低 |
| 偏离代理 | 集中度、资源租金等可从公开数据库获取 | 低至中 |
| 计量检验 | 标准面板协整、ECM | 中 |

该方案完全基于观测数据，不引入理论自由参数，可证伪，且结论将对理论提供强力支撑或明确反驳。建议先从单个国家（如中国、美国）的制造业面板数据入手进行初步验证，再扩展至跨国投入产出数据库（如 WIOD）以检验跨区域稳健性。
