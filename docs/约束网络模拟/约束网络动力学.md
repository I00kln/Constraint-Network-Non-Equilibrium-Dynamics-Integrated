# 约束关系网络动力学：从异步关联到涌现时空的统一框架

## 序言：第三代经典力学

本框架致力于构建一种比量子力学和广义相对论更底层的经典理论。其基本本体不是物质点，不是时空流形，也不是波函数，而是 **非交换的关系事件**。每个关系事件编码一对基本单元之间的因果影响幅度。整个网络的自组织动力学同时产生了时间、空间、度规和引力。这是继牛顿力学（质点与绝对时空）和爱因斯坦/玻尔力学（相对论时空与量子算符）之后的**第三代经典力学**。

引力不是基本力，而是关系网络在自组织过程中退相干所涌现的宏观几何印记。

---

## 第一部分：基本公设

### 公设 0（本体论）
世界由一个**有限的有向图** \(G=(V,E)\) 描述。节点 \(i\in V\) 不是空间点，而是纯粹的区别性指标。每条有向边 \((i\to j)\) 携带一个**非交换的复数强度** \(\hat{R}_{ij}\)，编码从 \(i\) 到 \(j\) 的因果影响振幅。在最低能量近似下，\(\hat{R}_{ij}\) 可以分解为：
\[
\hat{R}_{ij} = \sqrt{q_{ij}} \; e^{i A_{ij}}
\]
其中 \(A_{ij}\in U(1)\)（相位），\(q_{ij}>0\)（强度模方），并满足非交换泊松括号 \(\{A_{ij}, E^{kl}\}=\delta_i^k\delta_j^l-\delta_j^k\delta_i^l\)，其中 \(E^{ij}=q_{ij}\) 的共轭动量。这实现了最初构想的非交换性（通过泊松括号）和复值关系。

### 公设 1（动力学：局域异步更新）
系统**没有全局时间参数**。演化由一种**局域异步随机过程**驱动，该过程在算法上等价于对每个节点/边随机顺序执行三种基本操作：
1. **对称化**：调整 \(A_{ij}\) 和 \(A_{ji}\) 使其趋于共轭（模拟因果互反性）；
2. **剪枝**：删除强度低于阈值的边；
3. **重连**：以一定概率在节点间添加新边或改变连接。

该异步过程的数学等价描述是**随机辛积分器**：在连续时间极限下，它退化为由哈密顿量 \(H_{\text{tot}}\) 生成的经典动力学方程，其中时间参数 \(\tau\) 是约束面内的序参数（见公设 3）。

### 公设 2（约束系统）
所有物理状态必须满足以下第一类约束（泊松括号在约束面上闭合）：
- **高斯约束**（规范对称）：
  \[
  \mathcal{G}_i = \sum_{j\in\text{star}(i)} E^{ij} \approx 0
  \]
  生成局域 \(U(1)\) 变换，消除冗余自由度。
- **扩散约束**（标度对称，强制网络趋向稳定标度）：
  \[
  \mathcal{S}_{ij} = p^{ij} q_{ij} - \hbar_{\text{eff}} \approx 0
  \]
  其中 \(p^{ij}\) 是 \(q_{ij}\) 的共轭动量，\(\hbar_{\text{eff}}\) 是有效量子参数。
- **哈密顿约束**（时间重参数化对称）：
  \[
  \mathcal{H}_{\text{tot}} = \sum_{\langle ij\rangle} \frac12 \frac{(E^{ij})^2}{q_{ij}} - \beta \sum_{\triangle} \cos(F_\triangle) \approx 0
  \]
  其中 \(F_\triangle = A_{ij}+A_{jk}+A_{ki}\)。

**生死判据 1**：约束泊松代数必须无异常闭合，特别是 \(\{\mathcal{H}_i,\mathcal{H}_j\}\approx\) 离散微分同胚生成元。若出现异常，必须引入量子耗散项（即**退相干**）来吸收异常，这恰好对应最初构想中“退相干即弯曲”的机制。

### 公设 3（涌现时间）
在约束面上，哈密顿约束 \(\mathcal{H}_{\text{tot}}\) 作为虚拟演化生成元。物理态之间的序由**因果深度** \(\tau_i\) 定义：
\[
\tau_i = \max_{\text{paths from a reference}} \sum_{\text{edges}} \left(1 - \cos(F_\triangle)\right)
\]
该量度量从参考节点出发沿因果链传播的累积“作用”。在连续极限下，\(\tau_i\) 趋于一个光滑参数 \(\tau\)，且不同 slicing 的物理等价性由约束代数闭合保证（refoliation invariance）。异步更新的离散步骤与 \(\tau\) 的微小增量一一对应。

### 公设 4（涌现几何与维度）
- **空间距离**：两节点间的有效距离定义为
  \[
  d_{ij} = \min_{\text{paths}} \sum_{\text{edges}} |A_{ij}|
  \]
  即沿最小相位差路径累积的绝对相位。
- **空间维度**：在红外固定点上，随机游走返回概率 \(P(t) \sim t^{-d_s/2}\) 定义**谱维度** \(d_s\)。理论的现实性要求 \(d_s \to 4\)（连续极限下）且热力学维度 \(d \approx 3\)。
- **曲率**：每个三角形面上的离散 Ricci 曲率定义为
  \[
  \text{Ric}_\triangle = 1 - \cos(F_\triangle)
  \]
  它正比于该面的关系密度 \( \rho_\triangle = \frac{1}{\text{area}} \sum_{\text{edges}\in\triangle} q_{ij} \)。这直接实现了“关系密度凝聚生成曲率”的原始直觉。

## 第二部分：核心数学判据（生死线）

理论是否存活不依赖于哲学直觉，而只取决于以下可验证的判据：

1. **约束代数闭合**（离散 Dirac 代数）：在最小图（四面体）上显式计算 \(\{\mathcal{G}_i,\mathcal{G}_j\},\{\mathcal{G}_i,\mathcal{H}\},\{\mathcal{H}_i,\mathcal{H}_j\}\) 在约束面上的值。必须无异常项。若存在微小异常，则允许通过 **退相干耗散项** 吸收，此时异常的大小应正比于 \(\hbar_{\text{eff}}\) 且随 RG 流向红外衰减。

2. **重整化群固定点**：存在耦合参数 \(\beta^*\) 使得粗粒化变换 \(R(\beta)\) 在此点不动，且关联长度 \(\xi\to\infty\)。该固定点定义了连续极限，也是涌现洛伦兹对称性的必要条件。

3. **自旋‑2 集体模**：在固定点附近的线性谱中，必须存在无质量极化的横向无迹模式，对应有效引力子。Weinberg–Witten 定理被规避是因为基础层无洛伦兹对称性，且自旋‑2 模是集体激发而非基本粒子。

4. **维度涌现**：谱维度 \(d_s\) 在固定点必须趋近 4（而非 2、分形或其他值）。这是筛选物理现实性的最终条件。

## 第三部分：最小可计算模型（第一代实现）

为了测试上述判据，我们定义以下最小模型：

### 图结构
- **四面体图**：4 个节点，6 条无向边（视为两条有向边），4 个三角形面。
- **变量**：每条有向边 \((i,j)\) 携带相位 \(A_{ij}\in[0,2\pi)\) 和电场 \(E^{ij}\in\mathbb{R}\)，满足 \(A_{ji}=-A_{ij},\; E^{ji}=-E^{ij}\)。
- **哈密顿量**（Kogut–Susskind 型）：
  \[
  H = \sum_{\langle ij\rangle} \frac12 (E^{ij})^2 - \beta \sum_{\triangle} \cos(A_{ij}+A_{jk}+A_{ki})
  \]
- **高斯约束**：\(\mathcal{G}_i = \sum_{j} E^{ij} \approx 0\)。

### 经典动力学模拟（异步版本）
使用**随机异步辛积分器**：
- 在每个微观时间步，随机选择一个节点或边。
- 基于当前 \(H\) 的局部梯度更新其变量（类似 Metropolis 但保持辛结构）。
- 记录 \(\Delta_G(t)=\sum_i |\mathcal{G}_i|^2\) 和能量漂移。
- 成功判据：约束 violation 不随时间增长，能量漂移 < \(10^{-6}\) 每步。

### 代数闭合测试（解析 + 数值）
使用符号计算（SymPy）在四面体上计算 \(\{\mathcal{H}_i,\mathcal{H}_j\}\)，其中 \(\mathcal{H}_i\) 是局域哈密顿约束（例如定义在节点 \(i\) 上的平均贡献）。若结果出现非闭合项，引入退相干修正：
\[
\mathcal{H}_i \to \mathcal{H}_i + \gamma \sum_{j} \mathcal{G}_j^2
\]
调整 \(\gamma\) 使代数近似闭合。\(\gamma\) 的有限值解释为退相干强度。

### RG 粗粒化方案
- **块合并**：将四面体四个节点合并为一个超节点，新边由内部平均 \(A\) 定义。
- **有效耦合**：扫描 \(\beta\)，使用蒙特卡洛采样计算粗粒化后的 \(\beta'\)。
- **寻找固定点**：\(\beta_{n+1}=R(\beta_n)\) 的不动点。
- **关联长度**：在固定点附近计算两点关联函数 \(C(r)=\langle \cos(\sum_{\text{path}} A) \rangle\)，指数衰减长度 \(\xi\) 应随系统尺寸增大而发散。

### 谱维度测量
在固定点构型上模拟随机游走：
- 初始位置随机节点。
- 每一步等概率走到邻接节点（基于边的权重 \(e^{iA_{ij}}\) 的模方）。
- 计算返回概率 \(P(t)\)，拟合 \(P(t) \sim t^{-d_s/2}\)。
- 要求 \(d_s \to 4\) 当系统尺寸和外推至无穷时。

## 阶段 0：最小模型与代码框架（第1个月）

**目标**：实现一个可运行、可重现的四面体 U(1) 辛积分器。

### 具体任务（按周）
- **第1周**：定义图、边、面（四面体：4节点，6边，4面）。数据结构：字典 `A[(i,j)]` 和 `E[(i,j)]`，注意 `A[(j,i)] = -A[(i,j)]`，`E[(j,i)] = -E[(i,j)]`。
- **第2周**：实现哈密顿量 `H = Σ_e ½ E_e² - β Σ_△ cos(F_△)`，其中 `F_△ = A_ij + A_jk + A_ki`。
- **第3周**：实现**辛积分器**（Leapfrog / Verlet）：
  ```python
  def step(A, E, dt, beta):
      # 半步电场更新
      for e in edges:
          E[e] -= 0.5 * dt * dHdA(A, E, beta, e)
      # 全步联络更新（需 wrap to [-π, π)）
      for e in edges:
          A[e] += dt * E[e]
          A[e] = (A[e] + π) % (2π) - π
      # 再半步电场更新
      for e in edges:
          E[e] -= 0.5 * dt * dHdA(A, E, beta, e)
  ```
  其中 `dHdA` 可通过自动微分（JAX）或解析公式（对 plaquette 项求导）得到。
- **第4周**：编写观测函数：
  - `G_i = Σ_j E[(i,j)]`（注意定向）
  - `Δ_G = Σ_i G_i²`
  - 能量 `H(t)`

### 产出
- 可运行脚本 `sim_tetrahedron.py`，能输出 `(t, Δ_G, H)` 到文件。

## 阶段 Ⅰ：高斯约束漂移分析（第2–3个月）

**目标**：验证经典水平的约束保持性，确认数值伪影不会导致约束爆炸。

### 任务
1. **零初始条件测试**：从所有 `A=0, E=0` 开始，理论上 `Δ_G=0` 始终保持。验证积分器在机器精度内维持零。
2. **随机初始条件**：初始化小随机 `A, E`（例如 `A ~ Uniform(-0.1,0.1)`, `E ~ N(0,0.1)`）。计算 `Δ_G(t)` 和能量漂移。
3. **扫描 β**：β 从 0.0 到 1.5，步长 0.1。对每个 β，运行足够长时间（≥1000 单位时间，dt=0.01）。
4. **判断标准**：
   - **通过**：`Δ_G(t)` 不超过初始值的 10 倍，且无指数增长趋势；能量漂移每单位时间 < 1e-5。
   - **失败**：`Δ_G(t)` 在短时间内（< 100 时间单位）增长超过 3 个数量级 → 理论当前形式不可用，需修改哈密顿量或积分器参数（如更小的 dt）。

### 注意
- 必须使用**辛积分器**，不能用普通 ODE 求解器。若使用普通欧拉，约束漂移会很快淹没信号。
- 建议同时实现**投影法**：每若干步将 `E` 投影到满足 `G_i=0` 的子空间，作为对照实验。

### 第一个关键图
- 绘制 `log10(Δ_G(t))` vs `t`，多条曲线对应不同 β。如果曲线平坦或有界振荡 → 好消息；如果指数上升 → 理论需要修正。

## 阶段 Ⅱ：局域哈密顿约束与代数闭合（第4–5个月）

**目标**：明确构造 `H_i`，并用符号计算检验 `{H_i, H_j}`。

### 任务
1. **定义局域哈密顿约束密度**（试探性）：
   \[
   H_i = \frac{1}{\deg(i)} \sum_{e \ni i} \frac12 E_e^2 \;-\; \beta \sum_{\triangle \ni i} \cos F_\triangle
   \]
   其中 `deg(i)` 是节点 i 的度数（四面体中每个节点度为 3）。注意每个三角形面出现在三个节点上，但求和时每个三角形只计入一次。
2. **符号计算泊松括号**：
   - 使用 SymPy 定义符号变量 `A_e, E_e`（e=0..5），泊松括号 `{A_e, E_f} = δ_{ef}`。
   - 计算 `{H_i, H_j}` 的解析表达式（i≠j）。结果必然包含 `E_e` 和 `sin(F_△)` 项。
   - 检查表达式是否可写为 `Σ_k α_{ij}^k G_k + ...` 的形式。通常会出现不能表示为 `G_k` 及其导数的项 → **异常项**。
3. **数值评估异常项的大小**：在典型构型（如随机小 A,E）上计算 `{H_i, H_j}` 的数值，并与 `Σ_k G_k` 比较。
4. **决策**：
   - **如果异常项显著（>1e-3 量级）**：接受代数不闭合，转入阶段 Ⅲ，将异常解释为退相干源。
   - **如果异常项极小（<1e-6）**：可能发现了一个“意外闭合”的离散哈密顿约束，这是重大惊喜，需进一步验证。

### 关键洞见（来自你的建议）
> 大概率不闭合。这是正常现象，也是本理论可以脱颖而出的地方。

### 产出
- SymPy 笔记本 `hamiltonian_constraint_algebra.ipynb`，显示 `{H_i, H_j}` 的具体表达式。
- 一份简短说明：异常项的结构（例如是否包含 `sin(F_△)E_e` 等高阶项）。

## 阶段 Ⅲ：异常驱动的退相干与约束吸引子（第6–9个月）

**目标**：将代数异常转化为动力学耗散，并观察是否形成约束吸引子。

### 理论基础
如果 `{H_i, H_j} ≠ 0`（mod 高斯约束），那么经典哈密顿演化会违反哈密顿约束。但我们可以引入**退相干修正**：
\[
\dot{E}_e = -\frac{\partial H}{\partial A_e} - \gamma \frac{\partial}{\partial E_e} \left( \sum_i G_i^2 \right)
\]
或更简单地在辛积分器后添加一个**约束耗散步**：
\[
E_e \leftarrow E_e - \eta \cdot \frac{\partial}{\partial E_e} \left( \sum_i G_i^2 \right)
\]
这驱动系统向 `G_i=0` 流动。同样地，对哈密顿约束也可以加入类似项 `-μ ∂(Σ_i H_i²)/∂E_e`。

### 具体任务
1. **实现耗散修正**：在每个完整辛积分步后，对 `E` 执行梯度下降：
   ```python
   E[e] -= eta * d_dG2_dE[e]   # d_dG2_dE 可通过自动微分得到
   ```
   其中 `η` 是退相干系数（小正数，例如 0.01 * dt）。
2. **扫描 η**：从 1e-5 到 0.1，观察：
   - `Δ_G(t)` 是否随时间衰减并稳定在低水平。
   - `Σ_i H_i²` 是否也衰减。
3. **寻找吸引子**：在固定 `β` 下，开放长时间演化（>1e5 步），测量最终的平均约束 violation 作为 `η` 的函数。如果存在最优 `η` 使得两个约束 violation 同时最小，那可能是一个吸引子机制。
4. **RG 视角**：改变图大小（从四面体扩展到 8 节点、16 节点的小随机图），观察约束 violation 的稳态值是否随系统尺寸趋于零（即热力学极限下约束恢复）。如果是，则说明异常在红外被压制。

### 成功判据
- 存在 `η*` 使得 `Δ_G` 和 `Δ_H` 比无耗散时低 2–3 个数量级。
- 在固定点附近，系统对初始条件和 `η` 的选取不敏感（普适性）。

### 产出
- 代码 `phaseIII_decoherence.py`，能产生 `Δ_G(t, η)` 相图。
- 报告：异常驱动的退相干是否能够恢复有效约束代数。

## 阶段 Ⅳ：RG 流向与涌现几何的初步探索（第10–12个月）

**目标**：试探粗粒化对约束异常的影响，并开始测量谱维度。

### 任务
1. **定义粗粒化**：对稍大的图（例如 16 节点，规则网络或随机图），将相邻节点合并为一个块，新块的联络变量由内部平均相位定义（或更复杂的映射）。
2. **测量有效异常**：在粗粒化前后，计算局域约束泊松括号的异常项大小。看异常项是否随着粗粒化而衰减（**关键问题**：异常在 RG 下是相关还是无关算子？）。
3. **测量谱维度**（在固定点附近）：
   - 运行随机游走，计算返回概率 `P(t)`。
   - 拟合 `P(t) ~ t^{-d_s/2}`，得出 `d_s`。
   - 检查 `d_s` 是否趋近 4（对于 3+1 维涌现）。
4. **线性化谱**（可选）：如果系统已接近约束面，尝试在平均背景（`A≈0`）下展开，看是否存在无质量模。这需要构造有效度规，可能比较复杂，但可作为长期目标。

### 预期结果（乐观情况）
- 异常项在 RG 流下是**无关算子**，随着粗粒化而衰减 → 连续极限下恢复 Dirac 代数。
- 谱维度 `d_s ≈ 4` → 时空维度涌现正确。
- 存在无质量横向无迹模 → 引力子涌现。

- 阶段性报告：`RG_flow_of_anomaly.pdf` 和 `spectral_dimension.pdf`。
- 最终总结：当前模型是否满足“生死判据”？列出哪些已通过，哪些仍需改进。
