# 非线性元胞自动机的观测因子、观测熵与解码：形式化理论框架 V15

## 摘要

针对一维非线性元胞自动机在有限时空窗口下的观测推断问题，本文构建严格形式化理论，将其统一为“观测profinite因子——相对拓扑熵——约束满足解码”三层结构。在两端渐近静止构型空间上，通过有限观测划分的逆向极限得到紧致完全不连通的观测因子，其上具有诱导动力学。引入**观测时间熵率** \(h_{\mathrm{obs}}\) 与**因果溢出速率** \(\kappa\) 作为纯组合不变量，并严格证明了它们与纤维拓扑熵之间的普适不等式：
\[
h_{\mathrm{fib}}(F\mid\Theta_\infty) \;\le\; h_{\mathrm{obs}} + \kappa\cdot\log|S|.
\]
证明通过Bowen–Dinaburg生成集与**因果溢出集无干扰引理**实现，得到可分离的组合上界。建立了碰撞可逆性的约束满足问题(CSP)分类；对固定有限激发族，证明问题可归约为有限状态同步过程，有限延迟可解码性在等价关系稳定条件下可判定，一般复杂度猜想为 \(\Pi^0_2\)-完备。将观测因子嵌入 sofic 移位表示与有限型因子理论，给出有限状态表示的充分条件。全框架参数均由局部规则与观测参数通过有限步骤显式确定。Rule 110 的计算实验验证了框架的可操作性。全文严格区分已证明定理与形式化猜想。

## 1. 预备知识与基本设定

### 1.1 构型空间与全局映射

- **状态集**：有限集 \(S\)，\(|S|\ge 2\)，存在静止态 \(s_0\in S\) 满足 \(f(s_0,\dots,s_0)=s_0\)。
- **局部规则**：语法半径 \(r\ge 1\) 的映射 \(f: S^{2r+1}\to S\)，生成全局映射 \(F: S^{\mathbb Z}\to S^{\mathbb Z}\)，
  \[
  F(x)_i = f(x_{i-r},\dots,x_{i+r}),\qquad x\in S^{\mathbb Z}.
  \]
- **两端渐近静止构型空间**（紧致、完全不连通）：
  \[
  X = \Bigl\{ x\in S^{\mathbb Z} : \lim_{|i|\to\infty} x_i = s_0 \Bigr\}.
  \]
  \(X\) 是 \(F\)-不变的闭子集。记 \(X_{\mathrm{fin}} = \{x\in X : \operatorname{supp}(x) \text{ 有限}\}\)，其中 \(\operatorname{supp}(x)=\{i : x_i\neq s_0\}\)。\(X_{\mathrm{fin}}\) 在 \(X\) 中稠密。

所有讨论均在 \(X\) 上进行；观测与等价关系自动限制于 \(X\)。

### 1.2 最小依赖偏移与精确因果锥

**定义 1.1**（左、右最小依赖偏移）。对局部规则 \(f\)，定义
\[
\begin{aligned}
v_- &= \min\{\, k\in[0,r] \mid \forall a,b\in S^{2r+1}: a_i=b_i\ \forall i\ge -k \Rightarrow f(a)=f(b) \,\},\\
v_+ &= \min\{\, k\in[0,r] \mid \forall a,b\in S^{2r+1}: a_i=b_i\ \forall i\le k \Rightarrow f(a)=f(b) \,\}.
\end{aligned}
\]
\(v_-,v_+\) 通过穷举所有 \(a,b\in S^{2r+1}\) 可计算。当两个邻域在 \([-v_-, v_+]\) 上一致时，输出必然相同，且该区间是满足此性质的最小区间。

**引理 1.1**（一步依赖区间）。对任意 \(x\in X\) 与 \(i\in\mathbb Z\)，\(F(x)_i\) 完全由 \(x|_{[i-v_-,\, i+v_+]}\) 决定。

*证明*：由 \(f\) 的作用方式及 \(v_-,v_+\) 的定义直接可得。\(\square\)

对 \(t\ge 0\) 定义**精确依赖锥**：
\[
D_t(i) = [i - t v_-,\; i + t v_+].
\]

**引理 1.2**（依赖锥迭代与单调性）。对任意 \(x\in X\)、\(i\in\mathbb Z\) 与 \(t\ge 0\)，
\[
F^t(x)_i \text{ 仅依赖 } x|_{D_t(i)}.
\]
此外，\(D_t(i)\subseteq D_{t+1}(i)\) 且 \(D_{t+1}(i)=\bigcup_{j\in D_1(i)} D_t(j)\)。

*证明*：对 \(t\) 归纳。\(t=1\) 由引理 1.1 得。假设对 \(t\) 成立，则 \(F^{t+1}(x)_i = f(F^t(x)_{i-r},\dots,F^t(x)_{i+r})\)。由归纳假设每个 \(F^t(x)_j\) 仅依赖 \(x|_{D_t(j)}\)。注意到对 \(j\in[i-v_-,\,i+v_+]\) 有 \(D_t(j)\subseteq D_{t+1}(i)\)，因此 \(F^{t+1}(x)_i\) 仅依赖 \(x|_{D_{t+1}(i)}\)。\(\square\)

### 1.3 观测设定与一致性公理

取观测扩展速率 \(c\ge 1\)。观测者在时刻 \(t\) 可见空间范围 \([-ct, ct]\)。对 \(T\in\mathbb N\)，定义时空观测块：
\[
\begin{aligned}
\Theta_T &: X \to \mathcal Y_T = \prod_{t=0}^{T} S^{[-ct,\, ct]},\\
\Theta_T(x) &= \bigl( x|_{[-c\cdot0,\,c\cdot0]},\; F(x)|_{[-c\cdot1,\,c\cdot1]},\; \dots,\; F^T(x)|_{[-cT,\,cT]} \bigr).
\end{aligned}
\]

**观测一致性公理**（全文假设）：
\[
\boxed{c \ge \max(v_-, v_+)}.
\]

定义初值有效区域（所有可能影响观测窗口的细胞的最小连续区间）：
\[
E_T = [ -T(c+v_+),\; T(c+v_-) ].
\]
为便于处理对称边界，选用较宽的对称区域
\[
A_T = [-(c+v)T,\; (c+v)T],\qquad v = \max(v_-,v_+),
\]
显然 \(E_T\subseteq A_T\)。超出 \(E_T\) 的部分即为“因果溢出集”。

### 1.4 因果溢出集与无干扰引理

**定义 1.3**（因果溢出集）。对时间 \(T\)，定义
\[
I_\infty(T) = A_T \setminus E_T = \{\, i\in A_T : \forall t\in[0,T],\; D_t(i)\cap [-ct,ct] = \varnothing \,\}.
\]

**引理 1.4**（溢出集无干扰引理）。若 \(i\in I_\infty(T)\)，则对任意 \(x\in X\) 和任意状态 \(s\in S\)，修改后的构型 \(x^{(i\to s)}\)（仅在 \(i\) 处置为 \(s\)，其余与 \(x\) 相同）满足
\[
\Theta_T(x^{(i\to s)}) = \Theta_T(x).
\]

*证明*：取任意 \(t\le T\) 和 \(j\in[-ct,ct]\)。由引理 1.2，\(F^t(x)_j\) 仅依赖于 \(D_t(j)\) 上的初值。若 \(i\notin D_t(j)\)，则 \(x\) 与 \(x^{(i\to s)}\) 在 \(D_t(j)\) 上一致，从而 \(F^t(x)_j = F^t(x^{(i\to s)})_j\)。因 \(i\in I_\infty(T)\)，有 \(i\notin E_T\)，特别地 \(i\notin D_t(j)\) 对所有 \(t\le T, j\in[-ct,ct]\) 成立。故对所有 \(t,j\) 一致，\(\Theta_T\) 值不变。\(\square\)

**命题 1.5**（溢出集大小的精确公式）。设 \(v = \max(v_-,v_+)\)，则
\[
|I_\infty(T)| = T(2v - v_- - v_+) + O(1).
\]
特别地，当且仅当 \(v_-=v_+=v\) 时，\(I_\infty(T)=\varnothing\)；否则溢出集大小随 \(T\) 线性增长。

*证明*：由定义，\(i\in I_\infty(T)\) 当且仅当 \(i\in A_T\) 且对所有的 \(t\in[0,T]\)，\(D_t(i)\cap[-ct,ct]=\varnothing\)。条件等价于 \(i + t v_+ < -ct\) 或 \(i - t v_- > ct\) 对所有 \(t\) 成立。因 \(c\ge v\)，函数 \(t\mapsto -t(c+v_+)\) 与 \(t\mapsto t(c+v_-)\) 均关于 \(t\) 单调，最严格约束发生于 \(t=T\)。故 \(i < -T(c+v_+)\) 或 \(i > T(c+v_-)\)。同时 \(i\in A_T = [-(c+v)T,\,(c+v)T]\)。左溢出部分长度为 \(T(v-v_+)\)，右溢出部分长度为 \(T(v-v_-)\)，总和为 \(T(2v - v_- - v_+)\)，端点附近至多 \(O(1)\) 误差。\(\square\)

**定义 1.6**（因果溢出速率）。定义
\[
\kappa = \lim_{T\to\infty} \frac{|I_\infty(T)|}{T} = 2v - v_- - v_+.
\]

### 1.5 有效区域观测亏损与 \(\kappa=0\) 的特殊情形

为避免溢出区干扰，直接在有效区域上量化信息亏损。

**定义 1.7**（有效区域限制与最大亏损）。对每个 \(T\)，令 \(E_T = [-T(c+v_+),\, T(c+v_-)]\)。对观测块 \(w\in\Theta_T(X)\)，定义兼容纤维：
\[
\Phi_T^{\mathrm{eff}}(w) = \{\, x|_{E_T} \in S^{E_T} : \exists x\in X,\ \Theta_T(x) = w \,\}.
\]
定义
\[
\delta_T^{\max} = \max_{w\in\Theta_T(X)} \log_{|S|} |\Phi_T^{\mathrm{eff}}(w)|,
\qquad
\Delta_T^{\max} = \delta_T^{\max}\log|S|.
\]
直观上，\(\delta_T^{\max}\) 度量了给定时空观测下，有效区域内最多有多少个不可区分的初值状态（对数尺度）。

**命题 1.8**（投影兼容性）。若 \(T_1\le T_2\)，遗忘映射 \(p_{T_1,T_2}: \mathcal Y_{T_2}\to \mathcal Y_{T_1}\) 诱导投影 \(p_{T_1,T_2}: \Phi_{T_2}^{\mathrm{eff}}(w')\to\Phi_{T_1}^{\mathrm{eff}}(p_{T_1,T_2}(w'))\)。

**注记 1.9**（\(\kappa=0\) 的物理意义）。当 \(v_-=v_+\) 时，\(\kappa=0\)，此时溢出集最终为零，有效区域与包围区域一致（\(E_T = A_T\) 对所有充分大的 \(T\)）。这意味着不存在因果泄漏的自由度，所有观测不可逆性均来源于有效区域内部的组合退化，而非边界自由度。这一情形是本文熵分解结果中最清晰的情况，也是区分内部亏损与边界泄漏的分水岭。本节后续的所有一般上界在此条件下退化为纯粹的内部亏损上界。

## 2. 无限时间观测因子与纤维拓扑熵

### 2.1 全域等价关系与逆向极限

对任意 \(T\)，定义全域等价关系
\[
x \sim_T^{\text{glob}} y \iff \Theta_T(x) = \Theta_T(y),\qquad x,y\in X.
\]
商空间 \(\mathcal K_T = X/\!\!\sim_T^{\text{glob}}\) 同构于有限离散集 \(\Theta_T(X)\)。遗忘映射诱导满射 \(p_{T_1,T_2}:\mathcal K_{T_2}\to \mathcal K_{T_1}\)，构成满射逆向系统。

定义极限等价关系
\[
x \sim_\infty y \iff \forall T\ge 0,\; x \sim_T^{\text{glob}} y,
\]
商空间记为 \(\mathcal K = X/\!\!\sim_\infty\)，赋予商拓扑。

**引理 2.1**（闭等价关系）。关系 \(\sim_\infty = \bigcap_{T\ge 0} \sim_T^{\mathrm{glob}}\) 是 \(X\times X\) 中的闭子集，故商空间 \(\mathcal K\) 是紧致 Hausdorff 空间。

*证明*：每个 \(\Theta_T: X\to \mathcal Y_T\) 连续，\(\sim_T^{\mathrm{glob}} = (\Theta_T\times\Theta_T)^{-1}(\mathrm{diag})\) 是闭集（\(\mathcal Y_T\) 离散，对角线闭）。任意闭集的交仍闭。紧致空间商去闭等价关系所得空间紧致且 Hausdorff。\(\square\)

**定理 2.2**（profinite 观测因子）。\(\mathcal K\) 同胚于逆向极限 \(\varprojlim \mathcal K_T\)，因而是紧致、完全不连通、可度量化的空间。商投影 \(\Theta_\infty: X\to\mathcal K\) 是连续满射。\(\sim_\infty\) 是 \(F\)-不变的，故存在唯一连续映射 \(\widetilde F:\mathcal K\to\mathcal K\) 使下图交换：
\[
\begin{CD}
X @>F>> X\\
@V\Theta_\infty VV @V\Theta_\infty VV\\
\mathcal K @>\widetilde F>> \mathcal K
\end{CD}
\]
系统 \((\mathcal K,\widetilde F)\) 是 \((X,F)\) 的拓扑因子，称为**观测因子**。

*证明*：\(\mathcal K\) 紧致 Hausdorff，有相容满射族 \(\mathcal K\to\mathcal K_T\)，满足泛性质，故同胚于 \(\varprojlim \mathcal K_T\)。完全不连通性与可度量性由有限离散集的逆向极限标准性质给出。\(F\)-不变性：\(\Theta_T(F(x))\) 由 \(\Theta_{T+1}(x)\) 决定，故 \(\sim_\infty\) 被 \(F\) 保持。\(\square\)

### 2.2 纤维与观测相对熵的定义

**定义 2.3**（纤维）。对 \(z\in\mathcal K\)，纤维 \(\Phi(z) = \Theta_\infty^{-1}(z)\) 是闭 \(F\)-不变集。记 \(\Phi_{\mathrm{fin}}(z) = \Phi(z) \cap X_{\mathrm{fin}}\)。

**定义 2.4**（有限到一因子）。若存在常数 \(M\) 使得对一切 \(z\in\mathcal K\)，\(|\Phi_{\mathrm{fin}}(z)|\le M\)，则称观测因子在有限支撑上为**有限到一**。

**定理 2.5**（有限到一的充分条件）。若存在 \(T_0\) 使得对所有 \(T\ge T_0\)，\(|I_\infty(T)|\) 有界且序列 \(\{\delta_T^{\max}\}_{T\ge 0}\) 有界，则观测因子在有限支撑上为有限到一。

*证明*：见附录 A。

我们采用**纤维拓扑熵**作为观测相对拓扑熵的定义，因为它直接刻画纤维上的动力学复杂度。

**定义 2.6**（观测纤维拓扑熵）。
\[
h_{\mathrm{fib}}(F\mid\Theta_\infty) := \sup_{z\in\mathcal K} h_{\mathrm{top}}(F|_{\Phi(z)}),
\]
其中 \(h_{\mathrm{top}}(F|_{\Phi(z)})\) 是子系统 \(\Phi(z)\) 的拓扑熵。在紧致度量空间且纤维为闭集的标准条件下，此定义与基于生成覆盖的相对拓扑熵（Bowen 型相对熵）一致 [7,8]。特别地，标准分解不等式仍成立：
\[
h_{\mathrm{top}}(F) \le h_{\mathrm{top}}(\widetilde F) + h_{\mathrm{fib}}(F\mid\Theta_\infty). \tag{2.1}
\]

## 3. 观测时间熵率与熵分解定理

### 3.1 观测时间熵率

**定义 3.1**（观测时间熵率）。
\[
h_{\mathrm{obs}} = \limsup_{T\to\infty} \frac{1}{T}\Delta_T^{\max}.
\]

**命题 3.2**（基本性质）。\(h_{\mathrm{obs}}\ge 0\)，且不依赖于溢出区的自由度数。

*证明*：见附录 B。

### 3.2 相对熵的观测上界（主定理）

为使生成集构造严密，先建立因果包含引理。

**引理 3.3**（观测窗口的因果包含）。设 \(m = nT\)，\(A_m = [-(c+v)m, (c+v)m]\)。则对所有 \(k < m\) 及所有 \(j\in[-cT, cT]\)，有
\[
D_k(j) \subseteq A_m.
\]

*证明*：\(D_k(j) = [j - k v_-,\, j + k v_+]\)。因为 \(|j| \le cT\)，\(k \le m-1 = nT-1\)，故端点绝对值最大为
\[
cT + (nT-1)v \le cT + nT v = (c+v)Tn = (c+v)m.
\]
因此区间完全落入 \(A_m\)。\(\square\)

**定理 3.4**（纤维拓扑熵的观测上界）。在观测一致性公理下，
\[
h_{\mathrm{fib}}(F\mid\Theta_\infty) \;\le\; h_{\mathrm{obs}} + \kappa \cdot \log|S|,
\]
其中 \(\kappa = 2v - v_- - v_+\)。

*证明*：赋予 \(X\) 标准距离 \(d(x,y) = 2^{-\min\{|i| : x_i\neq y_i\}}\)（约定 \(\min\varnothing=\infty\) 则 \(d=0\)）。  
对任意 \(\varepsilon>0\)，选取 \(T = T(\varepsilon)\) 使得 \(2^{-cT-1} < \varepsilon\)。此时，若两个构型在区间 \([-cT, cT]\) 上逐点相等，则它们的距离 \(<\varepsilon\)。

固定任意纤维 \(X_z = \Phi(z)\)（\(z\in\mathcal K\)）。对任意正整数 \(n\)，取 \(m = nT\)。考虑区域 \(A_m\)、有效区域 \(E_m\) 与溢出区域 \(I_\infty(m)\)。  
由于 \(z\) 固定，观测块 \(w_{[0,m]} = \Theta_m(z)\) 唯一确定。令
\[
R_m = \{\, x|_{A_m} \in S^{A_m} : x\in X_z \,\}
\]
为纤维中构型在 \(A_m\) 上的投影全体。有效区域部分的投影数受控于 \(\delta_m^{\max}\)：
\[
|\{x|_{E_m} : x\in X_z\}| \le |\Phi_m^{\mathrm{eff}}(w_{[0,m]})| \le |S|^{\delta_m^{\max}}.
\]
溢出区域提供至多 \(|I_\infty(m)|\) 个额外自由细胞，因此
\[
|R_m| \le |S|^{\delta_m^{\max}} \cdot |S|^{|I_\infty(m)|} = |S|^{\delta_m^{\max} + |I_\infty(m)|}.
\]

为每个投影 \(\rho \in R_m\) **任意选择**一个代表元 \(y_\rho \in X_z\) 满足 \(y_\rho|_{A_m} = \rho\)（由定义存在）。令
\[
E = \{ y_\rho : \rho \in R_m \} \subset X_z.
\]
则 \(|E| \le |S|^{\delta_m^{\max} + |I_\infty(m)|}\)。

断言：\(E\) 是 \(X_z\) 的 \((m,\varepsilon)\)-生成集（关于 \(F\)）。任取 \(x\in X_z\)，其投影 \(\rho = x|_{A_m}\) 属于 \(R_m\)，取对应的 \(y = y_\rho \in E\)。则 \(x\) 与 \(y\) 在 \(A_m\) 上逐点一致。对任意 \(0\le k < m\) 和 \(j\in[-cT, cT]\)，由引理 3.3 知 \(D_k(j)\subseteq A_m\)，故 \(F^k(x)_j = F^k(y)_j\)。从而对所有 \(0\le k < m\)，\(F^k(x)\) 与 \(F^k(y)\) 在 \([-cT, cT]\) 内完全一致，于是
\[
d(F^k x, F^k y) \le 2^{-cT-1} < \varepsilon.
\]
因此 \(\max_{0\le k < m} d(F^k x, F^k y) < \varepsilon\)，\(E\) 确为 \((m,\varepsilon)\)-生成集。

由拓扑熵的生成集定义，
\[
h_{\mathrm{top}}(F|_{X_z}) = \lim_{\varepsilon\to 0}\limsup_{N\to\infty}\frac{1}{N}\log N_{X_z}(N,\varepsilon).
\]
对上述固定的 \(\varepsilon\) 和对应的 \(T\)，沿子序列 \(N = m = nT\)（\(n=1,2,\dots\)）可得
\[
\begin{aligned}
h_{\mathrm{top}}(F|_{X_z})
&\le \limsup_{n\to\infty} \frac{1}{nT} \log |E| \\
&\le \limsup_{n\to\infty} \frac{1}{nT} \bigl( \delta_{nT}^{\max} + |I_\infty(nT)| \bigr) \log|S| \\
&= \limsup_{M\to\infty} \frac{1}{M} (\delta_M^{\max}\log|S| + |I_\infty(M)|\log|S|) \\
&= h_{\mathrm{obs}} + \kappa \log|S|.
\end{aligned}
\]
该上界对所有 \(\varepsilon>0\) 成立（因上述不等式左端与 \(\varepsilon\) 无关，而右端独立于 \(\varepsilon\)），故对任意纤维 \(X_z\) 均有
\[
h_{\mathrm{top}}(F|_{X_z}) \le h_{\mathrm{obs}} + \kappa \log|S|.
\]
取关于 \(z\in\mathcal K\) 的上确界即得
\[
h_{\mathrm{fib}}(F\mid\Theta_\infty) = \sup_z h_{\mathrm{top}}(F|_{X_z}) \le h_{\mathrm{obs}} + \kappa \log|S|.
\]
证毕。\(\square\)

该定理实现了内部亏损与溢出自由度的**可分离组合上界**，但注意此为单向不等式，非等号。

**推论 3.5**（全系统熵分解）。结合 (2.1) 与定理 3.4，
\[
h_{\mathrm{top}}(F) \;\le\; h_{\mathrm{top}}(\widetilde F) + h_{\mathrm{obs}} + \kappa \cdot \log|S|. \tag{3.1}
\]
特别地，当 \(v_-=v_+\) 时 \(\kappa=0\)，
\[
h_{\mathrm{top}}(F) \;\le\; h_{\mathrm{top}}(\widetilde F) + h_{\mathrm{obs}}. \tag{3.2}
\]

**推论 3.6**（有限到一情形）。若观测因子在 \(X_{\mathrm{fin}}\) 上为有限到一且 \(\kappa=0\)，则 \(h_{\mathrm{fib}}(F\mid\Theta_\infty)=0\)，从而 \(h_{\mathrm{top}}(F) = h_{\mathrm{top}}(\widetilde F)\)，且必有 \(h_{\mathrm{obs}}=0\)。

### 3.3 观测亏损的空间密度

**定义 3.7**（归一化观测亏损密度）。
\[
\rho_{\mathrm{obs}} = \limsup_{T\to\infty} \frac{\Delta_T^{\max}}{|A_T|\log|S|} = \limsup_{T\to\infty} \frac{\delta_T^{\max}}{|A_T|}.
\]
取值范围 \([0,1]\)。与 \(h_{\mathrm{obs}}\) 分别提供空间与时间维度的信息亏损度量。

## 4. 碰撞可逆性的约束满足与解码分类

### 4.1 约束满足问题 (CSP) 的形式化

对有限集 \(A\subseteq A_T\) 与非零赋值 \(\sigma\in S^A\setminus\{s_0^A\}\)，若 \(\Theta_T(x_{A,\sigma})\neq\Theta_T(\mathbf 0)\)，则称构型 \(x_{A,\sigma}\) 为时间 \(T\) 内的**可见激发态**。  
**最小触发簇** \(\mathcal M_T\)：支撑按包含关系极小的可见激发态全体。

考虑有限个互不相交支撑的可见激发态 \(E=\{e_k=(A_k,\sigma_k)\}\)，联合支撑 \(\mathcal A=\bigcup A_k\)。固定观测时间 \(T\) 及观测块 \(w\in\Theta_T(X)\)，定义**碰撞纤维**：
\[
\Phi_T(\mathcal A; w) = \{ (\sigma_k)\in \prod_k S^{A_k} : \Theta_T(x_{(\sigma_k)}) = w \}.
\]
该集合是以下约束系统（CSP）的解：
\[
\forall t\in[0,T],\ \forall i\in[-ct,ct],\quad F^t(x)_i = w_{t,i}. \tag{4.1}
\]

**命题 4.1**（时间相容性）。若 \(T_1<T_2\)，自然限制映射 \(\rho_{T_1,T_2}: \Phi_{T_2}(\dots)\to\Phi_{T_1}(\dots)\) 是满射。解集构成逆向系统，极限
\[
\Phi_\infty(\mathcal A; w_\infty) = \varprojlim \Phi_T(\mathcal A; w^{(T)})
\]
为所有未来无限时间内产生兼容观测的初值赋值集合。

### 4.2 约束图与因果宽度

每个 CSP (4.1) 的变量为 \(\mathcal A\) 中的细胞。对每个时空约束 \((t,i)\)（\(0\le t\le T, |i|\le ct\)），令其**变量域**为
\[
V(t,i) = \mathcal A \cap D_t(i).
\]
得到一个以变量为顶点的约束超图 \(\mathcal H_T\)，其超边为所有非空的 \(V(t,i)\)。

**定义 4.2**（约束图的树宽）。令 \(\omega_T = \mathrm{treewidth}(\mathcal H_T)\)。由于变量集 \(\mathcal A\) 固定有限，对于任意 \(T\)，约束超图的顶点集不增，故树宽天然有界：
\[
\omega_T \le |\mathcal A| - 1.
\]

### 4.3 消歧性分类与有限状态同步

**定义 4.3**（消歧延迟）。给定真实构型产生的观测序列 \(w_\infty\)，若存在 \(\Delta\ge 0\) 使 \(|\Phi_{T+\Delta}(\mathcal A; w_{\le T+\Delta})| = 1\)（\(w_{\le T+\Delta}\) 为对应有限截断），则称该碰撞为 \(\Delta\)**-可消歧**。
- \(\Delta=0\)：**立即消歧**。
- 存在有限 \(\Delta\)：**有限延迟可解码**。
- 否则为**本质不可解码**。

**定理 4.4**（固定有限激发族的解码可归约性）。对于任意固定有限激发族 \(\mathcal A\)（\(|\mathcal A|=M\)），
1. **立即消歧** 可由 \(T\) 时刻 CSP 穷举判定。
2. 有限延迟可解码问题等价于在有限状态同步自动机中判定吸收态可达性。具体地，构造确定型有限自动机 \(\mathcal A\)：
   - 状态空间 \(\mathcal Q\) 为 \(S^{\mathcal A}\) 的所有子集（共 \(2^{|S|^M}\) 个），每个状态 \(q\subseteq S^{\mathcal A}\) 表示当前与所有观测兼容的赋值集合。
   - 初始状态 \(q_0 = S^{\mathcal A}\)。
   - 转移：对于状态 \(q\) 和新时刻的观测行 \(w_{t}\)（即 \(F^t(x)\) 在 \([-ct,ct]\) 上的限制），下一状态为
     \[
     q' = \{ \sigma\in q \mid \text{由 } \sigma \text{ 决定的构型在 } t \text{ 时刻的观测与 } w_t \text{ 一致} \}.
     \]
     由于观测序列由真实构型生成，每次转移后 \(q'\) 非空且单调不增。
   此自动机完整模拟了 CSP 逐时间步的兼容性筛选。真实构型对应一条无限观测序列，自动机状态序列最终收敛到极限集合 \(q_\infty \subseteq S^{\mathcal A}\)。有限延迟可解码当且仅当存在某个时刻 \(t\) 使 \(|q_t|=1\)，即自动机到达吸收态（大小为 1 的状态，且该状态在后续观测下不变）。
3. 若存在可计算的界限 \(T^*\) 使得对所有真实构型，\(q_{T^*} = q_\infty\)（即等价关系在有限时间内稳定），则有限延迟可解码性通过在此有限状态系统上搜索吸收态可判定。一般情况下，该稳定性不可保证，但有限延迟可解码性仍是半可判定的（不断递增 \(T\) 检查是否收敛到单点集）。

*证明*：归纳构造详见附录 C。\(\square\)

**推论 4.5**（因果分离零碰撞）。若激发态族支撑的依赖锥在观测窗口内互不相交，则 CSP 分解为独立子系统的直积，各子系统由零构型背景唯一确定，从而立即消歧。

### 4.4 局部拼贴引理

**引理 4.6**（局部拼贴封闭性）。设碰撞组 \(E_1,E_2\) 支撑分离。若其约束图不连通，联合 CSP 的解为直积 \( \Phi(E_1)\times\Phi(E_2) \)；若连通，则为沿共享变量的纤维积。该性质保证逐层解码的组合一致性。

## 5. 与符号动力学及因子理论的联系

### 5.1 观测语言与有限状态表示

**观测语言** \(\mathcal L_{\mathrm{obs}}\) 定义为所有有限时空块在乘积拓扑下的闭包，它是 \(\mathbb Z^2\) 上的有效闭子移位。

**定理 5.1**（有限状态表示的存在性）。若存在 \(T_0\) 使得 \(\Theta_{T_0}(x)\) 唯一决定 \(\Theta_\infty(x)\)（即等价关系在有限时刻稳定），则可构造以 \(\mathcal Q = \Theta_{T_0}(X)\) 为状态集的确定型有限自动机，其转移定义为
\[
\delta(q) = \Theta_{T_0}(F(x)),\quad \text{对任意 } x\in X \text{ 满足 } \Theta_{T_0}(x)=q.
\]
由唯一决定性质，转移是良定义的。将此自动机状态序列沿时间展开，得到的一维符号序列闭包是一个 **sofic 移位**。该 sofic 移位正是观测过程的轨道编码，给出了观测因子的一个 sofic 表示。

*证明*：若 \(\Theta_{T_0}(x)=\Theta_{T_0}(y)\)，则 \(\Theta_\infty(x)=\Theta_\infty(y)\)，进而 \(\Theta_{T_0}(F(x))=\Theta_{T_0}(F(y))\)，故转移函数不依赖代表元选择。状态序列空间是 sofic。该序列空间通过映射 \(x\mapsto (\Theta_{T_0}(F^t(x)))_{t\ge 0}\) 与观测因子拟共轭，构成 sofic 表示。\(\square\)

当不存在有限决定性时，\(\mathcal L_{\mathrm{obs}}\) 仍为有效移位，可表示为 sofic 移位的逆向极限。

### 5.2 观测单射与分离时间

**定义 5.2**（有界支撑分离时间）。对有限集 \(\Lambda\subset\mathbb Z\)，
\[
\tau(\Lambda) = \min\{ T\ge 0 : \forall x,y\in X_{\mathrm{fin}},\ \operatorname{supp}(x)\cup\operatorname{supp}(y)\subseteq\Lambda,\ x\neq y \Rightarrow \Theta_T(x)\neq\Theta_T(y) \}.
\]
若对一切有界 \(\Lambda\) 该最小值存在，记 \(\tau(\Lambda)<\infty\)。

**定理 5.3**（观测单射等价条件）。\(\Theta_\infty\) 在 \(X_{\mathrm{fin}}\) 上为单射当且仅当对每个有界集 \(\Lambda\)，\(\tau(\Lambda)<\infty\)。

*证明*：见附录 D。

定义增长函数 \(T_{\mathrm{sep}}(N)=\tau([-N,N])\)。若 \(\limsup_{N\to\infty} T_{\mathrm{sep}}(N)/N < \infty\)，系统具有线性分离时间。

### 5.3 与 closing 映射的联系

有限延迟可解码性本质上与因子映射的左/右 closing 性密切相关。本框架中的 CSP 分类直接关联到符号动力学中 closing 映射的经典研究。该联系的精细化仍待后续工作。

### 5.4 有限型观测因子

**命题 5.4**（有限型观测因子的充分条件）。若存在 \(T_0\) 使 \(\sim_{T_0}^{\mathrm{glob}} = \sim_\infty\) 在 \(X\) 上成立，且该等价关系满足某种平移不变的局部兼容性（即商系统可表示为有限窗口的合法状态序列），则观测因子拓扑共轭于一个高阶边移位（有限型移位）。必要性方向目前仍是开放问题。

*说明*：稳定等价关系允许将商系统表示为基于有限窗口约束的移位空间，构造方式类似于 higher-block 表示。完整的充要条件有待进一步研究。

## 6. 计算实验：Rule 110

本节提供计算实证，不构成形式化定理。

Rule 110：\(S=\{0,1\}\)，\(s_0=0\)，\(r=1\)。计算得 \(v_-=v_+=1\)，\(v=1\)，\(\kappa=0\)。取等速观测 \(c=1\)，满足一致性公理。

### 6.1 最小触发簇

固定 \(T\)，穷举非空 \(A\subseteq A_T\) 及非零赋值，计算 \(\Theta_T(x_{A,\sigma})\) 并与 \(\Theta_T(\mathbf 0)\) 比较，筛出可见者，按支撑极小化得 \(\mathcal M_T\)。所得结构呈现出与已知粒子轨迹相似的自组织模式，但此处未形式化“粒子”概念（见开放问题 7.5）。

### 6.2 碰撞消歧类别示例

以 \(\mathcal M_T\) 中元素为节点，依赖锥交叠则连边。对每条边构造联合构型，求解 CSP，判定消歧类别。

### 6.3 信息亏损的数值估计

\(\kappa=0\)，测量 \(\Delta_T^{\max}\) 主要源于内部碰撞不可区分性。有限规模下 \(h_{\mathrm{obs}}\) 数值极小，表明绝大多数碰撞迅速消歧，内部信息亏损微弱。这仅反映观测因子的分辨力，并非系统全局拓扑熵。

## 7. 开放问题与猜想

**猜想 7.1**（一般 CSP 判定复杂度）。当激发族大小随 \(T\) 增长、约束图树宽无界时，
- 有限延迟可解码性的判定预期为 \(\Sigma^0_1\)-完备；
- 本质不可解码性的判定预期为 \(\Pi^0_2\)-完备。
固定有限族情形定理 4.4 已给出半可判定性；完备性需通过将停机问题编码为二维 SFT 的构造来严格证明，目前尚未给出完整归约骨架。

**猜想 7.2**（熵等式充要条件）。寻求一般 CA 中不等式 (3.1) 等号成立的拓扑学或遍历论判据。当前结果仅为上界。

**开放问题 7.3**（典型与最坏情形观测熵）。定义基于自然平移不变测度的典型观测熵率，研究其与最坏情形 \(h_{\mathrm{obs}}\) 的变分原理关系。

**开放问题 7.4**（高维推广）。二维及以上 CA 中因果溢出集的几何结构更为复杂，碰撞分类需引入高维约束图的树宽或团宽。

**开放问题 7.5**（触发簇与粒子的形式等价）。建立极小触发簇 \(\lim_{T\to\infty}\mathcal M_T\) 与移位轨道等价类意义上“粒子”的严格对应。

**开放问题 7.6**（观测因子与 closing 映射的深入联系）。将有限延迟可解码性的精细分类与因子映射的定向 closing 性质完全统一，搭建 CA 观测理论与符号动力学 closing 理论的桥梁。

**开放问题 7.7**（有限型因子判定的必要性）。命题 5.4 给出了等价关系稳定则因子为有限型的充分条件；必要性是否成立仍待解决。

## 8. 结论

本文以严格形式化方法，将一维元胞自动机有限观测问题统一为 profinite 观测因子、纤维拓扑熵与约束满足解码三层理论框架。核心贡献包括：
1. 建立了有限观测划分的逆极限结构与 profinite 观测因子。
2. 提出并区分了时间观测熵率与空间亏损密度，通过溢出集无干扰引理与 Bowen 生成集方法严格证明了纤维拓扑熵的观测上界不等式，实现了内部亏损与边界溢出自由度的可分离量化。
3. 发展了碰撞的 CSP 分类，指出固定有限变量自然保证有界树宽，归约为有限状态同步问题，在稳定性条件下可判定；一般复杂度以猜想形式表述。
4. 将观测系统嵌入 sofic 表示及因子理论，给出有限状态表示的充分条件，并与 closing 映射建立初步联系。
全框架参数均由局部规则与观测速率有限计算，定理均附完整证明。Rule 110 的计算实验验证了框架的可操作性。该体系为有限精度测量下的动力学推断提供了严格的数学基础。

### A. 纤维有限到一的组合证明（定理 2.5）

*证明*：假设存在 \(T_0\) 使得对所有 \(T\ge T_0\)，\(|I_\infty(T)|\le B\) 且 \(\delta_T^{\max}\le B'\)。  
设 \(z\in\mathcal K\) 且其纤维包含有限支撑构型，取两个相异的 \(x,y\in\Phi_{\mathrm{fin}}(z)\)。令 \(\Lambda = \operatorname{supp}(x)\cup\operatorname{supp}(y)\) 为有限集。取 \(T\) 充分大，使得 \(\Lambda\subseteq A_T\) 且 \(T\ge T_0\)。  
由于 \(x,y\) 产生相同的无限观测 \(z\)，故在 \(T\) 内观测相同，即 \(x|_{E_T}\) 与 \(y|_{E_T}\) 同属 \(\Phi_T^{\mathrm{eff}}(w)\)，其中 \(w = \Theta_T(x) = \Theta_T(y)\)。  
由 \(\delta_T^{\max}\) 定义，该兼容集大小至多为 \(|S|^{B'}\)。因此 \(\Phi_{\mathrm{fin}}(z)\) 在有效区域上的投影基数有界。结合溢出区至多 \(B\) 个额外自由度，整个纤维基数至多 \(|S|^{B'+B}\)，即为有限到一。\(\square\)

### B. 观测亏损上极限的性质（命题 3.2）

*证明*：定义 \(h_{\mathrm{obs}}\) 为 \(\frac{1}{T}\Delta_T^{\max}\) 的上极限，非负。当 \(v_-=v_+\) 时溢出集最终为零，此时 \(\Delta_T^{\max}\) 仅与内部碰撞复杂度的渐进行为有关。一般情况下，上极限足以保证主定理证明中的不等式，因为仅需上极限估计，且 \(h_{\mathrm{obs}}\) 不依赖于溢出区自由度数（因为 \(\Delta_T^{\max}\) 本身定义在有效区域上，已排除溢出区）。\(\square\)

### C. 固定有限激发族的有限状态同步构造（定理 4.4 详细证明）

*证明*：设激发族支撑集为 \(\mathcal A\)，\(|\mathcal A| = M < \infty\)。所有可能的初值赋值集合为 \(\Sigma = S^{\mathcal A}\)，基数 \(N = |S|^M\)。  
构造同步自动机 \(\mathcal A} = (\mathcal Q, q_0, \delta)\)：
- 状态空间 \(\mathcal Q = \mathcal P(\Sigma)\)（\(\Sigma\) 的幂集），共 \(2^N\) 个状态。
- 初始状态 \(q_0 = \Sigma\)（无观测时，所有赋值均兼容）。
- 转移函数：对于状态 \(q\subseteq\Sigma\) 以及新时间层 \(t\) 的观测 \(w_t\)（即对构型施加 \(F^t\) 后在 \([-ct,ct]\) 上的配置），定义
  \[
  \delta(q, w_t) = \{\, \sigma\in q \mid \Theta_t(x_\sigma) \text{ 与观测历史一致且 } F^t(x_\sigma)|_{[-ct,ct]} = w_t \,\}.
  \]
  由于真实构型对应某个 \(\sigma^*\)，其产生的观测序列 \(w_0, w_1, \dots\) 必定满足 \(\sigma^* \in q_t\) 对所有 \(t\)，因此 \(q_t \neq \varnothing\)。

状态序列 \(q_0, q_1, q_2, \dots\) 单调不增（\(q_{t+1}\subseteq q_t\)），故由于有限基数，必然最终稳定于某个非空集合 \(q_\infty\)。  
有限延迟可解码等价于存在 \(t\) 使得 \(|q_t| = 1\)，即自动机到达一个吸收状态（大小为 1 且在该状态下任何合法的观测序列下不再改变）。  
若系统满足“有限稳定”性质，即存在可计算界限 \(T^*\) 使得对所有真实构型，\(q_{T^*} = q_\infty\)，则判定可在 \(\mathcal Q\) 中进行穷举搜索：考察是否所有从 \(q_0\) 出发并遵守动力学与观测约束的路径都最终落入大小为 1 的状态。这相当于检测该自动机是否必定同步。由于状态有限，可判定。  
若无此界限，但问题仍为半可判定：我们可递增 \(T\) 并检查 \(|q_T|\)，一旦发现为 1 则输出“可解码”；否则永不停止。这正是 \(\Sigma^0_1\) 半可判定性的标准模式。\(\square\)

### D. 观测单射与有界支撑分离时间的等价证明（定理 5.3）

*证明*：
(⇒) 设 \(\Theta_\infty\) 单射。取有界 \(\Lambda\)，则集合 \(\mathcal X_\Lambda = \{x\in X_{\mathrm{fin}} : \operatorname{supp}(x)\subseteq\Lambda\}\) 有限。对每对不同构型 \(x,y\in\mathcal X_\Lambda\)，由单射知存在 \(T_{x,y}\) 使 \(\Theta_{T_{x,y}}(x)\neq\Theta_{T_{x,y}}(y)\)。取 \(T=\max_{x\neq y} T_{x,y}\) 即得 \(\tau(\Lambda)\le T<\infty\)。
(⇐) 若所有有界 \(\Lambda\) 有有限分离时间，任取不同 \(x,y\in X_{\mathrm{fin}}\)，令 \(\Lambda=\operatorname{supp}(x)\cup\operatorname{supp}(y)\)，则 \(\Theta_{\tau(\Lambda)}(x)\neq\Theta_{\tau(\Lambda)}(y)\)，故 \(\Theta_\infty(x)\neq\Theta_\infty(y)\)。\(\square\)

## 附录 E  典型观测熵与相对测度熵（条件性定理与纲要）

本附录将正文中最坏情形分析提升到测度论层面，引入纤维条件测度熵与基于典型构型的观测熵率。由于目前缺少条件 Shannon–McMillan–Breiman 定理与纤维局部熵公式的直接支持，严格证明依赖于若干未在本文中建立的假设。因此，主要结论表述为**条件性定理**，并给出完整证明纲要，明确所有技术缺口。

### E.1 不变测度与观测因子上的诱导测度

取 \(X\) 上的一个 Borel 概率测度 \(\mu\)，满足  
\[
\mu\circ F^{-1}=\mu,\qquad \mu\circ\sigma^{-1}=\mu,
\]
且 \(\mu\) 支撑在两端渐近静止构型集 \(X\) 上。观测因子投影 \(\Theta_\infty:X\to\mathcal K\) 诱导因子上的测度  
\[
\nu=\Theta_\infty^*\mu,\qquad \nu(A)=\mu(\Theta_\infty^{-1}A).
\]

由紧致度量空间的 Rokhlin 分解，对 \(\nu\)-几乎处处的 \(z\in\mathcal K\)，存在纤维 \(\Phi(z)\) 上的条件概率测度 \(\mu_z\)，满足  
\[
\mu=\int_{\mathcal K}\mu_z\,d\nu(z),
\]
且对 \(\nu\)-a.e. \(z\)，\(\mu_z\) 是 \(F\)-不变的。

### E.2 纤维测度熵

**定义 E.1**（纤维测度熵）。对 \(\nu\)-a.e. \(z\)，定义子系统 \((\Phi(z),F,\mu_z)\) 的 Kolmogorov–Sinai 熵 \(h_{\mu_z}(F)\)。则系统 \((X,F,\mu)\) 相对于因子 \((\mathcal K,\widetilde F,\nu)\) 的相对测度熵定义为  
\[
h_{\mu}(F\mid\Theta_\infty):=\int_{\mathcal K} h_{\mu_z}(F)\,d\nu(z).
\]
在可测纤维分解存在的标准条件下，它与相对熵的生成覆盖定义一致。标准分解公式成立：  
\[
h_\mu(F)=h_\nu(\widetilde F)+h_\mu(F\mid\Theta_\infty). \tag{E.1}
\]

### E.3 典型观测亏损

对每个 \(T\)，定义局部观测亏损的典型值：  
\[
\Delta_T^{\mu}=\mathbb{E}_{x\sim\mu}\Bigl[\log|\Phi_T^{\mathrm{eff}}(\Theta_T(x))|\Bigr],
\]
其中 \(\Phi_T^{\mathrm{eff}}\) 如正文定义 1.7。\(\Delta_T^{\mu}\) 是平均意义下观测不能区分的有效区域状态数量的对数值。注意，目前未证明序列 \(\frac{1}{T}\Delta_T^{\mu}\) 的次可加性，因此我们仅使用上极限定义其渐近增长率。

**定义 E.2**（典型观测时间熵率）。  
\[
h_{\mathrm{obs}}(\mu)=\limsup_{T\to\infty}\frac{1}{T}\Delta_T^{\mu}.
\]
显然 \(h_{\mathrm{obs}}(\mu)\le h_{\mathrm{obs}}\)。

### E.4 典型相对熵不等式（条件性定理）

**定理 E.3**（条件性）。设 \(F\)-不变测度 \(\mu\) 满足以下条件之一：
1. \(\mu\) 是 Gibbs 测度，且纤维分解具有良好的条件局部熵渐近性；
2. 存在条件 Shannon–McMillan–Breiman 型定理，使得对 \(\nu\)-a.e. \(z\) 和 \(\mu_z\)-a.e. \(x\)，有
   \[
   \limsup_{T\to\infty} \frac{1}{T}\log|\Phi_T^{\mathrm{eff}}(\Theta_T(x))| \le h_{\mathrm{obs}}(\mu),
   \]
   并且组合多重度与条件 Bowen 球的测度对数渐近等价。

则在观测一致性公理下，有  
\[
h_\mu(F\mid\Theta_\infty)\;\le\; h_{\mathrm{obs}}(\mu)+\kappa\log|S|,
\]
其中 \(\kappa\) 为因果溢出速率。

**证明纲要**。  
对任意 \(\varepsilon>0\)，选 \(T\) 使 \(2^{-cT}<\varepsilon\)。对 \(\nu\)-典型 \(z\)，考虑纤维 \(X_z\) 上的条件测度 \(\mu_z\)。取 \(m=nT\)。由条件假设，对 \(\mu_z\)-典型 \(x\)，
\[
\limsup_{m\to\infty} \frac{1}{m} \log|\Phi_m^{\mathrm{eff}}(\Theta_m(x))| \le h_{\mathrm{obs}}(\mu).
\]

现在构造 \((m,\varepsilon)\)-生成集。由一致性公理与有效区域定义，若两个初值在 \(E_m\) 上给出相同的兼容赋值，并且在溢出区 \(I_\infty(m)\) 上取相同扩展，则它们在前 \(m\) 步的观测轨道在 \([-cT,cT]\) 上完全一致。结合局部依赖锥估计与度量定义，这些点落在同一个 \((m,\varepsilon)\)-Bowen 块中。因此，兼容集 \(\Phi_m^{\mathrm{eff}}(w)\) 的每个元素通过固定溢出区中的一个赋值，可生成一族候选轨道代表。溢出区有 \(|I_\infty(m)|\) 个细胞，每个可自由取值，故每个兼容赋值至多贡献 \(|S|^{|I_\infty(m)|}\) 个候选代表。从而可构造生成集 \(E\) 满足基数上界
\[
|E| \le |\Phi_m^{\mathrm{eff}}(w)| \cdot |S|^{|I_\infty(m)|}.
\]
取对数得
\[
\log|E| \le \log|\Phi_m^{\mathrm{eff}}(w)| + |I_\infty(m)|\log|S|.
\]
沿子序列 \(m=nT\)，由条件假设和溢出速率的定义，
\[
\limsup_{m\to\infty} \frac{1}{m}\log|E| \le h_{\mathrm{obs}}(\mu) + \kappa\log|S|.
\]
由测度熵的生成集定义（或条件 Brin–Katok 公式），若上述构造对几乎每个 \(z\) 都成立，则 \(h_{\mu_z}(F)\) 不超过该上极限。对 \(z\) 积分即得
\[
h_\mu(F\mid\Theta_\infty) = \int h_{\mu_z}(F)\,d\nu(z) \le h_{\mathrm{obs}}(\mu) + \kappa\log|S|.
\]
完整严格化要求建立条件测度熵与组合兼容集大小之间的局部渐近等价性，这已超出本文当前的技术范围，故定理以条件性形式给出。\(\square\)

结合 (E.1) 即得分解不等式  
\[
h_\mu(F)\;\le\; h_\nu(\widetilde F)+h_{\mathrm{obs}}(\mu)+\kappa\log|S|.
\]

### E.5 变分原理讨论

由定义，\(h_{\mathrm{fib}}(F\mid\Theta_\infty)=\sup_z h_{\mathrm{top}}(F|_{\Phi(z)})\)。通常有 \(\sup_\mu h_\mu(F\mid\Theta_\infty)\le h_{\mathrm{fib}}(F\mid\Theta_\infty)\)，等号在纤维上熵谱上半连续等条件下成立。研究其成立条件即为猜想 7.2 的测度论版本。

### E.6 进一步方向：条件局部熵与相对 SMB 定理

上述纲要中缺失的核心桥梁是：对条件测度 \(\mu_z\)，能否将组合兼容集大小 \(\log|\Phi_T^{\mathrm{eff}}(\Theta_T(x))|\) 与动态球测度 \(-\log\mu_z(B_T(x,\varepsilon))\) 的渐近行为联系起来。在经典理论中，Shannon–McMillan–Breiman 定理给出
\[
-\frac{1}{T}\log\mu(B_T(x,\varepsilon)) \to h_\mu(F) \quad \mu\text{-a.e.}
\]
而我们需要的是条件版本，即对 \(\mu_z\)-a.e. \(x\)，
\[
-\frac{1}{T}\log\mu_z(B_T(x,\varepsilon)) \to h_{\mu_z}(F).
\]
若能进一步证明 \(\log|\Phi_T^{\mathrm{eff}}(\Theta_T(x))|\) 与 \(-\log\mu_z(B_T(x,\varepsilon))\) 渐近等价（例如通过证明每个兼容类恰对应于一个条件 Bowen 球），则定理 E.3 可无条件成立。这一方向需要建立条件 Brin–Katok 公式和条件 Katok 熵框架，是观测亏损理论通向严格测度论基础的核心问题，留待后续工作。

## 附录 F  严格非零 \(\kappa\) 的构造实例（修正传播方向表述）

正文指出当 \(v_-\neq v_+\) 时，因果溢出速率 \(\kappa=2v-v_--v_+>0\)。此处给出一个局部规则不对称依赖的元胞自动机实例，其 \(\kappa>0\) 且溢出集线性增长。本修订版修正观测速率以严格符合一致性公理，并澄清溢出完全来自单侧泄漏的结构。

### F.1 规则定义

取 \(S=\{0,1\}\)，静止态 \(s_0=0\)，半径 \(r=2\)。定义局部规则 \(f:S^5\to S\) 为  
\[
f(x_{-2},x_{-1},x_0,x_1,x_2)=x_{-2}.
\]
输出仅取决于左侧第二个邻居，其余位置被忽略。该规则满足 \(f(0,\dots,0)=0\)，生成全局映射 \(F(x)_i=x_{i-2}\)（构型向右移动两位，即每个细胞的状态被来自其左侧第二个邻居的信息所替换）。

### F.2 依赖偏移计算

依据定义 1.1：
- **右最小依赖偏移 \(v_+\)**：寻找最小 \(k\in[0,2]\) 使若 \(a_i=b_i\) 对所有 \(i\le k\)，则 \(f(a)=f(b)\)。取 \(k=0\) 即满足，因为条件包括 \(i=-2,-1,0\)，迫使 \(a_{-2}=b_{-2}\)，故 \(v_+=0\)。
- **左最小依赖偏移 \(v_-\)**：寻找最小 \(k\) 使若 \(a_i=b_i\) 对所有 \(i\ge -k\)，则输出相等。\(k=0\) 时条件仅保证 \(i\ge0\) 相等，不约束 \(x_{-2}\)；\(k=1\) 保证 \(i\ge-1\) 相等，\(x_{-2}\) 仍可不同；\(k=2\) 保证所有位置一致，故 \(v_-=2\)。

因此 \(v=\max(v_-,v_+)=2\)，因果溢出速率  
\[
\kappa = 2v - v_- - v_+ = 4 - 2 - 0 = 2.
\]

### F.3 观测与溢出集（完全满足一致性公理）

取观测速率 \(c=2\)，满足 \(c\ge v=2\)。此时有效区域  
\[
E_T = \bigl[-T(c+v_+),\; T(c+v_-)\bigr] = [-2T,\; 4T].
\]
包围区域  
\[
A_T = [-(c+v)T,\; (c+v)T] = [-4T,\; 4T].
\]
溢出集  
\[
I_\infty(T)=A_T\setminus E_T = [-4T,\,-2T) \cup (4T,\,4T].
\]
右侧区间为空，故溢出集实际仅为左侧区间 \([-4T,-2T)\)，长度  
\[
|I_\infty(T)| = 2T + O(1),
\]
线性系数即为 \(\kappa=2\)，溢出完全来自左侧。

**严格溢出论证**：该规则下，时空点 \((i,t)\) 的回溯依赖锥（causal preimage cone）为单点 \(\{i-2t\}\)，即依赖锥向左方回退。对于初始位置 \(j\in[-4T,-2T)\)，其在时刻 \(t\le T\) 所影响的细胞位置为 \(j+2t\)（因为构型右移，状态向右传递）。观测窗口的左边界为 \(-2t\)。因此对任意 \(t\le T\)，
\[
j+2t < -2t \;\Longleftrightarrow\; j < -4t \;\Longleftrightarrow\; j < -4T \le -4t,
\]
即该细胞的影响永远位于观测窗口左侧之外。右侧溢出区为空，因为任何右方初值 \(j>4T\) 的状态保持不变（无右向依赖提供新信息），而观测右边界为 \(2t\le 2T<4T<j\)。故溢出完全由左侧依赖锥与观测锥的不匹配造成，且大小严格线性增长。

### F.4 意义

此例证实溢出项 \(\kappa\log|S|\) 可以严格为正且由纯粹的单侧依赖不对称产生。此类元胞自动机在物理上对应信息传播方向与观测窗口扩展方向不匹配的系统，确存在“因果泄漏”自由度。

## 附录 G  纤维拓扑熵严格为正的实例与隐藏自由度结构定理

本附录通过临时将构型空间放宽至全移位，构造一个纤维拓扑熵严格为正的系统，并在此基础上建立“隐藏自由度命题”，将实例提升为结构定理。最后，完整验证熵不等式，揭示观测亏损与纤维熵的本质区别，并讨论未来方向。

### G.1 空间放宽说明

正文构型空间 \(X\) 限定为两端渐近静止构型，在此空间上移位子的拓扑熵为零。为展示纤维拓扑熵可严格为正，**仅在本附录内**将构型空间替换为全移位 \(S^{\mathbb Z}\)。该空间紧致、完全不连通，保持所有动力学定义和观测映射。正文中关于 profinite 观测因子与熵不等式的结论（定理 2.2，定理 3.4 等）对 \(S^{\mathbb Z}\) 同样成立：原空间 \(X\) 是其闭不变子集，所有构造可自然限制，且证明中仅用到紧致性与依赖锥性质，与端点渐近条件无关。故本附录的实例在全移位上进行，所得结论可视为正文结果在更大空间上的实现。

### G.2 观测投影扩展

允许观测映射 \(\pi:S\to\Sigma\)，时空观测块定义为  
\[
\Theta_T^\pi(x)=\bigl(\pi(x|_{[-cT,cT]}),\pi(F(x)|_{[-cT,cT]}),\dots,\pi(F^T(x)|_{[-cT,cT]})\bigr).
\]
等价关系与观测因子均相对于此投影定义。所有先前定理可直接平移，只需将依赖性条件调整为“影响观测值 \(\pi\circ F^t\)”，并相应地计算 \(v_-^\pi, v_+^\pi\)。为简洁，本附录直接显式计算。

### G.3 实例构造（正纤维熵）

令 \(S=\{0,1\}^2\)，状态记为 \((a,b)\)，静止态取 \((0,0)\)（在全移位中无需特殊作用）。全局映射 \(F\) 定义为  
\[
F(x,y)_i = (x_{i+1},\; y_i),
\]
即第一坐标右移一位，第二坐标恒等。该映射连续且是 \(S^{\mathbb Z}\) 上的自同胚。  
观测映射取 \(\pi(a,b)=b\)，仅读取第二坐标；观测符号集 \(\Sigma=\{0,1\}\)。

### G.4 依赖偏移与一致性公理

考虑 \(\pi\circ F^t(x,y)_i = y_i\)，它只依赖于初始 \(y_i\)，与邻居无关。因此相对 \(\pi\) 的左右最小依赖偏移均为 0：
\[
v_-^\pi = v_+^\pi = 0.
\]
取观测速率 \(c=1\)，满足一致性公理。有效区域  
\[
E_T = [-T(c+v_+),\; T(c+v_-)] = [-T,\; T].
\]
包围区域 \(A_T = [-(c+v)T,\; (c+v)T] = [-T,\; T]\)，溢出集为空，\(\kappa=0\)。

### G.5 观测因子与纤维熵的严格计算

对任意 \(T\)，观测块 \(\Theta_T^\pi(x,y)\) 仅记录 \(y\) 在窗口 \([-T,T]\) 上的值。由于 \(y\) 不随时间变化，且 \(\bigcup_{t}[-ct,ct]=\mathbb Z\)，无限观测序列 \(\Theta_\infty^\pi(x,y)\) 完全决定整个构型 \(y\in\{0,1\}^{\mathbb Z}\)。因此 \(\Theta_\infty^\pi(x_1,y_1)=\Theta_\infty^\pi(x_2,y_2)\) 当且仅当 \(y_1=y_2\)。等价类由 \(y\) 唯一标记，商空间 \(\mathcal K^\pi\) 同胚于 \(\{0,1\}^{\mathbb Z}\)，投影为 \(\Theta_\infty^\pi(x,y)=y\)。诱导映射 \(\widetilde F(y)=y\) 为恒等，故 \(h_{\mathrm{top}}(\widetilde F)=0\)。

固定 \(z=y\)，纤维 \(\Phi^\pi(z)=\{ (x,y) \mid x\in\{0,1\}^{\mathbb Z} \}\)。其上的动力系统由  
\[
F(x,y) = (\sigma x,\, y),
\]
给出，与第一坐标的移位 \(\sigma\) 拓扑共轭。因此子系统 \((\Phi^\pi(z),F)\) 的拓扑熵为
\[
h_{\mathrm{top}}(F|_{\Phi^\pi(z)}) = h_{\mathrm{top}}(\sigma) = \log 2.
\]
由定义 2.6，
\[
h_{\mathrm{fib}}(F\mid\Theta_\infty^\pi) = \sup_z h_{\mathrm{top}}(F|_{\Phi^\pi(z)}) = \log 2 > 0.
\]

### G.6 观测亏损与熵不等式验证

全系统 \((S^{\mathbb Z},F)\) 为乘积系统，第一坐标全移位，第二坐标恒等，故
\[
h_{\mathrm{top}}(F) = h_{\mathrm{top}}(\sigma) + h_{\mathrm{top}}(\mathrm{id}) = \log 2 + 0 = \log 2.
\]
分解不等式 (2.1) 给出
\[
h_{\mathrm{top}}(F) \le h_{\mathrm{top}}(\widetilde F) + h_{\mathrm{fib}}(F\mid\Theta_\infty^\pi) = 0 + \log 2,
\]
等号成立，熵完全集中于纤维。

计算观测时间熵率 \(h_{\mathrm{obs}}\)。对给定的观测块 \(w\)（即 \(y\) 在 \(E_T=[-T,T]\) 上的一个特定赋值），兼容的有效区域初值包括任意 \(x\) 在 \([-T,T]\) 上的取值，因为 \(x\) 不影响观测。因此
\[
|\Phi_T^{\mathrm{eff}}(w)| = 2^{|E_T|} = 2^{2T+1},
\]
从而
\[
\Delta_T^{\max} = \log|\Phi_T^{\mathrm{eff}}(w)| = (2T+1)\log 2,
\qquad
h_{\mathrm{obs}} = \lim_{T\to\infty} \frac{1}{T}\Delta_T^{\max} = 2\log 2.
\]
定理 3.4 给出上界 \(h_{\mathrm{fib}} \le h_{\mathrm{obs}} = 2\log 2\)，而实际纤维熵为 \(\log 2\)，上界远非紧致。这揭示了一个重要事实：**观测时间熵率 \(h_{\mathrm{obs}}\) 衡量的是所有兼容自由度的总数，而纤维拓扑熵 \(h_{\mathrm{fib}}\) 只捕获那些真正贡献轨道指数分离的“活跃”自由度**。在本例中，尽管 \(x\) 坐标的移位产生全部动力学复杂度（贡献了 \(h_{\mathrm{fib}}=\log 2\)），观测亏损却将有效区域中 \(x\) 的所有瞬时取值视为同等可能的独立自由度，因此 \(h_{\mathrm{obs}}\) 达到 \(2\log 2\)。换言之，\(h_{\mathrm{obs}}\) 统计的是空间窗口内的瞬时不确定性，而 \(h_{\mathrm{fib}}\) 统计的是时间方向上的轨道复杂度增长率；同一移位轨道被空间窗口中的多个坐标重复计数，导致 \(h_{\mathrm{obs}}\) 可以严格大于真实的纤维轨道熵。

### G.7 隐藏自由度结构定理

上述实例可提炼为一般性命题，使该附录从单一例子升格为结构定理。

**命题 G.1**（隐藏自由度与纤维熵下界）。设紧致度量空间 \(X = X_1\times X_2\)，连续自映射 \(F = (F_1, F_2)\) 满足 \(F(x_1,x_2) = (F_1(x_1), F_2(x_2))\)。设观测映射 \(\pi: X\to\Sigma\) 仅依赖第二坐标，即存在 \(\pi_2: X_2\to\Sigma\) 使得 \(\pi(x_1,x_2)=\pi_2(x_2)\)，并且无限观测窗口能分离 \(X_2\) 的点，即
\[
\Theta_\infty^\pi(x_1,x_2)=\Theta_\infty^\pi(x_1',x_2') \iff x_2=x_2'.
\]
则
1. 观测因子 \((\mathcal K^\pi,\widetilde F)\) 拓扑共轭于 \((X_2, F_2)\)；
2. 对任意 \(z\in\mathcal K^\pi\)，纤维 \(\Phi^\pi(z)\) 上的动力学与 \((X_1, F_1)\) 拓扑共轭；
3. 纤维拓扑熵 \(h_{\mathrm{fib}}(F\mid\Theta_\infty^\pi) = h_{\mathrm{top}}(F_1)\)。

*证明*。条件 \(\pi(x_1,x_2)=\pi_2(x_2)\) 意味着观测仅依赖于第二坐标。分离条件确保两个构型产生相同无限观测当且仅当它们的第二坐标相同，故等价关系为 \((x_1,x_2)\sim(x_1',x_2')\iff x_2=x_2'\)。商映射即投影到第二坐标，因此 \(\mathcal K^\pi \cong X_2\)，且 \(\widetilde F\) 对应于 \(F_2\)。对固定的 \(z\)（即固定的 \(x_2\)），纤维 \(\Phi^\pi(z) = X_1\times\{z\}\)，其上的 \(F\) 作用等同于 \(F_1\)。因此子系统拓扑熵即为 \(h_{\mathrm{top}}(F_1)\)。对所有纤维取上确界仍为 \(h_{\mathrm{top}}(F_1)\)。\(\square\)

对于乘积系统，总有
\[
h_{\mathrm{top}}(F) = h_{\mathrm{top}}(F_1) + h_{\mathrm{top}}(F_2).
\]
在本例中 \(h_{\mathrm{top}}(F_2)=0\)，故
\[
h_{\mathrm{top}}(F) = h_{\mathrm{top}}(F_1) = h_{\mathrm{fib}}(F\mid\Theta_\infty^\pi).
\]
即全系统熵完全隐藏于不被观测的坐标中。

**注记 G.2**。命题 G.1 将具体实例升华为一般结构：只要存在完全不依赖观测的乘积分量，且该分量具有正拓扑熵，纤维拓扑熵便严格为正，且数值等于该分量的熵。这为探索“观测不可约复杂性”提供了严格的代数‑拓扑基础，并表明 \(h_{\mathrm{fib}}\) 可以作为隐藏动力学的量化指标。

### G.8 进一步方向：次可加性、上界紧致性以及与符号扩展理论的联系

本节简要指出当前框架中三个核心开放问题的推进思路，不构成完整定理。

**1. 观测亏损的次可加性与极限的存在性**  
当前 \(h_{\mathrm{obs}}\) 定义为 \(\limsup_{T\to\infty} \frac{1}{T}\Delta_T^{\max}\)。若序列满足次可加性 \(\Delta_{T+S}^{\max} \le \Delta_T^{\max} + \Delta_S^{\max} + o(T+S)\)，则极限存在且等于下确界。在一般设定下，次可加性未必成立，但可在额外条件下部分获证。考虑有效区域兼容集的性质：对时间 \(T+S\)，观测块可分解为前 \(T\) 步和后 \(S\) 步。给定 \(w_{[0,T+S]}\)，其兼容集 \(\Phi_{T+S}^{\mathrm{eff}}(w)\) 中的每一初值必须同时兼容前 \(T\) 步和后 \(S\) 步的观测。由此可得
\[
|\Phi_{T+S}^{\mathrm{eff}}(w)| \le |\Phi_T^{\mathrm{eff}}(w_{[0,T]})| \cdot \max_{v} |\Phi_S^{\mathrm{eff}}(w_{[T,T+S]}\mid \text{初值兼容 }v)|,
\]
其中最大值遍历所有中间相容状态。若系统满足某种 Markov 性或相容性的均匀衰减，则可望得到次可加上界。严格建立该性质将把 \(h_{\mathrm{obs}}\) 变为真实极限。

**2. 熵上界紧致性的条件**  
定理 3.4 给出了 \(h_{\mathrm{fib}} \le h_{\mathrm{obs}} + \kappa\log|S|\)。附录 G 的实例表明该上界可以严格不紧。寻求等号成立的充要条件是一个深刻问题。从信息论角度看，等号意味着观测亏损完全由纤维动力学贡献，没有额外的空间冗余。一个自然的猜想是：若系统在纤维上满足 expansiveness 且观测亏损具有某种“信息稳定性”，则等号成立。特别地，当纤维上的动力系统是 Kolmogorov 型的，且观测映射能生成纤维拓扑熵时，组合自由度与动力学复杂度可能完全匹配。

**3. 与符号扩展理论的可能联系**  
观测亏损函数 \(\Delta_T^{\max}\) 本质上度量了观测因子作为拓扑因子时，纤维上“新出现的图样复杂度”。它所扮演的角色与符号扩展理论（symbolic extension theory）中出现的增长函数（如相对拓扑熵的条件生成函数）非常相似。在 Boyle–Downarowicz 等的符号扩展理论中，一个因子的 extension entropy 衡量了为用符号系统忠实地表示该因子，扩展系统中必须引入的额外熵量。我们的 \(\Delta_T^{\max}\) 恰好对应了在有限时间窗口内，纤维上所能产生的额外图样的对数增长率。因此，探索 \(h_{\mathrm{obs}}\) 与某种适当定义的 relative symbolic extension entropy 之间的联系，将是本文框架未来最重要的理论升级方向之一，有望将观测亏损正式纳入标准因子理论的语言体系中。
